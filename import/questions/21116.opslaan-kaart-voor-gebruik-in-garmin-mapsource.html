Ik neem aan dat je de kaarten download van http://garmin.openstreetmap.nl In dat geval (en andere) kun je de tool JaVaWa GMTK gebruiken om de geinstalleerde kaart een andere family ID te geven. Daarna kun je een andere kaart installeren. Enzovoort. Zie ook punt 2.2. van de FAQ.

I assume that you download the maps from http://garmin.openstreetmap.nl In that case (and other cases) you can use the tool JaVaWa GMTK to give the already installed map a different family ID. After that you can install another map. Etcetera. See also point 2.2. in the FAQ.
