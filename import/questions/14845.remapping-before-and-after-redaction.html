Remapping before and after redaction

1

Why is there a difference in remapping before and after run of redaction bot? If I understand correctly, before the bot it was enough to edit particular object while having approved new contributor terms (as a user). Why is this not possible after the redaction?

You misunderstood. It was never enough to just edit an object; the only situation in which editing an object would have removed the original contributor's copyright is when none of the object's properties remained (for example, when you moved an un-tagged node a considerable distance).
