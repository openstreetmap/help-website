In general, "save a map" is not enough. You need an application or a dedicated device to show the map.

If you have a Garmin device, look here. If you have a smartphone, pick the correct OS from here to find an app.

If you need it for other purposes than the above mentioned two, e.g. printing, displaying on a website, etc. start reading our wiki.
