How to tag a small apartment building

1

Hi,

I'm tagging a whole lot of buildings, for now the main building tags I use are:

building=yes
building=house
building=hangar
building=garage(s)

This works well but I'm wondering how to tag a small apartment building, not the skyscraper one but the smaller version like this one for example: http://farm3.static.flickr.com/2405/2469087289_0bc4c39c48_b.jpg

I'm using

building=apartment

but mapnik renders it the same as building=yes, is there a convention for small aparment buildings like this one?

building=yes or your other suggestions are just fine.

Don't worry about how they are rendered or differentiated in any particular renderer. If you want to render them differently, you can do so by creating and using a modified style.
