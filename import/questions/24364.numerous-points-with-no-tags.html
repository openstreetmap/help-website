They seem to be leftovers from the TIGER import (see the history of one of these nodes) and can be removed. The removal process is probably the most easiest when using JOSM as editor by searching for type:node untagged -child.

"Empty" nodes of this sort may be helpful with spotting redacted and not re-mapped items.
