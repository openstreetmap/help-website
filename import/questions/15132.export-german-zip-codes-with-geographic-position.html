Take a look at overpass api: https://wiki.openstreetmap.org/wiki/Overpass_API

You can choose to dowload all nodes with a postal code in a certain relation (the german boundary).

Now you have an XML file with all German postal codes and their position. I guess you can script your way through to transform that XML file into a CSV.

See also this question: https://help.openstreetmap.org/questions/12554/is-there-a-web-service-exposed-to-get-all-cities-in-a-country
