You can use OpenStreetMap for anything you like. You only have to make sure you credit OSM in the manner requested, and that any improvements you make to the data are contributed back under the same terms: see http://www.openstreetmap.org/copyright.

In addition, bear in mind that the servers at openstreetmap.org have limited capacity and are run by volunteers. So if you expect significant usage, or 100% availability is crucial to your project, then you should set up your own server with OSM data or use a commercial provider. See http://wiki.openstreetmap.org/wiki/Tile_usage_policy.

By the way, it's called "OpenStreetMap" (one word, not plural), not "OpenStreet Maps" or anything. :)
