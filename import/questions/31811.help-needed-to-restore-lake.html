Some days ago I made some changes to the lake Lundevannet in Norway using Bing aerial to make the borders of the lake more correct. Somehow now the lake is rendered white and I need some experienced user to restore the status of the lake to what it was before those changes.

It appears that the lake consisted of one circular "way" and you split it into three ways. None of the three forms a complete circle, so none are colored blue as water.

I just fixed this by creating a "multipolygon" including all three ways. Since together they form a complete circle, the multipolygon as a whole is colored blue.

Eric, please watch out for the 2000 nodes limit editing such a large way. If you cross that number you won’t be able to save your work and delete some nodes by using the undo button until you’re able to save it. You’ll end up splitting the lakeshore into several ways again. So it’s easier to leave it into three different ways now.
