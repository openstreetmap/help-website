I can receive packets when I capture on "microsoft" interface, but I just see informations my own laptop has, and none of others (I can see GETS from my browser, but not from any other on the network !) I guess the "microsoft" interface is your wireless interface. On Windows you'll only see your own traffic unless you capture with an AirPCAP adapter since you can't toggle monitor mode for the WiFi card on any Windows OS.

No packet on the Gigabit interface probably means that there is no cable connected to the wired interface, or your PC prefers communicating over Wireless (you still should see some broadcasts) or there might be an aggressive personal firewall blocking away packets.
