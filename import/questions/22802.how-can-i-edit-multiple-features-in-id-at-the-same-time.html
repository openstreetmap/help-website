So, I can select the buildings, but not edit. What to do?

At the moment, you can select multiple features and do geometry operations on them - edit, join, and merge - but cannot edit the tags of multiple objects at the same time. We haven't really found a way to make that kind of editing user-friendly and free of corner cases yet, but it will be supported when we do.
