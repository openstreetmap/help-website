How to tag numbered sections of power line with given name?

0

I'm mapping Ontario power lines. This document gives detailed information about the grid https://www.hydroone.com/abouthydroone/RegulatoryInformation/txrates/Documents/G2-01-01.pdf

As one can see, there are multiple distinct sections of power line bearing the same name. For instance, we have B5D sections 2 through 7. How should this section number be tagged?

It seems incorrect to add an ad-hoc suffix to the name, like "name=B5D3" or "name=B5D section 3", etc. Nor do I want to make up a new tag without good reason, like "line:section=3", but I could find no examples on the wiki or taginfo.

Can someone please advise?
