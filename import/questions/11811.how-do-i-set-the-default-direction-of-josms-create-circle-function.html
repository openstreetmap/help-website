How do I set the default direction of JOSM's "create circle" function?

0

The default direction for creating circles in JOSM is clockwise. However, I edit almost exclusively in regions where it's substantially more useful for the default to be anticlockwise. How do I tell JOSM I prefer circles to be created anticlockwise?

After having a look at the advanced settings of JOSM (typing roundabout in the search field there) it seems that there is NO option to define the direction of newly created roundabouts.

To achieve this option you can

  1. do a mailing in the JOSM mailing list, or
  2. open a feature request at josm.openstreetmap.de
