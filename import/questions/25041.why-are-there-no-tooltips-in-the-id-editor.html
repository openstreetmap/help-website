I see tooltips on every last button in iD when I hover over them. The one on the "Undo" button which you clicked isn't entirely clear and should probably have the word "Undo" added to it somewhere. Other than that they seem to be pretty decent.

Also, did you do the "Walkthrough" when it first fired up? This feature explains how to use the basic controls very well. You can start it at any time from the help menu.
