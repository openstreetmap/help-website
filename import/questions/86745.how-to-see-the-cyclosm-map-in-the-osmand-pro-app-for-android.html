Yes, it is possible to show CyclOSM in OsmAnd:

  • Enable Plugins / Online maps
  • Configure map / Overlay map / Install more... / CyclOSM

Probably not. There is already a bicycle profile, multiple different map styles (personally I prefer the Touring style for cycling) and the possibility to show bicycle routes on the map. This won't look the same as CyclOSM but should display roughly the same features.
