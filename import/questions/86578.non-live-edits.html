From the changeset summary you seem to have created 34 relations all called "Bus 140: Earl Estate -> Chatham" each with one or two bus stops. These should all be one (or two) relation(s).

Given the name you have used I suspect you meant to do this PTv2 style with a relation for each direction in which case it would be unusual for the relation to contain bus stops on both sides of a road. The relevant roads should also be included in these relations.

There isn't a really a delay before edits go live they are all available in the database immediately. The length of time it takes for various services to update varies.
