I assume you're referring to the main Openstreetmap website map. Which features appear on the map depends on the zoom level; this is predetermined by the rendering software and can't be changed by users. However, you can select a different rendering engine by clicking the "layers" button, but the alternative rendering engines tend not to show administrative boundaries at all.
