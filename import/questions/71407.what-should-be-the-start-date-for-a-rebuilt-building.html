What should be the start date for a rebuilt building?

1

We have a castle built in the 19th century that was demolished and rebuilt in the early 20th century. The castle's name still comes from the family that built the first one. Which build date should start_date hold?

You can use was:*= or demolished:* to prefix *:start_date=* and *:end_date=* for the previous iteration. Make sure to fill in the rest of was:*= / demolished:*,and historic:*= tags.
