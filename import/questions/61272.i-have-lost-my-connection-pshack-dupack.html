Cloudshark Preview imposes a 2MB max upload size limit, so a 96 MB is too big. Some possible options:

  • Sign up for Cloudshark for Business or Cloudshark Enterprise, which give you a 150 MB and 4 GB limit on capture files, respectively
  • Try to post it elsewhere, such as to dropbox, pastebin, etc.
  • Filter the data you have so it's smaller and "postable"
  • Take another capture with capture options set that cause a smaller capture file to be generated, such as by setting an appropriate value for the snaplen (maybe tcpdump's old default value of 96 bytes, for example), and/or applying a capture filter to only capture traffic of interest (e.g. tcp port 4050).
