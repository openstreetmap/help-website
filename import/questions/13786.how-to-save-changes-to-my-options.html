How to save changes to My Options?

0

After logging on, I wanted OSM to be in English and add my location, so went to change My Options settings accordingly, but there's no Save button anywhere in the page.

How do I save the changes?

There is a "Save Changes" button in the bottom right of "My Settings", just below the home location map.

The button is apparently at a fixed offset to the right, so with a very small browser window you might need to scroll horizontally to the right to see it.
