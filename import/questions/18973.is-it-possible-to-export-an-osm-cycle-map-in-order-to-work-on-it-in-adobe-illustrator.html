(To others: This may not a a complete answer, feel free to edit.)

"Our" export seems to only export the standard style (regardless of which layer is selected), right. By the way: http://www.opencyclemap.org/ is the home URL of the OpenCycleMap - but I cannot find an export option.

Btw: I guess svg would be equally good for your use in AI as a pdf with vector graphics in it.

Probably https://help.openstreetmap.org/questions/9043/printing-osm-maps-to-paper helps you:

  • http://maperitive.net/ sounds suitable.
  • MapOSMatic claims to be able to export SVG, however I saw that it does not allow to select OpenCycleMap as style.
  • http://walking-papers.org allows you to export OpenCycleMap as pdf - but it just contains bitmap graphics.

Also see https://help.openstreetmap.org/questions/1455/using-map-in-adobe-illustrator and https://help.openstreetmap.org/questions/22/how-do-i-export-map-images-of-larger-areas (good!)
