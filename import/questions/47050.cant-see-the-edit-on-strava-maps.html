Floripa! Cool. A new bike shop. We like those. I found it and fixed it for you: https://www.openstreetmap.org/node/3882670884

The thing you were missing was... you'd set the name, but need to say what the thing is. A bicycle shop, by clicking the top type selector bit in the editor.

Now that the correct tag is on there, you should find it will appear (after a delay) on the strava maps. It also appears in all sorts of other places, such as OpenCycleMap.org for example, as lots of different people make use of this OpenStreetMap contribution.
