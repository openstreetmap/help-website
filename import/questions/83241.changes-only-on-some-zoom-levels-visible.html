Changes only on some zoom levels visible

1

I mapped an area of a railway station, where all buildings, paths and points have been removed and currently it is a big construction site. https://www.openstreetmap.org/#map=16/51.8268/7.2943

I updated OSM with the iD editor on 26. Jan. On the user map the changes appear correctly only in zoom level 100m. With other zoom levels the changes are only partly visible or not at all. Even polygons like landuse = construction are only partly rendered and different by zoom level. Does someone know why this is? Did I do something wrong when editing?.

Thanks for some expert input. Klaus

You are just catching stale tiles there. E.g. when I zoomed in, zoom level 16,17 and 18 were fine (construction site) while zoom level 19 had some construction site tiles (new) but some old tiles with the parking space in between. Doing a hard reload on zoom level 19 (ctrl+F5) got me the new tiles - only construction site now.

Please note: this does not work on zoom 0 to 12 where tiles are only re-rendered about once a month, but for higher zoom levels, this normally works.

If it does not work for higher zoom levels, the re-rendering queue is overloaded, but then it should work a day later or so.
