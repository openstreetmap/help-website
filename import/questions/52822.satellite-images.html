@VaslavN if, as scai and nevw have already asked the legal side of things is in the clear it really depends on in which format the imagery is available and how large the dataset is.

If it is available from a public WMS or tile server you can add it to the imagery index

If you actually have the imagery available as files you can either host it yourself (or ask on a suitable mailing list / forum if somebody is willing to host it for you), or you might make it available to OpenAerialMap (in both cases you would still need to create entries for imagey index.
