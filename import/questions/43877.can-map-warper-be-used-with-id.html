In the "Export" tab of Map Warper, there is a "Tiles" url. To use the map in iD, open the "Background Settings" tab (shortcut "B"), select "Custom" from the bottom of the list and copy the Tiles url into the popup. It may be necessary to click the magnifying glass if you already have a custom background set.
