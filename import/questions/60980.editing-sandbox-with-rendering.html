Is there a copy of the OSM API anywhere (either at OpenStreetMap.org or elsewhere) either with rendered maps based on the data added or with the planet files/diffs output that someone could use for rendering?

I'm asking because a related question has been around for a while, and a similar question has just popped up on osmf-talk again.

Currently the sort-of answers that I'm aware of are:

  • opengeofiction - not really an answer as I believe they have community rules about "not copying tthe real world".
  • iD's walkthrough is a sandbox, but there's no rendering.
  • Using a /map call to fetch from osm.org's dev API - not ideal because it's limited to small areas.

None of these are ideal though.
