New Zealand map install issue

0

have a Garmin Drive 51 with preloaded European maps. I am planning a trip to New Zealand. I have downloaded and unzipped the gmapsupp,img for New Zealand and installed it on the GPs using Basecamp and 'Install Map' in Basecamp's Map menu. It shows up on the GPs in the Settings -> Map&Vehicle -> myMaps tab. But when I try to change country to New Zealand to look at the map, only European countries show up. (Note: I am still in the UK, which may be significant). How can I view the New Zealand map?

Thanks again escada. I had tried the direct download first before using Basecamp. Prompted by your hint I looked where Basecamp had put the map. It turned out to be in a Map folder not the Garmin folder as advised in other posts. Deleting the version Basecamp had added, and moving the version I had downloaded from the Garmin to the Map folder seems to have fixed it.

So the answer seems to be: on this device download gmapsupp.img to a folder called Map, not to the Garmin folder.
