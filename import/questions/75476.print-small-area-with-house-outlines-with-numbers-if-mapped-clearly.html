This is fundamentally because the rendered maps are optimised for viewing on a screen and not on paper. Basically font sizes and amount of detail all need tweaking for a paper-oriented style. I'd suggest looking at OSM on Paper (https://wiki.openstreetmap.org/wiki/OSM_on_Paper - originally I placed a link to this node herehttps://www.openstreetmap.org/node/1154244400 ) or rolling your own style using QGIS (there are a number of existing stylesheets for QGIS which could be adapted).
