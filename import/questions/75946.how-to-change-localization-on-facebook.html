It seems that very few addresses have been mapped in this area. You could improve your data in OpenStreetMap by adding address tags. See this example of a nearby apartment block that does have address tags: https://www.openstreetmap.org/node/3945814167

However there is no guarantee this will have any impact on Facebook. I don't know if Facebook even uses OSM address data (if so it seems odd that they have placed your shop on a street with a completely different name rather than on Paseo del Faro).

Also just to mention, if it is a pharmacy where you can buy prescription medicine, the usual tag would be amenity=pharmacy rather than shop=chemist.

On the Facebook page, click on the three dots (next to Like, Follow, Share) and select Suggest Edits. Click Edit by the address, and you should then be able to edit not only the address but also the location of the map pin. You'll probably need to zoom the map in a bit to do that. Then click Save.

That's what I'd do as an ordinary Facebook editor. If you're admin for the page, it might look completely different. Note that Facebook's software can take a long time to accept a change. It takes into account edits by multiple users before deciding whether your edit is reliable.
