Search Openstreetmap.org does not see admin_level = 8 when searching for a village

0

there is a village Veresy I typed it in the openstreetmap.org search The search returned https://www.openstreetmap.org/search?query=%D0%B2%D0%B5%D1%80%D0%B5%D1%81%D0%B8%20#map=14/50.3321/28.7623

But as a result of the search there is no https://www.openstreetmap.org/relation/9085754

For your information: In Ukraine, admin_level = 8 is an administrative-territorial unit subordinate to which the village is located Question: why the search result is missing https://www.openstreetmap.org/relation/9085754

The search returns the admin centre of that relation, which is a node named Вереси (https://www.openstreetmap.org/node/337524166).

The relation itself is not called that (the name is Вересівський Старостинський Округ), so one would not expect the search to return the relation as a result.
