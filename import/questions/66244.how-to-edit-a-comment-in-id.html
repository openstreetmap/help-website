1

I am a beginning mapper. Yesterday I mapped for HOT, then continued mapping on OSM in the UK. I used ID. Can I edit my comments? The comment field for my UK edit contained the HOT comment (hastags) from my previous edit, this had been added automatically. I only noticed this while I was hitting the Submit button. I am sure there must be a way to correct this?

Also I think I didn't add a comment on all of my edits, so would like to review and add.

You cannot edit or amend the comments of changesets you have already closed, however you can log in to the OSM web site and add further comments in the "discussion" text box e.g. on https://www.openstreetmap.org/changeset/63324275
