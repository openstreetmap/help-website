Hi, I'd suggest adding the building:use tag, which describes what a building element is used for. It's especially useful when the form of the building doesn't match with the current use of the building, eg building=church + building:use=residential.

In this case maybe building=house (or =detatched, or =static_caravan, etc) + building:use=model_home (or =show_house if you prefer, but I think "model home" is the less ambiguous term.)

You could also add a freeform description using the description= tag.
