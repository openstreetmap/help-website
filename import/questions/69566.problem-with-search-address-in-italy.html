Hi, answer to your question is quite simple; this is happening because 100% of roads in Borgia are still unnamed in OSM (see link below), this is because we badly lack a mapper in the area. Same is happening quite often from Rome going south. If you're a local in Borgia and would like to spend some time working on it it'd be great.

This is how Borgia looks right now in OSM https://www.openstreetmap.org/#map=17/38.82620/16.50876
