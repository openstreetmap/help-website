incorrect removal of tag?

0

I recently added two hotels to OSM tags used were building yes, and then added the name, the tracing done from Bing, obviously so as the hotels were shown, however someone as removed the building yes tag, and added tourism hotel, therefore the buildings are no longer rendererd,is this realy acceptable?

Adding tourism=hotel is acceptable (whatever you mean with acceptable), because that is just adding missing information.

Removing building=yes is (in this case) an error. Since nobody is infallible, it is acceptable that people make errors. Nobody is stopping you from fixing the error (by adding building=yes again) though.

You could also write a friendly message to the other mapper explaining that he made an error, so he is less likely to repeat it.
