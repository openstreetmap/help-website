Export track from OSM to OSMand

0

Hi,

I'm new to OSM and still learning. My question: I found a track I like: https://www.openstreetmap.org/relation/9030#map=12/52.5253/13.2825

Now I would like to only export track data (GPS) from this map and import it as a new track in my OSMand app.

How do I export the track from the linked page? If I click on "GPS Traces" I only see random public GPS traces but not the ones I would like to use. Thanks for your help!

It is not a track, but an object in the OSM database, a relation of many ways, representing a cycling route.

For this kind of route relations, you can use waymarkedtrails.org, it allows export as gpx.
