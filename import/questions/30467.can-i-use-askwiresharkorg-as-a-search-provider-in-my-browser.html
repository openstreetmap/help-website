Can i use "ask.wireshark.org" as a search provider in my browser?

0

Hi all,

i'd like to use "ask.wireshark.org" as a search provider (in IE) or search engine(in FF) in my browser. Now adding it works fine, but no results to my Q's .. it seems to use a token?

Follow the instruction of your browser to add a new search provider, then add the following 'search engine prefix'

http://ask.wireshark.org/search/?q=

or

http://ask.wireshark.org/tags/?q=

for a Tag search.
