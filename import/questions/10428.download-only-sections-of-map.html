No, OpenStreetMap doesn't do that kind of data processing. You will have to download the full data set and then use a tool like osmosis or osmfilter (for both, see wiki.openstreetmap.org) to extract the bits you want. On downloads.cloudmade.com they also have pre-produced "highway" extracts for various countries/areas but these will include lower-level roads as well.
