From your description it sounds like you can get to the osm.org website fine but you aren't getting map tiles. Map tiles come from a different server than the website so it is not the same thing. According to system monitoring, map tiles are still being served up fine by the tile server in usual quantities so I don't think anything major is broken. It is possible that the caching server you are being sent to by geoDNS is having problems. That would affect more people than just you but not everyone.
