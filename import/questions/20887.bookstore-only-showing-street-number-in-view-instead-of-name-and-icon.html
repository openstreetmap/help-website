1
1

I'm having a problem with a bookshop only showing the street number (4650) on the map in View mode instead of the business name and icon: http://www.openstreetmap.org/?lat=32.953863&lon=-80.161958&zoom=18

The other businesses in the building that I've entered so far show correctly in View mode. The post office I entered in a nearby building is only showing an icon, no name.

In Edit mode, the icon appears for the bookshop and the barbershop next door, but no name. The restaurant (Philly's) in the same building displays icon and name in both View and Edit modes.

What determines when the icon and name or just street number show?

I've tried deleting them all, waiting a day and re-adding them, to no avail. Couldn't find anything in the wiki about it.

Thanks in advance for any help you can provide.
