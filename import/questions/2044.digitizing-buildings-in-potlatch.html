That's two questions.

Answering 1: in both version of potlatch, you close polygons by clicking once on the starting node while you're drawing the outline of the shape. In both versions you start making polygons by just clicking in an empty area. You need at least three nodes placed before you can close the polygon.

To answer question 2: in PL2, make sure you have the building outline selected. Then use the dropdown menu on the left hand side to choose "Building" (under "Buildings"). You may need to scroll down to see it. Names are optional.

In PL1, press the + sign, or type "+" to add tags. Enter "building" for the key part, and "yes" for the value part.
