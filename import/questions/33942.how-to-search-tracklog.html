The 'nxt' tag is a still a mystery to me, nevertheless, I shall press on....

If you want to download gpx tracks to your mobile phone or Garmin or similar and use these for hiking/cycling etc then the openstreetmap.org site is not ideal as many of the tracks that have been uploaded to this site have been created and uploaded primarily to assist with editing the map.

The only way I know it can be done is using JOSM by right-clicking on the 'downloaded GPX data'>'choose visible tracks' and if the one you choose seems suitable, then you can load it into a browser if it has a url and then download.

You can also select track segments that are already mapped, save to a new layer and then save as gpx. Here is a video I just spotted. Don't delete stuff on the osm...just select and copy.

It is much easier to use the sites such as the following examples that are designed for displaying downloadable gpx tracks

Wikiloc
EveryTrail
Ride with gps

The new Strava heat maps are great for looking for popular routes (takes a while to load) but you can't download a track directly AFAIK but you can use it as a guide when drawing your own track in another program such as Bike Route Toaster
