Building not accessed from nearest road

4

Having used GPS to access the Calcot Hotel (51.4418, -1.04857) the satnav wanted to approach the hotel via the nearest road (Gatcombe Close). However, the access is off the main Bath Rd. How does one resolve this problem?

Things that may help:

  1. Draw the highway=service road from Bath Road to the hotel.
  2. Put entrance=main on the main entrance.

Note that whether or not these things actually help depends on the navigation software that you use OSM data with.
