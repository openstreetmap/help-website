Not sure about Overpass, but in Overpass Turbo, you can do this:

[out:json]
[timeout:25]
;
(
  node
    ["tourism"="camp_site"]
    (10.65674,49.33228,11.71143,50.35948);
  way
    ["tourism"="camp_site"]
    (10.65674,49.33228,11.71143,50.35948);
);
out body;
>;
out skel qt;

which returns both nodes and ways tagged camp_site.
