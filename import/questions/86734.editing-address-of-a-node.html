Hi, You should be able to edit the tags via iD editor from a browser on the android device. I've just tried it with Firefox on Android. Paste your link into Firefox then edit it with the iD editor from the EDIT drop-down menu. Need more help or want me to do it for you please get in touch. Regards Bernard.
