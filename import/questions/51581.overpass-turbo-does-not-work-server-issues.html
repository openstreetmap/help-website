There appear to be problems with the overpass-api.de/api server at the moment. See https://wiki.openstreetmap.org/wiki/Platform_Status and https://wiki.openstreetmap.org/wiki/Overpass_API/status , and this message on the dev list.

This doesn't just affect Overpass Turbo; I'm seeing similar issues from queries from the main website (press the question mark and click somewhere - that request goes to the same server).
