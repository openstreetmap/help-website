Actually, if I saw an item added to OSM as "name=toilet" (in English or any other language) I'd contact the original mapper rather than just changing the data. If you or anyone else change the data without telling the original mapper, they'll continue adding "name=toilet" because they won't know any better.

If instead you explain that "amenity=toilets" makes more sense, because maps created from OSM data can display them properly, and people searching on e.g. a Garmin handheld will find them, then next time the'll map them properly as well.

In this particular example it's easy to explain as there's a ready-made specialist rendering of the data already.
