Here the approach to the runway is cleared with lights mounted on poles every so often. How should this be tagged?

aeroway=landing_light was the most common tag/value combination in August 2009, according to OSMDoc. The TagStat page for aeroway=* is worth consulting too.

According to the beacon wiki page, you can use manmade=beacon with airmark=beacon. Mind you, I'm not sure if that wiki page reflects actual usage anywhere.

I think it would be better to stick to using the aeroway key and tag them with aeroway=beacon, beacon=landing or something similar. I don't think any of these render yet, so if you want to see them you would need to talk to the people operating the various renderers.
