tshark -i 1

well, interface 1 is probably lo0 (the loopback interface) in your VM.

Please run tshark with the name of the interface (eth0,eth1, etc.) you want to capture on

thsark -i eth0 ....

or figure out the interface number with dumpcap

dumpcap -D -M

tshark -i 1 -qz hosts
