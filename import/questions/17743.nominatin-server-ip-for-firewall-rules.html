OpenStreetMap currently uses the following IP address ranges:

  • 128.40.168.64/26 (UCL)
  • 193.63.75.96/27 (Imperial)
  • 89.16.179.150 (Bytemark)

We also have a few distributed "CDN" tile.openstreetmap.org servers.
