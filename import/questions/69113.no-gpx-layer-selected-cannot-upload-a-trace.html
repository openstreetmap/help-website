If you have a GPX file on your hard disk, you just have to open it JOSM, via "File - Open" from the menu bar.

If you already have a GPX file opened in JOSM, then you can use that plugin to upload the file to the OpenStreetMap servers. The reason to do this is that others might use your trace as well (if you make the trace public)

So you "open" a GPX file in JOSM, you do not upload a GPX file to JOSM. The reason for this wording is that JOSM is a local program (or app), not something running on a remote server.

Hope this helps.
