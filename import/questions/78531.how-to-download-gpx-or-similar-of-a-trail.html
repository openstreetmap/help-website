AFAIK not from the OpenStreetMap website. But, with you can using other tools like Waymarked trails in which you can select a relation and download it as a GPX. That relation in particular seems to have some gaps though.

You can download a GPX with overpass-turbo too. Just select export and save it in GPX format.
