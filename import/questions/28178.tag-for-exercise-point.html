I note these two.. Tag:leisure=fitness_station and Proposed features/exercise point and find either would be suitable, but there appears to be little use of them yet and no guidance on the MapFeatures page.

Use what you think is best. Also feel free to comment pro and con at the proposal page. This help site is no consensus finding tool. ;-) However some hints:

Look at taginfo about leisure=fitness_station and amenity=exercise_point for current usage and note that wiki-documented tags are likely to be of more value since everybody understands the same. Just a dead proposal page is not really relevant (someone – in fact it was only one person, see page history – has just proposed to use a specific tag).
