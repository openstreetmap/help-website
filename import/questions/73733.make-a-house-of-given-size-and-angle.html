If you can do one wall can you estimate the rest of it? If you can plot a gpx line from buildings location with some form of gpx creating tool at the right bearing that should give you correct angle. Import that gpx as a vector file as a background.

I created the magenta Angle.gpx with a mapping app and dragged that file over the iD screen, i had previously drawn a rough rectangle which i then pulled into the angle and used the scale to size it. I am pleased you get the idea, it works but isn't that great.
