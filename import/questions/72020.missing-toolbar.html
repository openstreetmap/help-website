To edit the map woithout login you need a secret editing link. You can search your browser history if it is still there. If not you can try contacting one of the maintainers of the umap sit you are using and ask them to reestablish access for you.

If you had that feature enabled you can also download the full map data, create a new map and import the saved data there.
