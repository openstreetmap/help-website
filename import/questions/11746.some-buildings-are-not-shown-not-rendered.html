A quick way to see whether something is closed is to use the browse/way links you quote and see if the first and last node in the list has the same id. This won't help you spot self-intersecting ways though.

3

In all of the cases I checked you had not joined the end back to the beginning properly. My advice is zoom in more and hold the mouse pointer over the first point before clicking to be sure you join the way up. In many cases you were very close, but you had joined near the beginning, leaving a small piece sticking out. If you zoom in closely enough you can see this. If you zoom in enough it is also easier to trace the outline of where the building is on the ground. If you find a small stub sticking out you can select that node and just delete it to leave a closed way.
