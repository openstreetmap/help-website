How to connect a river to lake?

0

Where should the start node of river placed on a multipolygon lake?

In the middle of the lake or on a node on the lake polygon?

If the lake has one or more inflows you should map the river(s) continuing through the lake to form a continuous path (just like roads), e.g. with waterway=river. If no inflow start the waterway where you think it makes morst sense. At the point where the waterway intersects with the lake I always connect the two, although I think this is handled differently by some mappers.
