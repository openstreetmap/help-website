First I can't tell anything about Windows 8.1 compatibility, as OSM has to much FLOSS projects and support of newest windows version isn't a top priority. So you need to test the tools on your own.

I guess what you are looking for is a solution with

  • offline maps
  • offline routing and address lookup
  • browse the map
  • available on windows with new maps

Thus if you filter the desktop software you will find some candidates that are known to work for the most people:

  • Gosmore
  • Mapfactor free - not FLOSS but more user friendly?
  • Navipown
  • Navit

This is a subjective selection, also I hope that I didn't miss any important candidate.
Please keep in mind, that most FLOSS tools (esp. Navit) need some time for setup to work properly and it's usability is ...moderate. Anyway I can tell that they are able to do the job :)
