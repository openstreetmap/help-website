You could use these two tags: shop=clothes, and clothes=accessories

I think of Accessorize as a type of clothes shop ... but I'm not a big follower of fashion! Likewise for shops that sell fashion handbags. And then I've used shop=bags for shops selling travel bags, suitcases, etc.

If you like looking at XML, you can find all the things tagged name=Accessorize (and see their other tags) using this XAPI query: http://open.mapquestapi.com/xapi/api/0.6/*[name=Accessorize]
