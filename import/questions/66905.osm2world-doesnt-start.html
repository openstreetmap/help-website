As per the installation instructions, OSM2World requires Java. As you're receiving an error message such as

the command 'java' couldn't be found or has been misspelled

the most likely problem is that Java simply isn't installed on your computer yet. The OpenStreetMap wiki has a page with instructions on how to install Java.

I'm sorry that you are having trouble, by the way. The installation process isn't as easy as it could (and probably should) be.
