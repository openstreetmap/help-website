Modify default settings for Umap

0

Hi !

Every time I want to create a map in my Umap account I must change several settings (shape propierties, interaction options) and so on, as I use a totally different set of defaults for my maps. There is a way to preset these deafults I use instead of the common settings to avoid changing it every time?

Thanks in advance

I don't think it's possible. You can look at the issue tracker, and ask for it, but it's probably complicated to implement.

Otherwise you could just create a kind of "template", an empty map with the desired settings, and "clone" it each time you need to create a new map. Clone is available in settings, advanced actions. It should copy all settings to the new map.

Regards.
