The Uganda wiki page had versions going back to as early as 2009, which includes mention of an import, but generally speaking there are not hard and fast start dates for data in any particular country save that no OSM mappers will have been working on it before the start of the project in 2004.

The version history of each object will show the dates each element was added and the dates they were edited, but will not necessarily indicate if those objects were split from existing objects at that point in time or are entirely new.
