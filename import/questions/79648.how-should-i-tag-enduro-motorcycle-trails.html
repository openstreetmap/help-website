How should I tag Enduro motorcycle trails?

0

Tagging as Motorcross track makes no sense, so what should I do?

If what you want to map is a racetrack you can use highway=raceway with an appropriate sport= key.

If this is more some sort of marked trail not primarily for racing you should map the ways as highway=path or what better indicates their nature and add appropriate access tags, e.g. access=no and motorcycle=designated. You can then also add a route relation combining the individual highways into the complete trail.
