I have some nodes in my area tagged "fixme: OS Locator". http://keepright.ipax.at/report_map.php?schema=87&error=17799641 is an example.

What does this mean? I tried looking at the OS Locator background but that seems to be blank.

It is not a very helpful fixme tag, as it doesn't really explain what the problem is. But presumably it means the name in OSM doesn't match the name from OS Locator.

Keepright says it is tagged on way 4014473, so you can view the history for that. That road was originally tagged as name=High Street Waltham Cross, but was later changed to name=High Street and alt_name=High Street Waltham Cross.

You can view the names from OS Locator by using Musical Chairs. For that road, it says Name: HIGH STREET.

I suggest you survey that street, and see what it actually says on the street signs. The name on the street signs should usually be tagged as name in OSM. Alternative (correct) names can be tagged as alt_name etc. If OS Locator has an incorrect name, then you can tag this with not:name.

Once you have checked which names are actually correct, you can remove the fixme tag.
