0

So, I need that the interfaces of all my nodes appear in the same wireshark window.

So, I need that the interfaces of all my nodes appear in the same wireshark window.

You can use more than one -i statements on the commandline.

Example:

wireshark -i eth0 -i eth1 -i eth2 -n ....
