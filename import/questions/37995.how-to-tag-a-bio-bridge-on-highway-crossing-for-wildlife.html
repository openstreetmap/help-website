How to tag a bio-bridge on highway (crossing for wildlife)

4

Is there a special tag for marking a bio-bridge? They usually are on multi-lane highways and should allow wildlife to cross the highwas due to environmental reasons. There is no road on the bridge, only grass and maybe a fence and some trees to create a more "natural" habitat so that wildlife does not fear to cross. Public traffic is of course restricted to these bridges.

Examples from the Czech Republic:

  • R1 highway near Prague (the first 3 are bio-bridges)
  • R4 highway

You can use the following tags together at the centre of the wildlife bridge (draw it as a line):

highway=path
surface=grass
bridge=yes
width=[Number of metres]

And of course

barrier=fence

for the wildlife protection.

And maybe you can use this tags for the bridge area:

natural=grassland
layer=1

Highways have default access values which wouldn't be appropriate for one of these wildlife crossings. For example, the global default for paths is that foot, bicycle, and horse are allowed, which they certainly aren't in reality for a wildlife crossing. You'd probably want to add access=no if you're going to tag it with one of the highway tags.
