Automatically calculate distance using polylines (umap)

Hi! Is it possible Umap calculates distances while drawing lines and polygons? I know this function from share.mapbbcode.org and Google Maps.

The measuring tool (toolbox "more" on the left hand side, select ruler icon) displays the distance in km. You can't directly draw lines with it, but you can display distances of lines drawn with the line drawing tool.
