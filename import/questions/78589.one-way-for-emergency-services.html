One way for emergency services

0

Hello! iam looking for a way to correctly tag an one way street so that emergency services can use this road for their navigation and 'normal' drivers can not. I see there is an option to exclude cyclist, is there any way i can alter such an option? Or maybe there is another smart way to get the same result? for example; https://www.openstreetmap.org/edit#map=16/52.5427/5.8725 All help is welcome :-)

Yes you can specify exceptions just like for cyclists.
oneway=yes
oneway:emergency=no
