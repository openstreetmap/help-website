That's a 2.6 GB download, so it wouldn't surprise me if something went wrong somewhere. If your zip file isn't 2.6 GB, something definitely went wrong. If your file is exactly 2 GB in size, you may need to change whatever you're using to download / unzip it, as that might suggest that it can't handle files > 2Gb.
