Is it possible to select a particular background from the openstreetmap.org edit screen for my basecamp work?

0

I notice that at the openstreetmap.org edit screen, you have numerous backgrounds to choose from. I'd like a particular one for my basecamp work. Is it possible to select a particular background?

The backgrounds (e.g. the different aerial imagery) from the openstreetmap.org edit screen (likely you use the default editor iD) are not part of the OpenStreetMap data. We are just allowed to use them to add / change to our OpenStreetMap data (e.g. trace missing roads).

So, you can not get a OSM-based download/plugin for your basecamp which provides you those backgrounds. It could be that some aerial imagery providers offer ways themselves to be used in basecamp.
