It is possible the etrex will not display a map if the GPSes location is not on the map. Zoom out then you may see something then scroll to it. If that does not work create a waypoint that is on the map and display that to centre the screen on the map. I was confused when i loaded a map of another country until i realised the problem. Running Garmin Basecamp with the GPS connected via USB could prove the maps are on the Etrex and work. You can set the Etrex to decimal degrees and use OSM map to help you get waypoint coordinates, OSM uses minus degrees for WEST and SOUTH.
