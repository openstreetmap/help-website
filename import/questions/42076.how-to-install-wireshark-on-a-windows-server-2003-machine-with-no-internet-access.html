You presumably can connect to the Internet on some machine, as you were able to post this question. :-)

Try downloading the Windows installer for Wireshark on that machine, or some other machine with Internet access, but save it rather than trying to run it. Then copy it to the machine on which you want to install it, and run it on that machine.
