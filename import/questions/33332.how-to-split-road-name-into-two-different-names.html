Hi, I see you use Potlatch as your main editor so here are the relevant instructions...

Select way. Click to add new node at point where name changes. Click to select that node. Click on 'scissors' symbol at bottom right of screen to split way. Select section of way to be re-named. Edit name. Save changes.

Hope this helps. Graham :)
