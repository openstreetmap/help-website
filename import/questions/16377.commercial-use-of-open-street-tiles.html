The short answer is that using the data is acceptable. That carries the ODbL licence and produced works, such as a render of a map, can be any licence you choose, but you must attribute OSM. You can find more on the legal FAQ here: https://wiki.openstreetmap.org/wiki/Legal_FAQ

Using the map tiles from the OSM website is not quite the same. They too require attribution, but they use the licence CC BY-SA which may, arguably, force you to release all of the document as CC BY-SA. Use the data and render your own map to be sure. See http://switch2osm.org for how to do that.

If you want to use the tiles, rather than the data, there is a usage policy: https://wiki.openstreetmap.org/wiki/Tile_usage_policy

You are free to use the data and render your own tiles which can then be used freely, subject to the licence: https://www.openstreetmap.org/copyright Commercial use, within these terms, is perfectly acceptable.
