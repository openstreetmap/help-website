Hi Andy, The newer basecamp versions 4.7.2 breaks the mapinstall feature to export maps form basecamp to your device. Instead You can download the .IMG Format version from opentopomap en copy it to your device SD card manually. I think basecamp 4.6.2 is the latest version which still works. If I remember correctly, it also has been fixed if you install the mapinstall update package. https://www8.garmin.com/support/download_details.jsp?id=3291

kind regards, Joris
