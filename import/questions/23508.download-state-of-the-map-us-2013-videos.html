There's no supported way from Vimeo or OpenStreetMap but it's possible to grab the URL used by Vimeo and save the file by using an HTTP web debugger such as the tools built into Chrome (via Developer Tools→Network→ …) or Fiddler2. In particular, after obtaining the URL of the video, one can save the file.
