It would be nice to know whether you are stuck because you do not know how to start editing or because you do not find the shop=hat category.

Assuming you know how to edit with iD (the in-browser editor).

  1. Add a point
  2. Search for "shop" on the left and select that.
  3. Under "All tags" you will see "shop" and "yes". Replace the "yes" with "hat"
  4. Add the name as with any other shop
  5. save and add a comment on what you added and how you know the name and location of the shop (e.g. local knowledge, survey)

So far there are only 68 shops tagged as hat, and 43 as hats (see taginfo). OSM normally uses the singular form. Therefore there won't be a dedicated icon for the shop and you might not see it on the rendered map afterwards.
