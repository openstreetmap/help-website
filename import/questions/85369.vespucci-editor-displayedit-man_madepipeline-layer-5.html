Currently there is no specific styling for pipelines so you will get the default black color and width, but it is trivial to add it yourself, see http://vespucci.io/tutorials/data_styling/

If you do add your own colors etc, we would welcome a pull request https://github.com/MarcusWolschon/osmeditor4android/tree/master/src/main/assets/styles
