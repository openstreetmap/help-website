How do I add to (amend) an existing relation?

0

I have created a new relation for a long distance footpath which i'm adding. Now i'd like to flesh out the relation with some extra info. How do i do it?

Tags can be added to relations in the same way as they are added to nodes and ways. Suggestions as to what tags to use are found on: http://wiki.openstreetmap.org/wiki/Walking_Routes#Relations

You may also need to update an entry on the page that lists all the UK long distance paths and links to their relations. http://wiki.openstreetmap.org/wiki/WikiProject_United_Kingdom_Long_Distance_Paths

If you tag your route with network=rwn (or lwn if local) then it will be highlighted on this map: http://hiking.lonvia.de/
