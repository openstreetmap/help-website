Garmin GPSMAP 60CSx trouble viewing maps on device

0

Well I'm about to give up on my Garmin 60CSx. I've been trying to download maps on it and here's what I've done so far:

  1. Download free British Isles map from Talky Toaster
  2. Unzip file and rename to gmapsupp.img on desktop
  3. Plug-in Garmin to computer in USB mode
  4. Drag and drop gmapsupp.img from desktop into Garmin folder created on 16gb sd card

At this point, when I go into "setup" and "map" on the device, I can only see "Americas Marine Point Basemap, Jan 2001." This must be stored on the device as the 16gb sd card is new and does not contain any other .img files.

When I open up BaseCamp on the computer and connect the Garmin in USB mode, I can see the British Isles map appear in the internal storage with the map icon next to it. However, the problem remains that I see no evidence of the map on the device.

Any ideas of where to go from here?

-Nate
