Navfree has a reporting tool in the iPhone version (though if it makes it available to OSM is uncertain). The Android version states reporting will be implemented in the next version.

I would recommend Osmand to report bugs on Android, but it would require you to either download the areas beforehand, or have a internet connection.

If you want to report bugs afterwards you can use any of these tools.
