You should file improvement requests for each of the applications. Not all features are rendered on all maps. There are different reasons why features are not rendered. (e.g. because it is not an important feature in the country of the maker or just an oversight, etc.). Typically feature types that are not rendered should be logged as improvements, not bugs.

You could e.g. ask on https://github.com/gravitystorm/openstreetmap-carto to improve the default map you find on osm.org. Maps.me and other apps have different places to ask for improvements. As fas as the default map is concerned, this is maintained by a small team, so they appreciate any help you could give them by providing a patch or an icon to be rendered.
