There are several users reporting wrong altitude information with OsmAnd. This seems to be caused by using the PlayServices for obtaining the position (and height). You can check the latest nightly build (https://download.osmand.net/latest-night-build/OsmAnd-default.apk), it doesn't use the PlayServices to get the location, as far as I know. The same applies to the F-Droid version. Also, check https://github.com/osmandapp/OsmAnd/issues for similar reports (for example https://github.com/osmandapp/OsmAnd/issues/13805 among others).
