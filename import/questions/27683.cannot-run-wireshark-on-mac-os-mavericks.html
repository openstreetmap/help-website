If the DISPLAY environment variable isn't set in the OS X terminal (do env | egrep DISPLAY), that will prevent X11 apps from opening the display. After you install X11, you might have to close the OS X Terminal app and restart it in order to get DISPLAY set, or you might find that OS X Terminal windows opened after you install X11 have it set.
