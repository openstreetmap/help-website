It's a bit unusual for British postcodes to be so far out because Nominatim uses external sources (CodePoint Open) from the Ordnance Survey for postcodes if they are lacking on OSM.

No matter you've added the address yourself & this should fix the issue on OSM itself.

The issue has now been rectified. The Ordnance Survey code-point data https://www.ordnancesurvey.co.uk/business-government/products/code-point-open data that we use as a fallback in the search, was rather far off in this case, adding the postcode directly to our data fixed the issue there where just some unrelated technical issues that led to a delay.
