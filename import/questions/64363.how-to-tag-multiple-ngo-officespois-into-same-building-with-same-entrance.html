How to tag multiple ngo offices/POIs into same building with same entrance?

0

I have old building that was school, but now it is used for offices of many NGOs and some amenities. I know that building should be taged as old school with history value - not sure which tag should I use here - this is one question.

The other is about multiple offices in the same point. Do I actually need to tag separately points that may be on the same location? Or how do you do this?

The exact semantics of the building tag values are a bit murky, and often building "style" is mixed up with use. IMHO adding building=school and building:use=office (see https://wiki.openstreetmap.org/wiki/Key:building:use) is the best solution.

And yes, tagging individual points is the thing to do for multiple offices (add level tags if they make sense).
