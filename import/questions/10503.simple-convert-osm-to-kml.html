If you create a kml with your points, you can use a custom icon in the kml file by specifying it in the style block:

<Style>
  <IconStyle>
    <Icon>
      <href>http://www.domain.com/images/mymarker.png</href>
    </Icon>
  </IconStyle>
</Style>
