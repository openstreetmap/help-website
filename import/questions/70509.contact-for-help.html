Everyone can download small amounts of tiles remotely from the OSM server (this is how the openstreetmap.org web site works). Downloading larger amounts of tiles is discouraged as per the Tile Usage Policy.

If you need a lot of tiles, you will have to generate them yourself - tools like Maperitive.net (for smaller areas) or Mapbox Studio Classic can be used to produce tiles, or you could set up your own tile server as documented in www.switch2osm.org. If you would rather have someone else provide this service to you, check out the service providers on the wiki page Commercial OSM Software and Services.
