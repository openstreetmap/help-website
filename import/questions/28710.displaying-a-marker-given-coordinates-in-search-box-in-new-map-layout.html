2

In the old map layout I fould it useful that there was an icon displayed. Can this be implemented in (nice!) new map layout again?

You can manually create one marker or use http://owl.apis.dev.openstreetmap.org (while there is the old UI version). Indeed this is more useful than the newer behaviour!

Please report bugs at https://trac.openstreetmap.org/ (you can login with your usual www.osm.org login) for component "website". However, please note that your wish might not be respected.
