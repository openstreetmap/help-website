  • If downloading from http://garmin.openstreetmap.nl does not work for you, try another OSM-based Garmin map. Try downloading smaller extracts or at another computer. If you assume it is a server problem, then please report to http://garmin.openstreetmap.nl (or discuss in the forum - link is on the left side).
  • If you use a FAT32 file system on download target location,saving will fail if the file is bigger than 4 GiB (file system restriction). Same applies to unpacking. Memory cards are currently, usually formatted with a FAT32 file system.
