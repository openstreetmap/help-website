If you embed openstreetmap in your site, with script src="http://www.openstreetmap.org/..." tags, then the user's IP will be made known to openstreetmap. Even that alone may be considered personal data in some circumstances / jurisdictions. Only an attorney who practices in your jurisdiction can give an unqualified answer to your question.

If you are using tiles from the openstreetmap.org tile servers, that means: you are using standard style tiles, yes then the access are logged on our tile servers.

Note: naturally the same goes for any third party content (for example JS libraries etc) that you use on your web site that you do not host yourself. This is a topic that is largely ignored by data protection officials, but naturally can cause issues.
