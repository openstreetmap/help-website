Hi.

First thing is to check that you are allowed to use the data on this website. I tried but automatic translation is not so great. If you can get explicit authorization, you can use a GPX file from them.

If not, and the trail is signed on the ground, you'll have to record a GPX track of the trail, and then use that as source.

Then you'll have to read some parts of the wiki, I think starting here is a good place : Hiking. Import might be useful as well.

Best regards.
