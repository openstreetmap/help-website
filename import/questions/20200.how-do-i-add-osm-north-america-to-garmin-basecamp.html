Our docu wiki page "OSM Map On Garmin/Download" says: "As .gmapi for Mac OSx Roadtrip/Basecamp." So choose a map provider which also supplies a gmapi file for your BaseCamp on MacOS. (I don't use MacOS)

If that does not help, please better describe your problem (what/where goes something wrong) - just edit your question.

Note that there is not only one "OSM map". OSM is just data which can be used in various ways. Choose a map which best fits your needs.
