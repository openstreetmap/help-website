bus route names not displayed

0

I've updated and added bus routes in Beirut, Lebanon, but whereas at some parts the bus number/name is displayed next to the line it's not at others. This makes it difficult for users because they don't see where the bus continues after crossing another line. You need to log on and click on the route to see where the bus line actually continues. I've compared it to Berlin, Germany, and London, UK, and it seems this is the case everywhere. Is there anything that can be done about it to make it more user-friendly? How about using different colours?
