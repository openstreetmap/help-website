We can try - unfortunately (as you've noticed) the openstreetmap site isn't really designed with people just wanting to use the data in mind.

Maybe if we could try and take it one step at a time?

A previous answer to this question: http://help.openstreetmap.org/questions/1924/north-carolina-maps

suggested to look at the "OSM_Map_On_Garmin/Download" wiki page. On there there's a link to this site: http://daveh.dev.openstreetmap.org/garmin/Lambertus/

and in there are lots of folders, each with a different date on it. There's one created every couple of weeks. The most recent folder is this one: http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/

and in there there are 14 map downloadable maps (the ".img" files) and a "README" file: http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/README

which asks "Which file do I use?" and goes on to explain that the size of the map that you can install depends on how much space there is on the SD card that you have in your Nuvi.

So the first questions therefore are:

  • What sort of Garmin Nuvi do you have?
  • If you've already installed an SD or microSD card, how big was that?
