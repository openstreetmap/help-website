JOSM only suggests tags that are either included in JOSM's presets or have been used previously in the same geographic area. It is impossible for an editor to know all possible tags because you can use Any tags you like in OSM.

Whether or not a tag is shown on the "default" osm.org map style is unrelated. There is no direct connection between tags known to JOSM, tags documented in the OSM Wiki, and tags used for rendering any particular map. Also note that tags not visualized in the "default" map style can still be used by other maps and applications.
