Hi Brechmos, There 2 possible reasons for the picture you’re getting:

  1. There still have to be added several names to the streets in your town, district or region. In this case threes still a lot of work to do. But if you’re finished you’ll be satisfied with the result.
  2. On the other hand names are attached to the middle or centre of the way or road. So if you make a long distance road in one move it could end up with just one name and it appears to have no name. On the other hand adding a name to each part or section of a way could make the map less readable. Too many names are making the map or way invisible. Or at the end you could built your own separate map based on the OSM data and add all the names you need to plot the routes. There already maps for different users all over the world.

Greetz and watch out

@brechmos: if you are not satisfied with the street name placement or number of shown street names in a specific zoom level you can choose another map style or make your own map style. Our standard "mapnik" osm.org map style (I guess you are looking at that one) is not suitable for everyone. I is probably mainly for us mappers. There are other styles for every need.

See in the wiki: Renderer (e.g. Maperative) and/or OSM on paper (mostly about the already available styles). This web map service with customizable maps seems to be very easy to use, but I do not know if customizing to your satisfaction is possible: http://maps.cloudmade.com/ (see top menu: change style, edit style)
