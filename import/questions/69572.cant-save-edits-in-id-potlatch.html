Can't save edits in iD / Potlatch

0

When I want to save my changes (pressing export button), the following message occcurs: "Diese Seite bittet Sie zu bestätigen, dass Sie die Seite verlassen möchten – Daten, die Sie eingegeben haben, werden unter Umständen nicht gespeichert." In english: confirm that you want to leave this site, your changes might not be saved. This happens with iD editor as well as with Potlatch editor. No input field to type a comment.

I wondered if my edit is too complex (building shape with 40 nodes), but the message occurs even if I move a single node of an existing shape.

Can anybody give me a hint what leads to this behaviour ? Maybe an acciental change of the browser presets (firefox 67.0.1 (64-Bit)) ?

The export button is not the way to save your changes. You are clicking the wrong button. Both Potlatch and ID have their own save buttons. The export button leaves the editing session without saving, and exports the current map view.
