If you want to do photo mapping you're going to need to use an offline editor like JOSM. JOSM can open GEO tagged photos and can even let you uses a GPX track to add a GEO tag to a photo. There is an article on [Photo mapping}(http://wiki.openstreetmap.org/wiki/Photo_mapping) on the Wiki that you might want to read for more information.

PS I guess you can you Potlatch 1 for Photo-mapping but using Potlatch 1 is discourage so use JOSM or if you have strict no JAVA policy you can use Merkaartor
