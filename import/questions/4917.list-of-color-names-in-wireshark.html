List of color names in Wireshark

0
1

The Wireshark coloring rule dialogs show hexadecimal values for the background and foreground colors, for example, #FF0000 for red. At one of Laura Chappell's Sharkfest presentations, she showed that when creating or editing coloring rules, you can enter a color name and Wireshark will translate the name to the appropriate hexadecimal value. This is exceptionally convenient, but the supported names don't seem to be consistent. For example, both "yellow" and "light yellow" work, but "dark yellow" does nothing. "Red" and "dark red" both work, but "light red" does nothing. "Light blue", "blue", and "dark blue" all work.

Is there a list somewhere of all the supported color names (and perhaps their hexadecimal equivalents)?
