1

I am trying to map Kiwi orchard. I tagged them as landuse=orchard and crop=kiwi.Now I want to tag total tree count on orchard. There is 150 kiwi tree. How should I tag this 150 kiwi tree?

Manoj-MOBERA, if you really want to tag the number of kiwi trees, consider using the count key. Read this as well, https://wiki.openstreetmap.org/wiki/Tag:landuse%3Dorchard landuse=orchard, trees=kiwi_trees, trees=150 or kiwi_tree:count=150.

A friendly plea from New Zealand re correct use of the word 'kiwi'.......... Kiwi is firstly a flightless bird native to New Zealand and also a colloquial term for a New Zealander http://en.wikipedia.org/wiki/Kiwi

Although the fruit is often referred to as a 'kiwi' (generally outside of New Zealand) its correct name is 'kiwifruit' (originally known as Chinese Gooseberry). http://en.wikipedia.org/wiki/Kiwifruit

Please can the trees be tagged as 'trees=kiwifruit' and 'kiwifruit_tree:count=150'

If one wants to be really pedantic about it kiwifruit are actually grown on vines like grapes rather than on trees but that is probably being far too picky :)
