It sounds like what you are talking about is doing an import. There are two basic things to take into account when doing an import, procedural and technical. The procedural needs to be done first, basically you need to discuss what it is you are adding to the map, and if the community feels it is beneficial. If the community accepts your edits then you can move on to figuring out how to import the data from the table. You need to provide more information for us about the source of this table, and how it is currently formatted, for us to provide advice on if it is an appropriate import, and what software and method to use.
