A way to figure out router admin url with wireshark?

0

Hi all, I have a DG860 Arris router, but the router admin url isn't working. I have read the manual, and it says that I have to go to http://192.168.0.1/, but that brings up an unresponsive page.

I've tried other urls like 192.168.1.1, 10.0.0.1, 192.168.2.1, but I get the same result. I have tried it on wireless and on wired.

I can't reset the router because my roommates are afraid I'll break it. My roommates don't know how to access the admin url either.

Is there a way to use wireshark to figure out how to get to the admin url on the router?

You need the IP, not the URL I guess ;-)

If you can surf the internet, it usually means that you are using the router as a, well, router. Just run "ipconfig" on a command line prompt to check out what your default gateway is and use that IP in your URL.
