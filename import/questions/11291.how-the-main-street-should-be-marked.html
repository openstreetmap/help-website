While it is possible to add road signs as points to the map, most of the time it does not make a lot of sense. There is a tagging suggestion here https://wiki.openstreetmap.org/wiki/Key:priority_road for road priority, however it has only been used 2000 times which is a very small number for all of OSM.

You -should- tag the main road appropriately (primary, secondary, tertiary and so one) according to the rules in use in the area you are mapping in. These can vary slightly from country to country and it is best to consult the wiki on local practice.
