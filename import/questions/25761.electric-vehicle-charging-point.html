Hi, I think using a node tagged 'amenity=charging_station' is the preferred way. See this page in the wiki http://wiki.openstreetmap.org/wiki/Tag:amenity%3Dcharging_station On the Map Features wiki http://wiki.openstreetmap.org/wiki/Map_Features#Transportation the tag 'amenity=ev_charging' has a comment about "don't use" Graham
