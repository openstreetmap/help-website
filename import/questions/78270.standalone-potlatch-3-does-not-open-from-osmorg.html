It works for me. I downloaded it from a link here https://www.openstreetmap.org/user/Richard/diary/395259 when you run it you get a world map.One option is to zoom all the way in.. which isn't so convenient. A better solution is zoom into the map as we did with Potlatch2 then reduce this window to a half screen. Start Potlatch3 app and reduce that window to the other half of the screen. Go back to the zommed window and click on the Remote edit box. The Potlatch3 window will then be in edit mode, change it to full screen and it is just like using Potlatch2. See JPEG
