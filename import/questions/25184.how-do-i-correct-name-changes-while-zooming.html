How do I correct name changes while zooming?

0

In Rhode Island, USA, the main reservoir near the center of the state is called the Scituate Reservoir. It had been incorrectly tagged as the Quabbin Reservoir (which is actually in Massachusetts). I fixed the name on the edit map where I believed it could be changed, but it still shows up as Quabbin on the view map -- until you zoom in a few times and then the name is correct. Is there a way of correcting this behavior? Thanks.

Is there a way to correct? Yes - wait and/or clear your browser's cache.
