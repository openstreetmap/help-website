If you recorded those GPS coordinates using the waypoint feature of your GPS unit then it is possible to export them as a GPX file. Not sure about ID but you can import GPX files into JOSM and then draw in your new roads. I would guess that is possible with ID too but I haven't used it.

If you only have the numbers on a piece of paper it might be a bit more difficult. In the JOSM advanced mode there is a keyboard short cut (and menu item) for "Add Nodeâ¦" which allows you to specify the point via lat/lon. You should make sure your measurements use the WGS84 datum before keying them in.
