What tag to use for the place name in an address if place is not a city.

0

When writing addresses, I can not put "addr:city" because it is in a Municipality so should I do "addr:municipality" which does not exist?

You should not take the "city" part of the addr:city tag literally. The content of addr:city should be whatever place name is used in the postal address regardless of the status of the place.
