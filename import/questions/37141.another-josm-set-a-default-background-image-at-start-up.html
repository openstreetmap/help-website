Another Josm: Set a default background image at start up

3

Hi. How do I set JOSM to display Bing (or other) background imagery by default whenever I start JOSM. At the moment it starts with just the black background & I have to load it for every session. I've looked in the imagery preferences where I thought an option would be, but can't see it.

  1. Open josm with at least a bing layer, zoomed at your favorite location
  2. Go to file -> session -> save session as
  3. Tick only the bing layer, and save the *.joz file somewhere
  4. Change your josm shortcut (or however you start josm on your system) to add the *.joz as first argument.
