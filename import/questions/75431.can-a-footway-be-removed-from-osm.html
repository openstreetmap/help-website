Can a footway be removed from OSM?

0

OSM shows a footway on private property. Is it possible to remove the footway on OSM? This is private land not meant for public use.

on land private footway

If it actually exists, but isn't accessible by the public, I'd suggest changing the access tags to "private" rather than removing it altogether. That way, if it's visible from aerial imagery, no-one will add it back by accident.

You can do this yourself by zooming in to the area in question in a PC web browser and clicking "edit".
