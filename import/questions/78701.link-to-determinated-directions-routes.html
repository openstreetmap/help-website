0

Hi,

is it possible to create a Share Link to a determinated direction?

Yes, I can visualize a route on a website by creating a screenshot (I'm note allowed to upload an example) but what I want is to create something like "https://osm.org/go/..." that shows the route between two points, not just a certain point on a map.

Is this possible?

Sure, once a route is displayed you can share it by copying the address bar:

https://www.openstreetmap.org/directions?engine=graphhopper_car&route=27.43737%2C-82.45481%3B27.44282%2C-82.46421#map=16/27.4401/-82.4577
