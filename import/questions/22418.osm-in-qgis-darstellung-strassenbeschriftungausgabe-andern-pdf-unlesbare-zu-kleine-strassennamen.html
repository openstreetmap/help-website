Wenn man von Print Composer eine Ebene exportiert, die mit dem OpenLayer-Plugin erstellt wurde, verwendet QGIS nur die aktuelle Zoomstufe im Ansichtsfenster. Wenn du mehr Details möchtest, stelle sicher, dass du auf einer Zoomstufe bist, in der es lesbar ist.

(Ich kann mich nicht erinnern, ob man das im Hauptfenster von Print Composer machen muss, aber es ist ja einfach auszuprobieren).

English original text:

When exporting from Print Composer with a layer sourced from the OpenLayer Plugin QGIS only uses the current zoom level in the viewport. If you want more detail, make sure that you are zoomed in at a level which is legible.

(I can't remember if you do this in the main window of Print Composer, but it's easy to experiment).
