I don't know much about PPA's but I would assume installing software from some random PPA mentioned on the interwebs isn't a great idea.

There is a PPA created by one of the Wireshark core developers (who also produces the Wireshark packages for Debian) available here: https://launchpad.net/~wireshark-dev/+archive/ubuntu/stable

It's not quite up to date at the time I write this (1.12.1 as opposed to 1.12.3) as the maintainer has been having machine problems, but hopefully it will be updated soon.
