If you want to download raster map tiles, then please use one of this tools:

  • https://wiki.openstreetmap.org/wiki/Mobile_Atlas_Creator
  • https://wiki.openstreetmap.org/wiki/OSMtiledownloader

If you need instead raw vector geodata, then please get a Planet extract and maybe cut it with osmosis. Or you use the Overpass API that is read only.

If you otherwise refer to a specific application, please lookup there documentation on how to upgrade map packages.
