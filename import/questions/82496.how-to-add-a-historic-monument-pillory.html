According to Taginfo, currently there are 886 objects using the tag historic=pillory. I'm not sure which editor you're using but I presume you can add this tag manually if it's not present in the "map data pane".

If appropriate, you might also add

tourism=attraction

attraction=historic

Cheers

https://wiki.openstreetmap.org/wiki/Tag:historic=pillory is already documented to be with some significant usage. Absolutely good to use it.

iD, and other editing software won't have a preset for everything. You have to manually enter the tag at the bottom of the left panel.
