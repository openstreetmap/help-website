pure coordinates - how to extract from a given map ?

1

From an OpenStreetMap site I can extract an URL which has for example this form: https://www.openstreetmap.org/?mlat=48.7726&mlon=9.1688#map=12/48.7726/9.1688.

I can also rightcklick on the map an then "show address", it shows the coordinates with an separating comma: 48.78642, 9.05533. But when I copy and paste it, there is more: https://www.openstreetmap.org/#map=18/48.78642/9.05533 !

But this all is not the syntax to type in in the search field, which should be (as far as I know) of the form: 50°8'7"N,7°8'7"E or 48.641956, 9.474610 .

How can I extract from a given site the coordinates which I or another person can directly type into the search field, i.e. the pure coordinates (without "www.openstreet..." and so) ?
