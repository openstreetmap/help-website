I agree, but one thing you can use to make it a little better in iD is some of the keyboard shortcuts. For example, after you double-click to finish an area, click somewhere else to unselect it, then press 3 and you'll be in Area-drawing mode again. A list of the shortcuts is on the wiki: http://wiki.openstreetmap.org/wiki/ID/Shortcuts
