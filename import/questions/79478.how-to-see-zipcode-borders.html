There aren't any zip code borders in OSM. Zip codes are collections of addresses maintained by the post office and don't really have borders (the post office decides internally what zip code to assign an address to).

Census maintains areas created from their zip code data, but I don't really know where they publish it, https://www.census.gov/programs-surveys/geography/guidance/tiger-data-products-guide.html is probably an OK place to start.
