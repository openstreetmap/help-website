Hi

You simply need to create an account on openstreetmap.org and start adding any missing objects yourself.

Most things are very easy to add or modify. The default online editor has a good introduction to the concepts and there is lots of additional material to found here https://www.openstreetmap.org/help

However please do not add information from other maps or data sources (google, Here and so on) for legal reasons except if you have obtained permission to do so.

There is a very active Russian community that will be glad to help you with any issues, see https://openstreetmap.community/ (click on Russia to get a list of possible channels).
