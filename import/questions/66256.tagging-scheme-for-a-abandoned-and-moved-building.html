see https://wiki.openstreetmap.org/wiki/Tag:military%3Dbunker (especially "additional tags" section) and https://wiki.openstreetmap.org/wiki/Lifecycle_prefix . According to your description I would go for

  • abandoned:building=bunker
  • military=bunker
  • bunker_type=???

In a description tag you could add a description of it - including the fact that it is not the bunker's original location.
