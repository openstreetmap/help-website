I created a custom set of buttons for OsmTracker that has a button for each of the normally encountered speed limits in my area all on a single (default) screen. I can just tap the appropriate button as I pass the speed limit sign.

If desired, I could post the file and its associated button images but they are for US style MPH signs. No reason the concept couldn't be applied to other countries though.

Not quite that easy. It created a OSM file that had a point with the speed limit information. I still had to use an editor to place the max_speed=* on the road itself. And if it was a location where the speed limit changed I had to split the way.

That app worked on Android. I have since migrated to iPhone and been unable to find an equivalent app. So my current work flow is to use my dashcam. I have documented that at https://retiredtechie.fitchfamily.org/2018/05/13/dashcam-openstreetmap-mapping/
