You can see see details about how OSM handles disputes like this by reading this document on the foundation's website. As I suspected, the relevant claims in this area are indeed mapped, such as here.

The Halaib Triangle is currently administered by Egypt, so the creators of the OSM Carto map that you see by default at openstreetmap.org have chosen to show that. Different maps use different data, and as the PDF that I linked to suggests, if you want to create a map using that data you are encouraged to do so, and if you ask how to do that I am sure that lots of people will offer to help.
