Hi. What area are you looking at? I can see rivers on the cycle map (e.g. here near Dublin). Remember there are many many different map styles, each tuned to show (or not show) particular features. On the openstreetmap.org website, only a select few are shown. But there are more.

For example:

  • Hike & Bike Map which might be tuned to be a more "hike" friendly than the "cycle" map on osm.org.
  • OpenAndroMaps which are offline maps that might show more "hiking" stuff.
  • Waymarked Trails shows hiking routes.

The OpenStreeMap wiki lists several other hiking mapping options.

It all uses the same, underlying OSM database.
