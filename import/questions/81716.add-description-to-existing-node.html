Hi, it appears you are editing with maps.me. I have experienced some issues with editing with that tool myself.

Please try the iD editor on www.openstreetmap.org. Go to the area you want to edit the node and click the edit button at the top left. If not already done you have to log in with your OSM account (the one you also used in maps.me). It should be pretty intuitive how to add information to the node you created. If not you can follow the tutorial that is provided in the editor.

Please not that a description is only meant for a short descriptive text. But there are many more practical keys (fields) available to provide more information like opening hours, address, website etc.
