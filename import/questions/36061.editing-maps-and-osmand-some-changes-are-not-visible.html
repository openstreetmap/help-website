Editing maps and OsmAnd: some changes are not visible

0

Hello, I edited some keys like lanes, turn:lanes, lanes:forward, lanes:backward, traffic_calming and so on...

Today there was the second update of the OsmAnd offline maps after I have done this changes, but nothing is changed inside OsmAnd navigation.

I have also changed or added new streets, parks, trees, police offices, and POIs. - These data are availabe inside OsmAnd.

Maybe do you know why some changes inside OsmAnd are visible and some others not?

Near as I can tell, from using it rather than looking at the OsmAnd code, is that OsmAnd has only minimal support for lanes (I see it during navigation on motorways near exits at the top of the screen) but no support for lanes:forward, lanes:backward, traffic_calming, etc. From the discussion forum for OsmAnd I think there is some work to have those but no timeline.
