How much data do you need? If it's a very small area then the "export" tab on the main site should work. However, I suspect that you would be better with one of the country and area extracts described on this page.

I tend to use extracts from here, but other download sites are available.

If you try and export too big an area using the "Export" tab you may get issues such as this one.
