This can have several reasons:

  • be sure that you have flash player installed on your PC ... did you ever see tha map data displayed in any version of Potlatch?-
  • be sure that there is no problem with the OSM servers that deliver the raw OSM data to display the elements ... check Platform Status in the OSM wiki--

  • are you able to download the OSM map data with any offline editor like Merkaartor or JOSM? -

  • We also need to know whether you are using (old) Potlatch1 or recent Potlatch2 ... can you edit your question and tell us more?
