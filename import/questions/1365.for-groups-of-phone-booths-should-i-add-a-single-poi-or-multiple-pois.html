For groups of phone booths, should I add a single POI or multiple POIs?

1

If there is a group of two or more public telephone boxes (booths) all together with no spaces between them, should I add a separate POI for each phone or a single POI with a suitable tag to indicate the number?

Depends if you just want tag amenity=telephone, and no more. Or want detail information like add contact:phone=, ref= and so on.

You are free to choose your detail level, even if you don't want add detail for each phone you can separate it, that indicate number of phones available.
