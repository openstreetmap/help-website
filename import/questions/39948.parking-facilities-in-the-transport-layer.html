Thanks for the comment. This is a help system intended for questions - not for suggestions. You should use a 'bug-tracker' for the latter.

OpenStreetMap's standard bug-tracker is at http://trac.openstreetmap.org/ . You can post suggestions there. That said, they need to be filed against a 'component' (i.e. a part of the system), and the transport tiles aren't currently included. You could perhaps add the suggestion against the 'opencyclemap' component, as that's created by the same person (Andy Allan).
