Using OSM standard tile layer (via OpenLayers) in commercial application

0

Hi,

I am using OpenLayers library for embedding maps into a commercial browser application. I am using OSM as tile server (https://openlayers.org/en/latest/examples/simple.html ) with OpenLayers API. The embedded map contains some markers with tooltips. I want to embed OSM map () into a commercial browser application. Open Layers has BSD 2 licence and using the OSM tiles ( https://www.openstreetmap.org/ ) are under the CC-BY-SA license.

Below are my queries:

  1. Can I use OSM via OpenLayers in commercial application ? If yes, what should be the licence of the commercial product ?
  2. Should I display the credits "Â© OpenStreetMap contributors" in the embedded map ?

Thank you !

  1. You may if you adhere to the https://operations.osmfoundation.org/policies/tiles/ Tiles Usage policy and the the attribution guildelines as seen in 2. Be aware that OSM is not a tile service and your application may get blocked without any given reason. So this may not be the best business decision.

  2. See: https://www.openstreetmap.org/copyright - you not only should but you have to attribute OpenStreeMap contributors.
