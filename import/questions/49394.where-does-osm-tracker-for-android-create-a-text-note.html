Where does OSM Tracker for Android create a text note?

0

If I press a button to create a text note, at which moment will it actually be created: when I originally pressed the button or when I press OK after entering the note text? When moving at 100 km/h, the difference is important. I hope it is the former, but I would like to double check.

Hi Kotya!

The point is saved at the location where you hit "create a text note" - it does not matter if you don't type in that moment or later.

Normally, first I create the note and start to write a few seconds until 5 minutes later. Sometimes I turn off my screen and then type later. I tried while riding a bus or taxi, it's the same: the point is still where you first pressed the "text note" button.
