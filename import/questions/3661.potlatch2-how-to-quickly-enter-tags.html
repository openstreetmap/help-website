Potlatch2 - how to quickly enter tags?

1

When entering tags in Potlatch2, there doesn't seem to be an easy way of completing tags. For example, if I'm adding a Rugby pitch, after typing "le" I'd like to be able to select the one offered option in the drop box ("leisure"). What'd I'd expect to be able to do is to type down-arrow to select it and then tab to move the next field, but that doesn't seem to work. Currently it seems as if you have to leave the keyboard, find the mouse, select the option, back to the keyboard, hit tab, and repeat the procedure - but presumably I'm mistaken?
