Hi, That looks like No Fill (Wireframe) in the ID editor. To remove that effect click the Map Data symbol (shortcut U). In Style Options select Partial Fill which gives wider highways, you see the change instantly. There's lots of things in those side menus that can be played with.

Regards Bernard.
