The best place to start is with the man page! Either type "man tshark" on your linux/unix command line, or check it out on line: http://www.wireshark.org/docs/man-pages/tshark.html

It's not an "option" in the sense that you run Wireshark with a command-line option; it's a command of its own. As zachad suggested, "man tshark" should give you a UN\X man page on a UN\X system, and the URL in the other answer shows you an on-line version of the man page.
