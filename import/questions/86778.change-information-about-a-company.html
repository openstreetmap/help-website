Hello and welcome to the OpenStreetMap community, we're glad to have you with us!

  1. Please start by reading the Beginner guide and use the OSM wiki as a reference, most of your questions are already answered, as well as some much needed best practices.
  2. Check if there is existing data and update it with information you think is beneficial to the map.
  3. For a specific company, check the primary tags and the tags that can be used in combination. This page should get you started - https://wiki.openstreetmap.org/wiki/Tag:office%3Dcompany
