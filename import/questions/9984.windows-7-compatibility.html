The first version released!

I just downloaded and installed 0.99.2, and that was able to run and capture traffic on Win 7 SP1 64 bit.

There were some graphical glitches, but nothing to make it unusable, and I did the capture using the installed WinPCap 4.1.2 rather than revert to 3.1 which came with that version.
