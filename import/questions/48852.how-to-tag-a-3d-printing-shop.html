if you search https://taginfo.openstreetmap.org/keys/shop#values for "3d" you find 4 shops which are likely like yours – tagged with different tags. Similar to other tags (no caps, underscores instead of spaces), I would use shop=3d_printing for your shop. As always, just add (in addition) a description (e.g. description=shop for doing 3D prints) and its website.
