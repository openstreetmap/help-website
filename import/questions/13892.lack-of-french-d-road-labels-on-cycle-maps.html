Lack of French D road labels on cycle maps

0

Why do the French D etc road labels appear on the standard OSM but not on the cycle maps? For cycling users (particularly those using hand held devices)these would be particularly useful. is there any way they can be transferred?

By "D road", you probably mean the "Départementale" reference number. In OSM, we use the tag "ref" for this. If it is not rendered on cycle map, it is a choice of the style map designer. Go to the cycle map home page and try to contact the creators (email provided on the web site) and ask them why they decided to not render the "ref" tag.

You can lodge an enhancement request for the OpenCycleMap style at https://trac.openstreetmap.org/newticket - select "opencyclemap" from the "Component" dropdown. You'll need to log in with your standard OSM login/password. Please be polite, specific, and modest in your request.
