Adding a house number from a GPX waypoint

0

I happen to have the coordinates for a house in a new residential area. The street is on the map, but no houses are drawn, and I'm unable to pinpoint the house on the map.

There must be some .GPX file I can make from <wpt lat="59.6582207" lon="9.6543459"> <name>Tislegårdveien 24</name> </wpt> and upload somewhere?

(Going back there with a smartphone is now a 1000 km option.)

You have numerous options, but ths easiest is likely simply to use an editor that allows you to set the coordinates of a node (JOSM, vespucci, https://wiki.openstreetmap.org/wiki/Level0, others?). The hackish solution is to simply center the map on the coordinates and add the node there, as in

https://www.openstreetmap.org/#map=19/59.6582207/9.6543459
