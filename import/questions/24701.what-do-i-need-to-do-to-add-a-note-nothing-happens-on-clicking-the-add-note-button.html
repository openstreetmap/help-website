About 40 notes were created today also in the last minutes notes are created.

  • Could be that you (or you are) disallowe(/d) to make connections to the server api.openstreetmap.org from www.openstreetmap.org.
  • Could be that the API was not available (due to whatever problems) at that time your tried. Could be that there was a problem (e.g. read-only mode due to maintenance) with the database behind the API when you tried it.
  • Could be that your browser is not supported (you did not write which you are using)

In all cases a proper error message of the notes feature would be nice (apparently there is not in your case), so please help us to track down the problem.
