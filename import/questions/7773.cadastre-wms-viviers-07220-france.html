Cadastre WMS - Viviers 07220, France

0
1

Hi, I tried to upload Viviers cadastre (Postal Code 07220) to JOSM (rel. 4399) with Cadastre-fr pluggin (rel. 26606). I got only a set of colored rectangles but no buildings, nothing :( Any idea ? Rgds, Olivier

You have to change the projection. First, go to the JOSM preferences (F12), third tab (the grid symbol) then select the projection "Lambert CC 9 zones (France)" and next, the Lambert CC Zone "4 (44 to 46 degrees)". Then download the cadastre.
You can find more explanations on the wiki about which projection to use in JOSM for the french cadastre:
http://wiki.openstreetmap.org/wiki/FR:JOSM/Fr:Plugin/Cadastre-fr#Projections

If you have other questions about the french cadastre, contact the french mailing list talk-fr@openstreetmap.org or one of the french forums.
