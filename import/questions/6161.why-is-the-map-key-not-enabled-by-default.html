Why is the "Map Key" not enabled by default?

-1

This site is great - I've only just discovered it! So why was the "map key" not enabled by default? Why did it take so long for me to google and search on here in order to find out where it is? This feature is one that I would expect to be so simple for a user on a map-type interface.

I guess that's the standard behaviour of all popular web sites with maps on them - a map key is available on request, if at all, because it takes up too much screen real estate and one would rather activate it on the few occasions where one is interested, than deactivate it on every site visit because it blocks the map view!
