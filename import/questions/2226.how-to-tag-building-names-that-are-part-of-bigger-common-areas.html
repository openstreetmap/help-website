How to tag building names that are part of bigger, common areas?

5

I'm working on a block of different-sized buildings, which is effectively one big building. Each building has a name, which I want to tag, but it's impossible to accurately determine the areas for the individual buildings; it seems the best I can do is to tag the entrance doors. I've been looking at name= and addr:housename= , but I'm not sure if either of these is right. So, can anybody advise me about the right tags to use, or suggest a better system?

Thanks.

I would tag the entrance node with building=entrence addr:housenumber= addr:housename=

It might be hard to define different buildings in a block but from what I can understand two buildings can be considered the same if they are in the same block and have the same hight.
