<div class="answer-body">
<p>The Garmin Edge 800 is able to read GPX files, you just have to put them in the right place. So connect the Edge to your PC, and it should appear as a removable drive. Now go to that drive, then open the folder <code>Garmin\NewFiles</code>. Then copy the GPX file into that folder. Then disconnect the Edge from the PC, and switch it on. Then it should load the GPX file.</p>
<p>An alternative is to use <a href="http://www8.garmin.com/support/collection.jsp?product=999-99999-04">Garmin Training Center</a> or <a href="http://www.garmin.com/us/products/onthetrail/basecamp">Garmin BaseCamp</a>. Both are available as a free download for Windows or Mac. You can import the GPX file into Training Center or BaseCamp, then send it to the Edge.</p>
