<div class="answer-body">
<p>Two possibilities :<br />
- you noted down the coordinates with POI's in your Iphone application. Find a way to export the POI's is GPX format then either upload them into one of <a href="https://wiki.openstreetmap.org/wiki/Editing">the OSM editors</a> (<a href="https://wiki.openstreetmap.org/wiki/Potlatch_2">Portlatch2</a>, <a href="https://wiki.openstreetmap.org/wiki/JOSM">JOSM</a>, <a href="https://wiki.openstreetmap.org/wiki/Merkaartor">Merkaartor</a>). Then simply add a new node tagged with "<a href="https://wiki.openstreetmap.org/wiki/Key:place">place</a>" ("place=<a href="https://wiki.openstreetmap.org/wiki/Tag:place%3Dtown">town</a>" "<a href="https://wiki.openstreetmap.org/wiki/Tag:place%3Dvillage">village</a>" or "<a href="https://wiki.openstreetmap.org/wiki/Tag:place%3Dhamlet">hamlet</a>" depending on the definition of the tag place in this country (I found <a href="https://wiki.openstreetmap.org/wiki/Tr:Map_Features#Yer_s.C4.B1n.C4.B1flamas.C4.B1">this page here but not sure</a>). The limits between the different 'place' values are country specific; check the wiki or contact the <a href="https://wiki.openstreetmap.org/wiki/Mailing_lists">local mailing list</a> to be sure about the criteria). More tags can be required like the place name, a reference, a postcode, etc...<br />
- you don't have POI's but just a list of GPS coordinates (datum WGS84) : with JOSM editor, you have a simple feature which can help here, <a href="http://josm.openstreetmap.de/wiki/Help/Action/AddNode">the "Add Node.." tool</a> where you just type in the coordinates and the node is added at the right location. Then you just attach the related tags as described above and repeat the operation for the 10..15 locations you surveyed.</p>
</div>
