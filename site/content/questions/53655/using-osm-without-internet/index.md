<div class="answer-body">
<p>It largely depend on which device you want to use OSM. The website on osm.org cannot be used offline as far as I know.</p>
<p>However, since OSM is basicly a large database, there are some possibilities to use an offline map. The wiki page <a href="https://wiki.openstreetmap.org/wiki/Offline_Openstreetmap">Offline OpenStreetMap</a> lists a number of ways to do so.</p>
<p>There are several apps for smartphones that offer you take a part of the world in your pocket: Maps.me, OsmAnd to name a couple.</p>
