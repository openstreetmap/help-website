<div class="answer-body">
<p>I once lost my gpxes too. I found that the setting "Mapnik (true)" hid the gpx trace from view if it was set thus before opening a trace.</p>
<p><img src="/upfiles/Screen_shot_2013-11-02_at_10.18.31_PM.png" alt="screen" /></p>
<hr />
<p>Assuming that you have not loaded a gpx trace until now using JOSM, my procedure is ...</p>
<ul>
<li>Open JOSM</li>
<li>Select file . open then locate the .gpx file on your computer, select open</li>
<li>JOSM displays the track.</li>
<li>You should also see it listed in the layers box. <img src="/upfiles/Screen_shot_2013-11-02_at_10.28.09_PM.png" alt="layers screen" /></li>
<li>Right-click the track listed in the layers box and customise the colour to something bright, select 'customise track drawing' to change the 'drawing width of gpx lines' to 2 or 3</li>
</ul>
<p>and also add any photos taken by device with a gps chip whilst you made the gpx track and any .wav waypoints you made and these will be added as separate layers and will be seen along the track.</p>
<ul>
<li>By now you can are ready to Right-click the track listed in the layers box and select 'Download from OSM along this track'. You will see options to download 'openstreetmap data' and 'raw gps data' which others have already loaded into the osm database.</li>
</ul>
<p><img src="/upfiles/Screen_shot_2013-11-02_at_10.42.11_PM.png" alt="screen" /></p>
<ul>
<li>Now in the layers box you need to move your gpx to the top of the layers to see it overlay the others layers (select it and click on the up arrow)</li>
<li>You can now start following your gpx track, and add nodes to make the new ways and points of interest and tags. Once finished you upload the data to update the osm.</li>
<li>You can also edit the gpx track from the layers box by changing the gpx to a layer, selecting and deleting any points not wanted and then convert back to gpx and save over the original file on your pc if desired.</li>
</ul>
<hr />
<p>I normally upload my .gpx traces to OSM via the main OSM site <a href="http://www.openstreetmap.org"></a><a href="http://www.openstreetmap.org">http://www.openstreetmap.org</a> via the GPS Traces link at the left side <a href="https://www.openstreetmap.org/traces">GPS Traces</a> and select 'upload a trace'.</p>
<p>I hope this helps...If not, tell us as someone else will also help.</p>
<p>Cheers.</p>
