<p>You can just do an edit to <span>your</span> question text (login, then there will be a link "edit" below your text). Or add a comment below it (telling that you still are interested in an answer). In the default sorting the question then will be on top again.</p>
<p>If possible try to state your question clearly, maybe improve it to make clear what you mean.</p>
