<div class="answer-body">
<p>I have to say that you are likely doing something rather wrong.</p>
<p>You always need to download the area you want to edit in before you change/add anything. The download will include all data (including that from other users naturally) in the area selected.</p>
<p>Current vespucci versions will complain rather bitterly if you create or edit objects outside of the downloaded area, so you would normally notice the issue.</p>
<p>If we are misunderstanding you, could you supply a link to the area in question (from openstreetmap.org) and the vespucci version you are using (go to "Advanced preferences" to see that).</p>

<p>I noticed that you added a number of ways with a sole cycleway=lane tag, that is likely not what you wanted to do. cycleway=lane is an additional property of an existing way, not a standalone way.</p>
