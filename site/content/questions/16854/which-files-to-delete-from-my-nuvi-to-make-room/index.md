<div class="answer-body">
<p>If you want to be on the safe side, get an external SD card, create a directory called <em>Garmin</em> on it and put the new <em>gmapsupp.img</em> in this directory.</p>
<p><em>gmapprom.img</em> is the pre-installed map, you probably don't want to delete it. If you decide otherwise, better make a backup before deleting it. <em>gmapbmap.img</em> is the base map containing only large roads and larger cities, not really suitable for routing. You could delete it but as it is only about 50 MB you wouldn't gain much space.</p>
<p>Another option would be to go to <a href="http://garmin.openstreetmap.nl/">garmin.openstreetmap.nl</a> and select only the tiles you are interested in and see if the generated map fits on your device.</p>

<div class="answer-body">
<p>When I tried to use Garmin Express to update my maps on my Garmin Nuvi 660, it said I didn’t have enough space so it would only download a partial map of my choice (Southern US and Mexico or select areas of the U.S.</p>
<p>Here’s what I did to get the latest maps.</p>
<ol>
<li>Get a SD card for format it with FAT32. Garmin says it will only accept up to a 4GB card for the Garmin Nuvi 660, however I used an 8 GB card with no issues.</li>
<li>Create a folder named Garmin on the SD card.</li>
<li>Connect the Garmin device to the computer.</li>
<li>Copy the file named garmindevice.xml from the Garmin folder on the Nuvi to the Garmin folder on the SD drive.</li>
<li>Disconnect the Nuvi.</li>
<li>Launch Garmin Express. The SD card will now be recognized as the Garmin device.</li>
<li>Download the map(s) you want. I downloaded the lower 49 states.</li>
<li>After the download is complete, you will have 3 maps files in the Garmin folder of the SD card. They will be named gmapprom.img, gmaporom.unl and gmapprom.gma.</li>
<li>Delete the garmindevice.xml file from the Garmin folder on the SD drive so the SD drive will no longer be recognized as the Garmin device.</li>
<li>Plus in the Garmin Nuvi device. It will now be recognized as the Garmin device.</li>
<li>Delete the gmapprom.img, gmaporom.unl and gmapprom.gma files from the Garmin folder on the Nuvi. You have to delete them first because for some reason, you can’t just overwrite them. You’ll get the message that there’s no enough space.</li>
<li>Copy the three files from the Garmin folder on the SD card to the Garmin folder on the Nuvi.</li>
<li>Reboot the Nuvi. The new maps will now be loaded.</li>
</ol>
<p>I then deleted the files from the SD card and used it to load almost 3,000 MP3 files to take with me in the SD slot in the Nuvi.</p>
