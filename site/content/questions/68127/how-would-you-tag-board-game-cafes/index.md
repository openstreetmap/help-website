<p>I'd like to tag board game cafes.</p>
<p>I'd say there are two main types:</p>
<ol>
<li>it's primarily a board game cafe, which means customers usually pay by the hour. There are also drinks available.</li>
<li>it's primarily a coffee shop/cafe, but has some board games freely available for use</li>
</ol>
<p>For 1, I don't know what to tag at all, except <a href="https://taginfo.openstreetmap.org/keys/shop%3Agames"><code>shop: games</code></a> if there are also games for sale. But this doesn't mean it's a board game cafe.</p>
<p>For 2, I'd tag it as <code>amenity: cafe</code> with all accompanying tags, but I'm not sure what to add to specify that board games are available. <a href="https://taginfo.openstreetmap.org/keys/toys%3Aboard_games"><code>toys:board_games</code></a>? But it's barely being used at all...</p>
<p>Any inputs?</p>

<div class="answer-body">
<p>We have two board game cafes in Nottingham. Both are tagged amenity=cafe, cafe=board_game, with no additional tags. See:</p>
<ul>
<li><a href="https://www.openstreetmap.org/way/116810770">The Dice Cup</a></li>
<li><a href="https://www.openstreetmap.org/node/2535765175">Ludorati Café Bar</a> (<a href="http://www.ludoraticafe.com/">website</a>)</li>
</ul>
<p>I would think for general cafes, pubs etc which may have a small selection of board_games, something like board_game=yes, or board_game=yes (a few behind the bar) would work too. If a place specialises in a particular board game then replace yes with the name of the game, or for very widely played games such as chess, backgammon and go, use the explicit game in the same way (although for chess the use of sport=chess is already widely used). Note that in the UK, playing backgammon for stakes in a public place requires that the bar has an appropriate gambling licence.</p>
<p>None of these tags are <a href="https://taginfo.openstreetmap.org/search?q=board_game#values">heavily used</a>, but if one city with a 300k population has two there must be many more.</p>
