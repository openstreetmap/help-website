<div class="answer-body">
<p>To download OSM data around a gps track you can use the entry in the <a href="https://josm.openstreetmap.de/wiki/Help/Menu/GPXLayer">rightclick menu of the gps track</a> in the <a href="https://josm.openstreetmap.de/wiki/Help/Dialog/LayerList">layer dialog</a> after opening it with JOSM. If you cannot see your gps track anymore after this, you may want to change the layer order (gps layer on top) by using the <a href="https://josm.openstreetmap.de/wiki/Help/Dialog/LayerList#LayerUp">up/down layer panel buttons</a>.</p>
