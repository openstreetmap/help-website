<div class="answer-body">
<p>As far as I know, none of these information. Because osmdroid can only display <a href="https://wiki.openstreetmap.org/wiki/Tiles">tiles</a> but is incapable of processing raw data (<a href="https://wiki.openstreetmap.org/wiki/Elements">elements</a> and <a href="https://wiki.openstreetmap.org/wiki/Tags">tags</a>).</p>
<p>Take a look at <a href="https://wiki.openstreetmap.org/wiki/Routing">routing</a> in the wiki and maybe you can find a better suited library for your project.</p>
