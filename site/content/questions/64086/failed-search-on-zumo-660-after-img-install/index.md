<p>If you can pre-plan on a PC. Do the search on the PC, zoom in on openstreetmap with the destination in the center, note the Lat long see <a href="https://www.openstreetmap.org/#map=17/52.14031/-0.04495">https://www.openstreetmap.org/#map=17/52.14031/-0.04495</a> the N =52.14031 and the West (=minus sign) =0.04495 in this case. On my Garmins I do "where to" "Coordinates" set format "h ddd.dddd" ( as used by Openstreetmap) type in n 52.14031 w 0.o4495 "next" Does the map show the correct place yes? then "Go!" this works for me which is OK if you can pre-plan. If on the road "Where to" "Browse Map" browse and do a "Go"</p>
