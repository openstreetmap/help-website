<div class="answer-body">
<p><a href="https://wiki.openstreetmap.org/wiki/Keypad-Mapper_3">KeypadMapper3</a>, an Android App, is designed specifically for collecting addresses. Available on <a href="https://play.google.com/store/apps/details?id=de.enaikoon.android.keypadmapper3&amp;feature=search_result&amp;hl=en">Google Play</a> or from <a href="https://www.enaikoon.com/en/mobile/plugins/keypad-mapper-3/">Einakoon</a>.</p>
<p>It also collects mobile phone cells for OpenCellId: a feature which cant be switched off in the distributed version of the app. Some (not me) dislike this feature.</p>
