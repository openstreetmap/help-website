<p>While Simon's idea to use custom presets is perhaps the best way to enhance your JOSM experiences, it might be a bit intimidating.</p>
<p>But you can remember the keys over a restart. See the instructions on the <a href="https://josm.openstreetmap.de/wiki/Help/Dialog/AddValue">Add Value Dialog</a>.</p>
<p>Right-click on the dialog and check the "Remember last used tags" flag.</p>
<p>p.s. I can't remember I did it like this, I probably switched one of the variables in the settings pane under preferences by hand.</p>

<div class="answer-body">
<p>Yet another option (not exactly using JOSM) - key value pairs.</p>
<p>Save required tags and their values in a text file as follows:</p>
<pre><code>post_box:type=pillar
royal_cypher=VR
tag3=value3</code></pre>
<p>When you want to apply one of these tags, copy them and paste into the POI using the shortcut ctrl+shift+v</p>
