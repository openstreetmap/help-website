<div class="answer-body">
<p>API is an abbreviation of <a href="https://en.wikipedia.org/wiki/Api">Application programming interface</a> and the <a href="https://wiki.openstreetmap.org/wiki/Api">OpenStreetMap API</a> specifically describes how a program can send and retrieve OpenStreetMap data.</p>
<p>Thus, you may think of an API as a description of how computer programs should talk with each other to get the job done.</p>
