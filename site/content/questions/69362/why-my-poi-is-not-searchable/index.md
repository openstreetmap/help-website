<p>What are you using to search? Are you typing in "drinking water" on www.openstreetmap.org? That does not work, since you can only search for names on that site.</p>
<p>There are other maps like <a href="http://openpoimap.org/?map=userpoilayer&amp;zoom=15&amp;lat=43.86465&amp;lon=11.16677&amp;layers=B00T">OpenPOIMap</a> where you could do that. Your node is found there if you search for a user POI "amenity=drinking_water".</p>
