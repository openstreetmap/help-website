<div class="answer-body">
<p>iD doesn't have any plugins or tools. You just draw the building outline using the "Area" tool. If you want to make a multipolygon relation (for a building with a hole in it for example) then you can't use the current version of iD because it doesn't really support relations. The next version has some support for relations and you can use it from here: <a href="http://openstreetmap.us/iD/release/">http://openstreetmap.us/iD/release/</a></p>
