<p>Hello OSM-Community,</p>
<p>for switching between OSM and Google Maps I used for a very long time the plugin <strong><a href="https://addons.mozilla.org/de/firefox/addon/osmjumper">OSMJumper</a></strong>. One could open the current bbox in different map services easily by using the context menu and select "<em>Open map in ..</em>"</p>
<p>For me it was very useful and I loved it. But since Google introduced its new GUI and simply switched off the old one, OSMJumper does not work any more.</p>
<p>Does anyone know an alternative to switch between these two map-services?</p>
<p>Thank you very much :-)</p>
<p>Peer</p>
