<div class="answer-body">
<p>I think you have two steps here: first, identify which buildings is at that location. Second, get the building's dimensions.</p>
<p>For the first part, <a href="https://wiki.openstreetmap.org/wiki/Overpass_API">Overpass API</a> may help you. I do not much of it, but the "query features tool" (number 11 at <a href="https://wiki.openstreetmap.org/wiki/Browsing">https://wiki.openstreetmap.org/wiki/Browsing</a> ) lists enclosing features if you click the map at a location. Look at your browser's network console (e.g. ctrl+shift+E in Firefox) to see the issued request to Overpass API. There are many other <a href="https://help.openstreetmap.org/tags/overpass/">questions on Overpass API</a>.</p>
<p>For the second part, you can use Overpass API, too. It can return the building's edge nodes (which themselves contain coordinates). See the other question <a href="/questions/62275/">where-do-i-get-geographic-coordinates-for-osm-building-corners</a> for more info on OSM building objects.</p>
