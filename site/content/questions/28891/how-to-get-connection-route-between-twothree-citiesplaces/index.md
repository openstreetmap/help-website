<p>First of all, it's not a silly question. Adding routing support to the front page of OpenstreetMap.org has been a goal for <a href="https://wiki.openstreetmap.org/wiki/Top_Ten_Tasks#Routing_frontend">some time now</a>, but it's not available there currently. However, there are a number of other sites that use OpenStreepMap data that do support routing. <a href="http://map.project-osrm.org/">Here's one</a>, and <a href="http://open.mapquest.co.uk/">here's another</a>. And here are <span>more</span> (and info about routing with OSM).</p>
