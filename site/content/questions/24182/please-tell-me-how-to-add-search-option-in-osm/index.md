<p>What have you tried so far?</p>
<p>For searching OSM uses <a href="https://wiki.openstreetmap.org/wiki/Nominatim">Nominatim</a>. The result can be returned either via <em>XML</em> or <em>JSON</em>. And as long as you comply with the <a href="https://wiki.openstreetmap.org/wiki/Nominatim_usage_policy">usage policy</a> you can even use OSM's Nominatim instance. There are also <a href="https://wiki.openstreetmap.org/wiki/Nominatim_usage_policy#Alternatives">alternative instances</a> with a less restrictive usage policy and of course you can <a href="https://wiki.openstreetmap.org/wiki/Nominatim/Installation">install your own Nominatim instance</a> .</p>
