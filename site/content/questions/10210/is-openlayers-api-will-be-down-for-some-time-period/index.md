<p>You can always copy OpenLayers.js and the assorted images and CSS files to your local site. This is what many web sites do, including <a href="http://openstreetmap.org">openstreetmap.org</a></p>
<p>This is however not an OpenLayers support site. Most OpenLayers mailing lists are hosted on <a href="http://osgeo.org">osgeo.org</a> so they should still be available even if <a href="http://openlayers.org">openlayers.org</a> does not work. See <a href="http://trac.osgeo.org/openlayers/.">http://trac.osgeo.org/openlayers/.</a></p>
