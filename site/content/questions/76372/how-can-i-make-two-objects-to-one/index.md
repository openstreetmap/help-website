<p><strong>Using iD editor</strong>: if they are nodes, simply drag one element on another; if they are ways, copy the tags from the wrong one into the correct one and delete the wrong one.</p>
<p><strong>Using JOSM editor</strong>: if they are nodes, drag one node on top of the other, then press "M". If they are ways, , copy the tags from the wrong one into the correct one and delete the wrong one.</p>
