<p>Hi Whiskey1-11,</p>
<p>welcome to OSM! Great that you want to help! I will repeat some details which are available by searching this help site to make it easier for you:</p>
<p><em>First question:</em> your point does not show up on the default map because</p>
<ol>
<li>its category is not shown on the default map (note that there are many maps which are based on OSM),</li>
<li>or the point does not show up because the map has not yet been updated by the server</li>
<li>or your browser still shows an old cached part of the map. Usually hit Ctrl+F5 to request a fresh map part from the server. The exact key depends on the browser (see <a href="https://en.wikipedia.org/wiki/Wikipedia:Bypass_your_cache#Bypassing_cache">this Wikipedia Help page</a> for more info).</li>
</ol>
<p>See the older question <a href="/questions/102/">i-have-made-edits-but-they-dont-show-up-on-the-map</a> for more general details.</p>
<p>I had a look at your edit. Your travel agency <a href="https://www.openstreetmap.org/node/5330155121">https://www.openstreetmap.org/node/5330155121</a> shows up for me (as a purple icon showing a suitcase and a sun). So, try to view the map and then hit Ctrl+F5.</p>
<p><em>Second question:</em> after uploading you can exit the editor by just closing the browser tab/window or navigate to another page. E.g. you can click on the OSM logo in the top left corner to get back to the non-edit map view.</p>
