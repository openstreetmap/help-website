<p>Die üblicherweise für "rollstuhlgerecht" verwendete Eigenschaft heißt <a href="https://wiki.openstreetmap.org/wiki/DE:Key:wheelchair">wheelchair</a>. Eventuell unterstützt der von dir verwendete Editor (du schreibst nicht welcher) diese Eigenschaft nicht als Vorlage. So gut wie in allen Editoren kannst du aber manuell das Attribut/Tag/Eigenschaft an das Objekt (hier: "Punkt") manuell hinzufügen (hier z.B.: <code>wheelchair=yes</code>). Siehe unten auf <a href="https://wiki.openstreetmap.org/wiki/DE:Attribut">https://wiki.openstreetmap.org/wiki/DE:Attribut</a> .</p>
<p>Wenn du nur die Rollstuhlgerechtigkeit ändern willst, kannst du auch einfach den Editor von <a href="http://wheelmap.org">http://wheelmap.org</a> verwenden.</p>
