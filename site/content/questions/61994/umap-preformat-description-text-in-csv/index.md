<div class="answer-body">
<p>1) I capture my data in Google spreadsheet quite normally with Enter for line breaks.</p>
<p>2) I copy the data into a new worksheet and change the character (10) to "\N". =WECHSELN('Erfassung OCCU-NOTE-SOUR'!B1;ZEICHEN(10);"\n")</p>
<p>3) I export the worksheet into a csv file.</p>
<p>4) I import the csv file into uMap.</p>
<p>5) A change in uMap is not possible. A change destroys the LF.</p>
<p>Works reliably.</p>
