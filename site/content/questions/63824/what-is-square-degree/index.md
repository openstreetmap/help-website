<div class="answer-body">
<p>Simply degrees*degrees or put differently the South-North side of the bounding box measured in degrees times the West-East side of the bounding box measured in degrees. The value is simply an upper limit for size of the bounding box.</p>
<p>In practical terms you will not be able to actually retrieve (nor should you) such a large area in many places, please don't forget that the OSM API is an API for editors not for general data retrieval.</p>
