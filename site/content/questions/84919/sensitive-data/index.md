<p>You will need to build yourself something local with <a href="https://leafletjs.com/">https://leafletjs.com/</a> or <a href="https://openlayers.org/">https://openlayers.org/</a> if you want to use raster map tiles.</p>
<p>Or you can use vector tiles (which you could store locally) with <a href="https://maplibre.org/">https://maplibre.org/</a> or mapbox-gl.</p>
