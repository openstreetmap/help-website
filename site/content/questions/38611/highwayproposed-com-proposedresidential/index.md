<p>Como faço para definir que uma via está em construção e que também é residencial? ou<br />
que é projetada e que também é residencial?<br />
</p>
<p>Entendo que devo definir os 2 parâmetros simultaneamente, senão fica um tipo de via indefinido.<br />
</p>
<p>Só consegui da forma abaixo (conforme imagem) mas não sei se está correto:</p>
<ul>
<li>highway=proposed (projetada) e...</li>
<li>highway_1=residential</li>
<li>Será se está correto?</li>
</ul>
<p>E ainda para via em construção:</p>
<ul>
<li>highway=construction e...</li>
<li>highway_1=residential</li>
</ul>
