<p>There are typically two options for this:</p>
<ul>
<li>Split the road into smaller pieces and delete those. (also useful if you have only been able to confirm that one end of a road is gone)</li>
<li>Use another editor like JOSM that can deal with much larger areas.</li>
</ul>
