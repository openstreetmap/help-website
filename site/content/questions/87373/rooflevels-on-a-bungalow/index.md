<p>Hi, This page gives a good picture of building levels tagging:- <a href="https://wiki.openstreetmap.org/wiki/Key:building:levels">https://wiki.openstreetmap.org/wiki/Key:building:levels</a> From it you should see a bungalow with a pitched roof would be tagged building:levels=1 (the ground floor), roof:levels=1 (the roof).</p>
<p>With regard to roof levels please see this page:- <a href="https://wiki.openstreetmap.org/wiki/Key:roof:levels">https://wiki.openstreetmap.org/wiki/Key:roof:levels</a> If an attic/roof serves as an accommodation level it's still tagged as a roof level.</p>
<p>Regards Bernard.</p>
