<div class="answer-body">
<p>Routing data can take a long while to process. As a result, most routing engines only update their data occasionally. For example, OpenRouteService hasn't updated its data since 5th July (you can see the date at the top right of <a href="http://openrouteservice.org/">openrouteservice.org</a>).</p>
<p>However, note that OpenRunner only does Google Maps-based routing anyway, not OpenStreetMap. As their website says:</p>
<blockquote>
<p>This is an advanced feature of Openrunner based on one routing engines, GoogleMaps</p>
</blockquote>
<p>(They used to offer routing using OpenStreetMap data, but the company which provided that service, CloudMade, no longer does so.)</p>
<p><a href="http://cycle.travel/map">cycle.travel</a> provides routing for Western Europe and will be updated with fresh data this week, so from Tuesday onwards, you should be able to route via your new cycleways on that site. Disclaimer: I built it. :)</p>
