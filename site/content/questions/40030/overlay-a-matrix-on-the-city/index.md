<p>Take a look at <a href="https://wiki.openstreetmap.org/wiki/Humanitarian_OSM_Team">HOT</a> (Humanitarian OSM Team) and their <a href="http://tasks.hotosm.org/">OSM Tasking Manager</a> which does a quite similar job.</p>
<p>The OSM Tasking Manager allows multiple users to work collaboratively on a given area with a grid-like map. Users can lock tasks while working on them, unlock tasks afterwards, add comments to tasks, mark tasks as done and review tasks from other users. The tasking manager is open source and <a href="https://github.com/hotosm/osm-tasking-manager2">available on GitHub</a>.</p>
