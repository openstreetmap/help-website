<p>Looks like this is a <a href="https://phabricator.wikimedia.org/T248024">bug</a> in the software. Until it is fixed, you should be able to create new pages and add external links using the desktop version and the source code editor instead of the VisualEditor. Another user <a href="https://wiki.openstreetmap.org/wiki/Talk:Wiki_Translation#Confirmation_code_to_create_new_translated_pages.3F">reported the problem</a>, too.</p>
