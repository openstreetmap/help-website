I want to stay with what I know; I want to stay with iD. Additionally my PC is very slow and Java even more. It takes minutes to load JOSM. <p>Regarding the Java security: A Java plugin in the browser is usually not wanted and in fact is a security hole. However, JOSM does not need (and even cannot use) those browser plugins, please just <span>don't install plugins</span>. JOSM runs as Java program on your PC, like any other usual PC program would do (one could even argue that Java programs are more secure than the more usual C programs).</p>
<p>And I really cannot share your experience on slower PCs. It is the opposite for me (at least the actual mapping, startup may take a bit longer, yes).</p>
