<p>No. Please see</p>
<ul>
<li><a href="/questions/1973/">cant-use-google-maps-background</a></li>
<li><a href="/questions/706/">can-i-drawtrace-osm-data-from-a-google-map-or-yahoo</a></li>
</ul>
<p>Without knowing really better, please only use the imagery which is displayed by default in your OSM editor (JOSM, Potlatch2, iD, …).</p>
<p>If the resolution is not good enough for you, please <span>collect GPS tracks</span>; repeated tracking increases accuracy by enabling you to use a average track. Collecting and <span>uploading</span> them is always a good idea anyway (for checking the aerial imagery). And: if possible you should have visited the place which you want to map anyway.</p>
