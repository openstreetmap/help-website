<p>Find a tag or make one!</p>
<p>It's true that the <strong><code>telecom:medium</code></strong> tag describes the distribution side, ie, the interface for the telephone customers. I don't offhand know of any tag for specifying the network link type of a telecom exchange, but offhand I'd suggest <strong><code>telecom:network_link=microwave</code></strong>. No need to map it as a separate object.</p>
