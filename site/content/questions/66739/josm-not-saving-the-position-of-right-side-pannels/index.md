<p>Hi, please,</p>
<p>JOSM is not saving the heights (and widths) of right side pannels. It never worked for me, for any JOSM versions, even if fully re-installed. I suspect it happens with Windows, becasue I know fellows that use Linux and usually have their pannels saved (but I can't dismiss Windows-v7).</p>
<p>For GUI it saves OK last values:</p>
<pre><code>gui.geometry    x=-8,y=-8,width=1296,height=1010
gui.maximized   true</code></pre>
<p>But not for the right side pannel:</p>
<pre><code>validator.geometry         x=481,y=403,width=661,height=235</code></pre>
<p>Moreover, not availlable (just empty) for:</p>
<pre><code>propertiesdialog.geometry
selectionlist.geometry 
relationlist.geometry</code></pre>
<p>For any value I set for those, yes it saves, but does not work when opening new session:<br />
they always open with the same default heights, that is, with too much height for "validator" (used only in the end of session) and too short for "properties" (used constantly).<br />
Even though the values still are there as they were personally set by me.</p>
<p>So I have to manually drag down the pannels again every new session... since 5 years...</p>
<p>So, <strong>How to save personal adjusts of heights (and widths) for right side pannels?</strong>(in Windows)<br />
Thank you in advance.<br />
Sérgio - openstreetmap.org/user/smaprs</p>
