<div class="answer-body">
<p><strong>Yes, it has</strong>. There are numerous instances of this across the internet, often hosted by national mapping agencies (e.g., <a href="https://map.geo.admin.ch/?topic=swisstopo&amp;zoom=7&amp;lang=en&amp;bgLayer=ch.swisstopo.pixelkarte-farbe&amp;catalogNodes=1392&amp;layers=ch.swisstopo.lubis-luftbilder_farbe,ch.swisstopo.zeitreihen&amp;layers_timestamp=99991231,18641231&amp;E=2615838.88&amp;N=1094453.49&amp;layers_visibility=false,true">SwissTopo</a>) or libraries (e.g., <a href="https://maps.nls.uk/">National Library of Scotland</a>).</p>
<p>OSM itself hosts a number of historical maps for <a href="https://ooc.openstreetmap.org/">Great Britain</a> largely sourced from contributors map collections. Quite recently maps from Southern Africa dating to the 1970s were also collected.</p>
<p>Wikimedia have a historical map project which incorporates a <a href="https://warper.wmflabs.org/maps?show_warped=1">mapwarper</a> developed by a long-time OSM contributor. There is a standalone version of <a href="http://mapwarper.net/">MapWarper</a> which has been widely used by OSM projects such as <a href="https://www.openstreetmap.ie/projects/townlands/">Irish Townlands</a>.</p>
<p>There have been talks and blog posts on using historical maps with OSM since the first conference in 2007: far too many to provide even sample links.</p>
