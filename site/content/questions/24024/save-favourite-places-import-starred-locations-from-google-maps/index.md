<p>umap (<a href="http://umap.openstreetmap.fr">umap.openstreetmap.fr</a>) might be what you're looking for. It is a web application that stores your (personal) POIs on the server, and you can then embed and share your map. While it is theoretically possible that functionality like that might some day be offered directly on openstreetmap.org, I don't think it is very likely because openstreetmap.org does not aim to a feature-complete mapping portal.</p>

<p><strong>For those looking for an app</strong> you can use <a href="https://maps.me/">maps.me</a>.</p>
<p>It's available on Android and IOS, uses OSM data, makes it easy to add POIs (which you can then "star"), and naturally it's open-source and <a href="https://github.com/mapsme/omim">available on Github</a>.</p>
