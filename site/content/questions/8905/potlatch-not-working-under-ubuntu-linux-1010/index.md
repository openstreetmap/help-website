<div class="answer-body">
<p>Ubuntu comes with the open source flash player <em>gnash</em>. It does not have support for Action Script 3 that Potlatch 2 is written in. (<em>lightspark</em> have support for Action Script 3 but fails to run Potlatch 2.) You can confirm this by starting a flash program (like youtube) and right click. If the menu does not say <em>Adobe Flash</em> you are running the gnash plugin.</p>
<p>On my setup I got it running by uninstalling <em>gnash</em> and installing <em>flashplugin-installer</em>.</p>
</div>
