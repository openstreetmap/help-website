<p>It seems you successfully edit your local database, but view tiles (in the browser) from osm.org (instead of your own tiles). Please check which tiles are fetched by your browser (e.g. in Firefox via ctrl+shift+E or ctrl+shift+K and the opening your webpage <a href="http://localhost:3000">http://localhost:3000</a> ). Possibly you need to change the tile source for this webpage to point to your own tile server.</p>
