<div class="comment-text">
<p>To be clear, the planet file is only one set of (current) data, it is not one set of data and one set changes. The size difference you observe is almost certainly due to unit confusion (1.00 GB = 1.07 GiB, therefore 59.0 GB = 63.4 GiB) and has nothing to do with the file contents or any kind of "bloat". To obtain distance metrics you will need to load the data into a routing engine (e.g. GraphHopper, OSRM) and then use that engine's API.</p>
</div>
