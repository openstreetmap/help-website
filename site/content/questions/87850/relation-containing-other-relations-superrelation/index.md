<div class="answer-body">
<p>Super-relation is not really relevant because this kind of relations goes against an old rule : <a href="https://wiki.openstreetmap.org/wiki/Relations_are_not_categories">Relations are not categories</a>.</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Super-relation">Super-relation</a> and <a href="https://wiki.openstreetmap.org/wiki/Relation:superroute">Relation:superroute</a> are not much of a thing anyway, if you read the talk pages.</p>
<p>As for how to change that, you'd want to change the tag <code>type=route</code> to <code>type=superroute</code>, as you change other tags.</p>
<p>Regards.</p>
