<div class="answer-body">
<p>You should not be logging in with username/login in any case (and with any editor), the facility is solely provided to support private API installations.</p>
<p>You should remove your login credentials from the advanced preferences, and check if OAuth is checked for the selected OSM API URL.</p>
<p>If this is all correct, then when you try to upload you will be asked to login to the OSM website and authorize Vespuccis access to the OSM API, which you should do (you will only need to do this once). You can force this step without trying to uploading by selecting the corresponding menu item in the tools menu.</p>
<p>If you can't find the menus I'll add a couple of screenshots.</p>
