<div class="answer-body">
<p>It will be possible for such users to agree to the Contributior Terms after the license change is through. This will (likely) <em>not</em> automatically re-animate their data though; data marked as incompatible during the license change will remain inaccessible even if the user changes their mind.</p>
<p>It is unlikely that the API will have a means of accessing CC-BY-SA data after the license change (but you are right, the data will, at least initially, still be in the database).</p>
<p>However, anyone is free to download a full history planet file from planet.openstreetmap.org and run a read-only CC-BY-SA API on that. Or you could access CC-BY-SA data from fosm.org.</p>
