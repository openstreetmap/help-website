<div class="answer-body">
<p>I don't think you need to go down the tag proposal route for something as niche as this</p>
<p>I would recommend creating a wiki page and add a list of additional tags that make sense. For example, would tagging be appropriate for adding info in the same manner that amenity=place_of_worship almost always includes religion= <em>and denomination=</em> tags too by default.</p>
<p>The man_made tag is perfectly ok to use, but depending on the significance of the totem poles you should consider alternatives for the main tag, for example the historic=monument tag or others along those lines.</p>
<p>See <a href="http://taginfo.openstreetmap.org/search?q=totem_pole#values">here</a> for other tags used in relation to totem poles</p>
