<div class="answer-body">
<p>There are two tourism tags appropriate for self-catering holiday lets. These are <a href="https://wiki.openstreetmap.org/wiki/Tag:tourism%3Dchalet"><code>tourism=chalet</code></a> (for a building let entirely), and <a href="https://wiki.openstreetmap.org/wiki/Tag:tourism%3Dapartment"><code>tourism=apartment</code></a> for several apartments in a single building, with 17k and 1k uses respectively. Neither has a particularly obvious, or even highly appropriate name (I'd find it hard to label a gite or a farm holiday cottage as tourism=chalet, which dont fit the current description on the wiki), but these are what is in use.</p>
</div>
