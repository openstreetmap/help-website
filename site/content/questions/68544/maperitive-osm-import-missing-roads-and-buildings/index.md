<td><div class="item-right">
<div class="answer-body">
<p>Those features aren't showing up because the default rules aren't set up to show them.</p>
<p>In the case of the roads, the link ways aren't being shown because the rules don't include <code>highway=secondary_link</code> or <code>tertiary_link</code> ways. You can change this by adding rules similar to the existing ones for <code>highway=primary_link</code> or <code>motorway_link</code>.</p>
<p>In the case of the buildings, it looks like most of the buildings in that area have been given detailed building tags, like <code>building=apartments</code> or <code>building=school</code>. The default rules only look for <code>building=yes</code>. You could fix this by changing the building rule to the following:</p>
<p><code>building : building</code></p>
<p>...which would match any building tag.</p>
