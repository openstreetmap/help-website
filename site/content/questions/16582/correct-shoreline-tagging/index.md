<div class="answer-body">
<p>You can check for errors in the coastline with the <a href="http://tools.geofabrik.de/osmi/?view=coastline&amp;lon=-5.99339&amp;lat=57.27087&amp;zoom=14">OSMI coastline view</a>. It looks that there were no errors last night. This tool updates far more often than the coastline on tiles on the main website.</p>
<p>Since the updating of the coastline shapes on the main website is a partly manual process, there is no way of knowing when it will be done next.</p>
