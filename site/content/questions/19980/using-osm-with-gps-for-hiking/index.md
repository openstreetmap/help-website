<p>Depends on what sort of grid lines you want, but you could try <a href="http://wtp2.appspot.com/wheresthepath.htm">Where's The Path?</a>.</p>
<p>This can display OpenStreetMap maps (Standard style or Cycle Map), and has the option of displaying a grid on top. Available grids are Ordnance Survey Great Britain or WGS84 latitude and longitude.</p>
<p>You can then print a map from that.</p>
