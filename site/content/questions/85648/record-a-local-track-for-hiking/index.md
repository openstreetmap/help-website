<p>Hi.</p>
<p>First thing is to check that you are allowed to use the data on this website. I tried but automatic translation is not so great. If you can get explicit authorization, you can use a GPX file from them.</p>
<p>If not, and the trail is signed on the ground, you'll have to record a GPX track of the trail, and then use that as source.</p>
<p>Then you'll have to read some parts of the wiki, I think starting here is a good place : <a href="https://wiki.openstreetmap.org/wiki/Hiking">Hiking</a>. <a href="https://wiki.openstreetmap.org/wiki/Import">Import</a> might be useful as well.</p>
<p>Best regards.</p>
