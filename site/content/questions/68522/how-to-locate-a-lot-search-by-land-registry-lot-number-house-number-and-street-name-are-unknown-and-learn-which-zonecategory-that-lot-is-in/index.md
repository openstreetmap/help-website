<div class="answer-body">
<p>We don't usually map lots/plots/planning zones in OpenStreetMap. There are several reasons for this. One is that they are rarely visible on the ground, and we prefer mapping things that you can actually see and verify. So if two people fight over where exactly something is, we can simply go there and look. This is not the case with lot numbers. The second issue is that government publications will often be copyrighted, and not permitted as a source in OpenStreetMap.</p>
