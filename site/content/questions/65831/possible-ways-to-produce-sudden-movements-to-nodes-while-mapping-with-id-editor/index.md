<div class="answer-body">
<p>My guess is that the user is simply trying to pan the map, but unlike JOSM, iD uses the left mouse button for both map panning and dragging nodes. On a dense map and especially for new users, they try to pan the map not realising they are moving nodes.</p>
<p>See <a href="https://github.com/openstreetmap/iD/issues/4776">https://github.com/openstreetmap/iD/issues/4776</a></p>
