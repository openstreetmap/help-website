<div class="answer-body">
<p>There is currently no special tag to differentiate between shops selling recreational cannabis and those that sell only medically prescribed cannabis.</p>
<p>As for the Wiki, I wouldn't let its pronouncements restrict my use of the shop=cannabis key. If we can come up with a suitable new tag to make it possible to include medical only shops, that would be my favored way forward. This is a good question for the tagging list to kick around.</p>
<p>Perhaps</p>
<p>cannabis:medical=yes/only</p>
<p>or something similar?</p>
<p>Cheers, Dave</p>
