<p>A query like <code>way[~".*:lane$"~".*"];</code> should do it. It may require some adaptation to get exactly the result you want, there's some examples in the <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_QL#Key.2Fvalue_matches_regular_expression_.28.7E.22key_regex.22.7E.22value_regex.22.29">documentation</a> and you could consult any guide to POSIX extended regular expressions to help build the correct pattern.</p>
