<p>There is a list of tags for various types of vegetation <a href="https://wiki.openstreetmap.org/wiki/Vegetation">here</a>, although it doesn't include all green e.g. many sports pitches. For most it should not be necessary to check for nodes.</p>
<p>There are some additional overpass <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_API_by_Example">examples</a> on the wiki. <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_API_by_Example#Count_number_of_addr:housenumber_in_an_area">This one</a> shows how to use a city boundary relation instead of a boundary box for a search area.</p>
