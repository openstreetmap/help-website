<p>This looks like a raised boardwalk: commonly used through marshy areas and elsewhere. If the path is basically on the ground you just need to add surface=wood.</p>
<p>If it is raised I think the standard practice is still to tag it as bridge=yes or <span>bridge=boardwalk</span>, although boardwalk=yes has a paltry <span>8 values</span>. This was the first one I ever tried mapping:</p>
<p><img src="//upload.wikimedia.org/wikipedia/commons/thumb/d/dc/Bass-Hole-boardwalk_Yarmouth-MA-US.JPG/320px-Bass-Hole-boardwalk_Yarmouth-MA-US.JPG" alt="Bass Hole Boardwalk, Yarmoth, MA" /><br />
(<span>bigger and more info – CC BY-SA 3.0 by Costoa</span>)</p>
<p>All in all these are fairly common features in Nature Reserves, wetlands and coastal areas (dunes or salt marsh), so its a little surprising more are not marked explicitly.</p>

<p>You can use any tags you like. Many editors suggest common tags, but if one of those doesn't fit, you could use something else. If you want to see what other mappers have used, you can use taginfo to search:</p>
<p><a href="http://taginfo.osm.org/keys/surface#values">http://taginfo.osm.org/keys/surface#values</a></p>
<p>I believe that I've mapped things like your picture in the past as "highway=footway; surface=boardwalk" (taginfo sees 267 examples of that worldwide).</p>
<p>I beleive that you're using the Potlatch 2 editor, so you'll need to switch to "advanced" mode and manually enter a "surface" key with a value of "boardwalk".</p>
