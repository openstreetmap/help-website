<p>I have not noticed issues today (<a href="http://dns.openstreetmap.org/tile.openstreetmap.org.html">from</a> Germany), but looking at <a href="http://munin.openstreetmap.org/openstreetmap/tile.openstreetmap/index.html">the serving graphs</a> I spot an irregularity today (not happend the last week like it was today) in the outbound network traffic (apparently there was a switch-over to another server). <a href="/questions/38076/why-are-my-buildings-not-showing-up-on-renderer">Apparently</a> rendering updates are delayed too today.</p>
<p>So … please just wait until tomorrow or the day after tomorrow. I am quite sure that the server admins already are aware of these problems (by automatic server issue reporting and those performance graphs).</p>
