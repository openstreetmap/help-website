<div class="answer-body">
<p>You could use <a href="http://www.trekbuddy.net/forum/">TrekBuddy</a> on the phone, plus <a href="http://mobac.dnsalias.org/">Mobile Atlas Creator</a> to generate the maps for it.</p>
<p>See the README.HTM file included with Mobile Atlas Creator for details on how to add custom map sources, so you could set it to use your own rendered tiles.</p>
