<p>Hi Kotya!</p>
<p>The point is saved at the location where you hit "create a text note" - it does not matter if you don't type in that moment or later.</p>
<p>Normally, first I create the note and start to write a few seconds until 5 minutes later. Sometimes I turn off my screen and then type later. I tried while riding a bus or taxi, it's the same: the point is still where you first pressed the "text note" button.</p>
