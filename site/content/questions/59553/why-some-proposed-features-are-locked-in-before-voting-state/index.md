<p>Simple: not enough interest (by the proposer, the community, or both). Now and then the proposals have been superseded by others, it simply hasn't been documented.</p>
<p>Note: if no in use tagging scheme exists for whatever you are trying to map, using something that is "just proposed" is naturally perfectly OK (infamous example: amenity=childcare).</p>
