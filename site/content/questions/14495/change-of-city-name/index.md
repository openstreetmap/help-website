<p>You can use <a href="http://nominatim.osm.org/">http://nominatim.osm.org/</a> to get a detailed view of why Nominatim is returning the results that it is. Once you have done the search hit the details link and it'll give more info on the result. Take careful note of whether it is using nodes or polygons. Sometimes if a place has both a node and a polygon it can cause issues such as this. Greyed out lines in the search result page show that Nominatim is ignoring that detail.</p>

So if you know the area that city or town covers it needs a polygon mapping. Do a search of these questions for "town polygons" you will find out more.</p>
