<div class="answer-body">
<p>The OSM database does not map contours. The <a href="http://www.opencyclemap.org/docs/">contours</a> on the cycle map are generated from the <a href="http://www2.jpl.nasa.gov/srtm/">SRTM</a> data set from NASA, which you can download outside OSM.</p>
