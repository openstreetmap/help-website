<div class="answer-body">
<p>I'm a basic QGIS user and have had acceptable results using the OSM plugin.</p>
<p>If I remember correctly, there are 2 options (raster / tiles or vector data). The raster / tiles option provides the quick and simple solution.</p>
<p>However, if you want to manipulate the OSM data or render features not normally rendered, then you will need to use the Vector option and play around with it (I gave up on this and opted for the simple option, but your requirements may differ).</p>

<p>If you want to render the data in qgis, you have to import the data as new layer. You can achieve that with a DB connection (requires postgis database) or by using the OpenStreetMap plugin (in Extension menu) for importing data.</p>
<p>You can also take a look at <a href="https://bitbucket.org/springmeyer/quantumnik/">quantumnik</a> (<a href="http://www.youtube.com/watch?v=6Xn31jSg7YI">example video</a>) for a pretty and clean rendering with mapnik, but personnaly, I've try it but it never works :-(</p>
