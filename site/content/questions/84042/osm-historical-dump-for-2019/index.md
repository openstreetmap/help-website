<p>The OSM history file contains all history. If you only need it for a particular place, it will be easiest to download a regional history file from <a href="https://osm-internal.download.geofabrik.de/">https://osm-internal.download.geofabrik.de/</a> and then use the <code>osmium</code> command line tool to further limit the area.</p>
