<div class="answer-body">
<p>By the looks of it UTFGrids are just json tiles from the serving point of view. As mod_tile can support serving json tiles, it should probably be able to serve UTFGrid data. However, you need to get the UTFGrid data into the meta-tile file format for mod_tile to serve it and I am not sure if any application that generates UTFGrid can currently do this.</p>
