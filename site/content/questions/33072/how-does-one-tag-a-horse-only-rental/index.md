<div class="answer-body">
<p>There are a small number of places tagged <a href="http://taginfo.openstreetmap.org/tags/amenity=riding_school"><code>amenity=riding_school</code></a>, which is a typical (British) description of a place where one can ride a horse for a fixed time period. These places mainly cater for children but some adults use them. It may be difficult to separate such establishments from <a href="http://en.wikipedia.org/wiki/Livery_yard">livery stables</a> (<a href="http://taginfo.openstreetmap.org/tags/amenity=livery%20stable">1 example</a>) which offer rented stabling for horses. In practice many riding schools will also offer livery facilities.</p>
<p>Other types of establishment which involve horse riding, but on animals which have been hired, would not be included in the above tags. These might include: <a href="http://en.wikipedia.org/wiki/Pony_trekking">pony trekking</a> (<a href="http://taginfo.openstreetmap.org/tags/amenity=pony_trekking">1 example</a>), <a href="http://en.wikipedia.org/wiki/Dude_ranch">Dude Ranches</a>, and not doubt others.</p>

<div class="answer-body">
<p>If you <a href="http://taginfo.openstreetmap.org//search?q=rental#values">plug "rental" into taginfo</a>, then you can see that for most rental businesses the form <em>amenity=something_rental</em> is used. There are however currently only 5 <em>amenity=horse_rental</em> tagged in the world.</p>
<p>If for some reason the amenity tag is already used on the object you want to tag or if they rent out multiple kind of things, then the common form to tag rentals is <em>something_rental=yes</em>. In your case <em>horse_rental=yes</em>.</p>
<hr />
<p>Edit: In <a href="https://help.openstreetmap.org/questions/14748/how-to-correctly-tag-a-livery-stable-be-boarding-stable">this previous question</a> <em>tourism=trail_riding_station</em> is mentioned. It's <a href="http://taginfo.openstreetmap.org//search?q=trail_riding_station#values">used 287 times</a>.</p>
