<div class="answer-body">
<p>I've seen two different ways things like this have been handled:</p>
<ol>
<li>amenity=cooking_school (see also <a href="https://wiki.openstreetmap.org/wiki/Tag%3Aamenity%3Ddriving_school">driving_school</a>, <a href="https://wiki.openstreetmap.org/wiki/Tag%3Aamenity%3Dmusic_school">music_school</a>, language_school, dancing_school, etc)</li>
<li>amenity=training, training=cooking (see <a href="https://wiki.openstreetmap.org/wiki/Proposed_features/training">abandoned proposal</a>)</li>
</ol>
<p>Neither the <a href="https://overpass-turbo.eu/s/Fgu">former</a> nor the <a href="https://overpass-turbo.eu/s/Fgx">latter</a> are used very much yet, so you could go either way. Though driving_school is used more widely, so that might tip the scales a little towards that model?</p>
