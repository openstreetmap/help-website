<div class="answer-body">
<p>You can download the data and do (technically) whatever you want with it. If you want data for the GB there's a downloadable link at <a href="http://download.geofabrik.de/europe.html">http://download.geofabrik.de/europe.html</a>, if you want Northern Ireland as well you can download it from the "island of Ireland" link there and combine them.</p>
<p>What you do next is up to you - the OSM wiki has information about <a href="https://wiki.openstreetmap.org/wiki/Printed_maps">printing</a>, for viewing you'll need something to view the data with (perhaps a web server <a href="https://switch2osm.org/serving-tiles/">serving map tiles</a> and a web browser), for processing the data you might want to use something like <a href="https://wiki.openstreetmap.org/wiki/QGIS">QGIS</a>.</p>
