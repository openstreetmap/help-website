<div class="answer-body">
<p>Please see <a href="https://wiki.openstreetmap.org/wiki/Platform_Status">https://wiki.openstreetmap.org/wiki/Platform_Status</a> "Performance degraded until about 5th August (server move ongoing)".</p>
<p>Further, please note that the free service at <a href="https://www.openstreetmap.org">https://www.openstreetmap.org</a> (and its standard tiles) is driven <em>solely</em> by volunteers and donations. This service is primarily meant for contributing community members. There is no SLA. If you need that in your offices (and if you have a high usage volume!) please switch to a professional map service using OSM data.</p>
<p>Your offices are in the US, so this is likely related: OSM_tech team <a href="https://mobile.twitter.com/OSM_Tech/status/1024686258155671552">wrote</a> on 1. Aug. 2018: "The tile.openstreetmap.org cluster has been over capacity in North America for the last few days. Anyone able to offer a high-traffic server in North America?"</p>
