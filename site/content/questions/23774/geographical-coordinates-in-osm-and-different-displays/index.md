<div class="answer-body">
<p>To answer a subset of your questions:</p>
<p>Get coordinates using <a href="http://m.osmtools.de/index.php?lon=9&amp;lat=49&amp;zoom=6">http://m.osmtools.de/index.php?lon=9&amp;lat=49&amp;zoom=6</a> . There you can set a marker via clicking the link at the bottom left that says "Marker setzen", then clicking the map and then clicking "Permalink" at the bottom right. Now the URL contains lat/lon coordinates in decimal degrees.</p>
