<p>For the editor <a href="https://www.openstreetmap.org/user/Whiskey1-11/history">you</a> are currently using ("iD"), please see the older questions</p>
<ul>
<li><a href="/questions/22234/">how to display coordinates</a></li>
<li><a href="/questions/56687/">how to input coordinates</a></li>
</ul>
<p>To put it short (with the current aims and capabilities of iD): to work with coordinates it is better to switch to another editor. My personal recommendation is <a href="https://wiki.openstreetmap.org/wiki/JOSM">JOSM</a>. With JOSM you can input a new point by using coordinates and adjust it afterwards by dragging with the mouse pointer.</p>
