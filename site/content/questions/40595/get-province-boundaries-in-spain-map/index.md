<p>When and how features are displayed on an online map is controlled by the "style" (essentially the style controls the rendering process and lays down at which zoom level which elements are displayed). If you need a map with specific behaviour you will likely simply have to design the style yourself.</p>
<p>Search for example for <span>"TileMill"</span> and <span>"Maperitive"</span> on this site.</p>
