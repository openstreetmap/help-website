<p>The <a href="https://www.openstreetmap.org/user/OSMF%20Redaction%20Account">redaction bot</a> (more or less, isn't it? … IIRC) did what you describe, so this would not have been necessary to do manually.</p>
<p>To "roll back" you could use the tools for <span>change rollback</span> and just rollback the changes for this single object of course (for josm reverter: one by one, starting with the newest). Yes, that is not automatic and involves quite some manual work.</p>
