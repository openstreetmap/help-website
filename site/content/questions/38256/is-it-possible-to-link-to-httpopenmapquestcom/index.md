<td><div class="item-right">
<div class="answer-body">
<p>You can choose the Link option 3rd from left at Top Left of screen to get a permalink to a particular area on the <a href="http://open.mapquest.com">Open MapQuest site</a>. This link option outputs only a shortened link. If you want a full link (to construct it yourself), apparently the URL GET parameters <code>center</code> and <code>zoom</code> work (<a href="http://open.mapquest.com/?zoom=9&amp;center=6.5,2.1">example</a>).</p>
<p>Or you can show the Open Mapquest map as a "layer" on the <span>OSM main site</span> and use the share button to generate a link (or copy the link in the browser toolbar).</p>
