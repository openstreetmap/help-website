<div class="answer-body">
<p>It isn't clear which objects you're referring to, but I looked at the data in that area and you are missing some tags that could be causing your confusion. Both <a href="https://www.openstreetmap.org/way/348691820">the sports_centre building</a> and <a href="https://www.openstreetmap.org/relation/5195826">the cafe building</a> are missing building tags, so only the amenity icons are rendering. If you add appropriate building tags to these objects, they will then be visible.</p>
<p>There are also some things that could be changed with regard to relations:</p>
<ol>
<li>You've made 1-member multipolygon relations for the three swimming pools. These are simple closed-way objects so the relations are unnecessary and can be deleted.</li>
<li>Ways <a href="https://www.openstreetmap.org/way/348691865">348691865</a> and <a href="https://www.openstreetmap.org/way/348691918">348691918</a> have some tags that duplicate those on the building or relation, respectively. These tags should be removed from the multipolygon outer ways.</li>
</ol>
