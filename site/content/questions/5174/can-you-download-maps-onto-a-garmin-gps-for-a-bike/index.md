<p>The answer is yes. Go to <a href="http://talkytoaster.info/ukmaps.htm">http://talkytoaster.info/ukmaps.htm</a> and read carefully. The appropriate file can be downloaded, renamed and loaded onto your micro sd card in a folder called Garmin. Insert in GPSr and away you go. As to whether its any good for cycling - probably only as good as any other map.</p>

<div class="answer-body">
<p>One good option for Garmin maps for the UK is <a href="http://ravenfamily.org/andyg/maps/">Andy Gates' Munky map</a>.</p>
<p>Just download the file <code>OSM-Garmin-Munky-UKIE.zip</code>, then unzip it, and copy the <code>gmapsupp.img</code> file to a folder named "Garmin" on the MicroSD card.</p>
<p>These maps are for all of the UK and Ireland, and include contour lines and support routing. These maps have a customised style, which uses standard UK colours for roads etc, plus they highlight some things which are useful for cyclists, eg cycleways and bike shops.</p>
