<p>I'm not sure about a source for carrier routes, but there's really no such thing as ZIP code boundaries. ZIP codes are assigned to streets or individual addresses, and they often can't be easily grouped into nice polygons. See this site for more info: <a href="https://georeference.org/doc/zip_codes_are_not_areas.htm">https://georeference.org/doc/zip_codes_are_not_areas.htm</a></p>
