<div class="answer-body">
<p>Hi, I'd suggest adding the <strong><code>building:use</code></strong> tag, which describes what a building element is used for. It's especially useful when the <em>form</em> of the building doesn't match with the current use of the building, eg <strong><code>building=church</code></strong> + <strong><code>building:use=residential</code></strong>.</p>
<p>In this case maybe <strong><code>building=house</code></strong> (or <strong><code>=detatched</code></strong>, or <strong><code>=static_caravan</code></strong>, etc) + <strong><code>building:use=model_home</code></strong> (or <strong><code>=show_house</code></strong> if you prefer, but I think "model home" is the less ambiguous term.)</p>
<p>You could also add a freeform description using the <strong><code>description=</code></strong> tag.</p>
