<div class="comment-text">
<p>If you're asking about what some text on a particular map rendering looks like you'll need to post a screenshot for people to be sure, but assuming that you're talking about the "standard" layer on osm.org there are two candidates:</p>
<p><a href="https://www.openstreetmap.org/node/60655918">https://www.openstreetmap.org/node/60655918</a></p>
<p>(the city, in black)</p>
<p><a href="https://www.openstreetmap.org/relation/2921154">https://www.openstreetmap.org/relation/2921154</a></p>
<p>(the admin level 4 area, in purple)</p>
<p>The code to decide what gets shown at what zoom level is a bit convoluted. For admin levels it's <a href="https://github.com/gravitystorm/openstreetmap-carto/blob/dd096af4f566eb9c31e50ac447215f68e45b563f/admin.mss#L207">https://github.com/gravitystorm/openstreetmap-carto/blob/dd096af4f566eb9c31e50ac447215f68e45b563f/admin.mss#L207</a> - note that not everything gets shown at every zoom level.</p>
<p>An effective way of finding where a name is used is to search using taginfo, in this case <a href="https://taginfo.openstreetmap.org/tags/name=%E8%87%BA%E4%B8%AD%E5%B8%82">https://taginfo.openstreetmap.org/tags/name=%E8%87%BA%E4%B8%AD%E5%B8%82</a> finds exactly 2 matches (the ones shown above).</p>
