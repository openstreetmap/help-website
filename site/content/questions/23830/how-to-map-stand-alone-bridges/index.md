<div class="answer-body">
<p>That's fine. If there are no connecting paths to the bridge (so people walk just over a grass area on some route), then, well, that's how the real world is there.</p>
<p>And even <em>if there would be</em> paths around it is fine to start mapping at the bridge. <span><code>fixme=continue</code></span> could to be tagged to the end nodes to indicate that there is a path which should be mapped.</p>
