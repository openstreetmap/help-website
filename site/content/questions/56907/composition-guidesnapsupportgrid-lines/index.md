<p>Many classical buildings and gardens have been designed using a strict (and sometimes complicated) grid.</p>
<p>While the lines themselves are of course not visible, they are very clearly there. For example the footpaths in a park, or the many parts of a building, are evenly spaced and nicely aligned.</p>
<p>Is it ok to draw these "invisible" guidelines in the map? Of course they should not be rendered in the normal map view, but they help in aligning features properly, also with later edits.</p>

<p>You should only put things into OSM which are physically there (with a small number of exceptions). If there are actual footpaths, then map them. But if it's not a real thing that's not really there, then it probably doesn't belong in OSM.</p>
<p>Do you have an example or photo of the thing you're talking about so we can understand more clearly what you mean?</p>
