<p>Ways in OSM are split at arbitrary positions (usually if some property changes, like maxspeed, lane count, name, surface etc.). So a certain road with a specific name can consist of multiple ways in OSM, each having a unique ID. However, ways aren't split at every intersection. So if you need shorter stretches, split them in your client software. Take a look at the OSM wiki to learn more about <a href="https://wiki.openstreetmap.org/wiki/Elements">OSM elements</a>.</p>
</div>
