<div class="answer-body">
<p>If any grocery outlet specialises in selling products from a particular region, the usual way I use for tagging is to tag it as a shop=grocery or shop=convenience and add a cuisine tag. So for example there are many convenience stores in the UK run by Poles which will sell a whole range of items which are distinctively polish. I tag these <code>shop=convenenience</code>, <code>cuisine=polish</code>.</p>
<p>The cuisine tag was created for restaurants and fast_food, but serves equally well here.</p>
