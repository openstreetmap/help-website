<div class="answer-body">
<p>The data is there whether it is displayed on the map or not. One way to view specific data is to use the Overpass Turbo tool. <a href="https://overpass-turbo.eu/s/QBf">This query</a> will return features tagged "landuse=forest" for the bounding box of the screen, but you can modify that if you want different tags. Note that if you zoom out too far, it will return too much data. You can use the wizard to modify the query, for example to limit the query to particular geography, like a country, canton, district, etc.</p>
