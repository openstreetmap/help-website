<div class="answer-body">
<p>The easiest way to achieve what you want is currently likely to create a RSS feed from <a href="https://simon04.dev.openstreetmap.org/whodidit/">https://simon04.dev.openstreetmap.org/whodidit/</a> for the areas you are interested in.</p>
<p>An alternative might be <a href="https://osmcha.mapbox.com/">https://osmcha.mapbox.com/</a> it doesn't generate an RSS feed, but you can define an area of interest.</p>
<p>You can naturally always simply look at an objects history on openstreetmap.org if you can remember the objects you are interested in.</p>
