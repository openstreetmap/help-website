<p>Based on mapnik, tirex and german mapnik style I have rendered tiles from a planet file. All is fine for the zoom level 3 and up. But for the zoom levels 0 to 2 there are only blank tiles, that is, the PNGs are all empty/transparent and of size 103 Bytes.</p>
<p>Is this normal? Looking at the map at <a href="http://www.openstreetmap.de/karte.html?zoom=2&amp;lat=41.93455&amp;lon=-8.64009&amp;layers=B000TT">http://www.openstreetmap.de/karte.html?zoom=2&amp;lat=41.93455&amp;lon=-8.64009&amp;layers=B000TT</a> it should be possible to have a zoom level 2. Actually, I only need this one, the zoom 0 and 1 is not really needed.</p>
<p><em>Update:</em> This seems to be a tirex problem. Rendering zoom level 2 with generate_tiles.py works fine.</p>
