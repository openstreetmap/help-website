<div class="answer-body">
<p>Overpass supports all tags that are stored in the database. Many of them (a.o. the most commonly used) can be found on the <a href="https://wiki.openstreetmap.org/wiki/Map_Features">OSM Wiki</a>. Via <a href="https://taginfo.openstreetmap.org/">TagInfo</a> you can see which tags are used, even those that are not documented.</p>
<p>The tag for attractions is <a href="https://wiki.openstreetmap.org/wiki/Tag:tourism=attraction?uselang=en">tourism=attraction</a>.</p>
