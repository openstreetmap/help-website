<p>If you have windows, you can try my batch script which is based on mkgmap: <a href="http://www.openfietsmap.nl/downloads/osm_combi">http://www.openfietsmap.nl/downloads/osm_combi</a></p>
<p>It can combine tiles from <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a> and contours from sites like <a href="http://osm.thkukuk.de/">http://osm.thkukuk.de/</a></p>
