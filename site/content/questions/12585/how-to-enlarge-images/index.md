<div class="answer-body">
<p>Agree that the map text is too small.</p>
<p>Desktop computer, Win7, Firefox 12, 24" 16:10 LCD monitor at the monitor's default 1920x1200 resolution.</p>
<p>The map text is around 5 point at the map's default zoom (5 steps below maximum). At the map's maximum zoom level, the text is slightly larger, but it's still too small to make out some characters, and only readable at all by moving closer to the screen and squinting. Perhaps someone younger would find it easier to read, but not everyone is in their twenties.</p>
<p>The Firefox "Zoom all" works, but since it does zoom everything, it also considerably reduces the amount of the map that can be displayed on a single screen. "Text zoom" zooms the left column (containing the "Help Center", "Documentation", etc. menu) and the top menu ("View", "Edit", etc.), but does not affect the size of the text within the map itself.</p>
<p>Changing the font size in the Firefox options has no effect on the size of the map text.</p>
