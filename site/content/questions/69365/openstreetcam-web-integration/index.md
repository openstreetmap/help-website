<div class="answer-body">
<p>The motivation for OpenStreetCam is to assist improving OpenStreetMap. There is <a href="http://api.openstreetcam.org/api/doc.html">an API</a> so that developers can incorporate the images into map editors. You could potentially use that API for a webpage, although I'm not sure if this is legally allowed by Telenav.</p>
<p>You may be interested in an alternative project, Mapillary, which is similar to OpenStreetCam. There is <a href="https://mapillary.github.io/mapillary-js/">Mapillary JS</a> which sounds like what you want.</p>
