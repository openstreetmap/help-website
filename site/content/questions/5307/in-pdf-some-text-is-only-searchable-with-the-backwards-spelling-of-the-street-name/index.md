<p>Thanks for the clear answer! I also saw something in the SVG and did some experimenting. When I look at my SVG in Notepad for example, the TEXT objects that make up a street name will look something like this for the word "WOOD" (pseudo code):</p>
<p>id=2986 text = "d" id=2990 text = "o" id=2994 text = "o" id=2998 text = "w"</p>
<p>And no matter what order these lines are in, they will spell "Wood"....however, if you reverse the order of the lines, it will THEN become searchable by "Wood", not by "dooW".</p>
<p>I don't know if this gives you more clues. Forgive my redundancy if you already know this.</p>
