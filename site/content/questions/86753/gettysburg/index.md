<p>I don't know the import capabilities of OGIS but you can get a whole state extract for Pennsylvania <a href="http://download.geofabrik.de/north-america/us/">from Geofabrik</a> in either a shp file or a OSM pbf file. You can use a tool like <a href="https://wiki.openstreetmap.org/wiki/Osmium">osmium</a> to make an extract that covers only the area of interest, maybe the whole battleground, and then import that to your GIS program.</p>
