<div class="answer-body">
<p>OpenStreetMap works slightly different. We are not just drawing images, but instead we are trying to create a database consisting of so-called <a href="https://wiki.openstreetmap.org/wiki/Elements">elements</a> (mainly <a href="https://wiki.openstreetmap.org/wiki/Nodes">nodes</a> and <a href="https://wiki.openstreetmap.org/wiki/Way">ways</a>). This has the advantage that you can use the map for a lot of different things, not just for displaying it. You can query for POIs and addresses, calculate routes between locations and so on.</p>
<p>Consequently you can't use a regular drawing program like LibreOfficeDraw. But don't worry, contributing to OSM is still easy. Just register an account (you already seem to have one), browse to the area where want to change something and then press the large <em>Edit</em> button at the top. This will open OSM's web-based editor <a href="https://wiki.openstreetmap.org/wiki/ID">iD</a>. It is easy to use and also features a short introduction.</p>
<p>But keep in mind that you are not allowed to contribute data from copyrighted resources! Please check whether the map provided by your local authority is compatible with OSM's license. Most of these maps are copyrighted and must not be used to copy information from. See the <a href="https://wiki.openstreetmap.org/wiki/Legal_FAQ">Legal FAQ</a> for more information.</p>
<p>If you still have questions, come back again.</p>
