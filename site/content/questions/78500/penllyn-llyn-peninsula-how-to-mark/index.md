<p>This is another 'naming' example which I can't see and don't see how to mark, especially is it should (normally, on most maps) be viewed as a prominent item.</p>
<p>"Penllyn" or the Llyn Peninsula is the bit that sticks out on the top left of Wales. It is what the local people living there call where they live. Songs are sung about it and poems written. It features on any commercial map you can find.</p>
<p>It covers the <a href="https://www.openstreetmap.org/#map=10/53.0049/-4.4653">area west of the A487</a>, roughly. I cannot see it marked via search, nor can I see it displayed. I could add a marker, somewhere in the middle - but how would one make this sufficiently prominent - say about as prominent as the name for Ynys Môn / Anglesea?</p>

<div class="answer-body">
<p>It's already mapped as <a href="https://www.openstreetmap.org/relation/9237365">https://www.openstreetmap.org/relation/9237365</a> . In general, if you wonder why a particular map does not show that as you would expect, you'll need to ask the map style developers about that. For OSM's "standard" map, that is <a href="https://github.com/gravitystorm/openstreetmap-carto">https://github.com/gravitystorm/openstreetmap-carto</a> .</p>
<p>In this case, it's broken because there was a gap in the outside edge, which you could see in JOSM's validator. I've fixed that in <a href="https://www.openstreetmap.org/changeset/98147860">https://www.openstreetmap.org/changeset/98147860</a> . Various maps will start updating in about 5 minutes.</p>
<p>This isn't a National Park or AONB (well, there is a separate Llyn AONB), but what I wrote at <a href="https://www.openstreetmap.org/user/SomeoneElse/diary/395232">https://www.openstreetmap.org/user/SomeoneElse/diary/395232</a> mostly applies here too.</p>
