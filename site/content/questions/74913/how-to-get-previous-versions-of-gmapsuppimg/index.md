<p>There is no "one current version" of a gmapsupp.img. That's just the generic name for a map for a Garmin handheld. Lots of places allow you to download maps of this type - see <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">here</a> for a list. You can also create your own - see <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin">here</a>.</p>
<p>Occasionally we see complaints here about <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a> (one of the sites on that list). That site isn't in any way associated with the main OpenStreetMap project, and given that it seems to be up and down as frequently as the undergarments of a person of negotiable virtue, it is probably best avoided.</p>

<p>It's too late for your problem but as the micro SD cards that are available now tend have quite a big capacity why not just add a another map file to it. I realize that some older devices only recognize the name gmapsupp.img. Alternative maps can just be renamed with a date prefix, for example, then edited to select it. This way if the new map doesn't meet your requirements it is fairly easy to go back the previous version. I have been using several micro SDs for maps which only get deleted when I've seen the newer and hopefully better map working.</p>
