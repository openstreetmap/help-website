<p>If you can live with statically generated imagers your best bet is likely to use Maperitive <a href="http://maperitive.net/">http://maperitive.net/</a> or TileMill (if you have a rendering DB setup somewhere) <a href="http://www.mapbox.com/tilemill/.">http://www.mapbox.com/tilemill/.</a></p>
<p>Maperitive reads OSM files directly however will not handle large areas gracefully.</p>
