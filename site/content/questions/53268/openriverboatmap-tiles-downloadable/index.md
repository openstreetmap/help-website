<p>In answer to "How do I render such tiles and host them at my server?" there's some information at <a href="https://switch2osm.org/serving-tiles/manually-building-a-tile-server-14-04/">https://switch2osm.org/serving-tiles/manually-building-a-tile-server-14-04/</a> (although at the <a href="https://wiki.openstreetmap.org/wiki/Servers/December_2016_Maintenance">time of writing</a> that server is in the process of being moved from one location to another). I also wrote some info <a href="https://wiki.openstreetmap.org/wiki/User:SomeoneElse/Ubuntu_1604_tileserver_load">here</a> - you'd need to follow those, but use ybon's "riverboat" style instead. There are a few details in the <a href="https://github.com/yohanboniface/OpenRiverboatMap">readme</a>, but I might be tempted to try a style with more comprehensive documentation first.</p>
