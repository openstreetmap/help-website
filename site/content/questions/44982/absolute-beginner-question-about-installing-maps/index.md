<p>I'm a bit confused. I swear I visited openstreetmap.org earlier this month and in years past and I could download maps, but had to to so by clicking to select tiles and then the data would be converted to a Garmin-readable format and emailed to me. Now I can't figure out how to use this site. I can navigate to the particular geographic region I'm interested in, but when I click "export" it says the area is too big and recommends other sites that can provide bulk data downloads. So I click on the API Overpass link and it automatically downloads a file called "map" with no file extension. What am I supposed to do with this file? I have a garmin GPS unit and am using Garmin Basecamp to load maps on my device.</p>
<p>So my questions are, is it possible to download maps from openstreetmap and install them on my unit? And how do I do this? The help wiki doesn't appear to offer any relevant instructions for this task.</p>
