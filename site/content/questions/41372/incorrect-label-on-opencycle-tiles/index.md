<p>opencyclemap uses OSM data plus its styles. So, is there an error in the OSM data (which is used by many maps)? Or is it a style error? For me this is not that clear in your text.</p>
<p>If it is a data error you can fix it yourself (but please do not try to optimize only for the cyclemap but instead for correct data).</p>
<p>If the error is in the style of the cycle map, please contact the map maker Andy, see <a href="http://www.opencyclemap.org/docs/">http://www.opencyclemap.org/docs/</a> .</p>
<p>I had a look in the data: I only found two objects in that area which have this name: <a href="https://www.openstreetmap.org/node/151696303">https://www.openstreetmap.org/node/151696303</a> and <a href="https://www.openstreetmap.org/relation/111138">https://www.openstreetmap.org/relation/111138</a> . Both are more north So, I cannot explain from where this label is coming. You could wait for more comments here, but likely Andy has to be asked.</p>
