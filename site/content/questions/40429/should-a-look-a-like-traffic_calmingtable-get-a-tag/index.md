<div class="answer-body">
<p>I would say that a virtual table is no real table and should not be tagged as such. For example it will not make the street impassable to very low sports cars (which normal tables will possibly do).</p>
<p>You could tag it <code>traffic_calming=yes</code> if you want to stay within the established tags. Or invent a <a href="https://wiki.openstreetmap.org/wiki/Any_tags_you_like">new one</a> – e.g. <code>traffic_calming=virtual:table</code> (just a possibly bad example, see comments below).</p>
</div>
