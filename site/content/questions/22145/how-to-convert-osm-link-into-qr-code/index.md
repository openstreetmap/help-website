<p>Hi,</p>
<p>I'd like to convert an openstreetmap link (containing a marker) like: <a href="http://osm.org/go/0HhCQk8-?m">http://osm.org/go/0HhCQk8-?m</a> into QR code (mobile phone usage).</p>
<p>The "export" tab allows to export PNG/SVG/JPEG/PDF but no QR code.</p>
<p>The Firefox add-ons "QrCodeR" or "QRLink maker" do not work for me for different reasons.</p>
<p>Both happily convert URLs on a page itself (e.g. bottom right "ShortLink") but they do not convert an URL in the browser address bar.</p>
<p>The "QrCodeR" generated huge QR code immediately vanishes on pressing the Print Key so I cannot get a snapshot.</p>
<p>The "QRLink maker" generated QR code is <strong><em>very</em></strong> small and seems to generated via chart.googleapis.com. It is so small that I assume Google sits in the middle both when the code is first generated and then later if anyone uses the code.</p>
<p>If my assumption is right he/she would be exposed to google and linked to me. Evidently not an option.</p>
<p>What would do the trick?</p>
<p>Greetings,</p>
