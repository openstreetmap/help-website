<div class="answer-body">
<p>When you run this <a href="http://overpass-turbo.eu/s/133">http://overpass-turbo.eu/s/133</a> overpass query, you'll find a few highway stops. The tagging is highway=stop. You place it on a node of the street on which you have to stop, close to the crossing to which it applies.</p>
<p>You will not see icons on the map of www.openstreetmap.org</p>
<p>So highway=stop or highway=give_way are nodes on the highway. For maxspeed, we typically do not map the sign, but place maxspeed=&lt;speed&gt; on the street. The same for max weight, max length, max width.</p>

<p>two hints:</p>
<p><a href="http://wiki.openstreetmap.org/wiki/JOSM/Plugins/RoadSigns">http://wiki.openstreetmap.org/wiki/JOSM/Plugins/RoadSigns</a></p>
<p>That is a plugin for the offline OSM editor JOSM.</p>
<p>and</p>
<p>go to <a href="http://taginfo.openstreetmap.org">http://taginfo.openstreetmap.org</a> and enter any value for a street sign you want to use in its search box, like traffic_sign=DE:206</p>
<p>Proceed by klicking on one vaue in the Value column, and in the next screen there is a little symbol like a steering wheel (next to XAPI and JOSM button), klick it.</p>
<p>This will lead you to overpass-turbo.eu ... zoom out so that you can see whole Germany or similar, and press "Ausführen" ... thus you can see how that sign is used.</p>
