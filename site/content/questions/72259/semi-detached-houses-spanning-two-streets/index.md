<div class="answer-body">
<p>This is extremely common in Nottingham, there are numerous examples of ways to do it:</p>
<ul>
<li>Split the house in two, add the addresses separately as <a href="https://www.openstreetmap.org/node/338396751#map=19/52.94925/-1.19017">here</a>.</li>
<li>just add the addresses as nodes over the building</li>
<li>Add the main entrance to each house &amp; add the address to this node.</li>
</ul>
<p>We've usually done the first (and many examples of the other kinds have been replaced, we usually keep the third style for places where the division between living units is not clear).</p>
