<p>You have to press the button "Start Mapping" then you get to a list with the possible <a href="http://mapgive.state.gov/projects/">tasks</a>. It seems like the video tutorial <a href="http://mapgive.state.gov/learn-to-map/">MapGive</a> made, does not entirely match their current website layout.</p>
<p>As others have mentioned, the MapGive website is created by "someone else". This is one of several initiatives that individuals and organisations take to get people adding data to OpenStreetMap. That data is then used by a whole bunch of organisations, developers, etc.</p>
<p>The MapGive page has a contact button at the bottom. It links to an email address. I might be a good idea that you drop them a message explaining your problem. They might be able to fix their website (to match the video), so others would get not stuck there.</p>
<p>Hope you will succeed soon in completing the tutorials and enjoy mapping. feel free to ask new questions when you have problems.</p>
