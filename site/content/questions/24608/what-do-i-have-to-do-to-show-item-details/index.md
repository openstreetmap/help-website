<div class="answer-body">
<p>Not a dumb question. Don't know if it was asked before or not. . .</p>
<p>Google maps, Bing maps, etc. provide a map for an end user.</p>
<p>www.openstreetmap.org provides a view of an open map database with the goal of being a portal for citizen cartographers. Which, for better or worse, is not quite the same thing.</p>
<p>Another example is the lack of trip routing.</p>
<p>However the licensing of OSM data is quite generous (simply acknowledge the source of the data) so anyone can build a site providing similar services to Google maps using the data. Not that it is an easy task to do a great job in pulling together all the pieces for a great user experience, but at least the data is available.</p>
<p>If you wish to have something approximating a commercial end user map experience but access the openstreetmap data the best I know of is <a href="http://open.mapquest.com/">http://open.mapquest.com/</a> And there are probably others.</p>
<p>Why should you use something like <a href="http://open.mapquest.com/">http://open.mapquest.com/</a> rather than Google Maps? Two reasons come to my mind immediately: 1) You can fix the problems (e.g. in the US in rural areas neither Google nor OSM are accurate so both need fixing). 2) The fixes you provide in OSM are available for others to use and will make the world a better place since many low budget/non-profit organizations can use it freely. If you use Google's citizen mapping to fix things, Google owns the data so you are doing free work for a multi-billion dollar company which charges big bucks for map licensing.</p>
