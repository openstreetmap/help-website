<div class="answer-body">
<p>There is no "translation". The data in OSM is wrong.</p>
<p>See the <a href="https://www.openstreetmap.org/node/4196292201">place node for Ghayl Ba Wazir</a>. It only has a <em>name</em> tag in English whereas it should have it's local name in the <em>name</em> tag and the English name in <em>name:en</em>. See <a href="https://wiki.openstreetmap.org/wiki/Names#Localization">localization</a> and <a href="https://wiki.openstreetmap.org/wiki/Multilingual_names">multilingual names</a> for more information.</p>
<p>The same applies to <a href="https://www.openstreetmap.org/way/378468019">Al-ghail Main Street</a>. It has the English name in the <em>name</em> tag and the Arabic name in <em>name:ar</em>. This is wrong since the Arabic name is the local name. Thus the Arabic name has to be in the <em>name</em> tag and the English name in <em>name:en</em>.</p>
<p>It would be nice if you or someone else could fix these issues in the OSM data. Also try to contact the user who did it wrong and tell him/her how names in OSM work by pointing at these wiki pages.</p>
