<p><span class="small">The described behaviour matches what you would see in the current default editor iD, so I will comment on this:</span></p>
<ol>
<li>You need to click (to "select") the street once (instead of just hovering/mouseover on it)</li>
<li>move your mouse into the side bar</li>
<li>do your name change</li>
</ol>
<p>As always:</p>
<ol>
<li>Do other changes which are related (e.g. name changes of other parts of this street)</li>
<li>click the save button</li>
<li>enter a short <a href="https://wiki.openstreetmap.org/wiki/Good_changeset_comments">comment what, why and with which sources</a> you did</li>
<li>click the blue save button in the side bar</li>
</ol>
