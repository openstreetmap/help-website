<div class="answer-body">
<p>You can use <a href="https://wiki.openstreetmap.org/wiki/Conditional_restrictions">conditional access restrictions</a>. They support conditions based on the time and date which have to be specified using the <a href="https://wiki.openstreetmap.org/wiki/Key:opening_hours">opening_hours</a> syntax.</p>
<p>To forbid any <a href="https://wiki.openstreetmap.org/wiki/Key:motor_vehicle">motor vehicle</a> from December to June, use</p>
<pre><code>motor_vehicle:conditional=no @ (Dec-Jun)</code></pre>
<p>Alternatively you could disallow motor vehicles by default and only allow them between July and November:</p>
<pre><code>motor_vehicle=no
motor_vehicle:conditional=yes @ (Jul-Nov)</code></pre>
<p>But be aware that the second solution completely disallows motor vehicles for all applications that can't handle conditional restrictions.</p>
