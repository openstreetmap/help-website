<div class="answer-body">
<p>I guess you are talking about Potlatch's presets which offer only 10 km/h, 20 km/h and so on. You don't have to use these presets, instead you can switch to the <em>advanced tab</em> (at the bottom of the window) and enter whatever tag you like. In your case just add key <em>maxspeed</em> and value <em>15</em> (without the <em>km/h</em> because it is the default unit for maxspeed, see the <a href="https://wiki.openstreetmap.org/wiki/Key:maxspeed">maxspeed wiki page</a>).</p>

<p>Please don't add a speed limit to Dutch living streets. This speed limit is implicit by the road type. (The only roadtype in the Netherlands were this actually works.)</p>
