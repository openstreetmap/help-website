<div class="answer-body">
<p>It seems that you have to get familiar with the basic structure of OSM and what you can do with the raw data by using conversters, map formats and display frameworks.</p>
<p>So take three days or more and read many many pages in the OSM wiki at <a href="http://wiki.openstreetmap.org">http://wiki.openstreetmap.org</a> ... espacially about OSM elements, XML data, development, openlayers, framework, software, map display, rendering etc.</p>
<p>There are many opensource solutions for many ideas. YOU have to pick one and ask details about it here or at OSM forum or mailing lists when you get stuck.</p>
<p>If you want a service like the one from mapquest that you linked to: have you already asked mapquest how they did it?</p>
