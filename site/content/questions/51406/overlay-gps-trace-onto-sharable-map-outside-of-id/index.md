<p>Hello,</p>
<p>I'm looking to find a way to have a GPS trace overlaid onto the basic OSM online map (and thus sharable). Essentially I'm using OSM to share a specific GPS route (not tied to existing roads) and I need to be able to send it to others just using the OSM interface, not iD. For instance I'm using the "share" functionality on the right of my browser so I'm able to send a specific crop and also URL link.</p>
<p>Is this possible?</p>
<p>Within iD obviously I've been able to insert my GPS trace and edit the maps, however when I click "export" so I can get back to the basic OSM interface with the share key, it removes my GPS trace.</p>
<p>Is this possible to do, or am I using the functionality of OSM/traces different then intended.</p>

<div class="answer-body">
<p><a href="https://wiki.openstreetmap.org/wiki/Export">"export"</a> is to download raw OSM data ("export" it to your computer).</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Upload">uploading</a> gps traces to osm.org is to aid mapping — not to share the traces with your friends.</p>
<p>Likely you are looking for one of those: <a href="https://wiki.openstreetmap.org/wiki/Track_drawing_websites">https://wiki.openstreetmap.org/wiki/Track_drawing_websites</a></p>
