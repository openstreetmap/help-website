<div class="answer-body">
<p>Currently it is 0.25 degrees squared (e.g. 1°x0.25° or 0.5x0.5° etc), or an area containing 50,000 nodes, whichever limit is reached first. In densely mapped areas, the node limit will be reached well before the extent limit.</p>
<p>Use third party services like Overpass API or <a href="http://extract.bbike.org/">http://extract.bbike.org/</a> for larger custom downloads, or <a href="https://mapzen.com/metro-extracts/">https://mapzen.com/metro-extracts/</a> and <a href="http://download.geofabrik.de/">http://download.geofabrik.de/</a> to download ready-made larger extracts.</p>
