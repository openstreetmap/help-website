<p>For maps of this scale, I'd suggest getting small scale country shapes from <a href="https://www.naturalearthdata.com/downloads/">Natural Earth</a>. A map like this can be achieved with a mapping software such as QGIS or, if you want interactive maps, with web mapping tools such as <a href="https://d3js.org/">D3.js</a> or <a href="https://leafletjs.com/">Leaflet</a>.</p>
