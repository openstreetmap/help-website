<div class="answer-body">
<p>see <a href="https://wiki.openstreetmap.org/wiki/Tag:military%3Dbunker">https://wiki.openstreetmap.org/wiki/Tag:military%3Dbunker</a> (especially "additional tags" section) and <a href="https://wiki.openstreetmap.org/wiki/Lifecycle_prefix">https://wiki.openstreetmap.org/wiki/Lifecycle_prefix</a> . According to your description I would go for</p>
<ul>
<li><code>abandoned:building=bunker</code></li>
<li><code>military=bunker</code></li>
<li><code>bunker_type=???</code></li>
</ul>
<p>In a <code>description</code> tag you could add a description of it - including the fact that it is not the bunker's original location.</p>
