<div class="answer-body">
<p>Potlatch 2 keeps the current changeset open while you're working. This should encourage you to save early and save often. :)</p>
<p>If you want to start a new changeset for your next save, you'll need to close the current one first. That's easy. When you want to close a changeset, just press 'C' for Close. A little blue confirmation message will float up once it's been successfully closed.</p>
<p>Next time you save, a new changeset will be created, and so you'll be asked for a new edit summary.</p>
