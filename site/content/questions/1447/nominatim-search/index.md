<p>Nominatim works with URL's. You can find many examples on the wiki at:<br />
<a href="http://wiki.openstreetmap.org/wiki/Nominatim"></a><a href="http://wiki.openstreetmap.org/wiki/Nominatim"></a><a href="http://wiki.openstreetmap.org/wiki/Nominatim">http://wiki.openstreetmap.org/wiki/Nominatim</a></p>
<p>You already asked the same question <a href="http://help.openstreetmap.org/questions/1441/how-can-i-use-search-on-osm-in-my-program">here</a>. Once you read the specification, you have to adapt your code to handle URL's requests and returned XML data. This depends on your development/application and the question is not anymore specific to OSM and shouldn't be posted here. You might give a last try by asking on the OSM developers list ("dev"), see the details in <a href="http://wiki.openstreetmap.org/wiki/Contact">the wiki</a>.</p>

<p>here is an example i've beenplaying with, this is using codeigniter</p>
<pre><code>$xml-&gt;open(&#39;http://nominatim.openstreetmap.org/search?q=&#39;.$searchstring.&#39;&amp;format=xml&amp;polygon=0&amp;addressdetails=1&amp;countrycodes=gb&#39;);
$addrlist = $this-&gt;_xml2assoc($xml, &quot;root&quot;);
&#10;            $data[&#39;addressescount&#39;] = count($addrlist[0][&#39;childs&#39;]);
            $addresses = $addrlist[0][&#39;childs&#39;];
&#10;            if ($data[&#39;addressescount&#39;] == 0)
            {
                $data[&#39;addrfrommsg&#39;] = &#39;Sorry, we could not find any location for this address.&#39;;
            }
            elseif (($data[&#39;addressescount&#39;] &gt; 0) &amp;&amp; ($data[&#39;addressescount&#39;] &lt; 20))
            {
                // put each address from nominatim into an array
                foreach ($addresses as $k =&gt; $inside)
                {
                    $addrlist[$k][&#39;place_id&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;attr&#39;][&#39;place_id&#39;];
                    $addrlist[$k][&#39;lon&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;attr&#39;][&#39;lon&#39;];
                    $addrlist[$k][&#39;lat&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;attr&#39;][&#39;lat&#39;];
                    $addrlist[$k][&#39;road_x&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;childs&#39;][0][&#39;childs&#39;][0][&#39;text&#39;];
                    $addrlist[$k][&#39;city_x&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;childs&#39;][2][&#39;childs&#39;][0][&#39;text&#39;];
                    $addrlist[$k][&#39;display_name&#39;]= $addrlist[0][&#39;childs&#39;] [$k][&#39;attr&#39;][&#39;display_name&#39;];
                    $data[&#39;addrfromlist&#39;][]=$addrlist[$k];
&#10;            // insert the location in database
                    $country = &#39;UK&#39;;
                    $direction = &#39;from&#39;;
                    $postcode = &#39;&#39;;
                    if ($addrlist[$k][&#39;place_id&#39;] == 0 )
                    {
                        $addrlist[$k][&#39;place_id&#39;] = $k;
                    }
                    $this-&gt;locations_mdl-&gt;insert_location($data[&#39;session_id&#39;], $addrlist[$k][&#39;place_id&#39;], $data[&#39;username&#39;],
                                        $addrlist[$k][&#39;lon&#39;], $addrlist[$k][&#39;lat&#39;], $addrlist[$k][&#39;road_x&#39;], $addrlist[$k][&#39;city_x&#39;],
                                        $postcode, $country, $addrlist[$k][&#39;display_name&#39;], $direction);
&#10;                }
            }
            elseif ($data[&#39;addressescount&#39;] &gt;= 20)
            {
                $data[&#39;addrfrommsg&#39;] = &#39;Sorry, too many results please refine your search.&#39;;
            }
            $xml-&gt;close();
            return $data;
    }</code></pre>
