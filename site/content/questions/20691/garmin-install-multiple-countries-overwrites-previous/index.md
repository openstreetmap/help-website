<p>Good to see you found the FAQ :)</p>
<p>Merging maps is possible, but you will need to generate a new overview map to reflect the added tiles. Mkgmap can generate an overview map, but is an commandline tool. Other tools can do this as well, like <a href="http://www.geopainting.com/">GPSMapEdit</a> or <a href="http://www.gmaptool.eu/en/content/merging-maps">GMapTool</a>.</p>
