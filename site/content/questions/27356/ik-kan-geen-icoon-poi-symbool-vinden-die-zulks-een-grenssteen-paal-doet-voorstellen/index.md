<p>TonenEllen, nee er is geen symbool voor een grenspaal. Een dubbele klik geeft in P2 een groene en een klik in JOSM een rode punt. Deze punt of node krijgt dan de kenmerken of tags. In P2 via het geavanceerde menu. ps zie hier de algemene beschrijvingen. <a href="http://wiki.openstreetmap.org/wiki/Help:Inhoud">http://wiki.openstreetmap.org/wiki/Help:Inhoud</a> en de Wiki <a href="http://wiki.openstreetmap.org/wiki/NL:Kaarteigenschappen">http://wiki.openstreetmap.org/wiki/NL:Kaarteigenschappen</a> voor de kenmerken. Sorry er ontbreekt een link, deze dus <a href="http://www.openstreetmap.org/#map=19/51.82779/4.98991">http://www.openstreetmap.org/#map=19/51.82779/4.98991</a> of deze Banpalen langs de Lek <a href="http://www.openstreetmap.org/#map=18/51.99988/5.06787.">http://www.openstreetmap.org/#map=18/51.99988/5.06787.</a> In P2 kun je waarden niet copieren, maar in JOSM kan je het onderwerp, node of punt kopieren.</p>
