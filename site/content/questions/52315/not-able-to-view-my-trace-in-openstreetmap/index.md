<p>From the the OSM slippy map at <a href="https://www.openstreetmap.org/#map=14/18.7695/73.811">https://www.openstreetmap.org/#map=14/18.7695/73.811</a><br />
select the 'Edit' button at top left, choose edit with iD (in-browser editor)<br />
Read the 'walkthough' help and when you know it all, select 'Edit Now'<br />
Drag and drop the gpx file you want to use from your file manager onto the iD screen and you should see your trace.<br />
You can then use your gpx trace and satellite imagery to add/modify a way.<br />
You can add other 'OpenStreetMap GPS traces' via the 'background settings' icon on the right side or choose other satellite imagery that might be more recent.<br />
Your trace will be amongst the 'OpenStreetMap GPS traces' when you make your trace public for others to view.</p>
<p>After using the iD editor for a while, get familiar with the other editors offered.</p>
