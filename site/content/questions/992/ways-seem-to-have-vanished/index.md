<p>I do some mountain biking and try to update OSM with anything that was missing. Some tracks in North Wales which I am sure as I can be (without having made notes) used to exist, do not currently exist. (Whether I added them or found they were already there is not the issue.) These are not obscure illicit paths, they are nationally know purpose built MTB trails.</p>
<p>Is there any way to search for deleted objects in a given area (a few square km) ?</p>
<p>It is of of course possible that I have a bad memory for what used to exist.</p>

<div class="answer-body">
<p>Open <span>Potlatch 1</span> (via the Edit tab and manual URL editing) for a particular area, and press U (for "undelete"). This will show you any ways that existed but have since been deleted. You can click any of them and 'unlock' them, then save, should you want to restore the ways.</p>
