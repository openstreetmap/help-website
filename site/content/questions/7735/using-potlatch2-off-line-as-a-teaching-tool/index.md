<div class="answer-body">
<p>P2 won't run offline, unless you have an OSM server set up on your own machine - which definitely <em>doesn't</em> count as "easily"!</p>
<p>However, you can use it on other databases other than the main OSM one. In particular, you might like to take a look at the test intallation at <a href="http://random.dev.openstreetmap.org/potlatch2/potlatch2.html">http://random.dev.openstreetmap.org/potlatch2/potlatch2.html</a> . This is connected to a testing database (that at <a href="http://api06.dev.openstreetmap.org">api06.dev.openstreetmap.org</a>), and so you can happily play and save whatever you want without damaging data.</p>
