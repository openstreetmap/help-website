<div class="answer-body">
<p>There are no routing services provided by the OSM / on openstreetmap.org All that can be used are simply linked to via the UI, you could for example use routing.openstreetmap.de which is based on OSRM and can be used similarly as the example you are quoting.</p>
<p>See <a href="https://wiki.openstreetmap.org/wiki/Open_Source_Routing_Machine">https://wiki.openstreetmap.org/wiki/Open_Source_Routing_Machine</a> and <a href="https://wiki.openstreetmap.org/wiki/Routing">https://wiki.openstreetmap.org/wiki/Routing</a> for more information.</p>
