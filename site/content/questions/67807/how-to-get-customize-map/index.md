<div class="answer-body">
<p>There are many ways to do this with varying degrees of customisation. While OpenStreetMap.org is the source for the data, the services using that data are normally run separately.</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Deploying_your_own_Slippy_Map">This page</a> on the wiki has a list of tools to get you started, or you might be interested in some of the <a href="https://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services">commercial services</a> that do a lot of the server side work for you.</p>
<p>I think it is worth mentioning that the default style you see at OpenStreetMap.org is subject to a <a href="https://operations.osmfoundation.org/policies/tiles/">tile usage policy</a>. It is mainly there to help contributors and generating a lot of traffic will probably get you blocked. The bottom of that page also has a short list of some of the larger providers.</p>
