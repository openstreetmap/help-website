<p>Apart from Simon's answer, …</p>
<p>you could use <a href="https://wiki.openstreetmap.org/wiki/JOSM">JOSM</a> to create a new OSM layer, do your imaginary additions, save into an .osm file (locally on your computer). Then you can start with your SUMO stack (I do not know which format SUMO ingests – .osm XML files?). That would be very easy.</p>
