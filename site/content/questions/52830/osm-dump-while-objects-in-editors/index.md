<div class="answer-body">
<p>Yes. Changes to the OSM database will not be available before</p>
<ul>
<li>the changes have been uploaded with the API to the database</li>
<li>the database changes have propagated to whichever operator is providing the OSM data</li>
<li>the operator has created the dump/extract</li>
</ul>
<p>The same goes naturally for anybody downloading data, for example for editing, via the API.</p>
