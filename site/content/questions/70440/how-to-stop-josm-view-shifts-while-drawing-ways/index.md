<div class="answer-body">
<p>This is called "Viewport Following" and it is superb for tracing long and straight features like border fences or rivers. I can see how it would interfere with everyday mapping though, so try Ctrl+Shift+F to disable it, or uncheck the appropriate box in the "View" menu.</p>
