<p>For the basic policy on such matters see <a href="https://wiki.osmfoundation.org/w/images/d/d8/DisputedTerritoriesInformation.pdf">https://wiki.osmfoundation.org/w/images/d/d8/DisputedTerritoriesInformation.pdf</a></p>
<p>If you have a local OSM database that you use for map rendering changing what is used for name rendering should be comparatively simple (use the name:vi tags instead of name for example). Changing the borders is slightly more involved but possible, but naturally the details depend strongly on what kind of setup you actually have.</p>
