<p>I was stuck with the same task; get all housenumbers of a given street.</p>
<p>Found this hacky way:</p>
<p><code> [out:csv("addr:housenumber")][timeout:25]; {{geocodeArea:Gänserndorf}}-&gt;.searchArea; ( node["building"]["addr:street"="Hauptstraße"]["addr:housenumber"](area.searchArea); way["building"]["addr:street"="Hauptstraße"]["addr:housenumber"](area.searchArea); relation["building"]["addr:street"="Hauptstraße"]["addr:housenumber"](area.searchArea); ); out body; &gt;; out skel qt; </code></p>
<p>I have to say, the learning curve can be a wall and I'm not totally happy with the result. Maybe someone else has a nicer version.</p>
<p>Afterwards I used "sort -g" under Linux to sort the list.</p>
