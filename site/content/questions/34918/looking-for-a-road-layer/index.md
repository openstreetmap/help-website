<p>You might want to try the shapefiles of <a href="http://download.geofabrik.de/north-america.html">http://download.geofabrik.de/north-america.html</a> But keep in mind that usually you loose tagging details and maybe some object classes due to the limitation of shapefile processing.</p>
<p>If you want to control it yourself, you need to can use OSM planet extracts, filter with osmosis and convert the remaining road geometries to shapefiles.</p>

<div class="comment-text">
<p>Sure, ok OSM has data dumps called <a href="https://wiki.openstreetmap.org/wiki/Planet">Planet extracts</a> that can be filtered e.g. with <a href="https://wiki.openstreetmap.org/wiki/Osmosis">Osmosis</a>. Last but not least we have <a href="https://wiki.openstreetmap.org/wiki/Shapefile">various solutions</a> to convert the geometries and metadata shapefile.</p>
<p>If you just want a road layer , you should check <a href="https://wiki.openstreetmap.org/wiki/ITO_Map">ITO maps</a> before and think about if a OSM renderer as <a href="https://wiki.openstreetmap.org/wiki/Maperitive">maperitive</a> might do the job (will save you converting etc.).</p>
