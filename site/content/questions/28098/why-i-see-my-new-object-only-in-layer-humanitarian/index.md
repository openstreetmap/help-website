<div class="answer-body">
<p>No, there is no command which defines in which layer an object should show. Also note that our OSM data is used for <span>far more maps and services</span> than just the few layers presented on osm.org.</p>
<p>See <a href="/questions/102/">i-have-made-edits-but-they-dont-show-up-on-the-map</a> ("Not everything that is in the data, is also shown on every map." and "cache"!) for a longer answer. It may either be that only the humanitarian layer shows this type of object, that the other layers were not updated yet, or that you are viewing an older version of the other layers from a cache.</p>
