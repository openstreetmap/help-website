<p>The data's different, which allows people to do things with the data that you wouldn't want to do with the live data (like <a href="http://api06.dev.openstreetmap.org/browse/way/4295036579">this</a>!).</p>
<p>However, if you want to use editing software in development with the live data, you can. You can use a bleeding-edge version of <a href="http://josm.openstreetmap.de/">JOSM</a>. There's also a similar version of <a href="http://openstreetmap.us/iD/release/">iD</a> available.</p>
