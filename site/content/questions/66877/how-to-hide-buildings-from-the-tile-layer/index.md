<p>I'd follow <a href="https://switch2osm.org/manually-building-a-tile-server-18-04-lts/">the switch2osm guide</a> and set up a tile server exactly like OSM's standard layer then I'd remove the bit from that that displays buildings.</p>
<p>That probably means editing <a href="https://github.com/gravitystorm/openstreetmap-carto/blob/master/project.mml#L422">this file</a> so that the layer corresponding to buildings (which I've highlighted the start of) is removed. There may be a bit more to do than that (specifically generic building labels) but that should start you off.</p>
