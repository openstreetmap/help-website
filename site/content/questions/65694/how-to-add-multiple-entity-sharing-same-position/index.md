<p>There is a "correct way" to add in the same position multiple entities.</p>
<p>On the real place there is a covered area (but not a building as it is open on three sides) in which are placed:</p>
<ul>
<li>A little "library for the traveler" holding some turistic guide and book of varrous item</li>
<li>Tables to eat an read (Added as Covered "Picnic Area")</li>
<li>On a wall there is a defibrillator (Added as Defribillator)</li>
<li>Various "trash cans" and reciclying cans (Added as Reciclying)</li>
</ul>
<p>I didn't know how to add this "free library" as it is not a "building" and in this library there are other things like a place where you can "leave a thinking" or "poems" or other "notes" You want to share to other travelers (hickers and cyclers).</p>
<p>I have done multiple "points" placed near each others, and this seems a "quite good" solution.</p>
