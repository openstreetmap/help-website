<div class="answer-body">
<p>There are already a lot of well-established bicycle tags available in OSM. The <a href="https://wiki.openstreetmap.org/wiki/Bicycle">bicycle wiki page</a> explains the tagging of cycleways. Additionally there are relations for <a href="https://wiki.openstreetmap.org/wiki/Cycle_routes">cycle routes</a> and various interesting tags for <a href="https://wiki.openstreetmap.org/wiki/Mountainbike">mountain bikes</a>.</p>
<p>Your second point is about rendering these tags. The main map only shows very few cycle-related tags such as generic cycleways. But via the sidebar you can select the bicycle layer showing the <a href="https://wiki.openstreetmap.org/wiki/OpenCycleMap">OpenCycleMap</a>. Then there is <a href="http://www.cyclestreets.net/">CycleStreets</a> for the UK, <a href="https://wiki.openstreetmap.org/wiki/BBBike_@_World">BBBike</a> for specific cities all over the world, the <a href="https://wiki.openstreetmap.org/wiki/Bicycle_tags_map">bicycle tags map</a> already mentioned by Hendrik, <a href="http://cycling.waymarkedtrails.org/de/">Lonvia's cycle</a> and <a href="http://mtb.waymarkedtrails.org/de/">mtb map</a> showing bicycle and mountain bike routes, and lots of other interesting <a href="https://wiki.openstreetmap.org/wiki/List_of_OSM_based_Services#Biking.2C_Geocaching.2C_Hiking.2C_Sport">OSM based cycle maps</a>.</p>
