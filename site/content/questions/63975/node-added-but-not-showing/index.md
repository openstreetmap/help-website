<p>Welcome to OSM! Different maps update at different rates; it'll appear on OSM's "standard map" shortly and the cycle and transport layers within a couple of weeks . It already appears at <a href="https://map.atownsend.org.uk/maps/map/map.html#zoom=17&amp;lat=51.482912&amp;lon=-0.01304">this map</a>.</p>
<p>Incidentally, if it's more of a pub than a bar you might want to use "<a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dpub">amenity=pub</a>" rather than "<a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dbar">amenity=bar</a>". Also if it serves real ale then add a tag such as "<a href="https://wiki.openstreetmap.org/wiki/Key:real_ale">real_ale=yes</a>" (I'm guessing that it does if it's a Fuller's pub).</p>
