<p>As noted by InsertUser, you can use the <code>building=construction</code> But I prefer the <a href="https://wiki.openstreetmap.org/wiki/Lifecycle_prefix">lifecycle tagging</a> and would use <code>construction:building=yes|house|detached|etc</code> as it can impart a little more information.</p>
