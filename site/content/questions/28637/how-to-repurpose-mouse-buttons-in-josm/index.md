<p>AFAIK there is no standard in GIS/CAD that says anything about how pan/drag actions should be invoked by which mouse button.<br />
Even in the "expert" preferences dialog you are <strong>unable to alter the default behavior</strong>, so I recommend to start a feature request ticket: <a href="http://josm.openstreetmap.de"></a><a href="http://josm.openstreetmap.de">http://josm.openstreetmap.de</a></p>
