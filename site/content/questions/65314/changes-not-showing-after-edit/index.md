<p>(sorry, not much time now)</p>
<p>Welcome! :-) Your changes will appear magically after some time. The update process just takes some time. Please the top questions at see <a href="https://help.openstreetmap.org/tags/update/?sort=mostvoted">https://help.openstreetmap.org/tags/update/?sort=mostvoted</a> and <a href="https://help.openstreetmap.org/tags/not_shown/?sort=mostvoted">https://help.openstreetmap.org/tags/not_shown/?sort=mostvoted</a> . Your changes to the coastline take much longer (weeks), that is normal - see <a href="https://help.openstreetmap.org/questions/276/why-do-the-changes-i-have-made-to-coastline-not-appear-on-the-map">https://help.openstreetmap.org/questions/276/why-do-the-changes-i-have-made-to-coastline-not-appear-on-the-map</a> .</p>
<p>Other contributors (like you) will, hopefully, look at your changeset if you requested review. If that happens depends if there are enough active people. Yes, you can do reviews, too: just add a changeset discussion comment with your feedback. However, I would not regard this as very newbie-compatible task. :-) See <a href="https://help.openstreetmap.org/tags/review_requested/">https://help.openstreetmap.org/tags/review_requested/</a> and <a href="http://neis-one.org/2017/09/review-requests-osm/">http://neis-one.org/2017/09/review-requests-osm/</a> .</p>
<p>If you have some specific other questions, please ask them all on separate question entries on this site.</p>
<p>Happy mapping!</p>
