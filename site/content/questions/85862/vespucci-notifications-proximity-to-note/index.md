<p>The mechanism is the same as described here <a href="https://lists.openstreetmap.org/pipermail/talk/2022-October/087822.html">https://lists.openstreetmap.org/pipermail/talk/2022-October/087822.html</a></p>
<p>Aka you will need to enable one of the continuous/auto download options (and keep the areas small). Note that there is currently no offline data format for notes as there is for OSM data.</p>
