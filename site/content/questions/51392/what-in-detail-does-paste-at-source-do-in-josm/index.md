<div class="answer-body">
<p>In older versions of JOSM, copying and pasting an object placed it at the same location as the source object. The most recent version changes this to copy the object either to the location of the mouse or the center of the visible map area (when the cursor is outside the area).</p>
<p>So a new command was added to give access to the old behavior.</p>
