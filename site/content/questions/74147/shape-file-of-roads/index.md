<p>If I understood correctly you are looking for Shape files for roads of the whole world and since you are posting this on OpenStreetMap help forum, you are try to get the data from OpenStreetMap.</p>
<p>If so, there are several sources from which you can do it. For example, you can use <a href="https://planet.openstreetmap.org/planet/full-history/">PlanetOSM</a> which backs up whole OSM data and you can also get historical data.</p>
<p>Another source is <a href="http://download.geofabrik.de/">Geofabrik</a> where you can get the data directly as shape file and then use a software like QGIS or ArcGIS (if you have it) to extract only the highways.</p>
<p>You can also check out this <a href="https://wiki.openstreetmap.org/wiki/Downloading_data">link</a> for other OSM data download sources.</p>
