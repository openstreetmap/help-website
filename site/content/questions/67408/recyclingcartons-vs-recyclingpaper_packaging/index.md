<div class="answer-body">
<p>Checking the <a href="https://wiki.openstreetmap.org/w/index.php?title=Tag:amenity%3Drecycling&amp;offset=&amp;limit=500&amp;action=history">history of the amenity=recycling wiki page</a>, the recycling:&lt;material&gt; tags date from the early days of OpenStreetMap (the page was started in 2006). I suspect that for a time people were trying to map recycling and come up with suitable tags where there weren't any suitable on the wiki page.</p>
<p>recycling:paper_packaging was added on <a href="https://wiki.openstreetmap.org/w/index.php?title=Tag:amenity%3Drecycling&amp;oldid=203838">5th January 2009</a> with the comment "(paper pacaging is a seperate recycling category in Iceland. Adding because I'm using it in the Reykjavík area)"</p>
<p>recycling:beverage_cartons was added on <a href="https://wiki.openstreetmap.org/w/index.php?title=Tag:amenity%3Drecycling&amp;oldid=285808">11th June 2009</a> with the comment "Materials: beverage cartons" and a description of "Paper Based Beverage Cartons"</p>
<p>recycling:cartons was added on <a href="https://wiki.openstreetmap.org/w/index.php?title=Tag:amenity%3Drecycling&amp;direction=next&amp;oldid=407641">18th January 2010</a> with the comment "Materials: UK recycling categories compleated <a href="http://www.leics.gov.uk/index/environment/waste/recycling_household_waste_sites/wastesite_loughborough.htm"><code>http://www.leics.gov.uk/index/environment/waste/recycling_household_waste_sites/wastesite_loughborough.htm</code></a>"</p>
<p>The modern version of that last link is probably <a href="https://www.leicestershire.gov.uk/environment-and-planning/waste-and-recycling/how-do-i-dispose-of">https://www.leicestershire.gov.uk/environment-and-planning/waste-and-recycling/how-do-i-dispose-of</a> and what they list under Cartons is the <a href="https://en.wikipedia.org/wiki/Tetra_Pak">Tetra Pak style</a> cartons which the wiki now gives as an example next to beverage_cartons.</p>
<p>So my personal interpretation is that if it is just paper based packaging, use the paper packaging tag, but if it is the tetra-pak based waxed paper stuff then use beverage cartons. Use cartons if you aren't sure, maybe.</p>
