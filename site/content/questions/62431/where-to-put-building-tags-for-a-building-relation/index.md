<div class="answer-body">
<p>You should put the other tags for a building on that object which represents the "building" - the object which has the <code>buidling=*</code> tag. This <em>the building outline.</em></p>
<p>The building relations do not have the main tag <code>building=*</code>. "Relations of <code>type=building</code> are used to group elements that form a building in simple 3D modelling." ( <a href="https://wiki.openstreetmap.org/wiki/Relation:building">https://wiki.openstreetmap.org/wiki/Relation:building</a> )</p>
