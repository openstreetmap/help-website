<p>Apologies for such a late response... the latest version (1.4.4) of gpsbabel reads garmin fit files from an Edge 800 as both input (and output). It loses some information (something I think may be resolved) like date/time (taking 'now' instead of trace metadata... but I don't know if that's fit or just gpsbabel), but the gps trace/route IS intact.</p>
<p>e.g. an example import</p>
<p>gpsbabel -i garmin_fit -f /mnt/GARMIN/Garmin/Activities/activity20120918.fit -o gpx -F ~/tmp/output.gpx</p>
<p>Hope that helps!</p>

<td><div class="item-right">
<div class="answer-body">
<p>I'm posting this here because Google likes this and I thought it might be useful if your running linux and lazy like me and want a one-click solution. This finds fit files in anything mounted called "garmin" and puts gpx files with the same name into a folder in your home folder called "Tracks" Thanks to spaceyjase for the gpsbabel syntax.</p>
<pre><code>#!/bin/bash
# Convert any new files to GPX from the garmin fit format.
# existing files will not be overwritten
# FROM is the name of the mount point for the garmin, my garmin is called &quot;GARMIN&quot; and I called the sd card &quot;garmin-map&quot; so this finds both if they&#39;re mounted with automount
FROM=garmin
TO=~/Tracks/
&#10;FROMPATH=`mount | awk &#39;{print $3};&#39; | grep -i $FROM`
GARMINEXT=&quot;.fit&quot;
TOEXT=&quot;.gpx&quot;
GPSBABEL=&quot;gpsbabel&quot;
&#10;# Check if gps babel exists
hash ${GPSBABEL} 2&gt;/dev/null || { 
    echo &gt;&amp;2 &quot;I require ${GPSBABEL} but it&#39;s not installed.  Aborting.&quot;
    exit 2
}
&#10;# there may be more than one matching path
for FROM in ${FROMPATH} ; do
# check if the path is valid
    if [ -d &quot;${FROM}&quot; ]; then
        # create the destination if it doesnt exist
        if [ ! -d &quot;$TO&quot; ]; then
            echo &quot;Creating $TO&quot;
            mkdir &quot;$TO&quot;
        fi
        echo &quot;${FROM} =&gt; ${TO}&quot;
&#10;        # find valid folders that we can use
        # we cant process all .fit files
        for SEARCHPATH in `find ${FROM} -type d -iname Activities -o -iname Courses` ; do
            # find all the fit files in the folder
            for INFILE in ${SEARCHPATH}/*${GARMINEXT} ; do
                if [ -f &quot;${INFILE}&quot; ]; then
                    # work out the destination file name abc.fit =&gt; abc.gpx
                    TOFILE=`basename &quot;${INFILE}&quot; | sed s/$GARMINEXT/$TOEXT/`
&#10;                    # only do the conversion if the destion doesn&#39;t exist
                    if [ ! -f &quot;${TO}${TOFILE}&quot; ]; then
                        echo &quot;Converting `basename &quot;$INFILE&quot;` to $TOFILE...&quot;
                        ${GPSBABEL} -i garmin_fit -f &quot;${INFILE}&quot; -o gpx -F &quot;${TO}/${TOFILE}&quot; || {
                            echo &quot;Failed to convert ${INFILE}&quot;
                        }
                    else
                        echo &quot;${TO}${TOFILE} alread exists.&quot;
                    fi
                fi
            done
        done
    else
        echo &quot;$FROMPATH does not exist.&quot;
        exit 1;
    fi
done</code></pre>
