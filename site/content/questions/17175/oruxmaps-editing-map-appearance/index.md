<p>The raw OSM data (what you have in the .xml file) does not have any specific attributes that are used for rendering (how the map appears on the output media). Typically this is acheived via a "style file".</p>
<p>Since you don't actually specify with what you are viewing the vector map data it is difficult to help you any further.</p>
