<p>Why are sorted tags a problem to you ? Note that tags are not ordered in any way as far as OSM data is concerned, so you shouldn't rely on any particular display order.</p>
<p>Even if an editor didn't sort new tags immediately, it'd probably sort them alphabetically once you click away from the object and back, because that's the only sensible way to order/display tags.</p>
