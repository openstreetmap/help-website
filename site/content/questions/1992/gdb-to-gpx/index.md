<td><div class="item-right">
<div class="answer-body">
<p>There is no way to upload GDB files directly. You can however open them in MapSource and then chose "Save As..." from the "File" menu, where you can then select the type to save the file as.</p>
<p>You can also use the open source utility <a href="http://www.gpsbabel.org/">GPSBabel</a> (available for Windows, Linux, OS X) to convert from GDB to GPX.</p>
