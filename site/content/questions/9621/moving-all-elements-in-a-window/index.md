<p>I would like to move all calles at once, but what is the best way to move things without breaking relationships and without performing a double move on intersection points?</p>

<div class="answer-body">
<p>In Potlatch 2 (the default editor on <a href="http://openstreetmap.org">openstreetmap.org</a>), you can select several objects at once by either:</p>
<ul>
<li>holding Control and clicking each one individually, or</li>
<li>holding Control and drawing a box around all the objects you want to select.</li>
</ul>
<p>Then simply drag them to the place you want.</p>
<p><em>(On a Mac, you'll need to use the Command key - the one with a picture of an Apple - rather than Control.)</em></p>
