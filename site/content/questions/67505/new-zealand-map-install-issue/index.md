<div class="answer-body">
<p>Thanks again escada. I had tried the direct download first before using Basecamp. Prompted by your hint I looked where Basecamp had put the map. It turned out to be in a Map folder not the Garmin folder as advised in other posts. Deleting the version Basecamp had added, and moving the version I had downloaded from the Garmin to the Map folder seems to have fixed it.</p>
<p>So the answer seems to be: on this device download gmapsupp.img to a folder called Map, not to the Garmin folder.</p>
