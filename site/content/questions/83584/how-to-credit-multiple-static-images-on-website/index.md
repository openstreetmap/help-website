<p>Hi there, I am making a website with multiple motorsports calendars, for which I want to create track maps based on Openstreetmap data. The main calendar page will look similar to <a href="https://www.formula1.com/en/racing/2022.html">the Formula 1 calendar page</a> (The track maps will look similar as well). As far as I understood everything I have to give credit using "Map data © OpenStreetMap" linking to www.openstreetmap.org/copyright. I also read that I can give credit once for multiple images on one site. Am I allowed to put this credit at the bottom of the page or does it have to be directly below the image?</p>
<p>Thanks in advance.</p>
