<p>In my city, we have HOV (High-occupancy vehicle) lanes that you can only merge in and out of in certain sections. I would like to map this, but I can't find any way to map merging restrictions. There is no physical barrier, only paint/signs, so I'm quite certain that it should be mapped as a single way.</p>
<p>There are 5 total lanes, 1 HOV (2+) and 4 regular. I've mapped this as:</p>
<ul>
<li>highway=motorway</li>
<li>lanes=5</li>
<li>motor_vehicle:lanes=no||||</li>
<li>hov=designated||||</li>
<li>hov:minimum=2</li>
</ul>
<p>How do I map the no merging sections? How do I map the sections where you are allowed to merge?<br />
Bonus: there are signs that tell you to exit the HOV lane to make specific exits. Can I tag those as well?</p>

<p>I figured it out. There's a proposed key change:lanes that's pretty well used.<br />
<a href="https://wiki.openstreetmap.org/wiki/Proposed_features/change">https://wiki.openstreetmap.org/wiki/Proposed_features/change</a><br />
So additionally I'd tag the restricted sections as:<br />
change:lanes=no|not_left|yes|yes|yes</p>
<p>and the non-restricted sections as:<br />
change:lanes=yes|yes|yes|yes|eys</p>
