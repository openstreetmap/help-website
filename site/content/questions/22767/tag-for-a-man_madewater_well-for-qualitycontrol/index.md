<div class="answer-body">
<p>It's better to use 1 or 2 tags that actually describe what something is, in stead of putting numerous tags on something in the hope that anybody understands what you mean.</p>
<p>Simply <a href="http://taginfo.openstreetmap.org/search?q=water#values">browsing Taginfo for values containing "water"</a> found monitoring=water_quality on the second page. According to a simple search with <a href="http://overpass-turbo.eu/s/eK">Overpass Turbo</a>, this is mostly used as a secondary tag with man_made=monitoring_station</p>
