<div class="answer-body">
<p>Despite the name, OsmAnd is not officially part of OpenStreeetMap. OsmAnd does use maps based on OpenStreetMap, but it is a separate project. So it may be more useful to check the guides on the <a href="http://www.osmand.net/">OsmAnd website</a>, or ask on their forum.</p>
<p>Anyway, to answer your question: For the "Show GPS Status" option to work you need to install the <a href="https://play.google.com/store/apps/details?id=com.eclipsim.gpsstatus2">GPS Status and Toolbox app</a>. This is a separate app, available to download for free on Google Play. Then when you choose the Show GPS Status option in OsmAnd, it will show this app.</p>
