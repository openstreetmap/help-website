<p>Even if the government provides data there are usually multiple reasons why we can't or don't want to use it:</p>
<ul>
<li>It might not be compatible with our license</li>
<li>The accuracy might be poor</li>
<li>Most importantly: It has to be <a href="https://wiki.openstreetmap.org/wiki/Import">imported</a> by following the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">import guidelines</a>, requiring extensive planning and consultation with the local community and <em>a lot</em> of <em>manual</em> work to merge it with the already existing data without creating duplicate information or destroying existing data of higher quality</li>
</ul>
