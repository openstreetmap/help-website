<div class="answer-body">
<p>What URL is the Landsat WMS using? I think the server JOSM has as default is <a href="http://onearth.jpl.nasa.gov"><code>http://onearth.jpl.nasa.gov</code></a>, but it seems NASA are no longer providing a WMS service for Landsat. If I copy that URL into my browser, it gives an error "This server no longer provides full WMS services!".</p>
<p>In the JOSM WMS/TMS preferences, I have an option named <code>Landsat (mirror)</code>. It is using a different URL ( <code>wms:</code><a href="http://irs.gis-lab.info/?layers=landsat&amp;"><code>http://irs.gis-lab.info/?layers=landsat&amp;</code></a> ), and it seems to be working fine for me.</p>
