<div class="answer-body">
<p>Use the <a href="https://wiki.openstreetmap.org/wiki/Tag">tag</a> key prefix <a href="https://wiki.openstreetmap.org/wiki/Key:demolished:"><code>demolished:</code></a> . E.g. instead of <code>building=yes</code> use <code>demolished:building=yes</code> for the building osm object.</p>
</div>
