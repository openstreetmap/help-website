If I click now on "edit with Potlatch 2" I see the last place I have edited in Potlatch but not the place where I have just added the GPS trace.</p>

<div class="answer-body">
<p>If you change your default editor from Potlatch 1 to Potlatch 2 using "My Profile" / "My Settings" / "Preferred Editor", "edit" will by default open your GPS trace in Potlatch 2, where you'll see that (and can view any other traces you want to), and also be able to see the Bing background imagery which isn't available in Potlatch 1.</p>
<p>There's also a new editor that you can try called "iD".</p>
