<p>You can set up your own WMS server with our free data (using Free Software like e.g. Apache with mod_mapnik_wms, or Mapserver). The WMS can project OSM data into any projection.</p>
<p>If you would like to use a WMS server run by someone else, see <a href="https://wiki.openstreetmap.org/wiki/WMS#OSM_WMS_Servers">https://wiki.openstreetmap.org/wiki/WMS#OSM_WMS_Servers</a> for options.</p>
