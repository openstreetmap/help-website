<div class="answer-body">
<p>Speed limits in OpenStreetMap are entered by the many volunteers that update the map data. Mostly that will be done based om what has been observed riding along a road. Over time speed limits might have changed. If no limits are mapped apps might assume default limits.</p>
<p>You can start correcting wrong limits yourself. Give it a try. Zoom to the offending road on <a href="https://www.openstreetmap.org">www.openstreetmap.org</a>, create a user account, hit the <em>edit</em> button on the top left and start editing. The editor is quite self explanatory and offers a new user tutorial. But should you still have questions come back here for further help.</p>
