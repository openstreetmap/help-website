<div class="answer-body">
<p>The first description seems to come from the internal translation of <a href="https://github.com/openstreetmap/openstreetmap-website/blob/master/vendor/assets/iD/iD/locales/de.json">editor iD</a>. It says "wide bends" while the second description from the wiki says "tight bends". This is an issue that should be addressed at <a href="https://github.com/openstreetmap/iD,">https://github.com/openstreetmap/iD,</a> not here.</p>
<p>However there is another contradiction. The English wiki describes mtb:scale=0 and mtb:scale=1 with wide bends while the German wiki describes mtb:scale=1 as tight bends. This is a general problem with translations having different meanings and possibly being outdated.</p>
<p>help.openstreetmap.org is not suited for such discussions. Better address this issue at the <a href="http://forum.openstreetmap.org/">forum</a> or <a href="https://wiki.openstreetmap.org/wiki/Mailing_lists">tagging mailing list</a>.</p>
