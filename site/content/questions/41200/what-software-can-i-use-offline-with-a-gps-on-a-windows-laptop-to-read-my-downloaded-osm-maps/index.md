<p>I do not use any myself, but see the older question <a href="/questions/5374/">offline-map-software-for-windows</a> (some support GPS).</p>
<p>Regarding your oziexplorer: After I had a quick look at <a href="https://wiki.openstreetmap.org/wiki/Oziexplorer">https://wiki.openstreetmap.org/wiki/Oziexplorer</a> it seems that it only can display raster maps. Likely you have downloaded <span>raw OSM data</span> (from which URL exactly? which files do you have?) which you cannot use directly with oziexplorer (needs <span>rendering</span> before).</p>
