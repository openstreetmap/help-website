<div class="answer-body">
<p>Nominatim is a geocoder. It is designed to convert addresses into locations (geocoding) and locations back into addresses (reverse geocoding).</p>
<p>Nominatim can handle some basic POI searches via <a href="https://wiki.openstreetmap.org/wiki/Nominatim/Special_Phrases">special keywords</a>, like "bakery near Altstadt, Dresden". Yet there might be better suited tools for your job, like <a href="https://wiki.openstreetmap.org/wiki/Overpass_API">Overpass API</a> where you can search for specific tags. See the <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Language_Guide">language guide</a> for various examples.</p>
