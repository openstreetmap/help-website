<div class="answer-body">
<p>You can adjust the scale on the export tab; enter a scale of 1:3000 and you get an export with house numbers. However using such a detailed scale will only be possible for relatively small areas too. This is because producing very large images consumes a lot of computing time on the server. What you can do is download the data, and then create the map from that yourself using suitable rendering software; there are many questions and answers about this here (search for "render own map" and similar).</p>
