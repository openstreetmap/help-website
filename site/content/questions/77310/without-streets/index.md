<p>It is completely possible to render an OSM data based map without streets. However the OSM data model does not have a layer abstraction and it would be a good idea to read up on some of the basics before progressing further. See for example <a href="https://wiki.openstreetmap.org/wiki/Elements">https://wiki.openstreetmap.org/wiki/Elements</a></p>
