<div class="answer-body">
<p>You can drag the start point over the other way, then press "J".</p>
<p>Or click on the start point to start extending that way, then hover over the way you want to join to, and you will see a little "+" icon, click there and the ways will be joined. You can go back and delete the original start point if it is unnecessary now.</p>
<p>In the future, when you start creating a trail which starts at a junction with an existing trail, select that way, then shift click on an existing point to create a branching way.</p>
<p>If there isn't a convenient point, you can create one by selecting the way and shift-clicking where you want it. You can also use this to add extra points and increase precision.</p>

<p>Hi, theres another way. An excisting route or road has blue dots. The selected (Made) one has red dots. Select the excisting route (it becomes yellow) and add by using shift and enter at the same time. Youll end up with a new dot (red), drag your own point (dot) over the new one and touch 'j' Youl get an joined at the bottom of your screen, control by dragging the new point to and from.
