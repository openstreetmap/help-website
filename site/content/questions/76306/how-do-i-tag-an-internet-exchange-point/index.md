<p>Who says it is not a primary tag? Osmose? Then you can safely ignore it I would say.</p>
<p>But it might be appropriate to add a <code>building=yes</code> or <code>building=service</code> tag if you are tagging the complete building. If you are just adding an area inside a building I would leave it with the <code>telecom=*</code> as the main tag. Also have a look at the <a href="https://wiki.openstreetmap.org/w/index.php?oldid=1687727#How_to_map">proposal page for the usage</a>.</p>
