<p>How can I add a building to OSM without the assistance of satellite images?</p>
<p>There's logs of great tutorials on the Internet that show how to add a building to OSM by tracing its outline from satellite images on openstreetmap.org -- but how can I do it <em>without</em> satellite images?</p>
<p>In my case, the satellite images are useless because they're blurry or there's a giant cloud in the atmosphere that's obstructing the view of the ground.</p>
<p>Is there some way for me to, with a GPS-enabled smartphone, literally walk to each corner of the building and tap an "add node" button until I've returned to the first node? And then use the corresponding node's bounded area as the building?</p>
<p>Or is there some way for me to fly a drone on a clear day to take a photo -- then somehow overlay that on-top of the OSM data and use that photo (instead of the satellite image) to add draw areas around buildings?</p>
<p>What options do I have to add a building to OSM without satellite imagery?</p>

<p>You should be able to use a GPS device to get the coordinates of the building corners. I suspect that unless you have a professional GPS or do way point averaging over a few days that the accuracy will be low. But it should work.</p>
<p>I don't have the link, but I saw a write up recently on making a background aerial image tile set from GPS aligned drone video. I was envious of the results: Sharp, clear and totally up to date. Almost makes me want to get a drone and figure out how to use it. My quick websearch isn't picking up the article I read but I think that <a href="https://en.wikipedia.org/wiki/OpenDroneMap">OpenDroneMap</a> was part of the tool chain.</p>
<p>While searching for the above, I found a OSM diary entry with another possible method: <a href="https://www.openstreetmap.org/user/tordans/diary/401804">Scalable Aerial Imagery Generation from Phone Lidar and 360° Point Clouds</a></p>

<p>I have drawn missing buildings by taking a few pics for reference. I then just estimate position and size using the reference photos. You can use a number of GPS way points that are aligned with the projected building sides. I did a diary entry that may give a few ideas. see <a href="https://www.openstreetmap.org/user/andy%20mackey/diary/38774">https://www.openstreetmap.org/user/andy%20mackey/diary/38774</a> note using the existing Bing (or other) images can show the footprint of the building plot.</p>
