<div class="answer-body">
<p>The easiest way to get it on the map is probably to add it yourself.</p>
<p>OSM only maps physical presences, so from your website it would appear that that is just the warehouse?</p>
<p>If you allow walk in trade to make purchases then it might be <a href="https://wiki.openstreetmap.org/wiki/Tag:shop%3Dflooring"><code>shop=flooring</code></a>, if not then it is probably just <a href="https://wiki.openstreetmap.org/wiki/Tag:office%3Dcompany"><code>office=company</code></a> and <a href="https://wiki.openstreetmap.org/wiki/Tag:industrial%3Dwarehouse"><code>industrial=warehouse</code></a>.</p>
<p>A <a href="https://wiki.openstreetmap.org/wiki/Addresses">address</a> and contact details would be useful too. Ad copy is not generally considered verifiable information and is usually removed.</p>
<p>Most editing software has a good set of presets so you don't need to add raw tags for most things but most allow you to add custom tags. In the "default" iD editor on OpenStreetMap.org these can be added below the preset on the left panel.</p>
<p>I'd say have a go with the default editor, try to be as precise as possible with the positioning of items. For the most part I think it's fairly straightforward, but learnosm.org give a good step by step introduction if you prefer to learn the software that way.</p>
