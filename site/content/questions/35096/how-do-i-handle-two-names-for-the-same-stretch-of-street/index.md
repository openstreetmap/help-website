<p>Hi,</p>
<p>in Ireland I occasionally encounter long roads/streets which have the same name for several kilometres but which have additional names (usually ... Terrace") for short stretches, usually between neighbouring side streets.</p>
<p>I looked through the help section but couldn't find anything on such a situation.</p>
<p>Example:</p>
<p><a href="https://www.openstreetmap.org/history#map=18/52.06111/-9.50599">https://www.openstreetmap.org/history#map=18/52.06111/-9.50599</a></p>
<p>The R876 is called Lewis Road for quite a way. Additionally, there are street signs calling the bit just south of Sunnyhill lower "York Terrace", the bit to the north "Emmet's Terrace".</p>
<p>I've no idea how to handle this.</p>

<div class="answer-body">
<p>If a road has two names, tag the less common name using <em>alt_name</em>.</p>
<p>To tag only a section of a road, you need to split the osm way first (using the scisors on a node in iD or "p" in JOSM... can't recall how this is done in Potlach2 but it shouldn't be hard to find).</p>
