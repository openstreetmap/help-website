<div class="answer-body">
<p>It is not fully clear for which tags you are asking, but I will try:</p>
<p>The main tags could be based on the <span><code>abandoned:</code></span> (or disused:, depending on the state of the structure) prefix. So, if it was previously a <code>man_made=chimney</code>, I would suggest to use <code>abandoned:man_made=chimney</code>. An <span>example</span>.</p>
<p>If you want to tag the operator (a industrial monument conservation organization or the city government?) of this old chimney, then use the <code>operator</code> tag as usual, there is nothing special about this operator. If it is still a building, then I would use <code>building=industrial</code>, too.</p>
<p>Somehow similar to <span>this old quarry</span> which is now a protected area.</p>
