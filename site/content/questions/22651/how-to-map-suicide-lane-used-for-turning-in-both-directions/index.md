<p>For those outside of the US, here is a link describing how it is used and another with some images about the "center turn lane": <a href="https://www.dmv.ca.gov/pubs/hdbk/traffic_lanes.htm#cltl">https://www.dmv.ca.gov/pubs/hdbk/traffic_lanes.htm#cltl</a> <a href="https://encrypted.google.com/search?q=center+turn+lane&amp;client=ubuntu&amp;hs=c5L&amp;channel=fs&amp;tbm=isch&amp;tbo=u&amp;source=univ&amp;sa=X&amp;ei=paueUbbuFaLCigL0q4HgCg&amp;ved=0CDYQsAQ&amp;biw=1751&amp;bih=850">https://encrypted.google.com/search?q=center+turn+lane&amp;client=ubuntu&amp;hs=c5L&amp;channel=fs&amp;tbm=isch&amp;tbo=u&amp;source=univ&amp;sa=X&amp;ei=paueUbbuFaLCigL0q4HgCg&amp;ved=0CDYQsAQ&amp;biw=1751&amp;bih=850</a></p>
<p>I think they parallel ways in the original poster's example should be replaced by a single way as there is no median, curb or barrier to keep the traffic separate. And that way should have a tag of "centre_turn_lane=yes". It looks like that is already used even if not widely, see <a href="http://taginfo.openstreetmap.us/keys/centre_turn_lane#overview">http://taginfo.openstreetmap.us/keys/centre_turn_lane#overview</a></p>
<p>I checked a few highways in my area to see how that was tagged and found that it is not tagged at all but in each case there was only one way as I would expect.</p>
