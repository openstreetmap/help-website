<div class="answer-body">
<p>You can download the Ukraine extract from the Geofabrik download server and then use any extract tool of your choice to count how many <code>name:en</code> and <code>name:ru</code> tags there are. (You can even download the .osm.bz2 file and then use bzgrep.) However, where people have (only) put a name in the <code>name</code> tag simply, it will be more difficult to analyse which language that is in.</p>
