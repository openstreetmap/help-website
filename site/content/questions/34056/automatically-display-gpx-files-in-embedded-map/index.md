<div class="answer-body">
<p>Adding GPX files to the embedded map obtained via the share button won't be possible without major modifications. But you can embed your map <a href="http://switch2osm.org/using-tiles/">the usual way</a> using <a href="http://switch2osm.org/using-tiles/getting-started-with-leaflet/">Leaflet</a> or <a href="http://switch2osm.org/using-tiles/getting-started-with-openlayers/">OpenLayers</a> (it seems like the embedded map also uses Leaflet).</p>
<p>As you mentioned Wordpress, there is a <a href="https://wiki.openstreetmap.org/wiki/Wordpress">wiki page about Wordpress</a> which lists some other possible solutions and plugins.</p>
