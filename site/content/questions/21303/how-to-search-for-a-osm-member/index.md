<div class="answer-body">
<p>Searching the help site will only return users that have registered for the help site. If you search for "SomeoneElse" it'll find me, but if you search for "SomeoneElse2" it won't (even though that's a valid ID that has edited).</p>
<p>I'd instead try <a href="http://textual.ru/whosthat/">Who's That</a>. As well as being able to find people by name it also does case insesitive searching.</p>
