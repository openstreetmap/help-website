<div class="answer-body">
<p>If I understand this correctly you want to sit in front of a computer and know where the trucks are.<br />
This has three tasks:</p>
<ol>
<li>Some tracking device in the vehicle must send its location repeatedly (SMS, internet)</li>
<li>Some server has to accept these messages and keep track of the vehicles</li>
<li>You need to display it.</li>
</ol>
<p>It shoudld not be hard to find some application for the first point (I am not sure if OSMtracker can do that). Third point is where OSM comes - OpenLayers might be what you are looking for.<br />
Your main problem will be number 2. I do not know about any out of th box solution for this, but it should not be too hard to create.</p>
