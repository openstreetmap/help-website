<p>Don't laugh, but on Windows 10 you may be able to simply use <a href="https://switch2osm.org/manually-building-a-tile-server-16-04-2-lts/">https://switch2osm.org/manually-building-a-tile-server-16-04-2-lts/</a> to generate local map tiles by <a href="https://www.howtogeek.com/249966/how-to-install-and-use-the-linux-bash-shell-on-windows-10/">installing Ubuntu on it</a>. I've not tried it, and as Windows 10 has a different filesystem to Linux it's not guaranteed to work, but it's worth a try.</p>
<p>If that doesn't work then a <a href="https://wiki.openstreetmap.org/wiki/User:SomeoneElse/Ubuntu_1604_tileserver_load">Linux VM</a> definitely will.</p>
<p>However that may be more work than you need just to "view an offline map" - there are some options <a href="https://wiki.openstreetmap.org/wiki/Using_OpenStreetMap_offline">in the wiki</a>.</p>
