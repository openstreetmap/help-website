<div class="answer-body">
<p>Displaying a map in an application has two sides: A legal side ("have you cleared the intellectual property rights to use this data") and a technical side ("is there a server that you can load the map from").</p>
<p>With OpenStreetMap, on the legal side it's totally free (under the ODbL license) and you are allowed to use it commercially.</p>
<p>On the technical side there are different providers of OpenStreetMap map tiles that you can consume in your app. Some are commercial and will charge you money to access tiles; the donation-funded www.openstreetmap.org server has a <a href="https://operations.osmfoundation.org/policies/tiles/">tile usage policy</a> that may change at any time; while limited commercial use is not discouraged, there is no promise of free commercial use for life.</p>
<p>You can also set up your own tile server using OpenStreetMap data, making you independent of third-party providers. Documentation about this is to be found on www.switch2osm.org.</p>

<div class="answer-body">
<p>The data provided by OSM is open source: You copy and use it all you want with no royalties or licensing. If you are modifying the data and redistributing, then you will have licensing issues, but it sounds like what you need are map tiles.</p>
<p>Map tiles are provided <a href="https://wiki.openstreetmap.org/wiki/Acceptable_Use_Policy">pretty much for demonstration purposes</a> and commercial use of those is not allowed. But instructions exist on how you can set up your own map server using OSM data. If you do that then you have no licensing and can even customized the appearance if desired. If that is beyond you desires, you can contract with a number of commercial operators who <a href="https://wiki.openstreetmap.org/wiki/Rendering">provide map tiles based</a> on OpenStreetMap data.</p>
<p>I suggest you start looking a the wiki page about <a href="https://wiki.openstreetmap.org/wiki/Introduction_to_OSM_for_Business_Users">business use of OSM</a>.</p>
