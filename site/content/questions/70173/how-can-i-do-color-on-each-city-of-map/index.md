<p>The cities seem to be already present in OpenStreetMap which is a good thing: <a href="http://overpass-turbo.eu/s/L2e">Overpass Turbo</a></p>
<p>Then it depends what you want to do exactly. You can extract the data from Overpass Turbo and use it in other tools. If you want the regions to be overlayed over a map you could use <a href="https://umap.openstreetmap.fr">uMap</a>. But you could also try to import data into a graphics program. Is this a one time task or do you need to recolor periodically or even dynamically?</p>
