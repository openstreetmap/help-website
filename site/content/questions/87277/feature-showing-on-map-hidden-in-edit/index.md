<div class="answer-body">
<p>It's been added twice - once as an <a href="https://www.openstreetmap.org/way/1068164496">area</a> and once as a <a href="https://www.openstreetmap.org/node/255010160">point</a>. To select the area, you'll need to click nearer the outer edge.</p>
<p>However. the area has more detail on it, so you won't want to delete that. To delete the node, just click on it and delete it.</p>
<p>To find both of these I used the <a href="https://www.openstreetmap.org/query?lat=51.58226&amp;lon=-0.11986#map=20/51.58224/-0.11979&amp;layers=H">query</a> button on the main OSM website.</p>
