<p>There appears to be a bug in the openstreetmap-website code, where it uses the API Keys correctly in the maps in most places they are used on openstreetmap.org, but not in the embed maps, which uses its own javascript.</p>
<p>I've <a href="https://github.com/openstreetmap/openstreetmap-website/issues/1293">opened an issue</a> and hopefully it'll be fixed.</p>
