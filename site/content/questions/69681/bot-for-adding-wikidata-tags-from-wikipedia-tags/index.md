<div class="answer-body">
<p>This is something that has been done in the past but it was highly contentious to the point of some of these mass updates having to be reverted outright because they would introduce too many errors.</p>
<p>It is not something that should be done automatically. Instead, a human being should verify whether a Wikidata entry really describes the same object as the OSM object, and only then add the Wikidata tag.</p>
