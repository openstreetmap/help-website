<p>Based on the Wikidata ID, it's this one: <a href="https://www.openstreetmap.org/way/60325575">https://www.openstreetmap.org/way/60325575</a></p>
<p>Someone edited it back in January and removed the <code>natural=water</code> tag, replacing it with <code>area=yes</code>. At that point, there weren't any tags indicating it was a lake (or really any kind of feature). I've changed the tag back, so the lake should show up again soon.</p>
