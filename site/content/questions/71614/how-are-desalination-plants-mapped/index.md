<div class="answer-body">
<p>I think the most common one is to simply tag it <code>man_made=water_works</code> and then the name of it usually includes the word "desalination" anyway, making it clear.</p>
<p>Imho the most correct would be <code>man_made=desalination_plant</code> that's been done in 17 places.</p>
<p>or maybe <code>water_works=desalination</code></p>
<p>could be combined with landuse=industrial e.g. <a href="https://www.openstreetmap.org/way/84258438">https://www.openstreetmap.org/way/84258438</a></p>
<p>17 - <a href="https://taginfo.openstreetmap.org/tags/man_made=desalination_plant">man_made=desalination_plant</a> | 2 - <a href="https://taginfo.openstreetmap.org/keys/water_works#values">water_works</a></p>
