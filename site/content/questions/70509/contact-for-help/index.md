<div class="answer-body">
<p>Everyone can download small amounts of tiles remotely from the OSM server (this is how the openstreetmap.org web site works). Downloading larger amounts of tiles is discouraged as per the <a href="https://blog.openstreetmap.org/2011/11/01/tile-usage-policy/">Tile Usage Policy.</a></p>
<p>If you need a lot of tiles, you will have to generate them yourself - tools like Maperitive.net (for smaller areas) or Mapbox Studio Classic can be used to produce tiles, or you could set up your own tile server as documented in <a href="https://www.switch2osm.org">www.switch2osm.org</a>. If you would rather have someone else provide this service to you, check out the service providers on the wiki page <a href="https://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services">Commercial OSM Software and Services.</a></p>
