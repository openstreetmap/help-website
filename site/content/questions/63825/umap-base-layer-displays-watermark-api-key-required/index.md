<p>The layer labeled "OSM Outdoors (Thunderforest)"?</p>
<p>These tiles are provided by <a href="https://www.thunderforest.com/">Thunderforest</a>. Thunderforest requires an API key. This has nothing to do with OSM in particular.</p>
<p>However, for me this uMap layer doesn't show a watermark. After inspection of the tile URLs I can see that uMap is already providing an API key.</p>
<p>Is this a new uMap project? Can you try selecting a different base layer and then switch back? Does "OSM OpenCycleMap" (also hosted by Thunderforest) also show this watermark?</p>
