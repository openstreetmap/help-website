<div class="answer-body">
<p>Es ist weder dein Fehler noch Bug von deinem Editor (Merkaartor). Es dauert von ein paar Sekunden bis mehr Stunden bis alle Änderungen in allen Zoomlevels korrekt dargestellt werden. Meistens ist es nötig die Browser Cache zu leeren, damit alle Teile vom OSM-Server neu geladen werden.<br />
Mit an die Teil-URL angehängtem <code>/status</code> bzw. <code>/dirty</code> kann man auch sehen wie alt das Teil ist, ob es für Rendering geplant ist bzw. ein neues Rendering verlangen.</p>
<p>Ein Teil (siehe auch <a href="/questions/54737/requesting-re-render-of-a-tile">wie man diese URL herausbekommt</a>): <a href="http://a.tile.openstreetmap.org/12/2236/1403.png">http://a.tile.openstreetmap.org/12/2236/1403.png</a><br />
Status: <a href="http://a.tile.openstreetmap.org/12/2236/1403.png/status">http://a.tile.openstreetmap.org/12/2236/1403.png/status</a></p>

<div class="answer-body">
<p>Es gibt nicht nur eine "OSM-Karte", sondern es werden aus unseren Rohdaten viele Karten (je nach Anwendungsfall) erstellt. Jede Karte für sich wird verschieden-häufig aktualisiert. Weiterhin ist es meist so, dass jeweils die <em>Zoomstufen und Kartenteile/kacheln</em> (die rechteckigen Teilbereiche) <em>für sich getrennt aktualisiert</em> werden (die größten Zoomstufen meist am schnellsten, bei openstreetmap.org typischerweise innerhalb von weniger als 5 Minuten). Also: <em>einfach abwarten</em> (man kann es eventuell manuell beinflussen/schneller machen, siehe <a href="/questions/8202/unterschiedliche-darstellung-abhangig-vom-zoomfaktor/8203">LM_1s Antwort</a>). Wieso das alles? Weil das Neuberechnen von Kartenkacheln unsere Server viel Rechenleistung kostet und daher versucht wird Neuberechnungen auf das nötige Minimum zu reduzieren.</p>
<p>Weiterhin kann dein Browser so eingestellt sein (das ist Standard), dass er Bilder zwischenspeichert, um ein Bild nicht erneut abrufen zu müssen, wenn du eine Seite/Bild erneut besuchst. Für osm.org-Kartenkacheln scheint zur zeit ein Wert von ca. 16 Stunden vorgegeben. Danach läd dein Browser das Bild neu, wenn du die Seite/das Bild wieder besuchst. Du kannst den Zwischenspeicher ("Cache") in den meisten Browsern auch manuell leeren oder eine einzelne Seite und ihre Bilder unter Umgehung des Zwischenspeichers neuladen (oft mit Strg+F5 möglich).</p>
