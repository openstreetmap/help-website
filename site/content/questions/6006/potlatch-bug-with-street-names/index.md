<p>I suspect that what would be required here would be a series of steps to reproduce the problem. Once you've got that, head over to <a href="http://trac.openstreetmap.org/">trac</a>, make sure that no-one else has already reported it, and log it there against the "potlatch2" component. Without steps the reproduce, the best the developers will be able to say is "works for me!" and carry on fixing other reported issues for which more information has been supplied.</p>
<p>It's possible, of course, that while you're figuring out exactly what you're doing before the problem occurs you might spot that e.g. you're inadvertently selecting lots of ways any applying a name tag, which would cause exactly the effect that you're describing.</p>
