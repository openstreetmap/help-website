<div class="answer-body">
<p>The most easy way if you only want the map on your GPS and not inside any Garmin software on your Mac: do <em>not</em> download the file for "MacOS", but instead osm_generic_gmapsupp.zip. See <span>Garmin.OpenStreetMap.nl:Manual#Installing</span>. Also see <span>OSM_Map_On_Garmin#Installing_the_map_onto_your_GPS</span> (you can usually just rename existing files, while keeping the file name extension).</p>
<p>To install the "MacOS" download you need to use <span>BaseCamp</span> (a Garmin software), which may be the more complicated way.</p>
</div>
