<p>Buenas Noches, mi duda es la siguiente: Estoy desarrollando un sistema localizador GPS, el cual pretende ser similar a muchos sistemas como el de control de flotas vehiculares, utilizando un dipositivo GPS y la erd GPRS para el envío y recepción de SMS. Pretendo vender el dispositivo y de esta forma, que el usuario necesite ingresar a un sitio web, en donde tiene quer registrarse y por ejemplo, el ingresará las coordenadas que recibió en su celular y el sistema le dirá su ubicación en el mapa. Como esta idea aun está en una etapa de prototipo, mi consulta es ¿Se puede utilizar el servicio de OpenStreetMap para realizar pruebas del prototipo? ¿Es necesaria alguna licencia de OSM si yo vendo los dispositivos y estos vienen con un ID el cual le permitirá al usuario ingresar al sistema para utilizar los servicios de localización en los mapas? Espero sus respuestas a mi correo. Gracias de antemano.</p>
