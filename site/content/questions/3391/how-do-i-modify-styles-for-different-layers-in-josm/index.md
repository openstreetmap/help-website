<p>I was able to modify line width by creating a map paint style involving an invented tag to be used only for display purposes when editing. Lacking xml skills, I did this:</p>
<p>download existing map paint style modify &lt;rules&gt; so that my invented tag paints a thick line for invented key/value condition modify name, author, etc. save xml file locally load local file as an active map paint style add invented tag to apply map style to objects. edit delete invented tag before upload - important</p>

<ul>
<li>Hit F12 in JOSM &gt; choose third tab (map projection &amp; styles) &gt; choose MapPaint-Styles</li>
<li><p>On the right side of the Dialog you will find a "+" that you can use to add URLs as well as files..</p></li>
<li><p>to activate the style use the checkbox on the left of loaded styles in the same dialog</p></li>
<li>or use MapPaint dock window in JOSM</li>
</ul>
