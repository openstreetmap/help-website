<div class="answer-body">
<p>I don't see anything wrong with using leisure=pitch + sport=basketball. Whether it's used exclusively for training or for playing an informal game with modified rules to support only one basket, it's still an area that's used for playing the sport of basketball.</p>
<p>Edit: To expand on my answer, <a href="https://wiki.openstreetmap.org/wiki/Tag:sport%3Dbasketball">the wiki page for sport=basketball</a> mentions the additional tag hoops=*, which can be used to define the number of hoops. Approximately 20% of <a href="https://taginfo.openstreetmap.org/keys/hoops#values">the objects with this tag</a> have it set to hoops=1, so many mappers are mapping single-hoop areas.</p>
