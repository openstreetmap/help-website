<p>It takes a while for new objects to appear on the map. But the shop <a href="https://www.openstreetmap.org/node/8649160147#map=18/4.62760/-74.07242">is there</a> in the data.</p>
<p>The problem in your case is that there are many shops close to each other. Not all of them can be displayed at once. The renderer needs to chose which to display and which to hide. On openstreetmap.org I see the new shop as an icon at zoom level 18. If you don't see it yet try reloading the page in your browser (possibly F5 or Ctrl+F5).</p>
