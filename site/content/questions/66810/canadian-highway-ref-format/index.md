<p>From a brief look at the current map, no. None of the highways seems to have the network code.</p>
<p>If you want to change this, it is best to discuss that with the local community, instead of a globel help forum like this. There is a mailing list for the Canadian community: <a href="https://lists.openstreetmap.org/listinfo/talk-ca">talk-ca</a>, but I don't know whether that is the prefered communication channel for them.</p>
