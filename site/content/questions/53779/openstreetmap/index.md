<p>Ik ga ervan uit dat je een kaart via garmin.openstreetmap.nl wil downloaden. Er zijn al een tijdje problemen met die site, maar de eigenaar is moeilijk te contacteren. Gelieve een van de alternatieven op <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download</a> te gebruiken.</p>
<p>zelf ben ik tevreden van <a href="http://extract.bbbike.org/">http://extract.bbbike.org/</a></p>
<p>English: I assume the user wants to download a map from garmin.openstreetmap.nl. This site has some problems for awhile, but the owners is hard to contact. Please use one of the alternatives on <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download</a></p>
<p>I am a happy user of <a href="http://extract.bbbike.org/">http://extract.bbbike.org/</a></p>
