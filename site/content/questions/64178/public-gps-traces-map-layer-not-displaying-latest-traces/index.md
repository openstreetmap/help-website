<div class="answer-body">
<p>This <em>may</em> shed some light on it: <a href="https://wiki.openstreetmap.org/wiki/Talk:Visibility_of_GPS_traces">https://wiki.openstreetmap.org/wiki/Talk:Visibility_of_GPS_traces</a></p>
<p>It says "Worth adding that the overlay map of GPS traces (in rainbow colours) is only calculated from time-to-time and is not automatically refreshed when traces are added." So I think the public GPS traces may only be added to the map when the developers do an updated release. It would be nice if a knowledgeable person would explain exactly when (meaning under what circumstances) we can expect to see traces we upload appear on the map.</p>
<p>Some traces I added for the Constellation Trails in the Prescott, AZ area have appeared, so I know it happens eventually. See <a href="https://www.openstreetmap.org/edit#map=15/34.6185/-112.4260">https://www.openstreetmap.org/edit#map=15/34.6185/-112.4260</a> and be sure to turn on OpenStreetMap GPS Traces in the map background.</p>
