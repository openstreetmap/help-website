<div class="answer-body">
<p>(ich habe das selbst noch nicht gemacht, aber ich hoffe es ist trotzdem hilfreich bzw. eine erste Klärung:)</p>
<p>Du hast zwei Möglichkeiten (die üblich sind):</p>
<ol>
<li>du bindest eine Karte in deine Website ein, die die Kreisgrenzen schon integriert hat
<ol>
<li>Dazu brauchst du eine solche Kartenquelle die als <a href="https://wiki.openstreetmap.org/wiki/Tiles">Tiles</a> verfügbar ist</li>
<li>du <a href="https://wiki.openstreetmap.org/wiki/DE:Erstelle_und_benutze_deine_eigene_Slippy-Map">bindest sie in deine Website ein</a></li>
</ol></li>
<li>du bindest eine Basiskarte in deine Website ein und legst die Kreisgrenzen obendrauf
<ol>
<li>dazu brauchst du eine Basiskarte, die möglichst leer ist und als Tiles verfügbar ist</li>
<li>und du brauchst die Kreisgrenzen in einem Format, was die Anzeigesoftware (nächster Schritt) versteht</li>
<li>du bindest die Karte samt Grenzen in deine Website ein. Z.B. <a href="https://wiki.openstreetmap.org/wiki/DE:UMap">umap</a> (basiert auf leaflet) oder <a href="https://wiki.openstreetmap.org/wiki/DE:Leaflet">leaflet</a> können sowas.</li>
</ol></li>
</ol>
<p>Sehr einfach ist das nicht. Das einfachste wird wohl umap mit einer Datenabfrage via <a href="https://wiki.openstreetmap.org/wiki/DE:Overpass_API">Overpass</a> sein.</p>
<p>Es liegt in der Natur der <a href="https://www.openstreetmap.org/copyright">frei verfügbaren</a> OSM-Daten, dass du fast alles machen kannst, was du dir vorstellen kannst. Allerdings könnte es häufig an Programmierkenntnissen (im weiteren Sinne) scheitern.</p>
</div>
