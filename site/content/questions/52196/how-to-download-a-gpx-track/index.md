<p>Unfortunately, there is no way that I know of to download a particular GPS trace that is visible on the OSM map. If you happen to find a trace you want on the <code>[GPS traces page][1]</code>, you can download it from there but there's no good way to search those traces for a particular area or keyword. Of course, you can download any of your own traces that you have previously uploaded to OSM.</p>
<p>Some other OSM guru may be able to add more help. I'll subscribe to this question and maybe learn something myself.</p>

<p>if the particular GPS track was (note: that's just my interpretation) <a href="https://wiki.openstreetmap.org/wiki/Visibility_of_GPS_traces">uploaded as</a> "Identifiable" or "public", then the OSM API <a href="https://wiki.openstreetmap.org/wiki/API_v0.6#Retrieving_GPS_points">returns</a> also an url with the set of points of this trace.</p>
<p>Most easy to use if you download the traces in <span>JOSM</span> (use a small area to preferably only get the trace you are interested in!), then look in the <a href="https://josm.openstreetmap.de/wiki/Help/Action/ChooseTrackVisibility">Choose Track Visibility</a> (screnshot there) window in the "url" column. Open that URL and you get to the OSM homepage view of this trace where you can download it in its entirety.</p>
</div>

<p>Thanks aseerel4c26, this was really useful. However, instead of opening the "Choose Track Visibility" dialog, once you know which track you need, it is better to open the "Info" dialog from the right-click context menu of the layer. The URLs displayed in the "Track Visibility" dialog cannot be selected/copied or clicked. In the "Info" dialog, you can at least select the text and copy it, and subsequently paste it in your browser to get to the actual track download.</p>
