<p>Hi,</p>
<p>I'm tagging a whole lot of buildings, for now the main building tags I use are:</p>
<pre><code>building=yes
building=house
building=hangar
building=garage(s)</code></pre>
<p>This works well but I'm wondering how to tag a small apartment building, not the skyscraper one but the smaller version like this one for example: <a href="http://farm3.static.flickr.com/2405/2469087289_0bc4c39c48_b.jpg"></a><a href="http://farm3.static.flickr.com/2405/2469087289_0bc4c39c48_b.jpg"></a><a href="http://farm3.static.flickr.com/2405/2469087289_0bc4c39c48_b.jpg">http://farm3.static.flickr.com/2405/2469087289_0bc4c39c48_b.jpg</a></p>
<p>I'm using</p>
<pre><code>building=apartment</code></pre>
<p>but mapnik renders it the same as building=yes, is there a convention for small aparment buildings like this one?</p>
