<div class="answer-body">
<p><span><code>barrier=bollard</code></span> is enough. But <span>since</span> "[the] bollard=* tag describes a barrier=bollard in greater detail" you can use <code>bollard=non-removable</code> or <code>bollard=irremovable</code> (which has already <span>5 usages</span>) in addition. I am not sure which one is the better English term.</p>
<p><span>Do not use the key <code>type</code></span> if possible.</p>
</div>
