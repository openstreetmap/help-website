<div class="answer-body">
<p>It would be nice to know whether you are stuck because you do not know how to start editing or because you do not find the shop=hat category.</p>
<p>Assuming you know how to edit with iD (the in-browser editor).</p>
<ol>
<li>Add a point</li>
<li>Search for "shop" on the left and select that.</li>
<li>Under "All tags" you will see "shop" and "yes". Replace the "yes" with "hat"</li>
<li>Add the name as with any other shop</li>
<li>save and add a comment on what you added and how you know the name and location of the shop (e.g. local knowledge, survey)</li>
</ol>
<p>So far there are only 68 shops tagged as hat, and 43 as hats (see <a href="https://taginfo.openstreetmap.org/keys/shop#values">taginfo</a>). OSM normally uses the singular form. Therefore there won't be a dedicated icon for the shop and you might not see it on the rendered map afterwards.</p>
