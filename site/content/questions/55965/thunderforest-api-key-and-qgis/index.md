<div class="answer-body">
<p>I'm not a QGIS user but there is a <a href="https://gis.stackexchange.com/questions/239773/add-an-api-key-to-qgis-for-qosm-layer/239827">similar question at gis.stackexchange.com</a>. I'm quoting the answer from <a href="https://gis.stackexchange.com/users/9159/andrej">AndreJ</a> here:</p>
<blockquote>
<p>I guess you are using the QOSM plugin to display the OpenCycleMap, since the standard OSM map does not have API keys.</p>
<p>You can obtain an API key from the map producer Thunderforest. They provide details <a href="http://www.thunderforest.com/docs/apikeys/">here</a>, including a sample tile request URL. You can add that to the properties of the QOSM layer, inserting the line</p>
<pre><code>http://a.tile.thunderforest.com/cycle/${z}/${x}/${y}.png?apikey=&lt;insert-your-apikey-here&gt;</code></pre>
<p>and clicking on <code>Add Custom Type</code>. Take care of the grammar for the variables that differs from the linked site.</p>
<p>Alternatively, you can use the QuickMapServices plugin, which does not show an API request for that site at the moment.</p>
</blockquote>
