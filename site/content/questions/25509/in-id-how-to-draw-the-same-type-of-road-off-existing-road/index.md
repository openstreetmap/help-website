<p>In iD editor, a very common situation. A large parking lot with lot of service roads/parking_aisles. I draw one line, select service road and then parking_aisle. From this I want to draw another parking_aisle. I click on a node in the existing parking_aisle but the new line is drawn as a regular line so I have to again pick service road and then parking_aisle.</p>
<p>Is there a way when drawing off the existing type of road that the new road will automatically be the same type of road? Sometimes it does work like this but I cannot figure out how to do it in every time. Thank you.</p>
