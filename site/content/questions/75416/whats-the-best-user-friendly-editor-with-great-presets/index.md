<div class="answer-body">
<p>You can actually use JOSM with presets it is not totally obvious but it is essentially an extra click on the icon in the tag key/value form. I would (but I'm biased) suggest using <a href="https://github.com/simonpoole/beautified-JOSM-preset">the Vespucci presets</a> instead of the default JOSM ones, but that's probably your best bet if you can't get use to iD.</p>
<p>To install in JOSM: copy JOSM download link from <a href="https://simonpoole.github.io/beautified-JOSM-preset/">https://simonpoole.github.io/beautified-JOSM-preset/</a> , start JOSM -&gt; Presets -&gt; Preset preferences -&gt; + sign on right hand side</p>
<p>Add the URL and a name to the popup, disable the default preset.</p>
