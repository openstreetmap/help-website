<div class="answer-body">
<p>There are many ways to skin a cat.</p>
<p>Limiting this to a web app boils down to either using raster tiles (as on openstreetmap.org) or vector tiles (which contain geometry information that is locally used to produce an imager in your browser).</p>
<p>There are no really supported ways to "just download" a bunch of image tiles (it can be done, but ...) so I would suggest first seeing if vector tiles will work for you, see for example <a href="https://openmaptiles.org/">https://openmaptiles.org/</a></p>
