<div class="answer-body">
<p>No, currently they are <a href="http://stefanosabatini.eu/doesitrender/#piste:type">not rendered</a> on the standard layer. But according to the <a href="https://wiki.openstreetmap.org/wiki/List_of_OSM_based_Services">list of OSM-based services</a> there is <a href="https://wiki.openstreetmap.org/wiki/OpenPisteMap">OpenPisteMap</a> which <a href="http://openpistemap.org/">does show</a> objects with these tags.</p>
