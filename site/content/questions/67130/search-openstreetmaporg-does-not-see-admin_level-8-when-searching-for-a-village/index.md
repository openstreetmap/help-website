<p>The search returns the admin centre of that relation, which is a node named Вереси (<a href="https://www.openstreetmap.org/node/337524166).">https://www.openstreetmap.org/node/337524166).</a></p>
<p>The relation itself is not called that (the name is Вересівський Старостинський Округ), so one would not expect the search to return the relation as a result.</p>
