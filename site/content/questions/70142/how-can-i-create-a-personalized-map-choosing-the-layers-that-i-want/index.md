<h4 id="overpass-turbo">Overpass turbo:</h4>
<p>Overpass turbo is a web based data mining tool for OpenStreetMap. It runs any kind of Overpass API query and shows the results on an interactive map. Using it can get technical quite quickly: beginners may find the "wizard" a good place to start.<br />
<a href="https://wiki.openstreetmap.org/wiki/Overpass_turbo">https://wiki.openstreetmap.org/wiki/Overpass_turbo</a></p>
<h4 id="qgis">Qgis:</h4>
<p>QGIS application enabling the user to visualize, manage, edit, analyse data, and compose printable maps. It runs on Linux, Unix, Mac OSX, Windows and Android and can be downloaded free of charge from qgis.org<br />
<a href="https://wiki.openstreetmap.org/wiki/QGIS">https://wiki.openstreetmap.org/wiki/QGIS</a></p>

<br />
Using this 2 features we can make customized map.</p>

<p>If you want to create your own layer on top of the OSM layers and shared after. There are online tools like <a href="https://umap.openstreetmap.fr/es/">uMap</a> that will let you do what you want. You can work with the online editor provided or work in an offline OSM editor like JOSM, save your railway layer in a .osm file and upload it to uMap. Please don't upload your layer directly to OSM.</p>
