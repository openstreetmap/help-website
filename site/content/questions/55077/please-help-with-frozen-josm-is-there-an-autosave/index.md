<p>Yes, there should be - see <a href="https://josm.openstreetmap.de/wiki/Help/Action/AutoSave#Preferences">https://josm.openstreetmap.de/wiki/Help/Action/AutoSave#Preferences</a> . I just tested it: there is a timestamped .osm file in <code>~/.josm/autosave/</code> (Linux) as expected.</p>
