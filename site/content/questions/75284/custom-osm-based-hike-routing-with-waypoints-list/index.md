<p>Hi all,</p>
<p>I am looking for a desktop interface for planning custom hiking and cycling routes, thus ideally based on OSM maps. I need this to be able to export the route as GPX, so I can then navigate it using OsmAnd on my Android phone.</p>
<p>The desktop version of OSM (osm.org) does not seem to really allow advanced routing (no dragging&amp;dropping), and Komoot seems to be full of bugs, such as not adding all waypoints defined on the map, to the list of waypoints.</p>
<p>I discovered brouter.de, but here I cannot seem to find a way to have all my waypoints listed, so that I can rename then, reorder them etc.</p>
<p>Can anyone recommend some other online tool for that?</p>
<p>Thanks in advance! Francesco</p>

<div class="answer-body">
<p>The <a href="https://graphhopper.com/maps/">Graphhopper</a> demo has several profiles including Hike Bike and Mountainbike. Additional via points can be added and re-arranged as required (not sure the limit on this). It has gpx export too.</p>
<p>For cycling specifically there is also <a href="https://cycle.travel/">cycle.travel</a> which supports also round trips and dragging and dropping an existing route to get a new route. This also supports GPX and PDF export, but is limited to the US &amp; Canada, UK, EU, Aus and NZ.</p>

<p><a href="https://www.alltrails.com/explore/map/new">AllTrails</a> suits all my needs for bike and hiking routing and I generally export the gpx for my etrex. You can alternate between routing along paths or roads or freedraw an offtrack section. As with all applications, it takes practice to become familiar with how it functions.<br />
I don't use waypoints much but note in the <a href="https://support.alltrails.com/hc/en-us/articles/360019245911-How-do-I-create-a-map-">instructions</a> that they can be added.</p>
