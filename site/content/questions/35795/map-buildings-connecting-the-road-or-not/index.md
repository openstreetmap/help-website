<div class="answer-body">
<p>Routers are clever enough to route to the "nearest point on a street". Don't connect roads to buildings unless you wish to make clear that motor vehicles can enter the building there - and personally, even when a driveway leads up to a garage, I let it end before it touches the building.</p>

<div class="comment-text">
<p>The road is modeled as a linear feature with the way as a line down the center of the actual road which actually occupies some area. So unless the building has upper levels that overhang the road, the building should not be touching the road.</p>
<p>I'm only really familiar with JOSM, but maybe the other editors work the same way in this regard: If the point jumps to the street, try zooming in some more before drawing the building.</p>
