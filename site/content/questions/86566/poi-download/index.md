<div class="answer-body">
<p>The download file for Europe on the Geofabrik download server contains all OSM data for Europe, including POIs. Be aware however that POIs can be modeled as nodes or ways in OSM, or sometimes even as relations. Serious processing (and not just simple filtering) of the PBF file is required to get a reliable list of POIs. One possible approach is loading the file into a PostgreSQL database and then extracting the POIs from there with suitable SQL queries.</p>
