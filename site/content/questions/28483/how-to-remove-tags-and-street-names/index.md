<p>Hi, nice that you want to use OSM :)</p>
<p>You need to keep <strong>2 things</strong> strictly seperated:</p>
<ol>
<li>OSM data (that contain tags and geometries</li>
<li>OSM (online) maps (that are rendered as raster images by applying a map style to the OSM data)</li>
</ol>
<p>So what you are looking for, is a <strong>map style without streetnames</strong>. You might use for example this online maps:</p>
<ul>
<li><a href="http://maps.stamen.com/#watercolor/">http://maps.stamen.com/#watercolor/</a></li>
<li><a href="http://maps.cloudmade.com/?styleId=3">http://maps.cloudmade.com/?styleId=3</a></li>
<li><a href="https://www.mapbox.com/editor/">https://www.mapbox.com/editor/</a></li>
</ul>
<p>If you just want to make a single image, you might also use offline OSM data and render a small area with a local renderer. I guess <a href="http://wiki.openstreetmap.org/wiki/Maperitive">Maperitive</a> suites well and maybe they also have a no-names style.</p>
</div>

<p>If you're working with leaflet the easiest way is to use <a href="https://github.com/mapbox/mapbox-gl-leaflet.">https://github.com/mapbox/mapbox-gl-leaflet.</a> Pay attention to the branches if you're using an earlier version of leaflet. You can link to the style in the basic.html in the examples folder (<a href="https://raw.githubusercontent.com/osm2vectortiles/mapbox-gl-styles/master/styles/bright-v9-cdn.json)">https://raw.githubusercontent.com/osm2vectortiles/mapbox-gl-styles/master/styles/bright-v9-cdn.json)</a> then copy and paste the info into your own file with its own name (such as style-cdn-json). From there you can modify the attributes in the style file to your liking.</p>
