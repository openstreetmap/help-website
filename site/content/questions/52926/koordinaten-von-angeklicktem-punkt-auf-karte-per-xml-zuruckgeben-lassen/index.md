<div class="answer-body">
<p>This has nothing to do with OpenStreetMap or Nominatim, it is a feature of OpenLayers or Leaflet or whatever library you use to display a map. For OpenLayers 2 see <a href="http://dev.openlayers.org/examples/click.html">this example</a> but Leaflet and OpenLayers 3 work similarly. The coordinates are passed as an arguement to a callback function, where you can format them in XML or any other way you want.</p>
<p>Das hat mit OpenStreetMap oder Nominatim nichts zu tun, sondern ist eine Eigenschaft von OpenLayers oder Leaflet oder welche Bibliothek auch immer Du benutzt, um die Karte anzuzeigen. Für OpenLayers 2 gibt es <a href="http://dev.openlayers.org/examples/click.html">dieses Beispiel,</a> aber Leaflet und OpenLayers 3 funktionieren ähnlich. Die Koordinaten werden einer Callback-Funktion übergeben, und dort kann man sie als XML oder beliebig anders formatieren.</p>
