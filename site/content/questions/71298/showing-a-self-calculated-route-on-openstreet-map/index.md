<p>If you have the capability to host a few html/javascript files yourself, then save your route as GeoJSON and throw together a small Leaflet web page that shows an OpenStreetMap layer plus your custom JSON. It shouldn't be more than 20 lines of code. If this is too much for you, use the web service umap.openstreetmap.fr and upload your GeoJSON file there.</p>
