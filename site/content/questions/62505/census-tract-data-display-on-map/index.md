<div class="answer-body">
<p>This isn't really an OpenStreetMap question at all.</p>
<p>You need geometries of Census Tracts which are not present in OSM. A suitable place to find them is <a href="https://www.census.gov/geo/maps-data/data/cbf/cbf_tracts.html">here</a>.</p>
<p>I would recommend using <a href="https://www.qgis.org/en/site/">QGIS</a> which can read both the geometries and your csv file and plot the relevant values. You will find extensive information about this software on the web, and specific guidance about particular problems on <a href="https://gis.stackexchange.com/">GIS Stack Exchange</a>.</p>
<p>The latter would have been a more appropriate place to ask the question in the first instance anyway, and this answer belongs there too!</p>
