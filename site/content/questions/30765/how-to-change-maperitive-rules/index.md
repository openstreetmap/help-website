<div class="answer-body">
<p>Maperitive rules are plain text files. Use your favourite editor, e.g. Notepad or Vim. Default osm-like rules should be in <code>Maperitive/Rules/Default.mrules</code> file. See <a href="http://maperitive.net/docs/">documentation</a> for the format description. Modified rules are loaded and applied immediately after saving.</p>
