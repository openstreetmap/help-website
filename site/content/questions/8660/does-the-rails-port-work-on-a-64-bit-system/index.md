<div class="answer-body">
<p>The rails port is built on "Ruby on Rails" which uses the programming language <em>Ruby</em> which is an interpreted language and platform independant. If you can run ruby on your platform you can run the rails port.</p>
<p>So the rails port can in theory run on Windows, Linux, Mac, FreeBSD, OpenBSD ... and on x86, x86-64, ARM, PowerPC ...</p>
