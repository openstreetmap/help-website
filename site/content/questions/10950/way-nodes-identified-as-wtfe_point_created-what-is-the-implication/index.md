<div class="answer-body">
<p>This means that 80n made the node that includes the coordinates that this way have. After the license change this node will be deleted and the way will then change geometry. If all nodes in a way is deleted in this way then there is not enough information in the licensed data to recreate the geometry of the way.</p>
<p>To prevent any problems with the lisence change you should remap the road with new nodes. You can not use the existing nodes, however you can use the way and all the tags as well as satelite photos of the area.</p>
