<p>If you're thinking of extracting a personal route, I like <a href="https://graphhopper.com/maps/">graphopper</a> for that. You can make a route, with many waypoints and then extract it to a GPX.</p>
<p>If you want the real segment, <a href="http://overpass-turbo.eu/s/lFM">you can query it by ID with overpass-turbo</a>. It allows for export to GPX.</p>
