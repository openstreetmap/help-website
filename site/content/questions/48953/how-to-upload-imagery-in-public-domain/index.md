<p>I would suggest contacting <a href="http://openstreetmap.us/">openstreetmap.us</a> that. if I'm correctly informed, run an imagery server for the US.</p>
<p>Alternatively the OSMF has some limited capacity to do the same.</p>
<p>One available you need to add the information on how to access the imagery to at least:</p>
<p><a href="https://josm.openstreetmap.de/wiki/Maps">https://josm.openstreetmap.de/wiki/Maps</a> for JOSM</p>
<p>and</p>
<p><a href="https://github.com/osmlab/editor-layer-index">https://github.com/osmlab/editor-layer-index</a> for iD</p>
