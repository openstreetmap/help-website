<div class="answer-body">
<p>I'm guessing here, but presumably you've downloaded a ".img" file from somewhere like <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a> . Your Windows computer sees the ".img" extension and thinks that it's a disk image (something that you might burn to CD), but it isn't. Try following instructions such as those at <a href="http://www.dcrainmaker.com/2013/05/download-garmin-705800810.html">http://www.dcrainmaker.com/2013/05/download-garmin-705800810.html</a>.</p>
<p>If you search this site for Garmin you'll get <a href="https://help.openstreetmap.org/search/?q=garmin&amp;Submit=search&amp;t=question">lots of similar questions</a> that you also might find helpful.</p>
