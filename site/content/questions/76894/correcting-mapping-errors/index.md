<div class="answer-body">
<p>A lot of water features in the UK were quickly traced from out-of-copyright mapping in the early days of the project and are ripe for replacement.</p>
<p>As long as you're sure you're not losing any information, then by all means delete and redraw. Object history is a "nice to have", not a must.</p>
<p>Do keep a wary eye on the full list of tags, though, to make sure that you're not removing any extra information. There might be details such as canoe access or weirs.</p>

<div class="answer-body">
<p>There is no need to ask someone if you want to delete it. The reason this is suggested or advised is to keep the history, so everybody can retrace the history of the object. There is a way to achieve this:</p>
<p>If you are using JOSM as editor with the installed utilsplugin2 you can draw the new line/object, select the old object and the new one (CTRL+Left Mouse) and press CTRL-SHIFT-G or "More tools - replace geometry". Basicly just <a href="https://wiki.openstreetmap.org/wiki/Keep_the_history">https://wiki.openstreetmap.org/wiki/Keep_the_history</a></p>
<p>After that everything including the history should be copied to the new object. You might want to try it first with a small object to be sure.</p>
