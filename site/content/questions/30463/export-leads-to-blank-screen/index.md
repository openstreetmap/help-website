<div class="answer-body">
<p>You can only export small regions from the main API, up to 50'000 elements. The main API is intended for editing, not for large scale downloads.</p>
<p>The area in question has currently more than 120'000 elements. It might have been smaller than 50'000 elements some years ago.</p>
<p>You can get the same data for the same area if you download it from a mirror instead of the main API. Just click the "Overpass API" link below the "Export" button.</p>
