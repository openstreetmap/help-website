<p>It looks like Mapnik have recently moved from svn to git. I think you can get the 0.7.1 release from <a href="http://trac.mapnik.org/changeset/3379/tags/release-0.7.1?old_path=%2F&amp;format=zip">this link</a> (zipped, so you'll need to unzip it afterwards, and probably then move the files into the correct path - where the svn co command would have put them). The link is the download link at the bottom of <a href="http://trac.mapnik.org/browser/tags/release-0.7.1">this page</a>.</p>
