<div class="answer-body">
<p>The OpenStreetMap Data is free, but the <a href="https://wiki.openstreetmap.org/wiki/Foundation">foundation</a> that runs the servers is small and not able to offer free hosting. For a deployed website you will either need to host the server side software yourself or use a company that does this as a service.</p>
<p>Self hosted would allow you to completely customise the look to your preference, but will require the most technical know-how. Some of the hosted services also allow a degree of customisation, especially if they are providing vector tiles.</p>
<p>Some information on getting started can be found at <a href="https://switch2osm.org/the-basics/">Switch2OSM</a>.</p>
