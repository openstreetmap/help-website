<p>One of the MAPS.ME developers has said (on IRC I think) that not all of the app store versions are up to date. This might be for technical rather than logistical reasons - I've no idea if the current version of MAPS.ME has dependencies that Amazon devices can't fulfil.</p>
<p>There is a "daily APK download site" somewhere but I've no idea if that's supposed to be public - I'm sure that one of the MAPS.ME developers will post that here if it is. Of course, there still might be technical reasons why later APKs won't run on your Kindle Fire.</p>
