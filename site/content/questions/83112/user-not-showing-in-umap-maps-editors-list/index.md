<p>Hi.</p>
<p>There are no settings in the uMap account.</p>
<p>Check with the user that they have successfully logged in on uMap, on the same instance.</p>
<p>The username can sometimes be modified from upstream, but <a href="https://github.com/umap-project/umap/issues/962">sometimes it isn't</a>, so maybe check the username displayed in uMap.</p>
