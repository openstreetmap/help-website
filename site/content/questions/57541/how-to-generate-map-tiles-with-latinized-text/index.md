<div class="answer-body">
<p>This really an FAQ and has been answered many times before.</p>
<p>Please see <a href="http://blog.gegg.us/2013/09/a-simple-way-to-localize-latinize-an-openstreetmap-style/">http://blog.gegg.us/2013/09/a-simple-way-to-localize-latinize-an-openstreetmap-style/</a> for an approach to solving this. If you want flexibility an alternative would be to use vector tiles (and add transliterated data to the tiles) see for example <a href="https://openmaptiles.org/languages/#0.94/0/0">https://openmaptiles.org/languages/#0.94/0/0</a></p>
<p>For other answers see <a href="https://help.openstreetmap.org/search/?csrfmiddlewaretoken=oNCqEd3VhaAlrAThNoJIBqzlxp5tmoGu&amp;q=transliteration&amp;Submit=search&amp;t=question">https://help.openstreetmap.org/search/?csrfmiddlewaretoken=oNCqEd3VhaAlrAThNoJIBqzlxp5tmoGu&amp;q=transliteration&amp;Submit=search&amp;t=question</a></p>
