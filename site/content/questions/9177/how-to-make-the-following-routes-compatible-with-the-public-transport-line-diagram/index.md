<p>There are no problems with your relations. However the <a href="http://www.overpass-api.de/public_transport.html">Public Transport Line Diagram</a> requires all routes to have the tags <em>network=</em> and <em>ref=</em> set. However not all routes have a network or a relation.</p>
<p>The only thing I notice odd in your relations is that you have invented the tag <em>route=jeepney</em>. I can not find any documentasion giving a verifyable description of this tag. What is the difference betwean a jeepney and a bus?</p>
