<div class="answer-body">
<p>The OSM Wiki says if it is also a place to donate blood it should be tagged as <a href="https://wiki.openstreetmap.org/wiki/Tag:healthcare%3Dblood_donation"><strong><em>healthcare=blood_donation</em></strong></a>. If only blood is stored there, but there is no donating opportunity it can be tagged as <a href="https://wiki.openstreetmap.org/wiki/Blood_bank"><strong><em>healthcare=blood_bank</em></strong></a> even though that tag is not documented.</p>
