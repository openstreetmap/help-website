<div class="answer-body">
<p>It is not necessary to add a separate tag for a way going below a bridge, assuming the bridge has already been tagged correctly. If there is a cutting then just add the <a href="https://wiki.openstreetmap.org/wiki/Key:cutting">cutting</a> tag. The <a href="https://wiki.openstreetmap.org/wiki/Key:tunnel">tunnel=building_passage</a> would be wrong because it should only be used in situations where the building in question is not a bridge itself.</p>
