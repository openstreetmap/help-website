<div class="answer-body">
<p>You need a <span>relation</span> for that. See the <span>Hiking page</span> on the wiki (at the bottom). And you need the <span>Route page</span> to find out how the ways, start points etc. can be added as members.</p>
<p>You add existing ways as relation members. If you just need a part of an existing way, you need to split it (into two ways) before adding the desired way part to the relation.</p>
