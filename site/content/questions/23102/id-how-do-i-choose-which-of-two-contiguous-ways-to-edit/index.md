<p>I ran into the same issue today.</p>
<p>An almost-solution: Select a node that's part of the way you want to select, then cycle through the ways using <code>\</code> (backslash). This way I can get the editor to highlight the desired way, but I haven't found a way to actually properly select and edit it.</p>
<p>A work-around: In the regular openstreetmap.org map view, select the way using e.g. the "Query features" tool, then click the "Edit" button and iD will come up with the way selected.</p>
