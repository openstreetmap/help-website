<td><div class="item-right">
<div class="answer-body">
<p>OSM is <strong>not</strong> a service to create personal maps. See the likes of <a href="http://umap.openstreetmap.fr/en/">umap</a> and <a href="http://share.mapbbcode.org/">mapbbshare</a> (which are osm-based) for that.</p>
<p>OSM is a project to gather the actual map data. Displaying rendered maps on osm.org is in a sense just a nice bonus. We all work on the same map. If you contribute a map changeset or a note, it is shared with everybody.</p>
</div>
