<p>The "Add imagery dialogue" is in JOSM. See <a href="http://josm.openstreetmap.de/wiki/Help/Preferences/Imagery#AddcustomWMSimagery">http://josm.openstreetmap.de/wiki/Help/Preferences/Imagery#AddcustomWMSimagery</a></p>
<p>You paste the WMS service URL (see above) in the "Service URL" field and click "Get Layers".</p>
<p>The URL you got from the export section of mapwarper is for old versions of JOSM, where it would append the parameters at the end. This might still work, but the only official way in the current version is full URL with substitution markers (like {width} and {bbox}). See <a href="http://josm.openstreetmap.de/wiki/Maps#Documentation">http://josm.openstreetmap.de/wiki/Maps#Documentation</a></p>

<div class="answer-body">
<p>Potlatch does not support WMS. Instead, you need to use the "Tiles (Google/OSM scheme)" link provided by MapWarper, and add this to Potlatch 2.</p>
<p>MapWarper will give you a URL like this:</p>
<pre><code>http://mapwarper.net/maps/tile/3/z/x/y.png</code></pre>
<p>You should add the $ symbol before z, x, and y, to make:</p>
<pre><code>http://mapwarper.net/maps/tile/3/$z/$x/$y.png</code></pre>
<p>Then add this to Potlatch 2's background imagery dialogue.</p>

<p>In JOSM, TMS is recommended as well, the format is</p>
<p><a href="http://mapwarper.net/maps/tile/313/%7Bzoom%7D/%7Bx%7D/%7By%7D.png"><code>http://mapwarper.net/maps/tile/313/{zoom}/{x}/{y}.png</code></a></p>
<p>If you like to use WMS, the URL template would be</p>
<p><a href="http://mapwarper.net/maps/wms/313?FORMAT=image/jpeg&amp;VERSION=1.1.1&amp;SERVICE=WMS&amp;REQUEST=GetMap&amp;Layers=image&amp;STYLES=&amp;SRS=%7Bproj%7D&amp;WIDTH=%7Bwidth%7D&amp;HEIGHT=%7Bheight%7D&amp;BBOX=%7Bbbox%7D"><code>http://mapwarper.net/maps/wms/313?FORMAT=image/jpeg&amp;VERSION=1.1.1&amp;SERVICE=WMS&amp;REQUEST=GetMap&amp;Layers=image&amp;STYLES=&amp;SRS={proj}&amp;WIDTH={width}&amp;HEIGHT={height}&amp;BBOX={bbox}</code></a></p>
<p>This can be retrieved by entering the WMS service URL (<a href="http://mapwarper.net/maps/wms/313?"><code>http://mapwarper.net/maps/wms/313?</code></a>) in the "Add Imagery URL" dialog.</p>

<p>One thing to note:</p>
<p>Going straight to the Imagery menu, to attempt: Imagery -&gt; Add Rectified Imagery</p>
<p>is almost certain to create frustration. It is <em>not</em> the same as doing this through: JOSM -&gt; Preferences or File -&gt; Import Image</p>
<p>JOSM -&gt; Preferences is for either WMS or TMS and will add the appropriate underlays to the Imagery drop-down menu. File -&gt; Import Image is for uploading a locally-stored GeoTIFF, etc.</p>
<p>Imagery -&gt; Add Rectified Imagery is for WMS only and is looking for a URL in a format that the JOSM-&gt;Preferences approach will create for you.</p>
<p>That all said, TMS will work much better than WMS for zooming, etc.</p>
