<div class="answer-body">
<p>In general, "save a map" is not enough. You need an application or a dedicated device to show the map.</p>
<p>If you have a Garmin device, look <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">here</a>. If you have a smartphone, pick the correct OS from <a href="https://wiki.openstreetmap.org/wiki/Software/Mobile">here</a> to find an app.</p>
<p>If you need it for other purposes than the above mentioned two, e.g. printing, displaying on a website, etc. start reading <a href="https://wiki.openstreetmap.org/wiki/Using_OpenStreetMap">our wiki</a>.</p>
