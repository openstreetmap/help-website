<p>Some options, depending on how much you want to get into micromapping:</p>
<ul>
<li>Choose one or the other for the way tag, maybe include a note too</li>
<li>Create two parallel line ways very close to each other<br />
</li>
<li>Create a line way for the fence, and an area way for the hedge (possibly sharing nodes with the fence way)</li>
</ul>
