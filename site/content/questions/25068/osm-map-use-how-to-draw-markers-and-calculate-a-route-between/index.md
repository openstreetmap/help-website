<div class="answer-body">
<p>Note that the openstreetmap.org homepage is (currently) not really targeted at end users but more at mappers.</p>
<p>See <a href="/questions/25/">how-do-i-add-a-marker-to-a-map</a> how to add marker(s) in general (you will see the marker coordinates in the openstreetmap.org URL - <code>mlat</code> and <code>mlong</code>). For routing between positions (markers) see the <span>OpenStreetMap-based routing services</span> (try e.g. <a href="http://open.mapquest.com/">http://open.mapquest.com/</a> ).</p>
