<p>Is there a special tag for marking a bio-bridge? They usually are on multi-lane highways and should allow wildlife to cross the highwas due to environmental reasons. There is no road on the bridge, only grass and maybe a fence and some trees to create a more "natural" habitat so that wildlife does not fear to cross. Public traffic is of course restricted to these bridges.</p>
<p>Examples from the Czech Republic:</p>
<ul>
<li><a href="https://www.facebook.com/flyfoto.cz/photos/a.749006175136017.1073741910.222760994427207/749006221802679/">R1 highway near Prague</a> (the first 3 are bio-bridges)</li>
<li><a href="http://www.komunikace-r4.cz/public/data/media/images/061107231303.jpg">R4 highway</a></li>
</ul>

<p>You can use the following tags together at the centre of the wildlife bridge (draw it as a line):</p>
<pre><code>highway=path
surface=grass
bridge=yes
width=[Number of metres]</code></pre>
<p><img src="//wiki.openstreetmap.org/w/images/7/7d/Rendering-highway_path_bridge_yes-mapnik.png" alt="highway=path with bridge" /></p>
<p>And of course</p>
<pre><code>barrier=fence</code></pre>
<p><img src="/upfiles/204964.png" alt="barrier=fence" /></p>
<p>for the wildlife protection.</p>
<p>And maybe you can use this tags for the bridge area:</p>
<pre><code>natural=grassland
layer=1</code></pre>
<p><img src="/upfiles/120px-Natural-Grassland.png" alt="natural=grassland" /></p>

<p>Highways have <a href="http://wiki.openstreetmap.org/wiki/OSM_tags_for_routing/Access-Restrictions">default access values</a> which wouldn't be appropriate for one of these wildlife crossings. For example, the global default for paths is that foot, bicycle, and horse are allowed, which they certainly aren't in reality for a wildlife crossing. You'd probably want to add access=no if you're going to tag it with one of the highway tags.</p>
