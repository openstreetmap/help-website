<div class="answer-body">
<p>All trackpoints from GPS traces that are uploaded to OpenStreetMap will be available to view in an editor, whether they are marked as <code>Identifiable</code>, <code>Public</code>, <code>Trackable</code> or <code>Private</code>. But only tracks marked as <code>Identifiable</code> or <code>Public</code> will show up on the traces page on <a href="http://openstreetmap.org">openstreetmap.org</a>. See <a href="http://wiki.openstreetmap.org/wiki/Visibility_of_GPS_traces">Visibility of GPS traces</a>.</p>
<p>If a trace is set is <code>Identifiable</code>, then the editor can link to it in the traces list. In JOSM, you can do this by downloading all GPS traces in an area, then right click on <code>Downloaded GPX Data</code> in the layers list, and click on <code>Info</code>. Any identifiable tracks will be listed with their URL, so you can then go to that page and download the original GPX.</p>
<p>For traces set as <code>Public</code> this is not possible. But you could manually look through the traces list and see if you can find it. Also try checking for any relevant tags.</p>
<p>For otherwise unidentifiable traces, you can still download them all into JOSM, then save the layer as a GPX file. But this will have all of the traces joined together, so probably not very useful.</p>
