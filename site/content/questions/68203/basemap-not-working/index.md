<div class="answer-body">
<p>The tile Layer is provided by Stamen, its main webpage is: <a href="http://maps.stamen.com">http://maps.stamen.com</a></p>
<p>It appears two of the tile servers: stamen-tiles.<strong>b</strong>.ssl.fastly.net and stamen-tiles.<strong>c</strong>.ssl.fastly.net are not responding to requests while stamen-tiles.<strong>a</strong>.ssl.fastly.net does. That's why you see only every third tile loading. You should open a ticket at <a href="https://github.com/stamen/maps.stamen.com/issues">Stamen map's github</a> for this. There is already a <a href="https://github.com/stamen/maps.stamen.com/issues/145">ticket open</a> regarding a different problem with the a server which is working for me.</p>
