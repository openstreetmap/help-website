<p>The wiki page you pointed shows an icon that is not used by the default osm.org tiles. This default style uses a red square. In France, the pharmacies are known to be represented by a green cross, so a green icon is used: <a href="http://tile.openstreetmap.fr/?zoom=20&amp;lat=48.84237&amp;lon=2.28131&amp;layers=B000000FFFF">example</a></p>
