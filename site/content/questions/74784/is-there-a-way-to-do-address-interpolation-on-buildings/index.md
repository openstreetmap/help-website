<div class="answer-body">
<p>Hi TheAdventurer64, you can encode address interpolations into the OSM database using the following technique:</p>
<ul>
<li>Create a node for the lowest-numbered address, tagged with <strong><code>addr:street</code></strong> and <strong><code>addr:housenumber</code></strong>. (Other <strong><code>addr:*</code></strong> tags can be added too.)</li>
<li>Create a similar node for the highest-numbered address</li>
<li>Draw a linear way from the low-numbered address node to the high-numbered address node, and add a <strong><code>addr:interpolation</code></strong> tag to that way, with an appropriate value.</li>
</ul>
<p>The most common values for <strong><code>addr:interpolation</code></strong> are <strong><code>"even"</code></strong>, (meaning all even-numbered addresses that fall between the low number and high number will be found along this way), <strong><code>"odd"</code></strong>, (meaning all odd-numbered addresses that fall between the low and high number will be found along this way) and <strong><code>"all"</code></strong> (meaning all addresses between the low and high number will be found along this way, for roads that don't put the even numbers on one side and the odd numbers on another or if you don't know what side the houses are on).</p>
<p>Here's an example:</p>
<ul>
<li><a href="https://www.openstreetmap.org/node/401884984">the low-numbered address node</a></li>
<li><a href="https://www.openstreetmap.org/node/401884983">the high-numbered address node</a></li>
<li><a href="https://www.openstreetmap.org/way/34609524">the interpolation way drawn between them</a></li>
</ul>
<p>You can read more about this on the wiki: <a href="https://wiki.openstreetmap.org/wiki/Addresses#Using_interpolation">https://wiki.openstreetmap.org/wiki/Addresses#Using_interpolation</a>.</p>
<p>Generally this technique is used for preliminary mapping. Once more precise locations for each address can be mapped, the <strong><code>addr:interpolation</code></strong> way can be deleted from the database.</p>
</div>
