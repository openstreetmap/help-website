<td><div class="item-right">
<div class="answer-body">
<p>When you're drawing a road, you can tell whether you're going to make the connection with another road by a number of visual hints:</p>
<ul>
<li>The pointer changes to a pen with a + (for a new node), an X (for an existing node), or an O (for an end node). If you've turned custom pointers off, it turns to a hand instead.</li>
<li>The nodes on the road light up blue.</li>
<li>If you're hovering over an existing node, a little tip appears at the top right, saying "over point".</li>
</ul>
<p>Once you've made the connection, it should show up with a black outline around the node. If you only see the node without an outline, you haven't made the connection.</p>
<p>You can also start drawing a new road with it already connected to another road. To do this shift-click on an existing node belonging to the other road. You then can continue to draw as normal. If there isn't a handy node available in the right place then shift-click on the way to create one. This is useful for roads you want connected at both ends.</p>
<p>If you're having difficulty making connections, then make sure "Use thin lines at all scales" isn't checked (in the preferences - the tick at the bottom left).</p>

<div class="answer-body">
<p>One more thing (to add to Richard's answer): if you're starting a way and want to connect the first node to another way, you'll have to start it at an existing node by shift-clicking on that node. If there is no existing node, shift-click on the way where you want the node, then shift-click again.</p>
<p>You can also check what a node is connected to by using the inspector (hit I) and selecting the node.</p>
