<p>Looks like you've already figured it out!</p>
<p>One thing to mention though - normally street names are stored as full names in OSM, so if the full name is "Hakea Plaza" that's what you'd set the name to, rather than "Hakea PL" - you'll notice that adjacent streets have "Grove", "Road", "Court" etc.</p>
