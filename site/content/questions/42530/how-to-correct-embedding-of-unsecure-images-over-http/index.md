<p>(Note: this specific image got lost during the last software update, it is not available on the server - I am just using it as example)</p>
<p>However, editing other people's comments is only possible for moderators AND even if that is possible it is not possible to save a comment which exceeds the comment length limit (it only has this length because it was an "answer" before and was converted into a comment).</p>
