<p>Are you referring to this object: <a href="https://www.openstreetmap.org/way/1115177718#map=15/45.8155/17.2784">https://www.openstreetmap.org/way/1115177718#map=15/45.8155/17.2784</a></p>
<p>If so, the reason it doesn't appear is that it has no main tag indicating what type of object it is, e.g. highway=track or highway=path.</p>
<p>Compare for example <a href="https://www.openstreetmap.org/way/342570410">https://www.openstreetmap.org/way/342570410</a> which has highway=track and does appear on maps.</p>
