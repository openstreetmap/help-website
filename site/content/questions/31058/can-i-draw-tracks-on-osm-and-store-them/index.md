<div class="answer-body">
<p>"on OSM"? – No. OSM is a project to create maps (or more strictly: to systematically collect geodata).</p>
<p>However, you can use tools with e.g. OSM-based maps and OSM-based routing (if you do not want to draw freehand). See <a href="/questions/884/">application-or-web-interface-for-freehand-plotting-tracks-on-osm-as-saving-as-gpx</a>, this is nearly the same question, isn't it?</p>
<p>Well, or just use BaseCamp (you can make or – more easy – download a <span>OSM-based map for Garmin</span> and import it in BaseCamp if you want a OSM map as support for your track drawing!).</p>
