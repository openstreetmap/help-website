<div class="answer-body">
<p>The search box is case-insensitive. If you find an example where case matters, you have found a bug in Nominatim. Russian script is <a href="https://trac.openstreetmap.org/ticket/2758">already known to have problems</a>. If you find examples in other scripts, you can help by creating a bug report for them in <a href="https://trac.openstreetmap.org/query?status=new&amp;status=assigned&amp;status=reopened&amp;component=nominatim&amp;order=id&amp;desc=1">trac</a>.</p>
