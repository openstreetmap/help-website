<div class="answer-body">
<p>When you upload GPS tracks to OpenStreetMap they usually become visible within a few minutes on the GPS traces overlay. However, this is only possible if you have either selected "Public" or "Identifiable" in the visibility dropdown.</p>
<p>The GPS trace layer sometimes also has other issues, where certain tracks don't render (See issues <a href="https://github.com/openstreetmap/iD/issues/5110">#5110</a> and <a href="https://github.com/openstreetmap/iD/issues/2450">#2450</a> on the OpenStreetMap repo). If you want to view all uploaded GPS tracks, you can use e.g. <a href="https://josm.openstreetmap.de/">JOSM</a> and download GPS data from OSM.</p>
