<div class="answer-body">
<p>This problem is described here:</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Landcover">https://wiki.openstreetmap.org/wiki/Landcover</a></p>
<p>In a nutshell, most of the time surface is an additional tag for the object and is tagged like surface=, but if you have no idea about anything beside it, you can use standalone landcover= tag, for example:</p>
<p><a href="https://taginfo.openstreetmap.org/tags/landcover=asphalt">https://taginfo.openstreetmap.org/tags/landcover=asphalt</a></p>
<p>You can also tag the color of the surface, if you know it:</p>
<p><a href="https://taginfo.openstreetmap.org/keys/colour">https://taginfo.openstreetmap.org/keys/colour</a></p>
<p>but the rendering may use its own colors.</p>
