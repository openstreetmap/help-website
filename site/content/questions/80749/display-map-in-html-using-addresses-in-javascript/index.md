<div class="answer-body">
<p>You can display a map in a html page using the LeafletJS library and show the address with a marker.</p>
<p>See <a href="https://leafletjs.com/examples/quick-start/">https://leafletjs.com/examples/quick-start/</a> for an example about integrating the map and displaying a marker.</p>
<p>If you don't have the coordinates for an address, you would have to geocode those first. Nominatim will help you in forward geocoding (getting coordinates for addresses), see <a href="https://nominatim.org">https://nominatim.org</a> for more information on geocoding).</p>
