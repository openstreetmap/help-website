<div class="answer-body">
<p>The documentation for reverting changesets is called <a href="https://wiki.openstreetmap.org/wiki/Change_rollback">Change Rollback</a> on the wiki.</p>
<p>Basically, nowadays, you should use the <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/Undelete">undelete</a> or <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/Reverter">reverter</a> plugins in JOSM, or the <a href="http://revert.osmz.ru/">RevertUI</a> web frontend of the revert scripts.</p>
<p>Please the doc carefully, especially as someone might have changed some things in the area in the meantime. One good practice is load the area in JOSM afterward and run the validator, to check that no errors were created.</p>
<p>Regards.</p>
