<div class="answer-body">
<p>I'm guessing that you either want to:</p>
<ul>
<li>know of a website that allows users to add markers to a map created from OSM data</li>
</ul>
<p>or</p>
<ul>
<li>create a website like that yourself.</li>
</ul>
<p>If it's the first of these, then have a look a <a href="http://open.mapquest.com/">Mapquest Open</a>. There's a feature on there called "My Maps" that may do what you want (although unfortunately "My Maps" seems to be down at the moment).</p>
<p>If it's the second, then the easiest place to start is probably <a href="http://switch2osm.org/using-tiles/getting-started-with-leaflet/">here</a>. That explains how to use the <a href="http://leaflet.cloudmade.com/">Leaflet</a> javascript library to display a slippy map similar to the ones on the OpenStreetMap and Mapquest Open front pages, but with the ability to display markers. You could also do something similar with <a href="http://switch2osm.org/using-tiles/getting-started-with-openlayers/">OpenLayers</a>.</p>
