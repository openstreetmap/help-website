<p>OpenStreetMap is created by editors around the world. Sometimes, a little a area might have been mapped by using only the aerial imagery available, so there's no fine detail editing (such as gates) because no local user has added them to the map. But you can add those gates, no exit on ways or one-way roads in your area.</p>
