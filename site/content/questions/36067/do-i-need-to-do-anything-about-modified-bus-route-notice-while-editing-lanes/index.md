<p>I think you don't have to care. When you split a way that is part of a relation, the editor should take care that both parts are placed in the relation. Since you get a warning that the relation has changed, this is probably the case.</p>
<p>I know for sure that JOSM handles this fine and I think that iD does this too. (now, there were some problems when it was first released).</p>
