<p>I've asked this on talk-legal but so far no response.</p>
<p>I need to cite OSM in an academic paper I contributed to: I used Overpass Turbo to extract some data and presented it in a graphical form.</p>
<p>Obviously I need to use the <a href="https://www.openstreetmap.org/copyright">correct attribution</a>.</p>
<p>I'm also required to use <a href="https://guides.library.uq.edu.au/referencing/vancouver/datasets">Vancouver style</a>. For datasets Vancouver requires:</p>
<p><em>Investigator(s) names. Title of dataset [medium type]. Host institution name/Producer: Physical location; Year of publication [Date accessed]. Available from: Identifier – DOI or URL</em></p>
<p>This is what I've decided on.</p>
<p><em>OpenStreetMap contributors. OpenStreetMap database [PostgreSQL via API]. OpenStreetMap Foundation: Cambridge, UK; 2021 [cited 22 Dec 2021]. © OpenStreetMap contributors. Available under the Open Database Licence from: openstreetmap.org. Data mining by Overpass turbo. Available at overpass-turbo.eu.</em></p>
<p>Can anyone comment - does this ^ look right?</p>
<p>Thank you.</p>
