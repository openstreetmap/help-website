<p>I have a Garmin Nuvi 265W. It does support SD cards, so the easiest way to add a reasonable-sized map is to buy an SD card for it.</p>
<p>It should work with a GMAPSUPP.IMG file copied to the \Garmin folder on the device, but you'll be very limited (mine currently has 1.8Mb of 1.8Gb free only).</p>
<p>To switch between maps, I choose tools / settings / maps / map info.</p>

<p>You can always use the <span>mass-storage mode of the device</span> (just connect it to your computer) and put a map at <em>Garmin/gmapsupp.img</em>. This works for both external sd cards as well as the internal memory. Don't forget to backup any existing <em>gmapsupp.img</em>.</p>
