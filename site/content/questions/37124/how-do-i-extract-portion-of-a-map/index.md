<p>You could use one of the ready made <a href="http://extract.bbbike.org/">BBBike garmin extracts</a> but you will need to re-extract and reload each time you think it is too out of date.<br />
You can save a permalink of the <a href="http://extract.bbbike.org/?lang=en&amp;sw_lng=-69.221&amp;sw_lat=11.997&amp;ne_lng=-68.702&amp;ne_lat=12.415&amp;format=garmin-osm.zip&amp;oi=1&amp;layers=B0000000T">Curacao area</a> at the bottom right for convenience.<br />
The BBBike data seems to be updated about once per week.</p>

<p>BBBike lists a few Garmin types<br />
<a href="http://extract.bbbike.org/extract.html">http://extract.bbbike.org/extract.html</a><br />
File size for the area is less than a Mb, so try a different one each time you do an update. They are using the same extract of data from the osm planet file and rendering a garmin suitable map using a subset of data in different ways to suit different uses. Hopefully you will find one that suits your needs. The file supplied will be a gmapsupp.img so be careful not to overwrite others you want to keep.</p>
