<div class="answer-body">
<p>Yes, you can use whatever tags you want with iD. At the bottom left, you should see "All tags", expand that and you can put in tags you want.</p>
<p>You really really should check what the standard/popular tags are for the feature you want to map. The OSM wiki can help (but it's not definitive). It looks like there is a <a href="https://wiki.openstreetmap.org/wiki/Tag:man_made%3Dwatermill"><code>man_made=watermill</code> tag</a> which is what you need.</p>
