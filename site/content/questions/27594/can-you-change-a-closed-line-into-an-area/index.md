<p>I was just wondering if there is a way to simply change an set of points that are actually closed (for example someone who traced the boundary of a lake) using a line and closing it as opposed to creating it as an area from the beginning. I find a number of lakes for instance that are not "areas" so cannot be tagged as a lake.</p>
<p>The only way around it that I can see is to delete the whole line and start again from scratch by tracing it again as an area then tagging it.</p>
<p>Is there a way that exists to simply convert the closed line to an area. And if not, where can I make the suggestion?</p>
