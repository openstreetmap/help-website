<div class="answer-body">
<p>There is a <a href="https://mapwith.ai">MapWithAI</a> project using automatically recognised features and human conflation. Their approach is most easliy seen with <a href="https://wiki.openstreetmap.org/wiki/RapiD">RapiD</a>.</p>
<p>There has also been attempts to make automatically recognised features from <a href="https://wiki.openstreetmap.org/wiki/Mapillary">Mapillary</a> available and <a href="https://wiki.openstreetmap.org/wiki/Facebook_AI-Assisted_Road_Tracing">AI assisted road tracing</a> has also been trialed.</p>
<p>Some of these approaches have been rather controvertial.</p>
<p>Even if you get community buy in for this you may need to be careful about licensing of the imagery. Just becasue permission has been given for manual tracing using OSM doesn't necessarily mean it has been granted for automatic detections.</p>
