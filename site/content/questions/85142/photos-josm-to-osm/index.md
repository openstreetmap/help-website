<p>OpenStreetMap does not store photos/images.</p>
<p>The photo-related tools in JOSM allow you to use images to help in adding map data, the photos themselves are not part of that data.</p>
<p>How did you intend to use the photos? If you want to use them to illustrate specific objects mapped in OSM, one way to do that is to upload to Wikimedia Commons and add a tag to the mapped objects to point to the photo:</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Wikimedia_Commons">https://wiki.openstreetmap.org/wiki/Wikimedia_Commons</a></p>
<p><a href="https://wiki.openstreetmap.org/wiki/Key:wikimedia_commons">https://wiki.openstreetmap.org/wiki/Key:wikimedia_commons</a></p>
