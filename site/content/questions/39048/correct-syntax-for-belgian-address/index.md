<div class="answer-body">
<p>addr:street=Louizalaan addr:housenumber=43-45 in case there is one entrance</p>
<p>addr:street=Dieweg addr:housenumber=61/2</p>
<p>addr:street=Joseph Wauters-plein (is there really a dash ?) addr:housenumber=16/1 Is there really only a bus 8 ?</p>
<p>addr:street=Bouillonlaan addr:housenumber=16B, same question for bus number</p>
<p>I advice you to subscribe to the <a href="https://lists.openstreetmap.org/listinfo/talk-be">Belgian mailing list</a>. We have recently been discussing a proposal to do an address import. Part of the discussion was on the housenumbers</p>
