<div class="answer-body">
<p>You can use any tag you want. If you go to advanced in Potlatch (since I guess you're using only presets), you can set the key as shop and then write the kind of shop as value. If you choose a random preset before you select advanced you can see how that tag is written, and just replace that value with the value of your choosing.</p>
<p>Note that odd shop values will probably not be rendered on the standard Mapnik rendering though.</p>
