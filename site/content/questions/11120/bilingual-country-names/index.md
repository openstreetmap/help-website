<div class="answer-body">
<p>There is no definite rule for handling multiple languages in the "name" tag (as opposed to the "name:cc" tags which are straightforward). In practice, whether to choose one of language or to put multiple ones in the same tag is usually decided by the country-local mapping community.</p>
<p>Concerning Ireland, the concensus has been to choose only one language, the most common one for the area. This means Irish is typically only used in the Gaeltach regions (even if there are some <a href="https://www.openstreetmap.org/browse/node/745762019">surprising exceptions</a>). Ditto for the name of the country, which is in English by default.</p>
<p>If you want to discuss this decision, try the <a href="http://lists.openstreetmap.org/listinfo/talk-ie">Irish mailing list</a> or the <a href="http://www.mibbit.com/chat/?server=irc.oftc.net&amp;channel=%23osm-ie">#osm-ie IRC channel</a>. You'll also find more <a href="https://wiki.openstreetmap.org/wiki/Ireland">Ireland-specific info</a> on the wiki.</p>
<p>If your question was not specifically about Ireland, but a general technical question, I'll just say that you need to put multiple values in the "name" tag of the country's <a href="https://wiki.openstreetmap.org/wiki/Tag:boundary=administrative">boundary relation</a>. But keep in mind that the same subjective decision-making by country-local mappers may apply, and that you should contact them before making any change.</p>
