<p>You can add a note to the map at the relevant location <a href="https://wiki.openstreetmap.org/wiki/ES:Notas">https://wiki.openstreetmap.org/wiki/ES:Notas</a></p>
<p>Hopefully an OpenStreetMap user will see the note and make the change, but note that we are all volunteers and there is no guarantee that there are active mappers in your area. It helps to add as many details as possible to the note so that other mappers can be confident the change is correct, e.g. is the correct name visible on street signs or on a map/callejero provided by the Ayuntamiento, was the name changed recently etc.</p>
<p>If you want to guarantee the change is made, you can edit OpenStreetMap yourself. Changing a name on an object that is already mapped is quite easy. More info at <a href="https://wiki.openstreetmap.org/wiki/ES:Gu%C3%ADa_de_principiantes">https://wiki.openstreetmap.org/wiki/ES:Gu%C3%ADa_de_principiantes</a></p>

<p>Login at <a href="https://www.openstreetmap.org/login">https://www.openstreetmap.org/login</a></p>
<p>Point the area</p>
<p>Once you are close enough, zoom until OSM elements appear</p>
<p>click "Edit" button in the upper left corner and "Edit with iD"; the url should change to something like <a href="https://www.openstreetmap.org/edit?editor=id#map=19/40.52847/-5.19783">https://www.openstreetmap.org/edit?editor=id#map=19/40.52847/-5.19783</a></p>
<p>clicking the way you want to rename and find the name tag in the form on the left, correct it and upload clicking the button on the upper left corner (or by shortcut CTRL-S)</p>
