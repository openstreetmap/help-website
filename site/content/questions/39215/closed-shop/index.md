<p>I was wondering about a bakery in my street that went out of business. Do I simply delete it or is there some tag to mark it as out of business?</p>
<p>This particularly bakery has been closed for the third time now in the past four years. In the past after being empty for a couple of months there was always a new bakery chain re-opening it. So chances are not bad this happens again.</p>
<p>So do I delete the whole data point or can I mark it appropriately somehow?</p>

<p>Do not delete it, use the <span><code>disused:</code> tag prefix</span>, then it is still available, e.g. potentially as navigation aid. <code>shop=bakery</code> → <code>disused:shop=bakery</code></p>

A "vacant" shop will be counted as "shop" <em>unless</em> the statistic creator cares for an exception for this special value. Similarly if you suppose a GPS user likes to navigate to the nearest shop (e.g. because he wants to find a telephone/toilet/water because of an emergency): shop=vancant will not help, so the POI search needs to take care of this exception.
