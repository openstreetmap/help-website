<p>I played a bit with <a href="https://taginfo.openstreetmap.org">taginfo</a></p>
<ul>
<li>90x man_made=calibration</li>
<li>3x man_made=calibration_target</li>
<li>12x calibration=area</li>
<li>5x calibration=point</li>
<li>4x calibration_point=...</li>
</ul>
<p>Personally I would use man_made=calibration_target ; calibration_target=satellite but I guess man_made=calibration ; calibration=satellite would work as well.</p>
<p>It would be nice if someone that mapped / imported those areas would have written a wiki page about it.</p>
