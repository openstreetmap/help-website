<p>I love <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a> but... have a Garmin device with very limited memory (Edge 520).</p>
<p>I travel a lot and use my Garmin for road cycling so would rather have a large area with less detail than have to take multiple maps with lots of detail I don't need and have to swap them all the time.</p>
<p>I have created a very rudimentary style for use with mkgmap that is great for road cycling and it's use results in maps only a fraction of the size of those currently created by garmin.osm.nl. I have placed this on Gitub for reference:</p>
<p><a href="https://github.com/raintonr/mkgmap-bike-onroad">https://github.com/raintonr/mkgmap-bike-onroad</a></p>
<p>It would be amazing to have some kind of alternate or user defined style used on garmin.osm.nl so I don't have to reinvent the wheel regards to downloading areas for conversion or sharing this.</p>
<p>Would that be possible?</p>
<p>If the site allowed for user defined styles then it would help others with making specialised maps for other activities and not just the single use case I describe.</p>
<p>Thanks :)</p>
