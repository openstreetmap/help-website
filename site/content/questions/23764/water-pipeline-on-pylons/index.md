<div class="answer-body">
<p>There is <a href="https://wiki.openstreetmap.org/wiki/Tag:man_made%3Dpipeline">man_made=pipeline</a> with <em>type=water</em> and <em>location=overground</em>. You can also set the <a href="https://wiki.openstreetmap.org/wiki/Key:operator">operator</a> if known. For the footway just additionally set the usual tags, like <a href="https://wiki.openstreetmap.org/wiki/Tag:highway%3Dfootway">highway=footway</a>, <a href="https://wiki.openstreetmap.org/wiki/Key:access">access=private</a>, <a href="https://wiki.openstreetmap.org/wiki/Key:bridge">bridge=yes</a> and <a href="https://wiki.openstreetmap.org/wiki/Key:layer">layer=1</a>. There is no need to map the pylons.</p>
