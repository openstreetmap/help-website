<div class="answer-body">
<p>Overpass can be used to query data that was last touched by a user or a set of users. You can use a query like:</p>
<pre><code>[out:json][timeout:25];
// gather results
(
  // query part for: “user:PlaneMad”
  node(user:&quot;PlaneMad&quot;)({{bbox}});
  way(user:&quot;PlaneMad&quot;)({{bbox}});
  relation(user:&quot;PlaneMad&quot;)({{bbox}});
);
// print results
out body;
&gt;;
out skel qt;</code></pre>
<p><a href="http://overpass-turbo.eu/s/orG">Try it</a>. Move the map and run again for any other area. The object counts are at the bottom. This should be fairly accurate if you run it everyday for edits in a 24 hour time range using a more specific query since it is unlikely for the objects to have been modified by another user so soon.</p>
<p>You can see some more query examples here:</p>
<ul>
<li><a href="https://www.mapbox.com/mapping/becoming-a-power-mapper/useful-overpass-queries/">https://www.mapbox.com/mapping/becoming-a-power-mapper/useful-overpass-queries/</a></li>
<li><a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_QL#newer">https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_QL#newer</a></li>
</ul>

<p>Hey you can try using <a href="http://kushanjoshi.com/owesome/">owesome</a> with the following query.</p>
<p><code>query ($dateFrom: String, $dateTo: String){ days(dateFrom: $dateFrom, dateTo: $dateTo) { day users(users:["Rajavelu_M"]) { user changeset nodeCount featureCollection { type features { id geometry } } } } }</code></p>
<p>Please note this project could be rough and only has data for 2017. Else you could simply use <a href="http://overpass-turbo.eu">overpass</a></p>
