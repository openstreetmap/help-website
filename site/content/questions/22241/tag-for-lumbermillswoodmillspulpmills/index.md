<div class="answer-body">
<p>A quick look on <a href="http://taginfo.openstreetmap.org/">taginfo</a> suggests that there is no established specific usage for tagging timber processing facilities. I've had a quick look to see if any I know about are tagged on OSM (for instance around Howe Sound, BC, Canada), but again drew a blank.</p>
<p>You are not alone with this issue because there are 1789 objects tagged <a href="http://taginfo.openstreetmap.org/tags/description=timber%20production#overview"><code>description=timber production</code></a>, which suggests we need a proper way of tagging this, and, indeed other specific industrial sectors.</p>
<p>I would suggest that you create an area for the bounds of the facility and tag this <code>landuse=industrial</code>. Individual buildings and processing areas can also be mapped if aerial photography is available.</p>
<p>Further tag the area with something like one of these : <a href="http://taginfo.openstreetmap.org/tags/industry=sawmill">industry=sawmill</a> (3 uses), <code>industry_sector=timber_processing</code>, <code>industry:code=2400</code>, <code>ref:industry_code=2400</code>. These different means of tagging have been used only a few times (industry=* is the most popular with a 100 uses), but all are trying to achieve the same thing. Most make use of the (International) <a href="http://en.wikipedia.org/wiki/Standard_Industrial_Classification">Standard Industry Classification</a>, which is a) public domain as originated by the US government and b) saves having to think of ones own classification. However, there is no obvious consensus yet.</p>
<p>As industry is the most widely used then probably the best course is to go with <code>industry=lumber_and_wood_products</code>, <code>ref:industry_code=2400</code>. Adding the ref tag would mean that you could use a more precise definition for the industry tag (e.g., <code>industry=pulpmill</code>), but anyone interested in the looking at timber processing in general could use the ref tag to identify all related industries.</p>
<p>You may gain further inspiration by looking at some of the existing things tagged with 'description=timber_production' (see taginfo link above).</p>
