<p>When you search on openstreetmap.org the search engine will use rough address data from the US TIGER dataset, this is however -not- in our data and can't easily be fixed nor would that make sense.</p>
<p>To add data to OSM go to openstreetmap.org, create an account, log in, and press the "Edit" button. There is an introduction to editing with iD (that is the editing "app" that you get on the website), which covers what you need to know, and off you go.</p>
