<div class="answer-body">
<p>Data for "Central America" is available from Geofabrik <a href="http://download.geofabrik.de/openstreetmap/">here</a> in a couple of formats; I'd guess that one of those would be what you want.</p>
<p>EDIT:</p>
<p>After a bit of digging about I don't think that you'll be able to use raw OSM XML data to update Navfree, not according to <a href="http://navmii.uservoice.com/forums/76347-navmii-/suggestions/1464265-download-maps#comments">this answer on their uservoice page</a>. Also, Jamaica isn't a region <a href="http://www.navmii.com/support/faq#q2">currently supported by Navfree</a>.</p>
