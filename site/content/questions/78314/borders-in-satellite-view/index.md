<p>Yes, you need to extract these borders from the OSM map data and then export this as a line vector layer, which can then be added as a third and fixed layer on top of your map. You can do this with QGIS. Probably you could also find vector EU border map files on the internet too. If you use Mapbox see this: <a href="https://www.mapbox.com/boundaries">https://www.mapbox.com/boundaries</a></p>
