<div class="answer-body">
<p>See <a href="https://wiki.openstreetmap.org/wiki/Tag:shop%3Dclothes">https://wiki.openstreetmap.org/wiki/Tag:shop%3Dclothes</a> ... I guess it is mainly (or at least 33%) clothes? That combination would do:</p>
<ul>
<li>shop=clothes</li>
<li>clothes=women;men;children;shoes;hats</li>
<li>shoes=yes</li>
</ul>
<p>The second tag's value is an example, please adjust "women;men;children" to what it really is. Add "leather", too, if it sells leather clothes.</p>
<p>Likely no data consumer will recognize that it also sells shoes... but this tagging is the only way to go since <code>shop=shoes</code> uses the same key. Same for <code>shop=leather</code> (for leather goods).</p>
