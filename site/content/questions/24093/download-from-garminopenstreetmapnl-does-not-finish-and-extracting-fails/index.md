<div class="answer-body">
<ul>
<li>If downloading from <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a> does not work for you, try <span>another OSM-based Garmin map</span>. Try downloading smaller extracts or at another computer. If you assume it is a server problem, then please report to <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a> (or discuss in the forum - link is on the left side).</li>
<li>If you use a FAT32 file system on download target location,saving will fail if the file is bigger than 4 GiB (file system restriction). Same applies to unpacking. Memory cards are currently, <a href="http://photo.stackexchange.com/questions/2730/what-file-system-are-memory-cards-formatted-with/35892#35892">usually formatted with a FAT32</a> file system.</li>
</ul>
