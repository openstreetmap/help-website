<p>you can ask for new iD features there: <a href="https://github.com/openstreetmap/iD/issues?q=is%3Aopen+is%3Aissue+label%3Aenhancement">https://github.com/openstreetmap/iD/issues?q=is%3Aopen+is%3Aissue+label%3Aenhancement</a> (note that this needs a login with a github.com account – an external company)</p>
<p>Note that you can use the bookmark feature of your browser to save locations (also with the editing mode already opened). However, that seems not to help much if you do not want to leave iD (keep one editing session).</p>
<p>In case you consider to use other OSM editors: currently <a href="/questions/40742/josm-save-bookmarks-or-initial-location">JOSM does not have a bookmark feature, too</a>.</p>
