<div class="answer-body">
<p>Your question is probably similar to <a href="/questions/8251/how-can-i-add-an-icon-to-the-map">"How can I add an icon to the map?"</a>.</p>
<p>The short answer is you cannot add new kinds of (what you call) markers to our <em>map</em> (at least not without talking to people and convincing them that these are important features to display), but you can add new markers to our <em>database,</em> from where you or anyone else can then extract them and show them on a map.</p>
<p>Some of the features you mention are already present in our database, e.g. there are about 5,000 <a href="https://wiki.openstreetmap.org/wiki/Tag:man_made%3Dwater_well">wells</a> even though these are not rendered on the main map. Same for ponds; there are about 40,000 points in OSM tagged <a href="https://wiki.openstreetmap.org/wiki/Tag:natural%3Dwater">natural=water</a> - these, too, aren't shown on the map at <a href="http://www.openstreetmap.org">www.openstreetmap.org</a> because that map only shows water <em>areas</em>, but you could set up your own map rendering that actually displays them.</p>
<p>If you have locations and names of ponds but not outlines, maybe it is possible to draw the outline with the help of aerial imagery in some places?</p>
