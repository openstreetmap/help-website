<div class="answer-body">
<p>Go to <a href="http://osm.org">opnestreetmap.org</a>, look for the cycle path you want to rename, click on "Edit" on the top left. When editor loads (you will get a short built-in tutorial), click on the cycle path and on the left column change the name. Then, click "Save" button on top-right.</p>
<p>If you have further doubts, you can find the <a href="https://wiki.openstreetmap.org/wiki/Beginners&#39;_guide">Beginners' guide</a> on the wiki.</p>

<p>Your changeset has already been saved and the database has already been edited. If you don't see the changes as soon as you've made them, that's because tile servers needs some time to update. But don't worry, the edit has been made. If you want someone to check your future edita, you can click the "I want someone to verify my edits" checkbox just above the "Cancel/Edit" buttons when you are saving. Owever, some check tools, like OSMChA, will tag <a href="https://osmcha.org/changesets/88047468">your recent edit</a> as a "New mapper" so that other users know that if you have made some mistakes, that's because of inexperience.</p>
