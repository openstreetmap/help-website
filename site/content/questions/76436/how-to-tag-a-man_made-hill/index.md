<div class="answer-body">
<p>Ones that look like they have a person underneath are <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Darchaeological_site"><code>historic=archaeological_site</code></a> + <a href="https://wiki.openstreetmap.org/wiki/Tag%3Asite_type%3Dtumulus"><code>site_type=tumulus</code></a> or <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Dtomb"><code>historic=tomb</code></a> + <a href="https://wiki.openstreetmap.org/wiki/Tag:tomb%3Dtumulus"><code>tomb=tumulus</code></a>.</p>
<p>There is an old proposal that <a href="https://wiki.openstreetmap.org/wiki/Proposed_features/peak">suggests</a> <code>man_made=peak</code> + <code>peak=knoll/hillock</code> but this <a href="https://taginfo.openstreetmap.org/tags/man_made=peak">doesn't seem</a> to have gained much traction. There is slightly more usage for <a href="https://taginfo.openstreetmap.org/tags/?key=natural&amp;value=knoll"><code>natural=knoll</code></a> and <a href="https://taginfo.openstreetmap.org/tags/?key=peak&amp;value=hillock"><code>peak=hillock</code></a> which are documented as tagging options for <a href="https://wiki.openstreetmap.org/wiki/IOF_mapping">orienteering</a>.</p>
