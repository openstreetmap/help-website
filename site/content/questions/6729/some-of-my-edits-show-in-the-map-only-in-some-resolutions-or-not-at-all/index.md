<p>It can take several days before your changes are fully rendered on all resolutions. If the server have a light load and the area are easy to render it might take as little as a few seconds to render. If your changes appears on some zoom levels then you have done everything right. If you have made errors in the tags it would not show on any zoom levels.</p>
