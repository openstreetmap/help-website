<p>This is a help forum for OpenStreetMap. OpenStreetMap is a project about data, not about digital raster imagery (tif-files). Have a look at the gdal project (specifically <a href="http://www.gdal.org/gdal2tiles.html">gdal2tiles.py</a>) to learn how to cut tiff-rasters into raster tiles.</p>
