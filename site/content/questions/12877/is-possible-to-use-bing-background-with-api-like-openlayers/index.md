<p>You can display Bing tiles in OpenLayers (I believe the layer you need to use may still be called VirtualEarth). But you seem to think that if you display Bing tiles through OpenLayers they will somehow magically be openly licensed and that's wrong; whenever you display Bing tiles - no matter where, in JOSM, in OpenLayers, or using the Bing API - they will always be under the same license. So, if you don't want the Bing license then you can't use the Bing tiles.</p>
