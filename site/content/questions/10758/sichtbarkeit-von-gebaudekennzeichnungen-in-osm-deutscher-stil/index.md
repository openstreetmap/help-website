<div class="answer-body">
<p>Man kann das selber einstellen, wenn man sich die Karten selber rendert - dann entscheidet man selbst, was erscheint und was nicht. Was auf der Karte von <a href="http://www.openstreetmap.de">www.openstreetmap.de</a> erscheint, das entscheidet die Gruppe von Leuten, die den Stil dafuer pflegt; die Entscheidungen werden dabei in der Regel anhand kartographischer Ueberlegungen getroffen (d.h. weder ist es so, dass alles, was auf <a href="http://openstreetmap.org">openstreetmap.org</a> sichtbar ist, automatisch auch auf <a href="http://openstreetmap.de">openstreetmap.de</a> sichtbar sein muss, noch ist es so, dass alle existierenden und im Wiki dokumentierten Tags automatisch auf <a href="http://openstreetmap.de">openstreetmap.de</a> sichtbar sein muessen).</p>
<p>Ueber den "deutschen" Kartenstil und seine Weiterentwicklung wird auf der Mailingliste <a href="https://lists.openstreetmap.de/mailman/listinfo/mapnik-de">"mapnik-de"</a> gesprochen. Hilfe ist dort immer willkommen.</p>
