<p>With <a href="http://www.mappa-mercia.org/2014/09/creating-an-always-up-to-date-map.html">umap</a> and <a href="http://www.mappa-mercia.org/2014/09/creating-an-always-up-to-date-map.html">Overpass</a> you can highlight any feature you want. See <a href="http://www.mappa-mercia.org/2014/09/creating-an-always-up-to-date-map.html">Mappa Mercia's blog post</a> on how to achieve this.</p>
<p>A umap can be included in your own website via an iframe.</p>
