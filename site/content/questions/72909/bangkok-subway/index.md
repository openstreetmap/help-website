<p>Maps.me updates its maps with every new app version. I can only estimate that they take the OSM data from two to one week before the app release to create the new maps.</p>
<p>So if you are missing any updates to the Bangkok subway that have been made since a few days before the last Maps.me release you have to wait for the next Maps.me version. Or are you missing something that has been in OSM for longer? Then you should get in contact with the Maps.me developers.</p>
<p>If you need up to date subway data you might try an online map (e.g. <a href="http://www.öpnvkarte.de/?lat=50.891&amp;lon=7.021&amp;zoom=10#100.52;13.7889;11">ÖPNV-Karte</a> (German website only but updating minutely) or <a href="https://www.openstreetmap.org/#map=12/13.7769/100.4856&amp;layers=T">transport map</a> (also lagging behind a few days in updates)) or use the OSMAnd app which has a subscription model allowing for daily or more frequent map updates.</p>
