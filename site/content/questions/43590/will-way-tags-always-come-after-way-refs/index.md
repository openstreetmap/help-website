<p>Depends where you get them from. The files on (say) planet.openstreetmap.org will usually conform to this. However there is no guarantee that this will happen. You might try to use a tool that will produce an XML file in a different manner. So you should be able to handle files with many different orderings.</p>
<p>So to answer your question: No, you cannot rely on that.</p>
