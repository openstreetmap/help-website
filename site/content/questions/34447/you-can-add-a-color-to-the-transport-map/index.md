<p>There are a few different varieties of transport maps listed near the bottom of <a href="https://wiki.openstreetmap.org/wiki/Public_transport">Public_transport wiki</a><br />
eg<br />
<a href="http://openmap.lt/#l=52.52023,13.39078,14,MT">openmap.lt</a><br />
<a href="http://openptmap.org/?zoom=11&amp;lat=51.19242&amp;lon=6.92888&amp;layers=B0000TFFT">openptmap.org</a><br />
<a href="http://öpnvkarte.de/?zoom=13&amp;lat=51.51525&amp;lon=7.13345&amp;layers=TBTTT">öpnvkarte.de</a></p>
<p>...but the bus or tram routes are all the same colour in each<br />
<a href="http://openmap.lt/#l=-27.47026,153.02352,17,MT">openmap.lt</a><br />
<a href="http://openptmap.org/?zoom=16&amp;lat=-27.47107&amp;lon=153.02813&amp;layers=B0000TFFT">openptmap.org</a><br />
<a href="http://öpnvkarte.de/?zoom=13&amp;lat=51.51525&amp;lon=7.13345&amp;layers=TBTTT">öpnvkarte.de</a><br />
</p>
<p>This demo <a href="http://demo.3liz.com/osmtransport/index.php?country=1&amp;location=1">http://demo.3liz.com</a> gives routes different colours...zoom in and check the buses and can click on map to see route details.</p>
