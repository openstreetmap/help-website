<p>I don't think it is possible to run the analysis for several points in one run. The web service returns some embedded <a href="https://en.wikipedia.org/wiki/Geography_Markup_Language">GML</a> for different areas. Maybe you can run several queries, extract the GML portions you're interested in and intersect the areas manually in QGis as an example.</p>
