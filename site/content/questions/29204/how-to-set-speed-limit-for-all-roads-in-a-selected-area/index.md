<p>Hold the <em>shift</em> key pressed while clicking on objects will selecti multiple objects in both iD and JOSM.</p>
<p>In JOSM you can also use <em>ctrl</em> so that the selection is toggleable, or even use <a href="https://wiki.openstreetmap.org/wiki/JOSM_search_function">the search functionality</a> to automatically select a lot of objects.</p>

<p>Hi, step by step; Please start JOSM, zoom in up to street level and push the ctrl button, keep it pressed until you’re ready selecting the desired ways. Select the ways with the mouse, they’re coloring red, press again to deselect, the color goes from red to white. On the right side you’re able to add a speed limit and value in the menu. Forget that the menu gives warnings about the different values of the different ways. Happy mapping in 2014.</p>
