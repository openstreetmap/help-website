<p>Ciclovía is a concept created in Bogotá, Colombia in the 80s, where normal streets are blocked for cars to only allow bikes and people to go there. This happens only on Sundays mornings.</p>
<p>In order to allow reduced traffic, there are detours, shared highways (dedicated lanes), etc. We have created a project to map all of these, initially in Bogota; but this same thing happens in all other cities where ciclovía (open streets) is being used. This is the <a href="https://wiki.openstreetmap.org/wiki/Colombia/Project-Ciclov%C3%ADas">wiki documentation of the project</a>.</p>
<p>As part of the project, we identified several restrictions and detours cases. Here is the issue of case 4.</p>
<p>This is the normal behavior of the highway:</p>
<p><img src="https://wiki.openstreetmap.org/w/images/thumb/3/39/Case4-normal.png/350px-Case4-normal.png" alt="alt text" /></p>
<p>This is how it looks during ciclovía (I drew the ciclovía as cycleway just for the image):</p>
<p><img src="https://wiki.openstreetmap.org/w/images/thumb/7/76/Case4-ciclovia.png/349px-Case4-ciclovia.png" alt="alt text" /></p>
<p>As you can see, the right highway has 3 lanes, with 2 of them for ciclovía and 1 still for cars.</p>
<p>How can we map all these conditions?</p>
<ul>
<li>Ciclovía is doble way.</li>
<li>Restriction on motor_vehicles, bicycles, and foot.</li>
<li>The highway is 3 ways, but just 2 are ciclovía.</li>
</ul>
<p>This is a <a href="https://github.com/MaptimeBogota/Varios/blob/main/proyecto-ciclovia/cases/cicloviaCases.osm">JOSM file to play with the highway</a> and propose a solution.</p>

<div class="answer-body">
<p>The street stays a highway type regardless of time, e.g. <code>highway=primary</code>. Then you can use <a href="https://wiki.openstreetmap.org/wiki/Lanes">lanes</a> and <a href="https://wiki.openstreetmap.org/wiki/Conditional_restrictions">conditional access restrictions</a> to mark the condition for the specified times. Combination of lanes and conditional restrictions is not documented in the wiki as far a I can see but I would do it like this:</p>
<p><code>highway=primary</code><br />
<code>oneway=yes</code><br />
<code>lanes=3</code><br />
<code>vehicle:lanes=designated|designated|designated</code><br />
<code>vehicle:lanes:conditional=no|no|designated @ (Su 07:00-13:00)</code><br />
<code>bicycle:lanes:conditional=designated|designated|yes @ (Su 07:00-13:00)</code></p>
<p>This works on the street above with separate carriageways for the two directions. If you have two-directional streets you need to add <code>:forward</code> and <code>:backward</code> suffixes.</p>
<p>You could also add a description to make the concept of Ciclovía known:<br />
<code>description=Ciclovía</code></p>
