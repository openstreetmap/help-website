<p>I'm looking for a way to streamline the entry of tags I use all the time. The Building preset in JOSM is the sort of thing I want although it does much more than tag building=yes. In my case I merely wish to tag farm ponds with three keys. I would draw the ways and connect them but I'm looking for a method to apply those three keys once that part is done by typing a simple keyboard shortcut like [Ctrl]-P for example.</p>
<p>natural=water</p>
<p>water=pond</p>
<p>source=Bing</p>
<p>Obviously, there are many other multiple key=value combinations that come to mind if I had this capability.</p>
<p>I'm experimenting with Autohotkey, a keystroke macro program, but so far I'm not having much luck. I did see an reference to Autohotkey somewhere in the OSM Wiki or Help but it was in German. Any AHK experts out there?</p>
<p>Alternatively, is there some non-programmatic way to create a Custom Preset in JOSM?</p>

<p>The message in the Version 5759 of JOSM says that "Ctrl-Shift-V now can paste tags from text buffer, e.g., a=2,c=3 and a 2 c 3"</p>
<p>I copied the following text from a text editor and pasted and into a way using Ctrl-Shift-V and worked as expected.<br />
highway=residential<br />
name=Street Name<br />
source=GPS<br />
</p>
<p>You can keep a set of tags in a text file and follow this method till JOSM get the macro functionality.</p>

<div class="answer-body">
<p>I think the best option is to use custom presets in JOSM. There is details of how to create presets here: <a href="http://josm.openstreetmap.de/wiki/TaggingPresets">Tagging presets</a></p>
<p>The easiest way is to copy an example, and modify it for the tags you want. So for your tagging ponds, try something like this:</p>
<pre><code>&lt;presets xmlns=&quot;http://josm.openstreetmap.de/tagging-preset-1.0&quot;&gt;
&lt;item name=&quot;Pond&quot; type=&quot;node,closedway&quot;&gt;
&lt;label text=&quot;Tagging a pond&quot; /&gt;
&lt;key key=&quot;natural&quot; value=&quot;water&quot; /&gt;
&lt;key key=&quot;water&quot; value=&quot;pond&quot; /&gt;
&lt;key key=&quot;source&quot; value=&quot;Bing&quot; /&gt;
&lt;/item&gt;
&lt;/presets&gt;</code></pre>
<p>Or on Pastebin if its not formatted properly here: <a href="http://pastebin.com/Dj5MpKSu">JOSM presets example</a></p>
<p>Copy that into a text editor, and save it as something like <code>ponds.xml</code>. Then open JOSM, and go to Preferences, then choose the 3rd icon down (Map settings), then select the tab for "Tagging presets". Then in the section for active presets, click on the "+" icon, and select your ponds.xml file. Then restart JOSM, and then Ponds should be available on the Presets menu.</p>
<p>It seems it is not possible to set a keyboard shortcut for a preset in JOSM. But you can add the preset to the toolbar - go to Preferences, then 5th icon down. Then you can just click on the toolbar button to add all of your tags.</p>
