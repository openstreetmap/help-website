<div class="answer-body">
<p>How to add points of interests (that is for example shops, restaurants and so on) depends a bit on the situation.</p>
<p>If it occupies the whole building you can add the attributes (tags) for the kind of POI and the name to the building outline, it is however equally valid to simply add a node (point) inside the building outline and add the tags to the point (if there are multiple POIs in the same building that is the usual way to add them in any case).</p>
<p>I suspect you should read one of the beginner tutorials on OSM, see for example <a href="https://wiki.openstreetmap.org/wiki/Beginners%27_guide">https://wiki.openstreetmap.org/wiki/Beginners%27_guide</a></p>
