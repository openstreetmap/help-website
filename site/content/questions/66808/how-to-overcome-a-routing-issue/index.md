<p>Routing issues are generally caused either by:</p>
<ul>
<li>Issues with the routing software you're using. Try the same route on the www.openstreetmap.org main site. Unfortunately, if the problem doesn't occur on the main site, we probably won't be able to help. You'll have to contact the producer of your routing software.</li>
<li>Issues with the map data. If the map data is to blame, you'll probably be able to reproduce the same routing problem on the www.openstreetmap.org main site. For example, sometimes two roads that intersect are incorrectly not linked to each other. If that's the case, you can edit the map to link them, or put a note on the map requesting an experienced mapper in your area to fix the issue. You can also ask here on the help site.</li>
</ul>
