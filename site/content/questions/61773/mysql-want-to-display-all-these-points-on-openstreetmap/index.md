<div class="answer-body">
<p>Assuming that the coordinates ar <a href="https://en.wikipedia.org/wiki/World_Geodetic_System">WGS 84</a> and that <strong>the data is your own data (not a third party's!!)</strong>: First, I would download the data table as <a href="https://en.wikipedia.org/wiki/Comma-separated_values">csv file</a>, changing the heading of <em>log</em> and <em>lat</em> to <strong>longitude</strong> and <strong>latitude</strong>. Then, open the csv file in JOSM (you need to have the <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/OpenData">opendata plugin</a> installed). In another layer, download the data of the area where you want to add the data. Merge carefully the data with the OSM data, tagging all objects acording to the <a href="https://wiki.openstreetmap.org/wiki/Map_Features">OSM wiki</a>.</p>
