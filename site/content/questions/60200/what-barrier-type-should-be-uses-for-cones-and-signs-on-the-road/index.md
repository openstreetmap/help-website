<td><div class="item-right">
<div class="answer-body">
<p>I'd suggest marking the <em>road</em> <code>motor_vehicle=no</code>, or access=no, and then specify what is permissible: <code>foot=yes</code>, <code>bicycle=yes</code>, etc. The advantage is this accurately reflects the situation on the ground, without using inaccurate tags for the barrier. (I'd not consider a cone to be a type of bollard)</p>
<p>You can still tag the node at the barrier itself, but you don't need to worry if the tag is standard or not. I'd add <code>barrier=cones</code> and a note. If the tags catches on, in Japan or elsewhere, great. If not that's ok too, because the access tags on the road are conventional, and will be recognized by other mappers, and routing software.</p>
<p>Also a "林道" (that's a category of road, not the roads name?) looks like a great place to cycle. Are they normally open to bicycles?</p>
