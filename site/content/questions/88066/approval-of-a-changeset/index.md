<p>No approval is necessary. As long as the changeset was uploaded without errors, you should see it in a few moments. More complex uploads, for example modifications to a large multipolygon, might take a few hours but eventually, they too will appear in the OSM slippy map.</p>
<p>Another exception is if you add an object that isn't currently being rendered on the slippy map. This is because some items simply are too little used to warrant the development required to render them.</p>
<p>Dave</p>
