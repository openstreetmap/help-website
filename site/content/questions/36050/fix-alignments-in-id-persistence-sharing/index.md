<div class="answer-body">
<p>Only for you and only during the current session.</p>
<p>The offset affects the whole imagery layer (Bing by default) and offsets are localized so I'm not sure it would make sense to persist offsets. There is however an <span>imagery offset database</span> which is intended to catalog offsets for different imagery types and locations. But I think you have to be using JOSM to make use of it.</p>
