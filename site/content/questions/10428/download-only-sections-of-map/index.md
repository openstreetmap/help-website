<p>No, OpenStreetMap doesn't do that kind of data processing. You will have to download the full data set and then use a tool like osmosis or osmfilter (for both, see <a href="http://wiki.openstreetmap.org">wiki.openstreetmap.org</a>) to extract the bits you want. On <a href="http://downloads.cloudmade.com">downloads.cloudmade.com</a> they also have pre-produced "highway" extracts for various countries/areas but these will include lower-level roads as well.</p>
