<p>You can use Overpass API instead. Just try instead</p>
<pre><code>http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=lon1,lat1,lon2,lat2]</code></pre>
<p>Please note that the order is (on both services) "lon, then lat". For example</p>
<pre><code>http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=7.0,50.6,7.3,50.8]</code></pre>
<p>gives all hospitals around my home (Bonn, Germany, roughly 50.6 north, 7.1 east). If you need meta data, use</p>
<pre><code>http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=7.0,50.6,7.3,50.8][@meta]</code></pre>
<p>instead.</p>
