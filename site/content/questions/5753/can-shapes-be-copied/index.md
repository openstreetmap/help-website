<div class="answer-body">
<p>There is a way, of sorts.</p>
<p>1) Select existing shape</p>
<p>2) press P (or click the 6-spot shape in the tools palette). This creates a "parallel" way, ie a geometrically similar shape (but adjustable size) with the same geometric centre, which will become visible as a thin black line when you move the pointer</p>
<p>3) "wobble" the pointer slightly, firstly to make the new shape visible and then to make it coincide with the old shape</p>
<p>4) click. The cloned shape is now highlighted and can be dragged to the required position.</p>
<p>Obviously it would be more elegant if you could omit step (3) thereby creating a shape which is automatically the right size, but for some reason this creates a shape missing the final edge. I would call that a bug.</p>
</div>
