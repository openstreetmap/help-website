<div class="answer-body">
<p>Set the <code>text-placement</code> property equal to <code>line</code>. The property <code>text-dy</code> provides a pixel offset. For example:</p>
<pre><code>#layer-id::labels {
  text-name: &quot;[NAME]&quot;;
  text-face-name:&#39;Norasi Oblique&#39;;
  text-fill:#bbb;
  text-size:15;
  text-placement: line; /* &lt;-- this is the property you&#39;re looking for */
  text-dy: -8;
}</code></pre>
<p>per <a href="https://www.mapbox.com/tilemill/docs/guides/styling-labels/">https://www.mapbox.com/tilemill/docs/guides/styling-labels/</a></p>
