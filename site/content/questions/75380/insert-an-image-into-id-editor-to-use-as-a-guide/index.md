<div class="answer-body">
<p>Copyright wise you'd need to make sure the map is either aged into public domain (unlikely if new) or you have explicit permission of the copyright holder to put into OSM.</p>
<p>JOSM can do this with the <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/ImportImagePlugin">relevant plugin</a>, but to do something like this in browser you might need to (ab)use <a href="https://wiki.openstreetmap.org/wiki/OsmInEdit#Floor_plan_.26_imagery">OsmInEdit</a> or similar alternate editor.</p>
<p>iD can use custom imagery URLS, but you will probably need to re-project the map and have them cut up into tile and hosted somewhere if you want to go down this route.</p>
