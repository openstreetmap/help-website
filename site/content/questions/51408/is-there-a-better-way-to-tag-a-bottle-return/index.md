<div class="question-body">
<p>Is there a better way to add a Bottle Return/authorized redemption center (a place not just a machine) to the map? As in a place where you can return cans and glass or plastic bottles to get back the refundable deposit that had been paid for them. See <a href="https://en.wikipedia.org/wiki/Container_deposit_legislation">container deposit legislation</a> for information<br />
recycling_type=centre<br />
recycling:glass_bottles=yes<br />
recycling:plastic_bottles=yes<br />
recycling:cans=yes<br />
Which doesn't work very well because it seems to indicate that a facility with recycling:glass_bottles=yes would take all or at least most glass bottles when bottle returns only take bottles that had a deposits paid for it (also if the return is in a store it might take/refund bottles that are sold at that store)</p>
<p>PS I just want to check if there already is a solution that I've missed before sending something to the tagging mailing list</p>

<div class="answer-body">
<p>Since 2017, "reverse vending" machine scheme has been documented at <a href="https://wiki.openstreetmap.org/wiki/Tag:vending%3Dbottle_return">https://wiki.openstreetmap.org/wiki/Tag:vending%3Dbottle_return</a> . It suggests:</p>
<ul>
<li>amenity=vending_machine</li>
<li>vending=bottle_return</li>
</ul>
<p>Additional "recycling" and "payment" tags are also documented on the linked wiki page.</p>
