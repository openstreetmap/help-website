<div class="answer-body">
<p>I don't think it's possible. You can look at the <a href="https://github.com/umap-project/umap/issues">issue tracker</a>, and ask for it, but it's probably complicated to implement.</p>
<p>Otherwise you could just create a kind of "template", an empty map with the desired settings, and "clone" it each time you need to create a new map. Clone is available in settings, advanced actions. It should copy all settings to the new map.</p>
<p>Regards.</p>
