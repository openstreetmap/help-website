<div class="answer-body">
<p>The Uganda wiki page had versions going back to as early as <a href="https://wiki.openstreetmap.org/w/index.php?title=WikiProject_Uganda&amp;oldid=260170">2009</a>, which includes mention of an import, but generally speaking there are not hard and fast start dates for data in any particular country save that no OSM mappers will have been working on it before the <a href="https://wiki.openstreetmap.org/wiki/History_of_OpenStreetMap">start of the project</a> in 2004.</p>
<p>The version history of each object will show the dates each element was added and the dates they were edited, but will not necessarily indicate if those objects were split from existing objects at that point in time or are entirely new.</p>
