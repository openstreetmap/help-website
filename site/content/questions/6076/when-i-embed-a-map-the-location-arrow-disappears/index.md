<td><div class="item-right">
<div class="answer-body">
<p>The iframe embedding (that you can generate from the Export tab) does not support arrows as markers. However you can use the "add marker" link before you copy the embedding URL to add a marker at a chosen position. This also answers your other question about moving the marker position.</p>
<p>If you really need an arrow then you will have to set up your own OpenLayers page that loads OSM tiles and places a marker.</p>
