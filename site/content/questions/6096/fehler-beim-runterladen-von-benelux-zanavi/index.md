<div class="answer-body">
<p>Zanavi basiert auf Kartenmaterial, das für die Routing-Engine "Navit" aufbereitet wurde. Auf der Seite <a href="http://wiki.navit-project.org/index.php/OpenStreetMap">http://wiki.navit-project.org/index.php/OpenStreetMap</a> sind verschiedene Downloadmöglichkeiten für Navit-Kartenmaterial beschrieben. OpenStreetMap selbst bietet keine fertigen Navit-Dateien und auch keine Daten-Auszüge für Benelux an. Finden Sie heraus, wer den Server betreibt, von dem Sie Daten herunterladen, und melden Sie das Problem dort!</p>
