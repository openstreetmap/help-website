<p>I noticed that several roundabouts that consist of multiple lanes are not modelled as such in OpenStreetMap. In practise, depending on the lane on which you enter the roundabout, you have access to different exits. In OpenStreetMap these roundabouts are modelled as single lanes. Is there a reason for this? It seems it would be useful to model these lanes for navigation purposes.</p>
<p>Examples: Prins Bernhardplein, Zaandam, The Netherlands <a href="https://www.openstreetmap.org/edit#map=18/52.45374/4.82275">https://www.openstreetmap.org/edit#map=18/52.45374/4.82275</a></p>
<p>Houtribweg (N307), Lelystad, The Netherlands <a href="https://www.openstreetmap.org/edit#map=19/52.54925/5.51436">https://www.openstreetmap.org/edit#map=19/52.54925/5.51436</a></p>
<p>Laan der Continenten/Donaulaan, Purmerend, The Netherlands <a href="https://www.openstreetmap.org/edit#map=19/52.50064/4.93888">https://www.openstreetmap.org/edit#map=19/52.50064/4.93888</a></p>

<div class="answer-body">
<p>The multi-lane roundabouts from your question simply haven't been tagged as such. In other areas multi-lane roundabouts do exist in OSM. You can <a href="https://overpass-turbo.eu/s/EW9">find multi-lane roundabouts</a> via Overpass API, for example <a href="https://www.openstreetmap.org/way/227772876">way 227772876</a> and <a href="https://www.openstreetmap.org/way/131245452">way 131245452</a>.</p>
<p>If you know the number of lanes then simply add it with the <a href="https://wiki.openstreetmap.org/wiki/Lanes">lanes key</a>. The <a href="https://wiki.openstreetmap.org/wiki/Key:turn#Turning_indications_per_lane">turn:lanes key</a> is also very helpful for navigation systems.</p>
</div>
