<div class="answer-body">
<p>If the lake has one or more inflows you should map the river(s) continuing through the lake to form a continuous path (just like roads), e.g. with <a href="https://wiki.openstreetmap.org/wiki/Tag:waterway%3Driver"><code>waterway=river</code></a>. If no inflow start the waterway where you think it makes morst sense. At the point where the waterway intersects with the lake I always connect the two, although I think this is handled differently by some mappers.</p>
