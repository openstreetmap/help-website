<p>I doubt that it's a deliberate policy. Presumably what needs to happen is that someone needs to raise an issue about it <a href="https://github.com/gravitystorm/openstreetmap-carto/issues/552">here</a>, and then someone else (or the same person) works out what code changes are needed to the rendering CSS to support storage tanks and silos.</p>
<p>OR you could just use another renderer that does show these - I'm not aware of one that covers Aalborg that does, but I bet there is one somewhere.</p>
