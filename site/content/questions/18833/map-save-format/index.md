<p>You can download an image in the MapQuest format using their <a href="http://open.mapquestapi.com/staticmap/" title="MapQuest static map API">static map API</a></p>
<p>You can also download a more traditional looking map using the MapQuest format with <a href="http://maposmatic.org/">MapOSMatic</a>. This site provides a nice frontend with a selection of map and file formats.</p>
