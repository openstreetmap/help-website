<p>The "ÖPNVKarte" layer has been removed from the OSM web site and nobody can help you getting it back. You might be able to replicate the past behaviour by setting up your own instance of Leaflet or OpenLayers and displaying tiles from http://%C3%B6pnvkarte.de; note, however, that the reason this map was removed from OpenStreetMap is that it was unstable and would frequently not work so if you want to continue using it, you are likely to experience outages.</p>
