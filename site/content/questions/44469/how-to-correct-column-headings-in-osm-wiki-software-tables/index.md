<div class="question-body">
<p>Prompted by an IRC conversation, I was looking at <a href="https://wiki.openstreetmap.org/wiki/Software/J2ME">this page</a> and noticed that the column headings weren't particularly meaningful, and didn't work at all well as table headings (particularly "Stores map-data on-board" and "Stores map-data off-board (downloads from internet)" - what I think that the original author meant to say was "works offline" and "can work online").</p>
<p>Below the table is a warning "This table is generated automatically by a bot. Do not edit it manually. <a href="https://wiki.openstreetmap.org/wiki/User:TTTBot">Learn more</a>.". That links to a user page which appears to have no information relevant to editing column headings on it. A <a href="https://wiki.openstreetmap.org/w/index.php?search=off-board&amp;title=Special%3ASearch&amp;go=Go">search for the offending text</a> finds only where it is used, not where it is defined.</p>
<p>What seems to be happening is that because the software tables in the OSM wiki are extemely cumbersome to edit, they don't ever actually get edited, and incredibly in some cases still recommend <a href="https://wiki.openstreetmap.org/wiki/B.iCycle">software</a> that is <a href="https://wiki.openstreetmap.org/wiki/Blocked_applications">likely to be blocked</a>!</p>
