<p>Hi, the validator in JOSM contains errors itself. I asked a similar <a href="/questions/32885/wrong-error-report-in-josms-validation">question</a> and User SomeOneElse (Andy) found the bug in the software. I then submitted a "ticket" and the programmers kindly corrected that bug. If you read the answers to my question you can see how it was done.</p>
<p>In the meantime:</p>
<ol>
<li>Could you post a link to your hiking route, so we can have a look?</li>
<li>Use an other <a href="https://wiki.openstreetmap.org/wiki/Quality_Assurance_Tools_script">validator</a></li>
<li>Use an other <a href="http://ra.osmsurround.org/">relation validator</a></li>
<li>Ignore the JOSM warning and continue the upload</li>
</ol>
