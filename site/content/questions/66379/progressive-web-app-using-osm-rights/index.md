<blockquote>
<p>Moreover I add a lot of info in this map, how I ensure that nobody else will use all the staff and abuse my own app?</p>
</blockquote>
<p>If I understand correctly you want to use OSM data for your map, but keep some of your own data private.</p>
<p>I think that the only way to do this and comply with ShareAlike is to use a Collective Database:</p>
<blockquote>
<p>If you publish a set of OSM data as a file or database, with other separate files or databases, this is generally known as a "Collective Database". The ODbL clearly states that Share-Alike does not apply to the other parts of a Collective Database.</p>
</blockquote>
