<p>We have an odd-even scheme for reducing air pollution in Iran. Based on this traffic plan, there is a restriction for accessing certain street based on car's plate numbers. For example only cars which their last digit is Even (0,2,4,6,8) can enter street "X" only on Even days of the week (Saturdays, Mondays, Wednesdays). How can I add this Conditional Restriction on OSM maps?</p>

<p>There is nothing stopping anybody from defining an license plate number based access value (which then in turn can be used in a conditional restriction).</p>
<p>But anybody doing so will have to drum up support for it in the current routing applications (I believe OsmAnd partially supports conditional restrictions, so they would be a good start).</p>
