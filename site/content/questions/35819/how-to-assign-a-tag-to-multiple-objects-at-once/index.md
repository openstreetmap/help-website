<p>Hi kotya, select all the ways you want to ad the tag to by pressing the ctrl button and keep it pressed, select them carefully with the mouse pointer and don’t add a node to the selection. When ready with selecting start adding the tags in the tag menu. If the ways are present don’t change anything else then the speed limits, save it afterwards and its done.</p>
