<p>Black nodes means that this node have tags. If you select the black node and click on the advanced tab you see that it have the tag odbl=clean. This means that although the creator of the node have not accepted the new contributor terms for the new license the node is considered by another mapper to be clean probably because the other mapper have done a resurvey of the area.</p>
