<div class="answer-body">
<p>Steps to install:</p>
<ol>
<li>Download the State's Cycle maps, as well as the Overview and TDB map from <a href="http://www.osmaustralia.org/garmincycle.php">http://www.osmaustralia.org/garmincycle.php</a></li>
<li>Download Gmapibuilder from <a href="http://bitbucket.org/berteun/gmapibuilder/downloads/Gmapibuilder.dmg">http://bitbucket.org/berteun/gmapibuilder/downloads/Gmapibuilder.dmg</a></li>
<li>Place the contents of the Overview and TDB download in the downloaded State folder</li>
<li>Open Gmapibuilder and click the three dots next to TDB file field, and select the TDB file in the State folder.</li>
<li>Gmapibuilder will automatically identify the IMG files.</li>
<li>Click 'Convert'.</li>
<li>Quit Gmapibuilder and open the State folder. Inside will be the new .gmapi file.</li>
<li>Double-click the .gmapi file to install it (it will automatically open Garmin MapManager - if you don't have it, install it)</li>
<li>Enjoy your maps.</li>
</ol>
