<div class="answer-body">
<p>Here are a few keyboard-centric solutions (I don't like double-clicking or clicking away: too much dexterity required :p).</p>
<p>Right after switching back to <em>add</em> mode you can :</p>
<ul>
<li>Hit <strong>Escape</strong> to unselect all before clicking</li>
<li>Use <strong>Shift+Click</strong> to start a new way as if nothing was currently selected</li>
<li>Use <strong>Alt+Click</strong> to start a new way connected to the currently-selected way/node</li>
</ul>
