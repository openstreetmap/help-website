<p>Thanks for the comment. This is a help system intended for questions - not for suggestions. You should use a 'bug-tracker' for the latter.</p>
<p>OpenStreetMap's standard bug-tracker is at <a href="http://trac.openstreetmap.org/">http://trac.openstreetmap.org/</a> . You can post suggestions there. That said, they need to be filed against a 'component' (i.e. a part of the system), and the transport tiles aren't currently included. You could perhaps add the suggestion against the 'opencyclemap' component, as that's created by the same person (Andy Allan).</p>
