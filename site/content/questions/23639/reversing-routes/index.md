<div class="answer-body">
<p>Your Garmin should be able to reverse the route. How to do this depends on your specific device.</p>
<p>Alternatively you can use a tool like <a href="http://www.gpsbabel.org/">GPSBabel</a> to reverse the route. This can be done by using the <em>reverse</em> filter. You can find it in the GUI at <em>filters -&gt; Routes &amp; Tracks -&gt; Reverse</em>. For the command line mode the correct flags are <code>gpsbabel -i gpx -f input.gpx -t -r -x reverse -o gpx -F output.gpx</code>.</p>
<p>Howto for the Garmin etrex 20/30: main menu -&gt; <em>Route Planner</em>, select a route and choose <em>Reverse Route</em></p>
