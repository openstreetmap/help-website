<div class="answer-body">
<p>No, the OSMF doesn't sell server access, but software and data are free, so you could set up your own Nominatim instance on your own server and run any number of queries you want. If it's just a one-time thing you could rent a big cloud server for a few weeks and run Nominatim there.</p>
<p>If you're looking for commercial providers of OSM-based geocoding services, check out the list at <a href="https://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services">https://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services</a></p>
