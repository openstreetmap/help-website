<div class="answer-body">
<p>The tags in use in OSM have evolved rather than being designed, so there are cases like this one where more than one tag might apply.</p>
<p>Personally, I'd use the one that you think "best fits", based on how other people are using those tags. That might mean you end up using both.</p>
<p>I wouldn't worry too much about whether any particular map renders a particular tag or not. There are many different map renderers, and the map that you see as the default at openstreetmap.org is seen by a small minority of those using OSM data.</p>
