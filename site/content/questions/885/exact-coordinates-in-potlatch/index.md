<p>You can avoid the uploading and deleting steps above. Create you gpx file. Save it. Then in Potlatch2 : Background: Vector File : File Open (your GPX): tic Show and tic Select. you will now see the GPX. note: it may be a little hard to see if only a very short trace, try turning off BING. and then adjust the dim and sharp settings.</p>
