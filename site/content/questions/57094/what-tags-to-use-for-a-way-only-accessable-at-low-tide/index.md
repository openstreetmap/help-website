<div class="answer-body">
<p>There is a reasonably well used tag of <a href="https://wiki.openstreetmap.org/wiki/Key:tidal"><strong>tidal=yes</strong></a> which can be applied to any element which is likely to be covered by water at some phase of the tides.</p>
<p>This is the minimum required, and I think rather less elaborate than using conditional restrictions. The main issue at the moment is that I doubt if people consume this tag along with highways, it certainly has not been added on some of the obvious candidates.</p>
