<div class="answer-body">
<p>In the Changeset List Dialog, <em>Incomplete</em> means JOSM has not downloaded all of the details for that changeset. So it is not able to show the changeset comment, or all of the objects affected by that changeset, or whether it is still open.</p>
<p>You can select a particular changeset from the list, then click the icon underneath to download details for that changeset (or right click on the changeset number, and choose 'Download'). You can then see details of the user, the changeset comment, whether it is still open, and select all of the objects affected by that changeset.</p>
<p>For more details, see the JOSM help: <a href="http://josm.openstreetmap.de/wiki/Help/Dialog/ChangesetListDialog">Changeset List Dialog</a></p>
