<div class="answer-body">
<p>I think it depends very much on which country or region you are looking at. Some country-specific pages may be listed on <a href="https://wiki.openstreetmap.org/wiki/Potential_Datasources">https://wiki.openstreetmap.org/wiki/Potential_Datasources</a></p>
<p>For example, I am most familiar with Spain, where there is extensive documentation about licences/permissions (including copies of letters granting authorization in some case) on various local wiki pages. These include references both to data sources used for imports and some used only to trace over as a background layer: <a href="https://wiki.openstreetmap.org/wiki/ES:Fuentes_de_datos_potenciales_de_Espa%C3%B1a">https://wiki.openstreetmap.org/wiki/ES:Fuentes_de_datos_potenciales_de_Espa%C3%B1a</a> <a href="https://wiki.openstreetmap.org/wiki/ES:Colaboradores_en_Espa%C3%B1a">https://wiki.openstreetmap.org/wiki/ES:Colaboradores_en_Espa%C3%B1a</a> <a href="https://wiki.openstreetmap.org/wiki/ES:Importaciones_en_Espa%C3%B1a">https://wiki.openstreetmap.org/wiki/ES:Importaciones_en_Espa%C3%B1a</a></p>
<p>That probably doesn't help you much, but it illustrates that there is potentially a lot of documentation that is not easy to find starting from a global page.</p>
