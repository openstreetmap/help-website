<div class="answer-body">
<p>This is a font issue. Check your Mapnik style file and make sure it mentions the "Unifont" font in addition to DejaVu Sans, and make sure that the ttf-unifont package is installed. (Newer versions of the style might reference a larger number of fonts for an even better result than just Unifont but Unifont already goes a long way to get rid of these boxes.)</p>

restart renderd
render_list ...</code></pre>
<p>This solves the problem with the square characters!</p>
