<h2 id="how-can-i-get-the-task-outline-gpx-from-hotosm">How can I get the task outline gpx from hotosm?</h2>
<p>A hotosm project states in its instructions (excerpt):</p>
<pre><code>    … the task shapes are irregular and will not load automatically in JOSM.
    To show the task shape, you will need to download first the boundary:
    Select a task Click on &#39;Download&#39; this task as a gpx file to see its boundary.
    Open it in the JOSM editor Click on &#39;Start editor&#39; and your task will load.</code></pre>
<p>When I select a task, I do not see the mentioned button or link; 'Download' this task as a gpx file.</p>
<p>I looked around the hotosm UI on my own and didn't find a .gpx download offered.</p>
<p>Looking elsewhere for how to get .gpx files for tasks, I found <a href="https://learnosm.org/en/coordination/tasking-manager/#potlatch-2">.gpx instructions for Potlatch 2</a> on LearnOSM. Part of that process is (excerpt):</p>
<pre><code>    select Edit with JOSM (if a window appears saying “JOSM remote
    control did not respond…” just click OK).
    Below the Edit with JOSM button, text will say
    “Tip: Download the following .gpx file…”. Download the .gpx file …</code></pre>
<p>When I ensure JOSM is not running, so I intentionally trigger the error above, I do not then however see a Tip appear offering the .gpx file.</p>

<p>I just tried it myself. It's much easier than described on that page.</p>
<p>Just open JOSM and when you select the task you want to map just make sure to select JOSM in the <em>Editor</em> field at the bottom. The map data and a border frame will be loaded into JOSM automatically. If you are already in the task you can achieve the same with the drop-down at the bottom of the right side panel labeled <em>Reload Editor</em>.</p>
