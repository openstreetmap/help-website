<div class="question-body">
<p>Is it possible to edit a subset of the objects in JOSM and hide the objects not belonging to this subset?</p>
<p>Examples use cases:</p>
<ul>
<li>while editing landuse areas, details like streets, POIs, buildings, etc should not be editable to avoid unwanted changes.</li>
<li>while editing buildings no landuse areas should be editable to avoid messing up these areas.</li>
<li>while editing streets no administrative boundary should be editable.</li>
</ul>
<p>Typical modes / filters for editing, I would like to see:</p>
<ul>
<li><strong>all</strong> for all details</li>
<li><strong>details</strong> for mapping buildings, amenities, POIs, ...</li>
<li><strong>roads</strong> for mapping streets, ways, paths, ...</li>
<li><strong>country</strong> for mapping landuse areas, water, ...</li>
<li><strong>boundary</strong> for mapping administrative and other kinds of boundaries</li>
</ul>
