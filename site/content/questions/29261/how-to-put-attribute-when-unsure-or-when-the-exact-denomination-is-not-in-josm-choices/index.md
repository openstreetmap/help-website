<p>I'm currently in Cambodia, and as most part of south asia, motorcycle is very widely used. Thus, motorbike repair station are a vital place that ought to be put on openstreetmap.</p>
<p>I'm very new to mapping in openstreetmap, and I'm using JOSM but must be concidered also as a total newbie on this topic.</p>
<p>I browsed the possible values to tag the place, and didn't find the correct denomination: It's not a garage (no roof, no cars, just people with a few tools and some knowledge).</p>
<p>Or if I'm in a bar-restaurant, should I put the attribute amenity to "bar" or "restaurant", or is it possible to put both ?</p>
<p>As a newbie, this sort of consideration must arise quite often. What to do when we don't find the exact tag in JOSM for the node we want to describe ?</p>
<p>The final objective is to publish on OpenStreetMap of course.</p>
<p>Sorry if this is widely known topic.</p>

<p>Since JOSM presets (and OSM data) reflect only what has been mapped heavily up to know it is a little difficult to start in a new region with features different from what has been up to now.<br />
I experienced this, too, for bars with restaurants but so far haven't made up my mind what to do.<br />
If you look for pobably appropriate tags you could also have a glimpse at <a href="http://taginfo.openstreetmap.org/">taginfo.openstreetmap.org/</a>. It is helpful to look up what kind of tags are in use.<br />
</p>
<p>For the motorcycle repairs: When the repair men are there day by day you could tag it as Hendrikklaas described it and add a tag as hint that it happens in the open space - maybe outdoors=yes.<br />
