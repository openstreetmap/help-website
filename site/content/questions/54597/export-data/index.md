<p>The easiest is by using overpass; if you don't need too much data. You can do it using QuickOSM in QGIS, or with the user interface at overpass-turbo. The wizard will even help you write basic queries. <a href="http://overpass-turbo.eu/s/mDz">This example loads lakes</a>. You can then export the output in whatever format you like.</p>
