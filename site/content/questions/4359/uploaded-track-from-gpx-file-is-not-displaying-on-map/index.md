</p>
<p>I suspect that I'm doing something obviously wrong, but after reading the help files, watching half of the long video, and looking for ideas here, I've reached a point where I just need to ask the question here.</p>

<div class="answer-body">
<p>The GPX track will not be overlayed onto off the map while in view mode and will only appear in edit mode after you told Potlatch to display GPX tracks. If you want to view the GPX track or show it to friends you should use one of the OpenStreetMap based webservices, not the main OpenStreetMap website for that.</p>
