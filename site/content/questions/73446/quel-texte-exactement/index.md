<p>Bonjour,</p>
<p>tout d'abord, je ne suis pas juriste, tout ceci est mon interprétation, des nombreuses discussions sur le sujet que j'ai pu lire.</p>
<p>La référence est la page <a href="https://www.openstreetmap.org/copyright">https://www.openstreetmap.org/copyright</a>.</p>
<p>Le texte minimal est "© les contributeurs d’<a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a>", avec un lien vers la page mentionnée.</p>
<p>Par contre ce texte doit aussi être dans la vidéo, et pas seulement dessous.</p>
<p>D'ailleurs dans la description de la vidéo, vous pouvez mettre un peu plus de détails, notamment la licence des images : CC-BY-SA.</p>
<p>Cordialement.</p>
