<div class="answer-body">
<p>Yes - it's not tremendously helpful when that happens is it? :)</p>
<p>If it's any consolation, that page is better than it used to be. Below the "export" button there's now some text that says:</p>
<pre><code>If the above export fails, please consider using one of the sources listed below:
&#10;Overpass API
    Download this bounding box from a mirror of the OpenStreetMap database
Planet OSM
    Regularly-updated copies of the complete OpenStreetMap database
Geofabrik Downloads
    Regularly-updated extracts of continents, countries, and selected cities
Metro Extracts
    Extracts for major world cities and their surrounding areas
Other Sources
    Additional sources listed on the OpenStreetMap wiki</code></pre>
<p>However, if you want files for a Garmin GPS, you actually don't want any of these - you want instead want one of <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin">these</a> or <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">these</a>. There are lots of options, but if you want a pre-made map the easiest to use and most popular is probably <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a>.</p>
<p>If you have a non-Garmin GPS, you'll need to search the wiki or search and then ask here saying "I have XYZ GPS and I'd like OSM maps on it; what do I need to do?"</p>
