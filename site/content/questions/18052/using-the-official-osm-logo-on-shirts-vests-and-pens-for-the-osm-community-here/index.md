<p>The OSM Foundation (who holds the trademark) hasn't published any guidelines for using the logo and might indeed take action if the logo was used in a way that is harmful for OSM, but using the logo in public while mapping is probably one of the best uses for the logo - nobody will complain if you print vests or shirts, or flyers or stickers, and use them in the context of mapping.</p>
