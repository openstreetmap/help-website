<div class="answer-body">
<p>I'm guessing by "transport map" you're referring to <a href="https://wiki.openstreetmap.org/wiki/Transport_Map">this</a> tile layer that appears at the OpenStreetMap.org website. If you want to use those tiles, see the links on that page for how to do so.</p>
<p>It isn't possible to choose "only some features" from an existing layer of raster tiles (though if you had access to the source of the map style you could create your own tiles to show what you want). Other map styles exist that might be closer to what you want - see <a href="https://wiki.openstreetmap.org/wiki/Public_transport/Tools#Renderers">here</a> for more information.</p>
