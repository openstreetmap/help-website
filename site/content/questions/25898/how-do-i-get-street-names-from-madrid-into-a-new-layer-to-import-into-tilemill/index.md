<p>Dear stephan75,</p>
<p>Finally I have reached a good solution for my issue. And it works! When you import the layer #roads into TileMill, firs of all check if this layer has the column called NAME (Here I am Supposing that you are importing your maps in shp format, and you get the #road file added)</p>
<p>If you have, you are lucky because you only must reply #roads for this other css code:</p>
<h1 id="roads">roads {</h1>
<p>text-name:'[name]'; text-face-name:"Arial"; text-placement:line; text-size:9; text-fill:#666; text-min-distance:60; text-size:11; }</p>
<p>Be sure, you have installed Arial into Tilemill Fonts. If not, look for other font and replace in the new code text-face-name: "your font";</p>
<p>PD: I'm getting all the maps from <a href="http://www.osm974.re/osm2gis/">here</a> and most of them with #roads layer included.</p>
