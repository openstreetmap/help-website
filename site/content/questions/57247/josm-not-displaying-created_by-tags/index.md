<p>This is by design. The created_by tag is deprecated.</p>
<p>JOSM removes this tag automatically on objects that have been edited by you. You should <em>not</em> remove it yourself since this is just not necessary. See <a href="https://wiki.openstreetmap.org/wiki/Key:created_by#Guidance">created_by#Guidance</a> for more information.</p>

<td><div class="item-right">
<div class="answer-body">
<p>JOSM automatically removes the created_by tag whenever an element with such a tag is edited. So you don't need to delete these tags by hand. Instead, they automatically disappear from the database as part of other edits.</p>
<p>Going out of your way to remove these tags is discouraged by the <a href="https://wiki.openstreetmap.org/wiki/Key:created_by">created_by wiki page</a>:</p>
<blockquote>
<p>You should not remove the created_by tag on objects you are not otherwise modifying. If we wanted to remove them in bulk we'd do it some other way.</p>
</blockquote>
<p>The concept of tags being automatically removed by editors is known as <a href="https://wiki.openstreetmap.org/wiki/Discardable_tags">"discardable tags"</a>. That page also links JOSM's list of discardable keys. For the most part, it contains superfluous tags from major imports.</p>
</div>
