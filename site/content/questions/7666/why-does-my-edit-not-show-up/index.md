<p>If the editor failed to close the changeset it remains open for 1 hour after you last edited. However your changes are uploaded when you save.</p>
<p>The map can take a while to render depending on the server load. Right now there has been a maintainence outage and the server are just now started to render again.</p>
<p>You can see that your changes are done correctly by opening the editor and confirm that they are there as the editor uses the raw vector data instead of the rendered map.</p>
