<p>why do you want to install java 7?<sup>(*)</sup> Install the newest you can, I bet JOSM will run! It just needs at least Java 7. In fact <a href="https://josm.openstreetmap.de/ticket/11390">JOSM's support for Java 7 is planned to be dropped in May 2016</a>. About 96% of all Mac OS 10.11 users use JOSM with Java 8 (stats from the last link), so … it will run. ;-) I run JOSM with openJDK 8.u77 on Linux.</p>
<p><span class="small">* maybe the wording on <a href="https://josm.openstreetmap.de/">https://josm.openstreetmap.de/</a> needs adjustment ("written in ​Java 7", mac: "requires ​Java 7"), I will take care of it.</span></p>
