<div class="answer-body">
<p>As far as I have observed on the <a href="https://github.com/openstreetmap/openstreetmap-website/commits/master">github repository</a> , the commits from translatewiki.net are triggered <em>manually</em> by user <a href="https://github.com/siebrand">siebrand</a> estimated once or twice in a month.</p>
<p>And about the wrong text in routing instructions: it is quite possible that this text is coming from OSRM sourcecode. So dig into that repository on github to get some hints about translations there.</p>
