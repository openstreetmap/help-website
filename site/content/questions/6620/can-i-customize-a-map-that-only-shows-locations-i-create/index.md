<div class="answer-body">
<p>Try using <a href="http://viking.sourceforge.com">Viking</a>. While designed primarily for use with GPS tracking, it really excels at making quick maps. It allows you to use any tile source (OSM Mapnik, OSM Osmarender, Google Maps, etc.). You can also create your own Waypoints to overlay the map.</p>
<p>Getting Specific, I would use the following procedure.</p>
<ol>
<li>Start Viking</li>
<li>Add a "Map Layer"</li>
<li>Goto the appropriate area (View --&gt; Goto Location...)</li>
<li>Add a "TrackWaypoint Layer"</li>
<li>Add the appropriate locations as Waypoints. You can even choose the icon or image file.</li>
<li>When you are happy with things, you can then "Generate Image File..." from the File menu.</li>
</ol>
<p>You may also find you may want to add different map sources. <a href="http://www.cloudmade.com">Cloudmade</a> has some nice map styles that may fit your needs.</p>
<p>I should also add that no matter what map tiles you use, you should make sure you follow the appropriate usage (read copyright) restriction and add the appropriate attributions.</p>
