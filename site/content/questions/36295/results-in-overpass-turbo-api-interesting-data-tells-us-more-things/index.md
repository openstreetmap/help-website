<td><div class="item-right">
<div class="answer-body">
<p>You looked for all <span>nodes, ways and relations</span> which have a <span>key</span> "school" (<span><code>school=*</code></span>) in a specific area specified by a coordinate <span>bounding box</span> (see s,w,n,e parameters in your query code on the left). The circles represent the location of those objects. You also could click on the circles to see this object's tags.</p>
<p>Note that just in case you are looking for schools you should rather search for all objects which are tagged <span><code>amenity=school</code></span> as <code>school</code> is just a subkey of this (see <a href="https://taginfo.openstreetmap.org/keys/?key=school#overview">its used values</a>).</p>
<p>In case you are unfamiliar with Overpass turbo, see <span>our wiki page about it</span>, specifically the "Map key" section of that page which explains what the different circle colours mean.</p>
