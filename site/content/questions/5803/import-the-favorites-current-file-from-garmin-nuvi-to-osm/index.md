<div class="answer-body">
<p>The current.gpx on garmin devices shows the current favorites (nüvi) and current track (Dakota, Oregon, Colorado), so make sure you want to make those public before you upload. Tracks and points of interest don't automatically get added to the map, but they do help folks who are editing the map find locations.</p>
