<p>Areas dedicated to religious purposes can be tagged <a href="https://wiki.openstreetmap.org/wiki/Tag:landuse%3Dreligious"><code>landuse=religious</code></a> in combination with <a href="https://wiki.openstreetmap.org/wiki/Key:religion"><code>religion=*</code></a> as appropriate. If multiple religions use the same area they can be listed using "<code>;</code>" as a separator. Denominations can be added <a href="https://wiki.openstreetmap.org/wiki/Key:denomination">similarly</a>.</p>
<p>The <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dplace_of_worship"><code>amenity=place_of_worship</code></a> or <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Dwayside_shrine"><code>historic=wayside_shrine</code></a> tags may also be suitable for the relevant objects. Within <a href="https://wiki.openstreetmap.org/wiki/Tag:religion%3Dshinto">Shinto shrines</a> <code>natural=tree</code> &amp; <code>religion=shinto</code> are suggested on the wiki, something similar may be appropriate here.</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dtaxi">Taxi ranks</a> and other features should probably be tagged separately as they would for other locations.</p>
