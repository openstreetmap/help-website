<div class="answer-body">
<p>It's possible that it's freezing as it tries to access Adobe's servers to load some Flash Player code.</p>
<p>One way in which this can happen is if your home network is badly set up. Many local networks use the 192.168.x.x IP address range. However, Adobe's website also has IP addresses beginning with 192. (e.g. 192.150.16.200). If your router is set up to think that all 192.x.x.x addresses are in your local network, then downloading the code from Adobe's servers will fail. You can easily test whether this is the case by trying to go to <a href="http://www.adobe.com"></a><a href="http://www.adobe.com">www.adobe.com</a>.</p>
