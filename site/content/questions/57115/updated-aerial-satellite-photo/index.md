<div class="answer-body">
<p>The main mission of OpenStreetMap is to create a map. If a building <a href="https://wiki.openstreetmap.org/wiki/Verifiability">exists on the ground</a> but is missing from the map, feel free to <a href="https://wiki.openstreetmap.org/wiki/Editors#Top_three_editors">add it to the map</a> right now, even if the satellite photos do not show it.</p>
<p>You can add a building by creating a node tagged "building=yes" and "name=<em>The Building's Name</em>" at the building's location. You can <a href="https://wiki.openstreetmap.org/wiki/WikiProject_Israel#Naming">record the name in English as well</a> by tagging "name:en=<em>Building's Name in English</em>" in addition to the local name.</p>
<p>If you have time to <a href="https://wiki.openstreetmap.org/wiki/Recording_GPS_tracks">take a GPS trace</a> of the building outline, that would be great too, but every contribution is valuable. If you don't have time to take a GPS trace, just add a node so routers (searches) would find the building.</p>
