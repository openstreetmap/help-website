<p>It seems like iD doesn't support showing the history of an element at the moment. I can't find any corresponding button and also the <a href="https://wiki.openstreetmap.org/wiki/ID/Shortcuts">list of iD keyboard shortcuts</a> doesn't contain it.</p>
<p>But after selecting an element there is a link <em>View on openstreetmap.org</em> in the lower left corner. You can follow it and click on <em>View History</em>.</p>
