<div class="answer-body">
<p>If you mean save or send a link so you or someone else can get the same view of the map that you are currently looking at, then all you need to do is save or copy the URL from your browser.</p>
<p>Notice that as you zoom and pan the map, the URL changes. These changes mean that it always represents what you are looking at.</p>
<p>See also <a href="https://wiki.openstreetmap.org/wiki/Browsing#Sharing_a_link_to_the_maps">Sharing a link to the maps</a> for more options.</p>
