<p>Anyone can use OSM data, but you need to use proper attribution.</p>
<p>About the osm.pbf file: if you repeatedly download large blocks of data (for instance if every user of your app must download a copy of whole OSM database) there can be a problem. You may want to check out this page too: <a href="https://operations.osmfoundation.org/policies/api/">https://operations.osmfoundation.org/policies/api/</a></p>
