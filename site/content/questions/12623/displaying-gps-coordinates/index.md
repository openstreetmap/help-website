<p>You can use <a href="http://umap.openstreetmap.fr/">uMap</a> for that. You can create a new map, and then you can import data as CSV, so for your example:</p>
<pre><code>lat,lon,description
50.1373102630358,14.4652336835861,test1
50.1368770323325,14.4654268026352,test2
50.1369939362421,14.4662743806839,test3
50.1373240163272,14.4661241769791,test4</code></pre>
<p>NB: I've added a third column to give a "description" to markers.</p>
