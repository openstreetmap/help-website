<div class="answer-body">
<p>Split it. A <span>way</span> can have a maximum of 2000 nodes. But better don't create a 2000 node way at all to leave some room for further changes. Just create a <span>boundary relation</span> consisting of multiple ways. See <span>Brecon Beacons National Park</span> (loading takes some seconds) for a rather complex example or <span>Binevenagh AONB</span> for a simple one.</p>
