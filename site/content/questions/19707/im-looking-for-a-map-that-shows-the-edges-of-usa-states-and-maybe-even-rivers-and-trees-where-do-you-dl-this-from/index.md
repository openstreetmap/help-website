<td><div class="item-right">
<div class="answer-body">
<p>Taking ArcGIS as an example, there's a <a href="https://wiki.openstreetmap.org/wiki/ArcGIS_Editor_for_OSM">nice wiki page</a> explaining how it can work with OSM data, with lots of links on it to other resources.</p>
<p>The "my aim is to get the map" statement is a little confusing. What you see on the front page of OSM's website is a map that's been created using OSM data. What you'd load into ArcGIS is that OSM data. The whole planet is available <a href="http://planet.openstreetmap.org/">here</a>, but that's a huge volume of data. It would make more sense to try a smaller extract of the data, perhaps from <a href="http://download.geofabrik.de/openstreetmap/north-america/us/">here</a>. From looking at the ArcGIS screenshots on the wiki page above it looks like it can also import direct from the OSM servers, but that is going to be limited to the area of a small town or so.</p>
<p>If you've not read it already, I'd have a read of <a href="http://wiki.openstreetmap.org/wiki/Beginners_Guide_1.3">this section</a> of the <a href="http://wiki.openstreetmap.org/wiki/Beginners%27_guide">beginners' guide</a> to see how OSM data is structured into nodes, ways and relations. You'll need to know that so that you can select e.g. rivers and trees.</p>
<p>Finally, if you need to filter out a certain subset of data from something that you've downloaded, you might find <a href="http://wiki.openstreetmap.org/wiki/Osmfilter">osmfilter</a> and <a href="http://wiki.openstreetmap.org/wiki/Osmconvert">osmconvert</a> helpful.</p>

<p>I think there is no ready made map with your desired features because no one is able to know your aims in advance.</p>
<p>but on this FAQ site there are already some hints how to get raw OSM data that consists only of some features.</p>
<p>Do a keyword search here for geofabrik, planet, osmfilter ... and you will get some results that maybe helps.</p>
<p>A further option is to use the overpassAPI webservice to get only map features that you need. Do a search about it in the OSM wiki.</p>
<p>If you have assembled your right OSM data you need, then choose a good Renderer to visualize that data and create a map graphic that you want.</p>
<p>About Rendering, ther are also some hints on this FAQ site or in the OSM wiki.</p>
