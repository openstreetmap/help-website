<p>I would say scale is hard to define or achieve as it will vary with screen size. I guess this is why we have a distance scale, as opposed to a fraction (1 to 25000) on the map page. On my WIn7 PC set-up pressing ctr + or - will allow some adjustment. If you wish to have printed map at that scale you could save a JPEG or PNG and then use a image processing program to adjust the size or scale. If I'm on the wrong track please give us a bit more information.<br />
</p>
