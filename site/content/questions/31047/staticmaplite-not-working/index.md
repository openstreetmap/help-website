<p>If you have been making several thousand requests to the "staticmap" API every day then it is likely that your access has been blocked for abuse. The "staticmap" API is run on donated infrastructure and not intended for high-volume production use. However both the OSM data and the software used for making maps are free, so you can run the software on your own infrastructure.</p>
