<div class="answer-body">
<p>After reading the Wikipedia articles of the systems in <a href="https://en.wikipedia.org/wiki/The_Plane_Train">Atlanta</a> and <a href="https://en.wikipedia.org/wiki/Denver_International_Airport_Automated_Guideway_Transit_System">Denver</a>, I think that both can be considered subways (<code>railway=subway</code>).</p>
<p>The system in Atlanta is a normal subway using steel wheels on rails made of steel. The only distinction from other subways is that it runs only with in the airport.</p>
<p>The system in Denver can be tagged as <code>railway=subway</code> as well although it uses rubber wheels on a concrete track. There is no special tag in OSM for these "rubber trains" and even a couple of Metro lines in Paris use railway=subway although they are rubber trains only.</p>
<p>However, not all people movers at airports are subways. For example, Düsseldorf airport uses a monorail and <code>railway=monorail</code> would be the correct tag.</p>
<p><code>railway=light_rail</code> would apply if the system crosses streets without bridges or tunnels. <code>railway=tram</code> would fit if the trains would run on the street and not have their own carriageway. However, the boundaries between subways, light rails and trams are not sharp. But that's a different topic …</p>
