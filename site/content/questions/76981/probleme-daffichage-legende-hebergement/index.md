<p>I see all tourism features appear at the same time as I zoom in. The ones with names appear at the same time as the one without.</p>
<p>Is it possible you have some cached tiles from your original visit to OSM? Tiles do not always render immediately especially when zoomed out. You can see the last render time if you add "<code>/status</code>" to the end of a tile's URL as described <a href="https://wiki.openstreetmap.org/wiki/FR:Slippy_Map#Rendu_des_pav.C3.A9s_par_Mapnik">here</a>.</p>
