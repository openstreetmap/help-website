<div class="answer-body">
<p>Hi, it appears you are editing with maps.me. I have experienced some issues with editing with that tool myself.</p>
<p>Please try the <em>iD</em> editor on www.openstreetmap.org. Go to the <a href="https://www.openstreetmap.org/#map=19/1.87478/-76.16593">area you want to edit the node</a> and click the <em>edit</em> button at the top left. If not already done you have to log in with your OSM account (the one you also used in maps.me). It should be pretty intuitive how to add information to the <a href="https://www.openstreetmap.org/node/9030203518">node</a> you created. If not you can follow the tutorial that is provided in the editor.</p>
<p>Please not that a <code>description</code> is only meant for a short descriptive text. But there are many more practical keys (fields) available to provide more information like opening hours, address, website etc.</p>
