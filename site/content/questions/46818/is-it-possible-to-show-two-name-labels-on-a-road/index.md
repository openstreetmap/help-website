<div class="answer-body">
<p>I was able to figure it out. On ingest, if you add an hstore column, it will add all the tags as key/value pairs. I was able to then edit the query slightly to include the native language name as well as the english name.</p>
<p>SQL query update:</p>
<pre><code>select way,highway,name,(tags -&gt; &#39;name:en&#39;) as my_column &lt;-- Add this piece
   from &amp;prefix;_line
   where waterway IS NULL
     and leisure IS NULL
     and landuse IS NULL
     and tags is not null //Also add this line
     and name is not null</code></pre>
<p>Styling (TextSymbolizer):</p>
<pre><code>&lt;&gt;size=&quot;9&quot; fill=&quot;black&quot; placement=&quot;line&quot; fontset-name=&quot;book-fonts&quot; halo-radius=&quot;1&quot; halo-fill=&quot;#fed7a5&quot;&gt;[name] + &#39; &#39; + [my_column]&lt;&gt;</code></pre>
