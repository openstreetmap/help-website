<p>There aren't any zip code borders in OSM. Zip codes are collections of addresses maintained by the post office and don't really have borders (the post office decides internally what zip code to assign an address to).</p>
<p>Census maintains areas created from their zip code data, but I don't really know where they publish it, <a href="https://www.census.gov/programs-surveys/geography/guidance/tiger-data-products-guide.html">https://www.census.gov/programs-surveys/geography/guidance/tiger-data-products-guide.html</a> is probably an OK place to start.</p>
