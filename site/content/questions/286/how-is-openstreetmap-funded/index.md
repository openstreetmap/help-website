<div class="answer-body">
<p>The main OpenStreetMap <a href="https://wiki.openstreetmap.org/wiki/Servers" title="OSM Servers">servers</a> that contain the database, the main <a href="https://wiki.openstreetmap.org/wiki/API" title="OSM API">API</a> and other services (like <a href="http://openstreetmap.org" title="OpenStreetMap Website">osm.org</a> and it's main map) are managed by the <a href="https://wiki.openstreetmap.org/wiki/Foundation" title="OSMF">OpenStreetMap Foundation (OSMF)</a>, which also takes care of other <a href="http://www.osmfoundation.org/wiki/OSMF:About" title="About the OSMF">"official" stuff</a>. The OSMF is mainly <a href="http://www.osmfoundation.org/wiki/Finances" title="OSMF Finances">funded</a> by <a href="http://www.osmfoundation.org/wiki/Membership" title="OSMF Membership">membership</a> fees as well as <a href="https://wiki.openstreetmap.org/wiki/Donations" title="Dontations">donations</a>.</p>
<p>However, by "OpenStreetMap" you may also mean the community in general, which itself provides <a href="https://wiki.openstreetmap.org/wiki/List_of_OSM_based_Services" title="List of OSM based Services">a lot of services</a>, that are funded in many different ways. Sometimes the hardware is provided by companies (e.g. the <a href="https://wiki.openstreetmap.org/wiki/FOSSGIS/Server" title="FOSSGIS Servers">FOSSGIS servers</a> that run several different community projects) other projects are completely privately financed by the people who run them.</p>
<p>You may also refer to how creation of the software that is used in OpenStreetMap is funded, like the API or the <a href="https://wiki.openstreetmap.org/wiki/Editors" title="Editors">editors</a>. All these software projects are usually written and maintained by random people in their free time.</p>
