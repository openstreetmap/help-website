<div class="answer-body">
<p>Consider using the tag system for languages in lieu of a proper multi-language feature on osqa.</p>
<p>Tag questions in a specific language with lang-(ISO 639-1 code). e.g. lang-de.</p>
<p>OSQA does allow users to have "ignored tags" ("blacklisting"), so users wishing to not see foreign language questions are able to do that in a rudimentary way.</p>
<p>Community tagging should allow us to get quite good coverage of language tags to questions.</p>
<p>The flaw with this is that there isn't (yet) a "whitelisting" feature for tags, meaning a (e.g.) German user wouldn't be able to filter out untagged (implying english) questions. But for now it's probably better than nothing.</p>
</div>
