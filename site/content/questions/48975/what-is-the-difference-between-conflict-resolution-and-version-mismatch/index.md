<p>I have been using the iD Tool to create edits on OSM. While uploading I have faced two situations:</p>
<ol>
<li>Before uploading changes the iD tool asks to resolve conflicts either by choosing "Keep theirs" or "Keep Mine" option for all conflicts.</li>
<li>The iD tool does not ask for any conflict resolution and tries to upload the changes and gives a version mismatch error saying "Version mismatch: provided: 1 server had: 2 of way w123456".</li>
</ol>
<p>What is the difference between these two?</p>
