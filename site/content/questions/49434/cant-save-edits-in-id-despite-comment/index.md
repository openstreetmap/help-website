<p>I tried to reproduce the problem but couldn't, the Save button wasn't greyed out anymore. So I guess the problem is solved.</p>
<p>Anyhow: If this happens to you right-click on the button and click "Inspect" (using Chrome). Find "disable=true" in the source code of the submit button HTML and delete it. The button becomes active again and you can submit your changesets.</p>
