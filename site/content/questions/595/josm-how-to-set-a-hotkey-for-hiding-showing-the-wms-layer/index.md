<div class="answer-body">
<p>No. At the current time not, as far as I am aware.</p>
<p>The <a href="http://josm.openstreetmap.de/wiki/Shortcuts">keyboard shortcuts list</a> has all the sneaky key presses available by default. There's additional ones which can be registered by plugins, but as far as I'm aware, there are no keyboard shortcuts for swapping between layers.</p>
<p>Could be a handy feature to add. One problem though, is that JOSM is running out of available keys for keyboard shortcuts! I guess it would be good to have 'select next layer' available as an action which you can bind to a key if desired.</p>
