<div class="answer-body">
<p>Usage is what makes really a name. If the colloquial name is used by many people and is even the only (not several names) colloquial name for the island, then I would tag this name as <code>name</code>.</p>
<p>To make clear that this is a local name set <code>loc_name</code> to the same value, too. Also you may add a note <code>note:name = officially unnamed by government. However it has a colloquial name</code>.</p>
