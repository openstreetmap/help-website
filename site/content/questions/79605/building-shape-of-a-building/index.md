<p>Hi, is there a way to tag the building shape of a building with hemisphere shape? I would like to map this building (<a href="https://www.openstreetmap.org/way/101292200">Camera Obscura</a>) with some 3D information. Here are some more information about this building: <a href="https://de.wikipedia.org/wiki/Museum_zur_Vorgeschichte_des_Films">Museum on the prehistory of film</a>, <a href="https://www.camera-obscura-muelheim.de/">Camera Obscura</a>, <a href="https://www.google.com/maps/place/Camera+Obscura+M%C3%BClheim+an+der+Ruhr/@51.429236,6.8678545,128a,35y,122.93h,20.33t/data=!3m1!1e3!4m5!3m4!1s0x47b8c110172e65af:0x5fc49bbc9018da26!8m2!3d51.4289901!4d6.8679397">Google Maps</a></p>
<p>I have a similar problem with this building (<a href="https://www.openstreetmap.org/way/101080073">Fernmeldeturm Mülheim-Speldorf</a>), the tower gets thinner and thinner towards the top but does not become pointed, as it is entered so far (<a href="https://osmbuildings.org/?lat=51.42650&amp;lon=6.82278&amp;zoom=17.8&amp;tilt=37&amp;rotation=1">see here</a>), is there a tag where you can specify the diameter at the top and bottom or something similar?</p>
<hr />
<p>Gibt es eine Möglichkeit, die Gebäudeform eines Gebäudes mit Halbkugelform zu kennzeichnen? Ich möchte dieses Gebäude (<a href="https://www.openstreetmap.org/way/101292200">Camera Obscura</a>) mit einigen 3D-Informationen abbilden. Hier fidedt man noch weiter Informationen über das Gebäude: <a href="https://de.wikipedia.org/wiki/Museum_zur_Vorgeschichte_des_Films">Museum zur Vorgeschichte des Films</a>, <a href="https://www.camera-obscura-muelheim.de/">Camera Obscura</a>, <a href="https://www.google.com/maps/place/Camera+Obscura+M%C3%BClheim+an+der+Ruhr/@51.429236,6.8678545,128a,35y,122.93h,20.33t/data=!3m1!1e3!4m5!3m4!1s0x47b8c110172e65af:0x5fc49bbc9018da26!8m2!3d51.4289901!4d6.8679397">Google Maps</a></p>
<p>Bei diesem Gebäude (<a href="https://www.openstreetmap.org/way/101080073">Fernmeldeturm Mülheim-Speldorf</a>) habe ich ein ähnliches Problem, der Turm wird nach oben hin immer dünner, wird aber nicht spitzt, so wie es bisher eingetragen ist (<a href="https://osmbuildings.org/?lat=51.42650&amp;lon=6.82278&amp;zoom=17.8&amp;tilt=37&amp;rotation=1">wie hier zusehen</a>) gibt es einen Tag, bei dem man den Durchmesser oben und unten angeben kann oder etwas ähnliches?</p>

<div class="answer-body">
<p>There is a tags for domed roofs (<a href="https://wiki.openstreetmap.org/wiki/Key:roof:shape"><code>roof:shape=dome</code></a>).</p>
<p>The <a href="https://wiki.openstreetmap.org/wiki/OSM-4D/Roof_table">roof tables</a> page also mentions <code>spherical_cutted</code> but I suspect this is less likely to be correctly interpreted even if it is more specific.</p>
<p>If your other example related just the the roof it would require two <code>building:part</code> areas to describe the shape adequately: one way for the central dome and a multipolygon for the surrounding cone. I'm not sure we have a method of describing buildings with angled walls that are distinct from the roof.</p>
<p>Note that there is also a pretty rare camera obscura tag <a href="https://taginfo.openstreetmap.org/tags/amenity=camera_obscura">in use</a> in the UK.</p>
