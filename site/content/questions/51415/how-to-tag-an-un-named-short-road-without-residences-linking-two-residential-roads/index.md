<p>If I tag it <code>highway=service</code> the routers will ignore it</p>
</blockquote>
<p>Really? Surely it depends on the router? If <code>highway=service</code> is the correct tag, then use that.</p>
<blockquote>
<p>If I tag it <code>highway=residential</code>, JOSM validation and the QC tools will nag me for a name. I</p>
</blockquote>
<p>JOSM validator and other QA tools aren't 100% accurate. It's very hard to write rules that apply 100% to one country, let alone the rest of the world! If the tagging is correct (e.g. there <em>is</em> no name), then that's the tagging. Just ignore the validator. It's a false positive.</p>
