<div class="answer-body">
<p>The rule "map what is on the ground" applies.</p>
<p>If there are roadsigns somewhere that tell the location of name change, then tag the names according to these signs. If there are no explicit roadsigns, then I also would assume without further evidence that the name changes at the border of the municipality.</p>
