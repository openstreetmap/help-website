<p>Bonjour,</p>
<p>si vous êtes réellement un débutant, c'est probablement une mauvaise idée de vouloir vous lancer dans des imports, la procédure est complexe, mal documentée, voire mal vue.</p>
<p>Plus d'infos sur le <a href="https://wiki.openstreetmap.org/wiki/FR:Import">wiki</a>.</p>
<p>Sinon <a href="https://wiki.openstreetmap.org/wiki/FR:Osmose">Osmose</a> propose l'ajout de données issues de nombreux ensembles de données libres ou compatibles avec OSM. Mais de façon bien encadrée, avec les bonnes correspondances de tags, etc.</p>
<p>Un exemple de <a href="https://forum.openstreetmap.fr/t/import-aires-de-covoiturage-avec-un-fichier-csv-sur-josm/8016">discussion</a> à ce sujet.</p>
<p>Cordialement.</p>
