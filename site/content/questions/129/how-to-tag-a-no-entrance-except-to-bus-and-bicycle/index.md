<div class="comment-text">
<p>Using transport mode keys, such as <code>psv=*</code>, for lane information prevents adding the actual access value (such as designated, yes, permissive, private, ...) for both the forward and backward directions.</p>
<p>Note that <code>cycleway=*</code> isn't a transport mode - that would be <code>bicycle=*</code> -, so <code>psv=opposite_lane</code> is <em>not</em> a logical extension of the widely used <code>cycleway=opposite_lane</code>.</p>
</div>

<p>In most cases where the entrance is in a junction relation:restriction should work straight away: (restriction=no_left_turn/no_right_turn, except=psv;bicycle, type=restriction, from=*, to=*, via=*)</p>
<p>In case the entrance is in the middle of straight street you'd probably need to split the way in two and use restriction=no_straight_on</p>
