<div class="answer-body">
<p>According to our <a href="https://wiki.openstreetmap.org/wiki/Garmin#N.C3.BCvi_series">Garmin Nüvi wiki page</a> the file must be located in a folder named <em>Map</em> (see the notes for the Nüvi 2350). Try to create such a folder and move your map to it. Additionally, the name of the file must be in the form of <em>gsupmap<strong>x</strong>.img (where <strong>x</strong> is anything, including nothing)</em>.</p>
