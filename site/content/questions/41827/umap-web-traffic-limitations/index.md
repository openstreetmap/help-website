<div class="answer-body">
<p>The best known uMap site is operated by the French OSM association, you would likely have to ask there for a definite answer if you are using umap.openstreetmap.fr . However in any case the uMap site only serves the JS and data files, any map tiles that you use would be subject to the usage conditions of the provider of said tiles.</p>
<p>Naturally you could run your own instance of uMap, the restrictions around tile usage would still apply though.</p>
