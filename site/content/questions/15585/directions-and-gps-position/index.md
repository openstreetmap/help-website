<div class="answer-body">
<p>OpenStreetMap is not a specific map nor application, it is a database of geo-information that enables you to create maps and applications. So the answer to both your questions is "yes".</p>
<p>Naturally that is not particularly helpful, so here are a couple of examples</p>
<p>1) <a href="http://hikebikemap.de/">http://hikebikemap.de/</a> has a continous coordinate display and it is easy to build your own website with something similar</p>
<p>2) there is currently no routing application directly integrated in the OSM website, but again there are numerous routing sites to choose from. Currently very popular is OSRM: <a href="http://map.project-osrm.org">http://map.project-osrm.org</a></p>
