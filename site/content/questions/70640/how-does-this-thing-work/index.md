<p>It's not that simple I'm afraid (assuming you talk about modifying the map on www.openstreetmap.org). If everyone would just upload tracks this way the map would be cluttered with duplicate and inconsistent data in the end.</p>
<p>That's why the map is edited almost exclusively manually. You can get learn more about how this is done on <a href="https://learnosm.org/">https://learnosm.org</a>. It's pretty easy actually and you are most welcome to give it a try.</p>
