<div class="answer-body">
<p>It's not a utility, so shouldn't have <code>utility=yes</code>. <code>marker=</code> is used for the marker structure, which should be <code>marker=post</code> here I believe.</p>
<p><code>boundary=marker</code> already expresses it is a border. The problem here is <code>boundary=</code> is already used, making reuse of eg <code>boundary=lot</code> (an alternative to <code>place=plot</code>) conflicting.</p>
