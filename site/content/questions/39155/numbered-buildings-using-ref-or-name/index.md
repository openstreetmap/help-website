<p>I worked at a site where all the buildings had codes largely used in internal documents (e.g. room 123 FE), but some of the buildings had the these codes prominently displayed outside (as seen here in Google Street View, the building is 62 Flurstrasse, the internal code is FE (displayed next to the number).</p>
<p>I imagine many large sites will use similar schemes of codes to denote buildings. In some cases these codes will get used as names: as in the Boots site where the names have been used for decades (<a href="https://www.openstreetmap.org/way/34116417#map=16/52.9263/-1.1877).">https://www.openstreetmap.org/way/34116417#map=16/52.9263/-1.1877).</a></p>
