<div class="answer-body">
<ol>
<li>Open josm with at least a bing layer, zoomed at your favorite location</li>
<li>Go to file -&gt; session -&gt; save session as</li>
<li>Tick only the bing layer, and save the *.joz file somewhere</li>
<li>Change your josm shortcut (or however you start josm on your system) to add the *.joz as first argument.</li>
</ol>
