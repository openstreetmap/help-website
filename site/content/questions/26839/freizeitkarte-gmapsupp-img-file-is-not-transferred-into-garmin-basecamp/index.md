<div class="answer-body">
<p>you need to download an installer .exe file for mapsource/basecamp. The image file you have downloaded now is not for basecamp but intended to be copied directly to your device's memory or SD card map folder.</p>
<ul>
<li>download the second link on <a href="http://www.freizeitkarte-osm.de/de/frankreich.html?ModPagespeed=noscript">http://www.freizeitkarte-osm.de/de/frankreich.html?ModPagespeed=noscript</a> - <em>not</em> the first one, as you did.</li>
<li><em>OR</em> directly copy the .img file to your device (if it has enough internal memory) or its SD card (to a directory called Garmin, or Map - depends on the device) without using basecamp (if the eTrex 20 <span>is at least firmware version 2.30</span>, if not the file first has to be named <code>gmapsupp.img</code>)</li>
</ul>
<p>Background info: The .img file was opened with "Windows Disc Image Burner" because you have set up your system to open .img files with this program. Here this does not work since the burner program expects a CD/DVD disc image - which the file you got is not.</p>

<p>Well I finally got there....this is how...</p>
<p>Uploading free Garmin maps to GPS:</p>
<ul>
<li>Use www.freizeitkarte-osm.de or equivalent.</li>
<li>Click on the first download for your chosen map ie. The one that is for your SD card.</li>
<li>When prompted choose the ‘save as’ option and save it direct to your Desktop</li>
<li>Once the zipped file has downloaded, Right click on it and select Extract (this will unzip it.</li>
<li>Once it’s unzipped it needs copying to the Garmin folder (you will need to open a folder called garmin on your SD card). This is where I encountered trouble; I found a right click copy and paste produced a copy of a folder, not a Disc Image File, but when I dragged and dropped it, it worked!! It may just be the way my laptop is configuered?</li>
</ul>
<p>NB This can still take some time, depending on your interweb speed; took me 40 ,minute to get France. I bough an 8Gb micro SD card (Sandisk) class 10 with adapter from www.amazon.fr for 12 € and downloaded the whole of France, Spain, Italy and the UK in an afternoon and still have about 4GB of space left on the card!</p>
