<p>The quick answer is that it doesn't, but it does use cookies to remember where you last looked. Perhaps you'd looked there previously?</p>
<p>I don't know whether there is any clever code but for me (UK based) I cleared all cookies (when the About page was open), re-opened the browser, and it showed a map of the whole UK.</p>
