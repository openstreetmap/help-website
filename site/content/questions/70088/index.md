<div class="answer-body">
<p>Some tag do not render on the default map. The <a href="https://wiki.openstreetmap.org/wiki/Tag:man_made%3Dwater_well">example wells from the wiki</a> don't show on the default style so your well will not show either. The humanitarian layer does show wells with a bucket icon. If it shows up on the humanitarian layer there soon, you have probably added it correctly.</p>
<p>The map style is open source, if you think this should render you can raise a feature request on its <a href="https://github.com/gravitystorm/openstreetmap-carto">github repository</a>.</p>
