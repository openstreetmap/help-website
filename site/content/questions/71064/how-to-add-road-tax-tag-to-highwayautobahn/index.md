<div class="answer-body">
<p>The conventional way is merely to add toll=yes to the relevant highway (see motorways in Switzerland which pioneered this approach: the Vignette).</p>
<p>There have been previous discussions on the wiki <a href="https://wiki.openstreetmap.org/wiki/Talk:Key:toll#Vignette:_differentiation_against_usual_toll">here</a> and <a href="https://wiki.openstreetmap.org/wiki/Proposed_features/toll">here</a> about extending the values of toll to specify that a vignette is required. The basic tag is fine for most uses.</p>
