<div class="answer-body">
<p>I got a quick response:</p>
<p>"The Edit Place button automatically disappears if the downloaded maps are older than two months. Usually we update maps before before the deadline, but, sometimes, a new update may be postponed.</p>
<p>It takes time to convert OpenStreetMap (the whole planet) to our data format. Plus, it requires time for testing as every release of the app contains new features and improvements and it requires to check if everything works with a new data. We postponed the new update as we require more time to fix all issues that we found during testing. We plan to release it, approximately, in the beginning of February."</p>
<p>Problem solved!</p>

<p>In the interim, if you have a mobile data plan, there are some OSM editors that run on <a href="https://wiki.openstreetmap.org/wiki/Android#Editing_the_map">Android</a>. <a href="https://wiki.openstreetmap.org/wiki/Vespucci">Vespucci</a> is probably the most powerful (with a steep learning curve) and <a href="https://wiki.openstreetmap.org/wiki/StreetComplete">StreetComplete</a> the friendliest for new users.</p>
<p>Offline navigation app <a href="https://wiki.openstreetmap.org/wiki/OsmAnd">OsmAnd</a> also has some editing capability (and can store edits until you reach WiFi). But it doesn't extend this ability to buildings without further tags.</p>
