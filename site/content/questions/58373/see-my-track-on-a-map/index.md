<p>This will work using iD Editor:</p>
<ol>
<li>Go to your trace</li>
<li>Click the "edit" link</li>
<li>Your GPS track will display in the editor</li>
<li>(iD specific-&gt;) Open Background Settings (shortcut B) and select OpenStreetMap (Standard)</li>
<li>(iD specific-&gt;) If you want to view with less data in the way, you can also go to Map Data (shortcut F), expand the Map Features section, and uncheck classes of features.</li>
</ol>
