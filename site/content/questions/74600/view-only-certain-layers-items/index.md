<p>You can use this kind of <a href="https://overpass-turbo.eu/s/TCe">query</a> on overpass turbo. I only searched for power=lines, but it will be easy to add any tag you wish. <p>In "Settings", "Map", you can set the tile transparency to 0 to hide the background. You can also export the data in various formats.</p>
