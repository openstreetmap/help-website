<div class="answer-body">
<p>If you search a bit on this website, you might notice that many similar questions have been asked in the past, search for <a href="https://help.openstreetmap.org/search/?q=disappear&amp;Submit=search&amp;t=question">disappear</a></p>
<p>The reason for the disappearing building</p>
<ul>
<li>not all zoom levels are rendered at the same moment</li>
<li>your browser has cached an older version of that zoom level, which did not yet contain the building.</li>
</ul>
<p>So wait a bit longer, and refresh your browser cache.</p>
