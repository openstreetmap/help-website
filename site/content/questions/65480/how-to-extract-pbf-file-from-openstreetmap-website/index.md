<p>The OSM backend uses <a href="https://github.com/zerebubuth/planet-dump-ng">https://github.com/zerebubuth/planet-dump-ng</a> to create the planet pbf from the database.</p>
<p>In general you can poke around in <a href="https://github.com/openstreetmap/chef">https://github.com/openstreetmap/chef</a> to see how various services are setup.</p>
