<div class="answer-body">
<p>The filenames you see at <a href="https://www.openstreetmap.org/user/chrisp9au/traces">user/chrisp9au/traces</a> are the names the files had during uploading. You can't edit them on the OSM website.</p>
<p>If you really want to change the names, then you can download them to your computer. Delete them from the website. Rename the files on your computer and reupload them using <a href="https://www.openstreetmap.org/trace/create">https://www.openstreetmap.org/trace/create</a>.</p>
<p>I wouldn't go through all that effort though. The description and the tags are what really matters.</p>

<p>Hi Chris, you probably mean not a "map" but a GPS trace/track, right? Here is a list of your traces: <a href="https://www.openstreetmap.org/user/chrisp9au/traces">https://www.openstreetmap.org/user/chrisp9au/traces</a> . <a href="https://www.openstreetmap.org/user/chrisp9au/traces/1399940">https://www.openstreetmap.org/user/chrisp9au/traces/1399940</a> is described "Black Hill Nature Reserve - Ridge Track". qqyxqpforptxvgyd.gpx is the file name. You cannot change the file name. You could delete the file ( at <a href="https://www.openstreetmap.org/user/chrisp9au/traces/1399940">https://www.openstreetmap.org/user/chrisp9au/traces/1399940</a> ) and <a href="https://www.openstreetmap.org/trace/create">re-upload it</a> with another file name. You can change the description and tags here: <a href="https://www.openstreetmap.org/trace/1399940/edit">https://www.openstreetmap.org/trace/1399940/edit</a> (you can get there via <a href="https://www.openstreetmap.org/traces/mine">your list of traces</a> → "more" beneath the relevant trace → "Edit this trace").</p>
<p>However, it is not necessary or recommended (by me) doing that. The file name is not important. What is important is that you recorded that trace yourself, that the trace quality (gps reception) is not bad and that it is useful. It is most useful if you <span>use your trace yourself while correcting our map data (e.g. ways) or adding new data (e.g. ways)</span>. GPS traces are also useful if you cannot edit our map data yourself - but not that much. To improve usefulness for other users add a meaningful description and tags (thanks for the pointer, <span><span>@cartinus</span></span>).</p>
