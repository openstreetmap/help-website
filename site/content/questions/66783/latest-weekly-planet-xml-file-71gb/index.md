<p>The XML planet file is raw OSM data and not suitable for your garmin device.</p>
<p>There are numerous places from which you can obtain garmin format maps based on OSM see <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download</a></p>
<p>For specific help on your Nuvi I would suggest asking on <a href="https://forum.openstreetmap.org/viewforum.php?id=26">https://forum.openstreetmap.org/viewforum.php?id=26</a></p>
