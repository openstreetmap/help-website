<div class="answer-body">
<p>You need to locate the position on <a href="https://www.openstreetmap.org/">https://www.openstreetmap.org/</a> then sign up to edit the map.<br />
You will see an edit tab at the top and then be offered an introduction which should enable you to shift the point of concern to the right position or delete it if it no longer exists.<br />
If using a smart phone you may need to use landscape mode to see the edit button.<br />
If you feel that you are unable to manage that then locate the problem on the map and paste the url here with some detail on what needs correcting and why.</p>
