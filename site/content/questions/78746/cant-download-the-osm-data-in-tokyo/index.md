<p>The file is in XML format and contains the following error message (you can open it with any text editor):</p>
<pre><code>runtime error: Query ran out of memory in &quot;recurse&quot; at line 1. It would need at least 514 MB of RAM to continue.</code></pre>
<p>Try with a smaller area or use another tool to export it. For example <a href="http://download.geofabrik.de/asia/japan/kanto.html">Geofabrik</a> or <a href="https://protomaps.com/extracts">Protomaps</a></p>
