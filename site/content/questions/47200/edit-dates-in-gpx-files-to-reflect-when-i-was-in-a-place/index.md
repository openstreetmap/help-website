<p>Hi,</p>
<p>I'm struggling to find a way of creating/editing GPX files for past and planned trips. Whilst there are many options to map <strong>locations</strong> as a track, none that I have found seem to include the facility to edit the <strong>dates/times</strong> easily. I have already drawn maps for most of my trips in Google Maps and exported the data to GPX files, so this gives me then <strong>where</strong> but not the <strong>when</strong>.</p>
<p>For example, for a trip to Argentina in 2001, the data (simplifying for the example) would be along these lines:</p>
<ol>
<li>London, 1/4/2001</li>
<li>Sao Paulo, 2/4/2001</li>
<li>Buenos Aires, 2/4/2001</li>
<li>Bariloche, 7/4/2001</li>
<li>Buenos Aires, 10/4/2001</li>
<li>etc.</li>
</ol>
<p>In some cases the place and date will be quite well defined, but many will be rather general, and that's fine for my purposes.</p>
<p>How can I edit my GPX files (preferably visually, on a map interface) to reflect dates/times when I was (or plan to be) in a specific location?</p>
<p>It just seems such an obvious thing to do?!?</p>
<p>Many thanks!</p>
<p>Stefan</p>

<div class="answer-body">
<p>GPX files are text files, you can edit them with any text editor ( I know that is not the map interface you ask for).</p>
<p>This is a fragment from a GPX file (waypoint file) from a Garmin Dakota 10. It represents 1 waypoint.</p>
<p>&lt;wpt lat="51.162456" lon="3.578275"&gt;<br />
&lt;ele&gt;1.713715&lt;/ele&gt;<br />
&lt;time&gt;2015-12-06T11:33:35Z&lt;/time&gt;<br />
&lt;name&gt;MB16-17&lt;/name&gt;<br />
&lt;sym&gt;Navaid, Green&lt;/sym&gt;<br />
&lt;/wpt&gt;<br />
</p>
<p>Just edit the value after &lt;time&gt; to travel through time :-)</p>
