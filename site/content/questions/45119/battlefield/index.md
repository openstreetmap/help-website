<div class="answer-body">
<p>If there is a <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Dmemorial">memorial</a> or <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Dmonument">monument</a> then you can add this kind of information in a <a href="https://wiki.openstreetmap.org/wiki/Key:description">description</a>. There is also <a href="https://wiki.openstreetmap.org/wiki/Tag:historic%3Dbattlefield">historic=battlefield</a> for which you can add a short description.</p>
<p>Otherwise such kind of information doesn't belong to OSM. We are not Wikipedia. Yet it will be of help to add a <a href="https://wiki.openstreetmap.org/wiki/Key:wikidata">wikidata</a> or <a href="https://wiki.openstreetmap.org/wiki/Key:wikipedia">wikipedia</a> tag so that the corresponding information can be found easily by end user tools, as noted by user SK53.</p>
</div>
