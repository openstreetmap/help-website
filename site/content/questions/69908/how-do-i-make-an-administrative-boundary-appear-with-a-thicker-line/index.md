<div class="answer-body">
<p>The boundary of Bobbing is represented in OSM by a relation object with the id 2421664. You can cause this object to be highlighted (albeit in orange) by using the URL <a href="https://www.openstreetmap.org/relation/2421664">https://www.openstreetmap.org/relation/2421664</a></p>
<p>I don't know if that is good enough for your purpose.</p>
