<p>What's "foundation"?</p>
<p>It's a structure. Keep using <code>bridge:support=</code> for <code>=pier</code> or whatever it is. Don't mix up with <code>=pylon</code> for cable towers.</p>
<p>Add eg <code>disused=yes</code> + <code>historic=yes</code>. Don't use <a href="https://taginfo.openstreetmap.org/tags/historic:bridge:support=pier">https://taginfo.openstreetmap.org/tags/historic:bridge:support=pier</a> since <code>historic:*=</code> is unclear. If it's designated with any status, add the appropriate <code>heritage=</code> + <code>protection_title=</code>.</p>
</div>
