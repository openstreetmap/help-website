<p>Hi.</p>
<p>There is a dedicated tag : <a href="https://wiki.openstreetmap.org/wiki/Tag:highway%3Dtrailhead">highway=trailhead</a>.</p>
<p>You might want to read more information on the wiki, for example starting with <a href="https://wiki.openstreetmap.org/wiki/Hiking">this page</a>.</p>
<p>Looking at your editing history, I see that trails in your area are weirdly tagged. Usually the trails names are in relations, of type route=hiking for example. Also there is a route=mtb tag on some paths, that tag is meant only for relation.</p>
<p>Regards.</p>
