<p>After having a look at the advanced settings of JOSM (typing roundabout in the search field there) it seems that there is NO option to define the direction of newly created roundabouts.</p>
<p>To achieve this option you can</p>
<ol>
<li>do a mailing in the JOSM mailing list, or</li>
<li>open a feature request at josm.openstreetmap.de</li>
</ol>
