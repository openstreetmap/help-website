<p>How would you name tag streets inside a camp site? As I understand the correct classification would of course be highway=service and some camp sites I know use <em>internal</em> street names for easier navigation among its guest. However, those roads are not public and their names are obviously not assigned by any authority. Is it therefore correct to use the name tag with the <em>internal</em> names or will this lead to incorrect results by some applications (e.g. Nomimati!)</p>

<p>There is no need for the name to be officially assigned - just that it is used in terrain. See <a href="http://wiki.openstreetmap.org/wiki/Names">http://wiki.openstreetmap.org/wiki/Names</a> : if there's a sign with a name, add the <code>name</code> in OSM.</p>
