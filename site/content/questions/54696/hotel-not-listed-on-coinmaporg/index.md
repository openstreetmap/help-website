<p>All we can tell you is:</p>
<ul>
<li>"pament:bitcoin=yes" is indeed the right way to indicate that a place accepts payment in Bitcoins (see <a href="https://wiki.openstreetmap.org/wiki/Bitcoin">Bitcoin</a> on the OSM wiki for details)</li>
<li>The linked hotel is tagged like that.</li>
</ul>
<p>coinmap.org is not part of the OSM project, so we cannot tell you why they do or do not show a given place - you will have to ask them about that.</p>
<p>One guess: It seems you only added that tag on 2017-01-20, about a month ago. Most projects that use OSM data only fetch fresh data with some delay, so maybe the data has not yet been updated.</p>
