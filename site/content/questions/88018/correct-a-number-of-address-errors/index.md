<p>No, as there is no linkage between the OSM addresses and the King County data. If you're lucky you'll find a volunteer to write some code to do this (and make the matches according to geometry) but it will be several hours of programming work so unless it's more than a few hundred addresses, doing it manually is probably fastest. Best way to find a volunteer is the OSM community forum linked in the header of this page.</p>
