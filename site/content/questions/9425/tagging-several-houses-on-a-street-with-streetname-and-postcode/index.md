<div class="answer-body">
<p>I do not use Potlatch, so better answers might come, but the easiest way to me seems this:</p>
<ol>
<li>Create all the address points</li>
<li>Create one complete address (even with numbers so that you do not have to add <code>addr:streetnumber</code> more times)</li>
<li>Click on each addresspoint and hit <code>R</code></li>
<li>Go through the address points again putting in the correct numbers</li>
</ol>
<p><strong>Remarks:</strong><br />
<code>R</code> copies the tags from the last object that was selected<br />
You have to unselect a newly created node for R to work (a bug maybe?)</p>
<p>Or (expecially if the street has consecutive numbers) consider using JOSM - it has address interpolation. If you do this, please place the numbers precisely beacause searching for the number on the map to correct this is really annoying.</p>
