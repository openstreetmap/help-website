<p>Hi, Use a GPS or app that can record gpx traces of routes. Make several traces of the paths to get average lines. You can then upload the traces to OSM and use them to draw the path lines. Some help is given here <a href="https://wiki.openstreetmap.org/wiki/Upload_GPS_tracks">https://wiki.openstreetmap.org/wiki/Upload_GPS_tracks</a> need more help please just ask.</p>
