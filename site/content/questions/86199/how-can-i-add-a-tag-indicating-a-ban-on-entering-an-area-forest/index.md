<div class="answer-body">
<p>Hello! You could provide the area in question with either the <a href="https://wiki.openstreetmap.org/wiki/Tag:access%3Dno">access=no</a> or <a href="https://wiki.openstreetmap.org/wiki/Tag:access%3Dprivate">access=private</a> tag depending on the limitations of the ban. See links for more comprehensive explanations of these tags. If access is granted for service personell on individual basis I would go with <em>access=private</em>. Note that there's also a <a href="https://wiki.openstreetmap.org/wiki/Tag:access%3Dpermit">access=permit</a> option. This tag should be used in cases where a permit is required for access, but when it's routinely granted to everyone requesting it.</p>
<p>You should also provide the possible roads and paths within the restricted area with the corresponding tags.</p>
</div>
