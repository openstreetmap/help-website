<p>There is an area with this name: details:</p>
<ul>
<li>name :Fareham</li>
<li>ref: Fareham</li>
<li>source: naptan_import</li>
<li>public_transport: pay_scale_area</li>
</ul>
<p>A quick search on the wiki gave this page : <a href="http://wiki.openstreetmap.org/wiki/Tag:public_transport%3Dpay_scale_area">http://wiki.openstreetmap.org/wiki/Tag:public_transport%3Dpay_scale_area</a> Problems should be reported to "the mailing list". But I have no idea which list is meant.</p>
<p>Maybe it can be discussed on that mailing to change the name to something that includes "pay scale area" ?</p>

<p>There are loads of these Naptan pay-scale areas in OSM. The rendering is a bit silly, in that the name is rendered along the boundary, but the boundary itself is not rendered. They're UK-specific, so perhaps Talk-GB <a href="http://lists.openstreetmap.org/listinfo/talk-gb)">http://lists.openstreetmap.org/listinfo/talk-gb)</a> would be the best mailing list to raise the issue on. Or possibly Talk-transit (<a href="http://lists.openstreetmap.org/listinfo/talk-transit).">http://lists.openstreetmap.org/listinfo/talk-transit).</a></p>
