<p>For the start see <a href="https://wiki.openstreetmap.org/wiki/Deploying_your_own_Slippy_Map">https://wiki.openstreetmap.org/wiki/Deploying_your_own_Slippy_Map</a> . Although I do not have that much experience with slippy map libraries like Leaflet, I am nearly sure that limiting the viewport is possible. And if you want an own map style, you can render your own tiles based on the OSM-data.</p>
<p>For an idea of tracking systems, start by reading <a href="https://help.openstreetmap.org/tags/tracking/?sort=mostvoted">old questions about tracking</a>.</p>
