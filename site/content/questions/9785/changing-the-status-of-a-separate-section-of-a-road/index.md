<td><div class="item-right">
<div class="answer-body">
<p>In order to split a way in Potlatch so that one bit can be renamed, click the edit tab on the area concerned, then click on the way that you want to split, so that the nodes that it's made up of turn red, then on the node where you want to split the way.</p>
<p>At this point the area of the editor to the left of the map will probably read "No tags set" (because there are no tags on the node that you've selected). You can then either press "<strong>x</strong>" or click on the "<strong>scissors</strong>" icon in the toolbox.</p>
<p>After doing that, you'll see two pieces of road, both currently called the same name. One part will be highlighted, so click on the other bit if that's the one that you want to rename. The name will appear to the left of the map, change it to the new one and save, and that should be it!</p>
