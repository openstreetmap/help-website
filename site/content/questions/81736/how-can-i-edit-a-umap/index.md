<div class="answer-body">
<p>There are two ways how you can edit a map.</p>
<ol>
<li>If the map has been created while you were logged in you have to log in again and open the map.</li>
<li>If the map has been created without having been logged in you need a "secret editing link" which gives you the editing rights. The link is provided in the top bar above the map when you edit it the first time. If you lost it your only chance is to still find it your browser history or to import the map data again into a new map.</li>
</ol>
<p>In both cases there is a pencil icon in the top right corner of the map which takes you to the editing mode.</p>
