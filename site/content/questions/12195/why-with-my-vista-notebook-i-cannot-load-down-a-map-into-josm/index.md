<p>Maybe you can get any error message when you start JOSM via Windows command line interface.</p>
<p>Choose Windows Start menu, there has to be something like "execute ...", type "cmd" there.</p>
<p>when the black window with the command prompt appears, change directory to where your josm.jar is saved, then type java -jar josm-tested.jar</p>
<p>Any output there from JOSM?</p>
