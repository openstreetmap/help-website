<td><div class="item-right">
<div class="answer-body">
<p>This site is running OSQA which is Free Software - licensed GPL3+. See <a href="https://github.com/dzone/osqa/blob/master/LICENSE">https://github.com/dzone/osqa/blob/master/LICENSE</a> or <a href="https://en.wikipedia.org/wiki/OSQA">https://en.wikipedia.org/wiki/OSQA</a> . So, what is "proprietary" about it?!</p>
<p>However, if you got a good proposal for another suitable software (and a good transition plan), let us know it (maybe over <a href="https://wiki.openstreetmap.org/wiki/Talk:Help.openstreetmap.org">there</a>).</p>
