<p>The tool <a href="https://osm-boundaries.com/">OSM Boundaries</a> gives you an easy way to download administrative boundaries.</p>
<p>You can then load that data into a custom map, either one you host yourself or by e.g. tools like <a href="https://wiki.openstreetmap.org/wiki/UMap">uMap</a>.</p>
