<div class="answer-body">
<p>MapCache is available as an nginx and an Apache module; both these web servers have configuration options that would let you serve tiles only if the request matches certain conditions (e.g. has the right User-Agent or Referer header, comes from the right IP range, carries the right cookie) which you could probably use to restrict access.</p>
<p>However, your question is a MapCache or web server configuration question that has nothing to do with OpenStreetMap and does not belong on this platform.</p>
