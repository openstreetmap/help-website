<div class="answer-body">
<p>OpenStreetMap is in fact only about the data. In the old days, there wasn't even a map to view.</p>
<p>Now, some features (like search and a rendered map) have been added to make it easier for the mappers, so the data gets better quality.</p>
<p>Off coarse, routing can also be some quality check, but it asks a lot of CPU power, for maybe very little gain for the mappers. So it will only be added when there is enough time/money left to develop it and have the servers. There have been several discussions about adding it, so I don't think it will take years from now.</p>
<p>For now, you can indeed use other routers, as Cartinus said, or download data and use offline software to calculate routes (on your computer, or on a mobile device).</p>
