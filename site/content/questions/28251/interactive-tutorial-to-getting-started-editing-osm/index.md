<div class="answer-body">
<p>Hmm, I guess you mean the built-in tutorial in our editor iD. It is centred on <span>Three Rivers, Saint Joseph County, Michigan</span>.</p>
<p>To access it:</p>
<ol>
<li>log in (on the <span>OpenStreetMap.org</span> homepage)</li>
<li>start iD (on the OpenStreetMap.org homepage click the "Edit" button or, if you have set another default editor in your user preferences, click "iD" inside the "Edit ▼" menu),</li>
<li>click the help icon (currently in the right panel, bottom),</li>
<li>click the biiig "Start the Walkthrough" button below the list of help chapters</li>
</ol>
<p>Note that this is mainly a tutorial to use iD. We have other <span>editors</span>. Other tutorials would be <span>Beginners' guide</span> in our wiki or <a href="http://learnosm.org/">LearnOSM</a> (and see <a href="/questions/6/">how-do-i-get-started-with-openstreetmap</a> for more).</p>
