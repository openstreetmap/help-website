<p><a href="https://hiking.waymarkedtrails.org"></a><a href="https://hiking.waymarkedtrails.org">https://hiking.waymarkedtrails.org</a> might be part of what you're looking for, but I suspect it depends on what you're actually expecting to be in the .shp files.</p>
<p>You can also download the raw OpenStreetMap data for a particular area from e.g. <a href="https://download.geofabrik.de/">https://download.geofabrik.de/</a> . If you do that you'll see that (for example) <a href="https://www.openstreetmap.org/way/65367546">this</a> bit of track has a lot more information stored with it than just "it is part of a hiking route".</p>
