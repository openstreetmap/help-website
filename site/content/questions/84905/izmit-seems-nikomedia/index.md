<div class="answer-body">
<p>OSM's Data Working Group** get lots of complaints about incorrect place names in Windy's apps and websites such as <a href="windy.com">windy.com</a>. I believe that in all of the place name reports that we've had so far, Windy has been at fault and not OSM.</p>
<p>In this case, Windy's website actually has the <a href="https://www.windy.com/?40.731,29.982,9">correct name</a> so hopefully their app will update when they get around to it.</p>
<p>Why Windy were using the name Nicomedia I've no idea - you'd have to ask them that. The Izmit node has had the same language names for a couple of years <a href="https://osm.mapki.com/history/node/8250973366">at least</a>. Nicomedia is in OSM as the latin name for Izmit, which is correct, but there's no evidence that Windy got their incorrect name from there. A number of Windy's other incorrect names don't have an obvious source in OSM either (one example was the same city name duplicated 30km to the east).</p>
<p>** I'm a member of that group</p>
