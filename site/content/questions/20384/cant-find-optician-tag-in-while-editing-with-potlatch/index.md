<p>Hi <span><span>@Ritwick</span></span>! The simple mode of Potlatch2 just interprets/shows some of the advanced mode's data (that is what is actually written/inside our database). Similarly you can only add/edit some map features with the simple mode.</p>
<p>You solved the optician problem with <span><span>@Hendrikklaas</span></span>' help already, fine! If you use a "name" field in the simple mode Potlatch2 actually adds a key "name" and assigns a value "whateveryouvetypedintothenamefield". You can see this if you look at existing map objects via the Advanced mode. I nearly always use the advanced mode. So to add a tag for the name of the shop just add this tag in the advanced mode: name=whateverthenameis (switch to the next/right field/column instead of typing the "=").</p>
<p>See <span>Potlatch_2#How_to_use_Potlatch_2 in our wiki</span> (I would like to emphasize the very helpful keyboard use) for help with Potlatch2.</p>
