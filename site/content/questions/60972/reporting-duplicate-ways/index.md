<p>How many are we talking about? If it's just a few, it would be better to just change them by hand. If it's a lot, you could make a <a href="https://wiki.openstreetmap.org/wiki/MapRoulette">map roulette</a> challenge with the problem ways. OSM doesn't really have a single place you can report bad data to. Since it's a wiki, the idea is that you would fix it yourself, rather than reporting it to someone. If the bad data is just in one area/country, you could try contacting the local community in that area/country, they might know the source of the problem and might be interested in fixing it up themselves.</p>
</div>
