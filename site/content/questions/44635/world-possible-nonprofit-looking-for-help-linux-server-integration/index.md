<div class="answer-body">
<p>If Raspberry Pies were super powerful, that could look like this. You download <a href="http://planet.openstreetmap.org/pbf/">the complete dataset</a>, then <a href="https://switch2osm.org/serving-tiles/">generate tiles</a> based on them. Next you will need <a href="http://wiki.openstreetmap.org/wiki/Deploying_your_own_Slippy_Map">a website to show these tiles on</a>. If you want more services, you could <a href="http://wiki.openstreetmap.org/wiki/Nominatim/Installation">install a geocoder</a>. And maybe <a href="https://github.com/Project-OSRM/osrm-backend/wiki/Building-OSRM">some routing too</a>? If you want your students to be able to make data extracts, you could <a href="https://github.com/tyrasd/overpass-turbo">install a very easy data extraction tool</a> too.</p>
<p>(Just using the first option that comes to mind, for every one of these services and steps there are several options.)</p>
<p>Keeping the limited hardware in mind, you would probably be better of <a href="http://wiki.openstreetmap.org/wiki/Software/Desktop">using a desktop application</a>. There are several available for linux.</p>

<p>Hi Jeremy</p>
<p>I suspect that if you want concrete "holding hands" help you will be better off asking on the OSM dev mailing list or the #osm-dev IRC channel.</p>
<p>I'm further fairly sure that you best bet would be to run a web server off a pre-rendered tile set likely max zoom level 12 with some interesting areas to higher zooms. Alternative would be to build something based on vector tiles with client based rendering, see <a href="https://github.com/systemed/tilemaker">Richard Fairhursts tilemaker</a> for a tool that would allow you to generate a static set of vector tiles. The downside is that you either need clients that can support on device rendering or you would have to render the vector tiles on demand to images on the server (which is likely to be rather slow). The other issue is that I don't believe anybody has tried to generate a worldwide tile set with tilemaker yet, but pre-generating could naturally be done on a super-hefty machine.</p>
