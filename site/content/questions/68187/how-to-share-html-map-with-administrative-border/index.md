<div class="answer-body">
<p>It is not simple but it is possible. What you would have to do is create a personal map with UMap (umap.openstreetmap.fr) that highlights the border. You can either draw the border manually, or you can export it from OSM and import into umap, or you can even configure your umap to dynamically load the border from OSM (in case it changes in the future). Some docs are here <a href="https://github.com/tyrasd/overpass-turbo/issues/95">https://github.com/tyrasd/overpass-turbo/issues/95</a> or search for the links related to Overpass in <a href="https://wiki.openstreetmap.org/wiki/UMap">https://wiki.openstreetmap.org/wiki/UMap</a> .</p>
