<p>Yes it's possible to add presets to your toolbar, but I'm not sure if they are expended by all previous values etc:<br />
<a href="https://wiki.openstreetmap.org/wiki/JOSM/Advanced_editing#Tagging_Presets">https://wiki.openstreetmap.org/wiki/JOSM/Advanced_editing#Tagging_Presets</a></p>
<p>You might also use F3 key, as the list is ordered by your editing history, so most recent preset comes first.<br />
Also sometimes it's useful to add an object with all tags and then just using D key to dublicate the object. So you need just to adapt very few tags.</p>

<div class="answer-body">
<p>You can add multiple key &amp; value pairs in the XML file of the preset.</p>
<p>I added key1 &amp; key2 as given below in the "Crafts" preset downloaded from <a href="https://josm.openstreetmap.de/josmfile?page=Presets/Crafts&amp;preset">https://josm.openstreetmap.de/josmfile?page=Presets/Crafts&amp;preset</a></p>
<pre><code>&lt;item name=&quot;beekeeper&quot; de.name=&quot;Imker&quot; fr.name=&quot;Apiculteur&quot;&gt;
    &lt;key key=&quot;craft&quot; value=&quot;beekeeper&quot; /&gt;
    &lt;key key=&quot;key1&quot; value=&quot;value1&quot; /&gt;
    &lt;key key=&quot;key2&quot; value=&quot;value2&quot; /&gt;
&lt;/item&gt;</code></pre>
<p>Selected a POI, pressed F3 and selected beekeeper and all the 3 key value pair were added to the POI.</p>
<p>Alternatively you can copy the tags from a text file and paste them into a POI. You need to have the tags and their values something like this:</p>
<pre><code>craft=beekeeper
key1=value1
key2=value2</code></pre>
<p>Copy these text and paste it into the POI using Ctrl-Shift-V.<br />
Hope this helps.</p>
