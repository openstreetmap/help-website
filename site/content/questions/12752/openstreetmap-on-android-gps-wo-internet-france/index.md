<p>You can create one easily. Try the following steps: 1) Get the france.osm.bz2 from <a href="http://downloads.cloudsmade.com">http://downloads.cloudsmade.com</a> 2) Install osmosis tool 3) Run the mapsforge file-writer plugin on the france osm file. This will create a vector map file 4) Install Advanced mapsforge APK viewer (download from <a href="http://code.google.com/p/mapsforge/)">http://code.google.com/p/mapsforge/)</a> 5) Copy the france vector map file to the mobile device for offline use. Use the advanced map viewer to open the vector map file.</p>
