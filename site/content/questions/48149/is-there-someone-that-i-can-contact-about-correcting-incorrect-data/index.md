<p>Hi knmille7, please see <a href="https://www.openstreetmap.org/about">https://www.openstreetmap.org/about</a> for how we work (in short: from volunteers like you!). And the best thing is to fix the errors yourself. Sign up, and <a href="https://www.openstreetmap.org/welcome">you're welcome</a>. Just, please keep in mind that we are not allowed to use other databases or maps for the contributions to our data (with a few explicit exceptions).</p>
<p>If, instead, you do not want to fix yourself or if you do not know the correct data, you could use the "notes" function (<a href="https://wiki.openstreetmap.org/wiki/Notes">help on it</a>) on <a href="https://www.openstreetmap.org">https://www.openstreetmap.org</a> to report wrong data.</p>
