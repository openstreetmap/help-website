<div class="answer-body">
<p>You can use the "Follow" feature in Potlatch 2 to help with drawing things with shared ways.</p>
<p>As exaplained by Steve Bennet in <a href="http://lists.openstreetmap.org/pipermail/talk/2011-January/056239.html">this post</a> on the 'Talk' mailing list, this is how to use it:</p>
<blockquote>
<p>Say you want to map a golf course that shares half its boundary with a landuse=industrial.</p>
<ol>
<li>Start drawing the golf course</li>
<li>Click on one of the nodes of the industrial area, as normal</li>
<li>Click the next node in the industrial area, as normal. (Without this tool you would just continue doing this all the way around)</li>
<li>Press "F". The way is now automatically extended to the next node.</li>
<li>Keep pressing F as far as necessary.</li>
</ol>
</blockquote>

<p>If i needed to map a polygon now, I would avoid using shared nodes by using the parallel line tool from the Potlatch2 toolbox. So create a parallel line for each section then join and trim them, ctrl click each section to select them all to make one polygon then pick the area type of choice. name it, source note it and save it. job done.</p>
