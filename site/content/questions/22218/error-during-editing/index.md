<p>According to comments on one of the IRC channels, there was a problem on one of the servers this morning. That's been fixed but things are running a little slower than normal while cache gets back to speed again. I had "map" call timeouts like yours just a few minutes ago. If you're using an online editor, try zooming in so that each map call fetches less data.</p>
<p>Entirely coincidentally, OSM has a <a href="http://blog.openstreetmap.org/2013/05/07/openstreetmap-launches-all-new-easy-map-editor-and-announces-funding-appeal/">funding drive</a> on at the moment to make the editing experience more resilient!</p>
