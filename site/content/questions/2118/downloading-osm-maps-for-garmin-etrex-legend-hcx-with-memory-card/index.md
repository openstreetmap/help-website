<div class="answer-body">
<p>Assuming you are referring to this website: <a href="http://garmin.na1400.info/routable.php">Free routable maps for Garmin brand GPS devices</a>. Checking now, it seems it does have the tile for the area you want.</p>
<p>I assume you are choosing "South Carolina", from the drop down list. The problem is this selection doesn't work properly, so it doesn't actually select all of the tiles in that area. But you can still tick the box for "enable manual tile selection", then click on the tile with Fripp Island in to select it. Then fill in your email address, and click on "build my map", and you should get emailed a link to download it. Note: apparently the server is rather busy right now, so it might take a while until you receive the download link.</p>
<p>Also, for Cloudmade: They do have maps in Garmin format, they are the ones with "img" in the file name. eg for South Carolina, go to this page <a href="http://downloads.cloudmade.com/north_america/united_states/south_carolina#downloads_breadcrumbs">CloudMade Downloads</a>, then download the file <code>south_carolina.img.zip</code>. This is a zip file that contains a <code>gmapsupp.img</code> file, you just have to unzip this and copy it onto the eTrex memory card, in a folder named <code>Garmin</code>. Though note the CloudMade Garmin maps do not support routing.</p>
