<div class="answer-body">
<p>note that using a service which is run by someone else is different from the data license:</p>
<ul>
<li>Using the service is covered by the usage policy of the person/organisation who runs the service. If possible you should (in case of much usage) run your own service – the <span>code</span> is under a GPLv2 license.</li>
<li>What you can do with the returned results is covered by the data license: ODbL. Please read the full license text and consult a lawyer if you need an evaluation of your own situation/use case. There have been many questions about using OSM data in commercial products in the past, please read over there: <a href="https://help.openstreetmap.org/tags/commercial/?sort=mostvoted">https://help.openstreetmap.org/tags/commercial/?sort=mostvoted</a></li>
</ul>
