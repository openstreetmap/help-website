<p>What is the best way to tag a contributing project name or website url so it is possible to track total user contributions from that particular project/website? Reporting these statistics can help with aiding future project funding and therefore increase OSM contributions.</p>

<div class="answer-body">
<p>The best way seems to add a proper tag to the <a href="https://wiki.openstreetmap.org/wiki/Changeset">changesets</a>, either as <em>comment</em> or <em>source</em> tag.</p>
<p>Please <em>don't</em> track your contributions by adding these tags to the OSM element! Each element can be edited by multiple users and usually has multiple versions. Yet adding a <em>source</em> tag to an element will only refer to one specific version or a specific set of tags, invalidating it over time. Add it to the changeset instead.</p>
</div>
