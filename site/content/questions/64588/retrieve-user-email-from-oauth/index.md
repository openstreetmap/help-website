<div class="answer-body">
<p>Nope</p>
<p>We do not provide e-mail addresses as part of the API or in any other form. This is something we guarantee our contributors <a href="https://wiki.osmfoundation.org/wiki/Privacy_Policy#Email_Addresses">https://wiki.osmfoundation.org/wiki/Privacy_Policy#Email_Addresses</a> and is very unlikely to change (for numerous, including legal, reasons).</p>
<p>There have been a couple of ideas around enabling messaging from 3rd party apps to the internal message system, see for example <a href="https://github.com/openstreetmap/openstreetmap-website/pull/1431#issuecomment-295668951">https://github.com/openstreetmap/openstreetmap-website/pull/1431#issuecomment-295668951</a> for a solution that would protect the privacy of our users. If you feel like implementing it, I'm sure a PR would be welcome.</p>
