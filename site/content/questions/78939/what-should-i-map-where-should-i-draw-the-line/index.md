<p>Most people who continue mapping soon find that there's only so much one can do at anyone one time: so there tends to be some self-limiting selection. That being said this might mean mapping everything in a small area, or going for some set of more significant features in a larger area, or focusing on a particular theme (hiking paths, power networks, etc).</p>
<p>An advantage of this self-limitation is that there comes a time when one wants to add more info &amp; revisiting &amp; revising earlier work keeps OSM fresher &amp; more up-to-date.</p>
