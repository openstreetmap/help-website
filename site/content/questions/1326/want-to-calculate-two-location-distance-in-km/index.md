<p>That depends what you mean by 'distance'. Distance can be the shortest path on the earth, much like a plane or a ship would use. You have many such calculators already like <a href="http://www.nhc.noaa.gov/gccalc.shtml">this</a>.</p>
<p>The other way to define distance is the shortest path along roads. This is a much more difficult task since you would have to calculate the route to take. You can find these to like <a href="http://maps.cloudmade.com/">this</a>.</p>
