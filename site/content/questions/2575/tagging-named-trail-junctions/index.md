<p>In mapping trails/paths here in Northern California, I've encountered a few instances where the junction of trails ("Six Trails" "Boy Scout Junction") is named. The name is usually referenced in mileage signs, as well as at the place itself, so a node seems appropriate. What's the best tag? I've experimented with place=locality, but wonder if there's anything more appropriate.</p>

<p>You can tag the intersection node as <code>junction=yes</code>, plus <code>name=</code> or <code>ref=</code> as appropriate. See <a href="https://wiki.openstreetmap.org/wiki/Key:junction">key:junction</a></p>
