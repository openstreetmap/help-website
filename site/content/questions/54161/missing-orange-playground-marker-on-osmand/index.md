<div class="answer-body">
<p>It seems as if you are displaying raster map tiles from somewhere as a background in OsmAnd, -not- data directly rendered in OsmAnd. Your problem is simply that the OsmAnd map data you are using, and which is used to display the POI markers, doesn't contain the newly added playground yet.</p>
<p>Verified with the current "official" OsmAnd map file for CH. Likely the OsmAnd map data from SOSM, see <a href="http://planet.osm.ch/">http://planet.osm.ch/</a> will show the POI.</p>
