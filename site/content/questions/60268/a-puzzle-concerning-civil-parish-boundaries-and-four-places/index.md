<p>I have just checked the others, and they are all admin_centres of parishes, except for one which is merely a "label." That parish is governed by a joint parish council, with an admin centre that is actually in another parish. A "label" node is used as a hint in case anyone wants to display a name of the individual parish, but it is probably rarely used.</p>
<p>The nodes you have found are not duplicates, but represent a place, i.e. a named settlement, which has a role in respect of the civil parish.</p>
<p>Does this provide sufficient explanation?</p>

<p>You are right that the data is probably/possibly incomplete if there are missing admin_centres, although there are cases when no admin centre can be assigned. Some parishes have so few electors that they don't have a council (just a so-called Parish Meeting). In that case there cannot be a "council office" so there also cannot be an admin centre. In other cases (such as Aston Tirrold/Upthorpe) a single council covers multiple parishes (which still exist individually in law) so there is only one admin_centre for the whole group of parishes.</p>
