<p>Hi, I wanted to add a construction site to an existing road. Position: Germany, 15834 Rangsdorf, Kienitzer Straße / Am Stadtweg. This junction has a temporary construction site blocking the road from 2011-10-24 to 2011-12-31.</p>
<p>I know the tags to use, but am unable to mark a section of an existing road ad add the tags to that section. I can add two new nodes on this road but when I shift-click from one new point to the other and tag it the new road segment is added to the existing one instead of replacing it.</p>
<p>I'd really like to see a simple way to add temporary blocked road to enable Skobbler to route correctly.</p>
<p>Please tell me how to do it right and please correct the existing entry. Thanks.</p>

<div class="answer-body">
<p>If the road is not closed, I would probably leave it as it is.<br />
</p>
<p>If the road is closed, I would consider the following:</p>
<p>Split the section that is closed for construction by selecting the end node and pressing 'x' to split. Do this for both ends of the construction area if needed.</p>
<p>Select the construction way and where highway=primary, change to highway=construction and construction=primary. Use other values if 'primary' is not appropriate.<br />
</p>
<p>Consider leaving the construction details / dates in a note tag for other mappers.</p>
<p>Be sure to go back and re-open the road when construction is complete.<br />
</p>
