<p>To check whether your edits have been saved correctly to openstreetmap: go to your user page on <a href="http://openstreetmap.org">openstreetmap.org</a>, and click the link for edits. ie for you it will be this page: <a href="http://www.openstreetmap.org/user/bdwillet/edits">http://www.openstreetmap.org/user/bdwillet/edits</a></p>
<p>From there, I can see that you have uploaded one changeset (a group of edits) from ArcGIS. I see this contains 10 nodes, but none of these nodes have any tags. For OpenStreetMap, you need to give objects tags to specify what sort of object it is. Otherwise there is no way for the map to know how to render it. Also note that not all tags are rendered on the main map on <a href="http://openstreetmap.org">openstreetmap.org</a>. And this map can sometimes take a while to update, so your edits may not appear immediately.</p>
<p>I'm not sure what you are trying to map, or how you add appropriate tags in the ArcGIS editor. I would suggest to start with you read the <a href="http://wiki.openstreetmap.org/wiki/Beginners_Guide_1.3">Beginners Guide</a>, and try using one of the popular editors, ie Potlatch 2 or JOSM. For how to use these, see the <a href="http://wiki.openstreetmap.org/wiki/Potlatch_2/Primer">Potlatch 2 Primer</a> or the <a href="http://wiki.openstreetmap.org/wiki/JOSM/Guide">JOSM guide</a>. This will give you a general idea of how editing OSM works, and if you have any problems you can get help here.</p>
