<div class="answer-body">
<p>There are multiple correct ways to map addresses, at least the following:</p>
<ul>
<li>add the information to the building outline</li>
<li>add it to a node enclosed by the building outline (best solution if there are multiple addresses in the same building)</li>
<li>add the information to an entrance node on the building outline</li>
</ul>
<p>Naturally this depends further on how addresses are allocated in the area in question, in some countries they are allocated per entrance, in others per building and in again others per plot.</p>
<p>In the end data consumers have to be able to deal at least with all the above variations so simply use what fits best.</p>
