<div class="answer-body">
<p>Most power line networks are above ground, and therefore fit well in OSM: features are verifiable by other mappers. Much of the fiber network is below ground, and would therefore need to come from an import. It would be important to check the license of the data source and work with the imports mailing list for this process.</p>
<p>Recording the topology as a network schematic - where the lines represent a virtual connection but have no physical counterpart, is not a customary use of OSM.</p>
