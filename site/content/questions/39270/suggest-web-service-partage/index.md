<p>Bonjour,</p>
<p>Est ce qu'il serait possible d'avoir une amélioration de la fonction partage sur le site. j'utilise et d'autre utilisateur demande si il serait possible de pouvoir personnalisez la zone de sélection en mettant notre taille d'image directement au lieu de déplacer le curseur sur la carte. ceci permettrai de gagner du temps pour le partage.</p>
<p>2e point est ce qu'il serait possible pour cette même fonction de sauvegarder une taille d'export sur son compte.</p>
<p>Cordialement.</p>
<p>translate --------------</p>
<p>Hello,</p>
<p>Is it be possible to have an improved "share" function on the "OpenStreetMap.org" website? I and another user were wondering if it would be possible to customize the selection area by putting the image size directly instead of moving the cursor on the map. This will make sharing quicker.</p>
<p>Also, would it be possible for "export" to work in the same way, to be able to define an "export size"?</p>

<p>This is not the best place for feature requests for the main web site.</p>
<p>Please see <a href="https://github.com/openstreetmap/openstreetmap-website">https://github.com/openstreetmap/openstreetmap-website</a> for that. The best way to get results is to set up an instance of the software yourself (see <a href="https://wiki.openstreetmap.org/wiki/Develop)">https://wiki.openstreetmap.org/wiki/Develop)</a> and build at least a prototype of the functinality yourself.</p>
