<p>This sort of thing is usually a Flash Player issue (good old Adobe, we love them dearly).</p>
<p>Chrome, I believe, actually has a built-in Flash Player, which is probably why it's behaving differently to the one with IE. If you go to the <a href="http://www.adobe.com/software/flash/about/">Adobe website</a> you'll see a little box telling you what version that is. Could you let us know? We can then test it out, see what's happening and try and fix it.</p>
