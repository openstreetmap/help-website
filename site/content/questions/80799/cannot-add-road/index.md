<p>I rarely use iD but it looks to me like the presets have failed to load somehow. Try reloading the page to see if this clears it. If you have work that you don't want to lose then you can manually add <code>highway</code> and <code>track</code> into the two boxes underneath the <em>Tags</em> heading and upload before reloading. To me this actually looks like it is <code>highway=service</code> and <code>service=driveway</code> rather than <code>highway=track</code> though so <code>service</code> may be more appropriate for the box on the right, with an additional row added for the second tag. <a href="https://wiki.openstreetmap.org/wiki/Highway_Tag_Africa">These</a> <a href="https://wiki.openstreetmap.org/wiki/East_Africa_Tagging_Guidelines">pages</a> have some Africa specific tagging examples.</p>
