<p>Your question is somewhat brief, but I'm assuming that:</p>
<ul>
<li>You're using the main openstreetmap.org website</li>
<li><del>You found an object (street, building, etc) using the search function, which highlights it on the map</del> You added a marker to the map using the "include marker" toggle (in the "Share" menu)</li>
<li>You exported a picture of the map (as JPG, using the "Share" menu), but the marker is not visible <del>object is not highlighted</del></li>
</ul>
<p>Is this correct? If not, please update your question.</p>
<p>If so, then I'm afraid that the export function removes markers. If you'd like a map with an object marked, then I suggest you take a screenshot. Alternatively, you can use a service like <a href="https://umap.openstreetmap.fr">UMap</a> to create a custom map with your own markers.</p>

<p>Hi Ralfo9380,</p>
<p>When you click the "Share" button, there are three different options for sharing:</p>
<ul>
<li>Link or HTML</li>
<li>Geo URI</li>
<li>Image</li>
</ul>
<p>The "<strong>Include marker</strong>" checkbox is part of the "Link or HTML" sharing -- it will generate a special version of the <a href="https://openstreetmap.org">https://openstreetmap.org</a> map URL which will include a blue marker at the center of the map.</p>
<p>The "<strong>Download</strong>" button at the bottom is part of the "Image" sharing. It creates an image file of the current map for you to download -- but it does not include the marker, since that's part of the "Link or HTML" share function.</p>
<p>If you want an image file that includes the marker, I'd suggest you just generate the link using the "Link or HTML" function, go that page, and take a screenshot.</p>
