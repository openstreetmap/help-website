<p>I'd second that but would make a couple of comments - "4wd_only" arose out of common Australian signage, so it's a clearly <a href="https://wiki.openstreetmap.org/wiki/Verifiability">verifiable</a> tag - something that all tagging should aspire to be.</p>
<p>Also, don't forget to add a surface description (whether or not "4wd_only=yes" is appropriate), as it will help users of all vehicles and none decide whether a particular road or track is for them.</p>
