<div class="answer-body">
<p><a href="https://extract.bbbike.org/">https://extract.bbbike.org/</a> provides extracts in many different formats and styles. An export for all of Saudi Arabia may however bee too large, which it definitely is for the export function on openstreetmap.org</p>
<p>Other options would include:</p>
<ul>
<li>getting an extract of Saudi Arabia (for example from GeoFabrik) and converting it to something printable yourself</li>
<li>contracting a commercial service provider (<a href="https://switch2osm.org/providers/">https://switch2osm.org/providers/</a>) to make a print out.</li>
</ul>
