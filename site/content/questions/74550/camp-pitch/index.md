<div class="answer-body">
<p>The map you see on osm.org does not show all possible tags. tourism=pitch is probably one of them. Please do not start fiddling with the tags so some lines appear in that style, we call that <a href="https://wiki.openstreetmap.org/wiki/Tagging_for_the_renderer">Tagging for the render</a>.</p>
<p>In some cases the tags are shown on other maps, but I do not know one that shows tourism=pitch.</p>
<p>You could try to create an issue on <a href="https://github.com/gravitystorm/openstreetmap-carto/issues">github</a>, but in the end the maintainers decide what they want to show on "their" map.</p>
</div>
