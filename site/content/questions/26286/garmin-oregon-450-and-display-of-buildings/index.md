<div class="question-body">
<p>I am trying to figure out how to use US maps in the city, especially display of buildings on my Garmin Oregon 450. First I tried maps from <a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/">http://daveh.dev.openstreetmap.org/garmin/Lambertus/</a> which does not seem to display any buildings although this was early in my investigation so I can be wrong here. Then I downloaded complete map of the US from osmmaps.com and experimented a bit. I realized that the display of the buildings depends on the active profile. Here are screenshots of the same area with different profiles.</p>
<p>Here is the Automobile profile:</p>
<p><img src="/upfiles/457.bmp" alt="alt text" /></p>
<p>and here is the Recreation profile:</p>
<p><img src="/upfiles/520.bmp" alt="alt text" /></p>
<p>and here is the Fitness profile which interestingly looks like the Automobile profile, at least in terms of buildings:</p>
<p><img src="/upfiles/622.bmp" alt="alt text" /></p>
<p>and finally Geocaching profile which I use most:</p>
<p><img src="/upfiles/589.bmp" alt="alt text" /></p>
<p>Each of the profiles has the same settings of maps (map selection, level of detail).</p>
<p>I can see that the buildings are there even on the last Recreation and Geocaching profiles however on the GPS screen itself the soft and bleak colors make it absolutely invisible (the above are screenshots taken with the Screen Capture feature of the GPS).</p>
<p>I have also tried with maps generated from <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a> and the effect is the same. The interesting thing is that I have no problem with maps of Czech republic generated from <a href="http://garmin.openstreetmap.cz/">http://garmin.openstreetmap.cz/</a> as the buildings are displayed just fine even in Geocaching profiles.</p>
<p>Has anybody encountered the same problem ? I did not find any way how to modify this kind of detailed setting anywhere in the GPS Setup menu. It seems that the profiles can be only cloned.</p>
<p>Or maybe there is some better forum to ask this question ?</p>
<p>update 10/2014: I tried this again (with Geocaching profile) and found out that so far the best is Generic with TYP file.</p>
<p>new style:</p>

<p>It depends if the maps use a TYP file or not. Without a typ file, buildings are almost invisible on most garmin devices, maybe except older ones. Try the generic new maps from <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a> it uses a typ file, or use the old/default style with mapnik typ file included.</p>

<p>The map display may be set up differently in each profile. You can change it in any one profile, when in that profile, by selecting:- setup: map: advanced map Setup: you then have four options, all will modify how a map is displayed, BUT I suggest Detail: Most.... See if that works. The Czech map is probably created with a different rendering style and that makes it look sharper.</p>
