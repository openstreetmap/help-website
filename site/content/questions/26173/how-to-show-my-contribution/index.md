<p>You can use the <a href="https://wiki.openstreetmap.org/wiki/Wikidata%2BOSM_SPARQL_query_service">new SPARQL service</a> that combines OSM and Wikidata. This example shows a map of all OSM objects with the last edit by the given user, made in a certain time period. Different object types (nodes/ways/rels) are shown in different color, and can be individually hidden/shown.</p>
<pre><code>#defaultView:Map
SELECT * WHERE {
&#10;  ?osmId osmm:user &#39;nyuriks&#39; ;
         osmm:loc ?loc ;
         osmm:type ?layer .
&#10;  # Filter by timestamp
  ?osmId osmm:timestamp ?ts .
  FILTER (&quot;2017-03-01T00:00:00Z&quot;^^xsd:dateTime &lt; ?ts &amp;&amp;
          ?ts &lt; &quot;2017-03-05T00:00:00Z&quot;^^xsd:dateTime)
}</code></pre>
