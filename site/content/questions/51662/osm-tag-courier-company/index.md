<div class="answer-body">
<p>I think there is not really an established tag for this. I would use <span><code>office</code></span><code>=courier</code> or <a href="https://wiki.openstreetmap.org/wiki/Key:shop"><code>shop</code></a><code>=courier</code>. I think <code>shop</code> would be more suitable if is something where customers really go to (on premises) and <code>office</code> if it is just an office with customers just phoning them.</p>
<p>See <a href="https://taginfo.openstreetmap.org/search?q=courier#values">https://taginfo.openstreetmap.org/search?q=courier#values</a></p>
