<div class="answer-body">
<p>If you find the image on Wikimedia commons there should be options to "Use this image in another wiki" these will give you pre-formatted markup for the markup editor.</p>
<p>For the visual editor you just need the raw name of the image. Choose insert image &amp; it offers "recently uploaded images" &amp; a search box. Place the name of the wikimedia image in the search box and it should appear. Various options are available in the advanced settings tab.</p>
