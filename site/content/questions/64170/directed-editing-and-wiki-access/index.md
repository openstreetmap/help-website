<div class="answer-body">
<p>What seems to have happened is that a page that started out on the <a href="https://wiki.openstreetmap.org/">OSM wiki</a> which anyone can edit has been moved to the <a href="https://wiki.osmfoundation.org/wiki/Directed_Editing_Policy">OSM foundation's wiki</a> without some of the context being clarified. When it says "create a wiki page" it means within the OSM wiki, which anyone can edit.</p>
<p>Just in case there's any doubt, the page at OSMF that you linked to is still undergoing consultation - there are many different communities within OSM and the way that they map things are different, which is partly why the policy has had such a long gestation period.</p>
