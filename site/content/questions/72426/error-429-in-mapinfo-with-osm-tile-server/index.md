<p>This is likely a side effect of the replacing Apache with nginx (for the tile caches) which is slightly more stricter with rejecting request that have missing required headers. With other words you probably need to get this fixed by your software vendor.</p>
<p>A little birdy has indicated that MapInfo Pro 16.0.4 might fix your issue.</p>
