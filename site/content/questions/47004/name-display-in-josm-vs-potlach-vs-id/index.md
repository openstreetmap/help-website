<div class="answer-body">
<p>You choose to use the tag <em>name:en</em> instead of <em>name</em> for <a href="https://www.openstreetmap.org/way/320361190">your building</a>. <em>name:en</em> is the name in English (see <a href="https://wiki.openstreetmap.org/wiki/Names#Localization">localization</a>). This seems to be fine in this case since the local language is non-English.</p>
<p>As you can see, some tools can handle the missing default name and some can't. If you know the local name of the apartments then please add it, too.</p>
