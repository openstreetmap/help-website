<p>To change the name of a garmin map image file the imo easiest way is to use <a href="http://www.gmaptool.eu/gmaptool.html">gmaptool or short gmt</a>.<br />
The command to use would be <em>gmt -w[option]</em>. Which options exist you can list by <em>gmt -w?</em>.</p>
