<p>It seems that this message was recently added to the data save feature because especially in Germany there have been several cases in the last past weeks where new users were not aware that they edited on the main data set of the OSM server when they deleted unwanted objects.</p>
<p>They thought their edits were only saved "locally" to their own map view.</p>
<p>Several changesets had to be reverted which isn't always easy.</p>
