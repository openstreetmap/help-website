<div class="answer-body">
<p>If the Garmin 350 uses the Garmin .img files you can create these with <a href="http://www.mkgmap.org.uk/page/main">mkgmap</a> from OpenStreetMap data. If you download an area of data, such as from <a href="http://download.geofabrik.de/openstreetmap/">Geofabrik downloads</a>, mkgmap can use it to create the .img files for your Garmin. If the area is too big for a single .img file you can use <a href="http://www.mkgmap.org.uk/page/tile-splitter">splitter</a> and then mkgmap to break the map data into chunks. The style of the Garmin map can be changed too.</p>
</div>
