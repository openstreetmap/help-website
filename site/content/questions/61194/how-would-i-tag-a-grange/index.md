<div class="answer-body">
<p>If it looks like a barn, map it as building=barn (or building=granary_barn).</p>
<p>Add a node for the "church": amenity=place_of_worship + religion + denomination and a node for the "community hall": amenity=community_centre inside the building.</p>
<p>The building tag refers to the type of building, not its usage.</p>
