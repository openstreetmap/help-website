<p>In several of my websites I implemented some level of integration of OpenStreetMap by dynamically generating javascript in PHP upon page requests (mostly setting call parameters of JS functions dependent on some calculation in PHP) and by dynamically generating kml file from PHP that are then used by OpenLayer vector layers. For an example see <a href="http://pgc.petschge.de/index.php">http://pgc.petschge.de/index.php</a> and linked files. Source is available by appending .txt to the filename.</p>
