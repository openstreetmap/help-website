<div class="answer-body">
<p>If there are convex edges from the corners that don't all meet at the same point then hipped.</p>
<p>The exception to this would be if there appears to be a change in slope part way up which would make it mansard.</p>
<p>See the illustrations <a href="https://wiki.openstreetmap.org/wiki/Key:roof:shape">here</a> for a nice summary.</p>
