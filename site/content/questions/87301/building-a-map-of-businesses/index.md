<p>You're not saying what platform you're using to make these pins - is it "umap" by any chance?</p>
<p>On OpenStreetMap we'd have some rules about what you can add to our map - a pickle maker shipping from dad's garden shed without being publicly visible would usually not be considered worthy of inclusion. However, if you are using umap then you can place pins wherever you want - it's your own map and it doesn't affect OpenStreetMap as a whole.</p>
