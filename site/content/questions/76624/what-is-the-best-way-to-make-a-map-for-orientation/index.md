<div class="answer-body">
<p>As an online slippy map there is for example <a href="https://www.opentopomap.org">OpenTopoMap</a>.</p>
<p>An offline map usable with apps like Locus or Orux could be obtained from <a href="https://www.openandromaps.org/">Openandromaps</a>.</p>
<p>There are certainly more out there. Browse through the <a href="https://wiki.openstreetmap.org/wiki/List_of_OSM-based_services">Wiki</a> for others.</p>

<p>There is no way of changing the appearance in a slippy map. It's all pre-rendered images loaded form the server. Maybe you find a map that looks more the way you like in the list on the Wiki.</p>
<p>A map using vector graphics could do what you want, but I don't know any topo online map from the top of my head.</p>
<p>Possibly some of the phone apps like Locus, Orux or OSMand can do it but I haven't checked.</p>
