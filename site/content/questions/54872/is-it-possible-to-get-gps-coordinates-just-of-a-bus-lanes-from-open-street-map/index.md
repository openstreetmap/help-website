<div class="answer-body">
<p>Before you attempt to query information from OSM, you should learn a bit about the tagging schema. What do you mean with a bus lane ? One the the examples you see on <a href="https://wiki.openstreetmap.org/wiki/Bus_lanes">this wiki page</a> ? Or a street (or service road) that can only be used by busses ? In the later case it might be tagged as highway=service;access=no;bus=yes (or some variations on the access part) Or do you mean bus routes ?</p>
<p>When you know how something is tagged, you can use e.g. <a href="overpass-turbo.eu">Overpass-Turbo</a> to extract data. Alternatively you can download extracts from cities, countries or the whole planet file and manipulate the data locally with e.g. osmfilter or put everything in a PostGis database and run queries there.</p>
<p>You can also download OSM data directly into QGIS and do manipulations in that software.</p>
<p>It all depends on the geographical area, the frequency with which you want to update the data and the use you will make of the "GPX coordinates" of the bus lanes.</p>
<p>Please note that depending on the country/city you are interested in, the information might be complete and up-to-date, a bit out-of-date all the way to non-existing (not mapped so far) in a particular area.</p>

<p>OSM is contributor driven. Any contributor can add the information they want. There is no board that dictates that you have to add this or that information. So you could add this information as well (preferable after visiting a place and not by importing it from some government database.</p>
<p>As I said, a road dedicated to busses could be mapped with highway=service; access=no ; bus = yes. This would mean that no other vehicle nor pedestrians are allowed on that road. But in case cyclists and pedestrians are allowed, it could be mapped as motor_vehicle = no ; bus = yes. Read the <a href="https://wiki.openstreetmap.org/wiki/Key:access">wiki page on Access</a> for more information. It depends on the local situation which access tags will be used. It is also possible that the mapper used highway=residential instead of service. O yes, people might have used psv instead of bus as well.</p>
<p>I also wonder how many roads are dedicated to busses. It might depend on the area you are interested in.</p>
<p>So with some OverPass query you can retrieve the ways. the nodes with have a long/lat. The query will have to cope with several different tagging possibiities (see above).</p>
