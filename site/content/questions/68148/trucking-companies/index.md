<p>There is a tag landuse=depot which does get used for bus depots and truck company depots. It doesn't fulfil every criterion, but is a useful start (particularly as such places can often be deduced from aerial imagery). Also there is company=logistics, and more generic approaches using industry_sector (ie., things like SIC codes).</p>
<p>However, it is well known that OSM tagging is not very rich when it comes to many blue-collar occupations.</p>
