<p>I have not used that model of Garmin, You could try one from <a href="http://garmin.opentopomap.org/">http://garmin.opentopomap.org/</a> i have not used them much in a car. You will probably find map search or lat lon easier to search for as most downloads don't seem to be set up for address searching. There is a download link on the opentopomap page. Do not download direct to the device as you may overwrite your existing map. USE a Memory card in an adapter. Assuming that that Garmin as a slot and maps can work from it.</p>
