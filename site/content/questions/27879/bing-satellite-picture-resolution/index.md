<div class="answer-body">
<p>If the Bing imagery isn't good enough where you live (and even if it is) you'll be able to contribute much more by collecting GPS data and using that. See the <a href="https://wiki.openstreetmap.org/wiki/Sv:Beginners%27_guide">beginner's guide (Nybörjarguide)</a>, especially the <a href="https://wiki.openstreetmap.org/wiki/Sv:Beginners%27_guide_1.2">GPS-data</a> section.</p>
<p>There don't seem to be many Swedish users of this help site, but the <a href="http://forum.openstreetmap.org/viewforum.php?id=4">Swedish users' forum</a> does seem quite popular.</p>
<hr />
<p>Och på svensk:</p>
<p>Om Bing bildspråk är inte bra nog där du bor (och även om det är) kommer du att kunna bidra med mycket mer genom att samla in GPS-data och använda det. Se <a href="https://wiki.openstreetmap.org/wiki/Sv:Beginners%27_guide">Nybörjarguide</a>, speciellt <a href="https://wiki.openstreetmap.org/wiki/Sv:Beginners%27_guide_1.2">GPS-data</a> sida.</p>
<p>Det inte tycks finnas många svenska användare av denna hjälpen webbplatsen, men <a href="http://forum.openstreetmap.org/viewforum.php?id=4">de svenska användarnas forum</a> verkar ganska populärt.</p>
