<div class="answer-body">
<p>You can also select lots of objects and then add tags to all of the selected object in one go.</p>
<p>Search for</p>
<ul>
<li>"id:0 type:way" will select all ways you have added since you started the program.</li>
<li>"id:0 type:way untagged" to find all newly created ways that are still untagged.</li>
</ul>
<p>Or just select them manually.</p>
<p>The <strong>Duplicate variant</strong> you mention works like this: if you want to create a new object with the same tags you press "ctrl-d", and you will get a node/way with all tags of the selected object at your mouse position.</p>
