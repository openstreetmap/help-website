<p>Assuming you're looking for a "traditional" slippy map, there are a number of "setting up your tile server" examples, such as <a href="https://switch2osm.org/manually-building-a-tile-server-16-04-2-lts/">this one</a> at switch2osm.org.</p>
<p>You'll also want to update your tileserver as people edit OSM, which you can do by following the instructions <a href="https://wiki.openstreetmap.org/wiki/User:SomeoneElse/Ubuntu_1604_tileserver_load#Updating_your_database_as_people_edit_OpenStreetMap">here</a>.</p>
