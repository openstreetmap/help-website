<p>If using iD editor, the answer to that question is 'yes'. If you create a field ref and set it to the #, you will see you have a tag ref set to that number listed below and making a tag should expand its equivalent field above the tags too; think of it as a nearly redundant way of expressing the same thing. The field can be more convenient to work with for individual and larger tags but the tags area groups them all together and presents them in a smaller space so you won't need to scroll so much as the number of created tags and their contents grows.</p>
