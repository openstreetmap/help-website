<div class="answer-body">
<p>The 'nxt' tag is a still a mystery to me, nevertheless, I shall press on....</p>
<p>If you want to download gpx tracks to your mobile phone or Garmin or similar and use these for hiking/cycling etc then the openstreetmap.org site is not ideal as many of the tracks that have been uploaded to this site have been created and uploaded primarily to assist with editing the map.</p>
<p>The only way I know it can be done is using JOSM by right-clicking on the 'downloaded GPX data'&gt;'choose visible tracks' and if the one you choose seems suitable, then you can load it into a browser if it has a url and then download.</p>
<p>You can also select track segments that are already mapped, save to a new layer and then save as gpx. Here is a <a href="http://www.youtube.com/watch?v=vl9nlBEI0as">video</a> I just spotted. Don't delete stuff on the osm...just select and copy.<br />
</p>
<p>It is much easier to use the sites such as the following examples that are designed for displaying downloadable gpx tracks</p>
<p><a href="http://en.wikiloc.com/wikiloc/map.do#lt=14.357775&amp;ln=121.337311&amp;z=9&amp;k=1&amp;tab=0&amp;a=">Wikiloc</a><br />
<a href="http://www.everytrail.com/world_map.php">EveryTrail</a><br />
<a href="http://ridewithgps.com/find#search/0/search%5Boffset%5D=0&amp;search%5Bstart_distance%5D=50&amp;search%5Bstart_location%5D=manila&amp;search%5Bkeywords%5D=&amp;search%5Blength_min%5D=0&amp;search%5Blength_max%5D=500&amp;search%5Belevation_min%5D=0&amp;search%5Belevation_max%5D=10000&amp;search%5Bsort_by%5D=">Ride with gps</a></p>
<p>The new <a href="http://labs.strava.com/heatmap/#12/121.09293/14.69590/yellow/bike">Strava heat maps</a> are great for looking for popular routes (takes a while to load) but you can't download a track directly AFAIK but you can use it as a guide when drawing your own track in another program such as <a href="http://bikeroutetoaster.com/">Bike Route Toaster</a></p>
