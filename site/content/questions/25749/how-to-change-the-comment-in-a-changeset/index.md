<div class="answer-body">
<p>Changeset comments (and all other changeset tags) cannot be changed afterwards by users. Unfortunately they can only be changed by manual database edits by the system admins (and that is not done under normal circumstances since it is taking too much time). If the comment would be very misleading then another edit to the relevant objects could be made with a correcting comment.</p>
<p>For general changeset reversion (the changes to the data), see <a href="/questions/731/">how-can-i-revert-a-changeset</a>. However, all known methods for this are not easy and should only be done with former experience with the used tool.</p>
