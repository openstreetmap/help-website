<p>The save popups in P2 suggest that you save after short edits. I split riverbank polygons into small polygons, this is helpful when the bank get used again for parks,woods or other polygons and if they get broken as they are easier to fix then. Scrolling across several screen widths also requires more data to be downloaded as well, which you end up doing to get back to the start of the polygon.</p>
