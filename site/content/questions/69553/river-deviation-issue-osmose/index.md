<p><strong>"The deviation is caused by sharp angle on nodes. By refining the way it should deseaper automatically on next refresh."</strong></p>
<p>So that is the thing. So when there is a sharp angle on a river path, Osmose detect it as an issue. The path has to be smooth as river paths are suppose to be smooth.</p>
