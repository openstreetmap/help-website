<p>Hi.</p>
<p>If the point is to display a big PNG as a slippy map, and allow users to draw polygons on it, <a href="https://leafletjs.com/">leaflet</a> might be a simple solution.</p>
<p>You can use an <a href="https://gis.stackexchange.com/questions/243497/image-as-a-background-map-in-leaflet">image as a background</a>. And there are a lot of <a href="https://leafletjs.com/plugins.html">plugins</a>, for example for <a href="https://github.com/geoman-io/leaflet-geoman">advanced edition</a>.</p>
<p>I might be available to help, but I would need a lot more details on your project. PM me through OSM (H@mlet).</p>
