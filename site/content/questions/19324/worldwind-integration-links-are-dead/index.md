<p>Our documentation page <a href="https://wiki.openstreetmap.org/wiki/World_Wind">https://wiki.openstreetmap.org/wiki/World_Wind</a> links to <a href="http://2bn.net/files/2/osm.xml">http://2bn.net/files/2/osm.xml</a> which is a dead link (<a href="http://web.archive.org/web/*/http://2bn.net/files/2/osm.xml">since Summer 2010</a>) and mentions Telascience. The <a href="http://web.archive.org/web/20100618081938/http://2bn.net/files/2/osm.xml">last working version</a> points to <a href="http://t2.hypercube.telascience.org/tiles">this tile link</a>, which is dead, too. So, this page needs updating with the info found out here (please leave a comment if done). Btw: <a href="http://www.worldwindcentral.com/wiki/Add-on:OpenStreetMap">http://www.worldwindcentral.com/wiki/Add-on:OpenStreetMap</a> is also old.</p>
