<td><div class="item-right">
<div class="answer-body">
<p>For individual pumps use <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dcompressed_air"><code>amenity=compressed_air</code></a>. Ideally specify the <code>valves=*</code>.</p>
<p>For pumps that are part of a <a href="https://wiki.openstreetmap.org/wiki/Tag:shop%3Dbicycle"><code>shop=bicycle</code></a> use <code>service:bicycle:pump=yes</code>.</p>
</div>
