<div class="answer-body">
<p>Unfortunately, Creative Commons Attribution 4.0 International License is not enough for import in OSM, mainly because we can't enforce attribution, if I understand correctly.</p>
<p>You have to request a specific authorization from the data publisher.</p>
<p>The list of already accepted requests is on the <a href="https://wiki.openstreetmap.org/wiki/Contributors#New_Zealand">wiki</a>, there is a specific <a href="https://wiki.openstreetmap.org/wiki/LINZ">LINZ page</a>.</p>
<p>Before any large-scale import, please be aware of the <a href="https://wiki.openstreetmap.org/wiki/Import">import guidelines</a>, it's not a task to be taken lightly.</p>
<p>You should first synchronize with the NZ community, for example through this <a href="https://wiki.openstreetmap.org/wiki/New_Zealand_data_import_projects">page</a>.</p>
<p>Good luck.</p>
