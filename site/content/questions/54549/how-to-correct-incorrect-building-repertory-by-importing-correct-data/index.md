<div class="answer-body">
<p>You can't use this data as far as I know. The "Geobasisdaten des Kantons Basel-Landschaft" are not compatible with OSM's <a href="https://wiki.osmfoundation.org/wiki/Licence">ODbL licence</a> according to the <a href="http://www.geo.bl.ch/fileadmin/user_upload/Geodaten/Nutzungsbedingungen_GBD_BL_V3p1.pdf">Nutzungsbedingungen</a>.</p>
<p>If you want to correct these buildings and addresses then you have to user other sources, e.g. your own photos, notes and GPS traces.</p>
<p>For more information about mapping see the <a href="https://wiki.openstreetmap.org/wiki/Beginners&#39;_guide">beginners' guide</a> and <a href="http://learnosm.org/en/beginner/id-editor/">iD editor</a>.</p>

<p>Current high-resolution imagery ~10cm/pixel is available for BL, which has a lot more detail that the old bing imagery the buildings were originally traced from.</p>
<p>There are no other data-sets available that we can use directly from the canton from a licence point of view.</p>
<p>Please note that the talk-ch mailing list is the far better place to ask such questions (see lists.openstreetmap.ch ).</p>
