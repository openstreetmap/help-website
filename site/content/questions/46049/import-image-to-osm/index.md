<div class="answer-body">
<p>That depends. You can't add pictures to the OSM database. This database just consists of raw geographic data. However you can edit this data and add URLs to specific pictures via the <a href="https://wiki.openstreetmap.org/wiki/Key:image">image key</a>. Or you can create your own map via <a href="https://umap.openstreetmap.fr/">uMap</a>, <a href="http://leafletjs.com/">Leaflet</a> or <a href="http://openlayers.org">OpenLayers</a> where you can embed images.</p>
