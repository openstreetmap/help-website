<div class="answer-body">
<p>If you want to display a circle then using symbol instead of an icon sounds more reasonable. See <a href="https://josm.openstreetmap.de/wiki/Help/Styles/MapCSSImplementation#Iconandsymbolstyles">https://josm.openstreetmap.de/wiki/Help/Styles/MapCSSImplementation#Iconandsymbolstyles</a></p>
<p>Regarding size calculations, you could have a look how its done in the Lane and Road Attributes mappaint style <a href="https://josm.openstreetmap.de/wiki/Styles/Lane_and_Road_Attributes">https://josm.openstreetmap.de/wiki/Styles/Lane_and_Road_Attributes</a></p>
<p>some extracts from there:</p>
<pre><code>/*------------------------------*/
/* Pixel per metre (on average) */
/*------------------------------*/
*|z16  { pixel_per_metre: 0.625; }
*|z17  { pixel_per_metre: 1.25; }
*|z18  { pixel_per_metre: 2.5; }
*|z19  { pixel_per_metre: 5; }
*|z20  { pixel_per_metre: 10; }
*|z21  { pixel_per_metre: 20; }
*|z22  { pixel_per_metre: 40; }
*|z23  { pixel_per_metre: 80; }
*|z24  { pixel_per_metre: 160; }
*|z25  { pixel_per_metre: 320; }
*|z26- { pixel_per_metre: 640; }
&#10;
/*-------------*/
/* Street lamp */  
/*-------------*/
node|z17-[highway=&quot;street_lamp&quot;]::Lighting_1
{
symbol-shape: circle;
symbol-size: eval(12 * prop(pixel_per_metre,&quot;default&quot;));
symbol-fill-color: white;
symbol-fill-opacity: 0.07;
}
node|z17-[highway=&quot;street_lamp&quot;]::Lighting_2
{
symbol-shape: circle;
symbol-size: eval(18 * prop(pixel_per_metre,&quot;default&quot;));
symbol-fill-color: white;
symbol-fill-opacity: 0.05;
}
node|z17-[highway=&quot;street_lamp&quot;]::Lighting_3
{
symbol-shape: circle;
symbol-size: eval(25 * prop(pixel_per_metre,&quot;default&quot;));
symbol-fill-color: white;
symbol-fill-opacity: 0.05;
}</code></pre>
