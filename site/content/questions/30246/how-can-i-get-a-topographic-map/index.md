<p>Enter "topographic" in the search box above and read some of the results. You will find that OpenStreetMap doesn't have topographic data - any OSM-based topo maps you see have the topographic component mixed in from some other source. Therefore, how you "get" these maps depends on which map it is and who the supplier is.</p>
