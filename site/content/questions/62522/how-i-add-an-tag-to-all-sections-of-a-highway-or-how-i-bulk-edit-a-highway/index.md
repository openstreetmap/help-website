<p>You probably don't - no-one has used that tag before: <a href="https://taginfo.openstreetmap.org/search?q=cycling%3Dpermissive">https://taginfo.openstreetmap.org/search?q=cycling%3Dpermissive</a> .</p>
<p>You may, however, mean "bicycle=permissive". See <a href="https://wiki.openstreetmap.org/wiki/Key:access">https://wiki.openstreetmap.org/wiki/Key:access</a> for what the "permissive" tag generally means. Again, you probably don't mean that - is this really a long private road on which the owner tolerates bicycles? If not, "bicycle=permissive" is not the right tag.</p>
<p>This, incidentally, is part of the reason why some editors don't support the "mass retagging" approach - it's too easy to make a massive number of errors.</p>

<div class="answer-body">
<p>In JOSM, the "Search" and "Filter" tools can be used to implement mass-editing.</p>
<p>I assume the highway in question has some tag that uniquely identifies it (most probably a <code>ref</code> tag). After downloading the relevant area, use Edit-&gt;Search and search for <code>ref=foo</code>. That will select all segments of the highway. <em>Make sure the selection is correct</em> (it may match both directions of the highway, for example), then add the tag as usual.</p>
<p>It might be necessary to use a more refined search query, for example, "highway=motorway ref=foo" to avoid matching bus stops with ref=foo.</p>
<p>Instead of using the "Search" tool, the "Filter" tool in 'inverse' mode (to grey out or hide <em>non</em>-matching elements) can also be used.</p>
<p>As always with mass edits, remember to discuss the edit with the local (in this case, Brazilian) community before making it.</p>
