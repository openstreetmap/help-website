<div class="answer-body">
<p>Wenn man von Print Composer eine Ebene exportiert, die mit dem OpenLayer-Plugin erstellt wurde, verwendet QGIS nur die aktuelle Zoomstufe im Ansichtsfenster. Wenn du mehr Details möchtest, stelle sicher, dass du auf einer Zoomstufe bist, in der es lesbar ist.</p>
<p>(Ich kann mich nicht erinnern, ob man das im Hauptfenster von Print Composer machen muss, aber es ist ja einfach auszuprobieren).</p>
<hr />
<p>English original text:</p>
<blockquote>
<p>When exporting from Print Composer with a layer sourced from the OpenLayer Plugin QGIS only uses the current zoom level in the viewport. If you want more detail, make sure that you are zoomed in at a level which is legible.</p>
<p>(I can't remember if you do this in the main window of Print Composer, but it's easy to experiment).</p>
<p>Sorry for not replying in German</p>
</blockquote>
