<div class="answer-body">
<p>This depends on the editor you are using:</p>
<ul>
<li>With <strong>JOSM</strong>, you can select multiple ways while pressing the shift key. Then you can modify tags for all of them at once.</li>
<li>With <strong>Potlatch 2</strong>, you can select multiple ways by pressing Control (or Command on a Mac); or you can select all ways in an area by holding Control and dragging.</li>
</ul>
<p>I'm not good enough with the other editors, so it would be good if others could join in.</p>
