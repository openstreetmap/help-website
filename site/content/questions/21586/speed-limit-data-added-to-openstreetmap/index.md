<div class="answer-body">
<p><a href="http://wiki.openstreetmap.org/wiki/Speed_limits">Speed limits</a> can be added to an osm way using the <a href="http://wiki.openstreetmap.org/wiki/Maxspeed">maxspeed</a> tag.</p>
<p>GPS are commonly used to get a "trace" of position data to help maping in OSM. Wether you collect that data using NMEA frames or using a gpx file doesn't matter much (except the gpx file is more practical).</p>
<p>However, you cannot use the nmea frames to gather the speed limit:</p>
<ul>
<li>It probably isn't included in the NMEA frame.</li>
<li>Evenif it was, it would be comming from map data that is copyrighted and <a href="http://wiki.openstreetmap.org/wiki/Copyright">cannot be used in osm</a>.</li>
</ul>
<p>You could look at which speed you were driving to infer the maxspeed, but that's very unreliable. Best thing is to survey the speed limit signs, using a camera for example.</p>

<div class="answer-body">
<p>As Ed and Vincent write you can add speed limits to OSM (and we do have some, but not many in <a href="http://osm.rasher.dk/?layers=B00000FFFTFFFFFFFFFFFFFFFFFFFFFFFF&amp;zoom=10&amp;lat=56.93601&amp;lon=9.01961">Thy</a>) and a GPS tracker can be useful for mapping, but from your question I guess you want to know more about how everything actually works.</p>
<p>To answer that I suggest you start with <a href="http://learnosm.org/">Learn OSM</a> and then send a message to the <a href="http://wiki.openstreetmap.org/wiki/Mailing_List#Country-Specific_Lists:">talk-dk mailing list</a> with a more detailed description of what you want to achieve with the project and what skills you have available.</p>
<p>Then it will be much easier to answer if the <a href="http://download.kortforsyningen.dk/">data</a> from <a href="http://www.gst.dk/">Geodatastyrelsen</a> is more useful (probably not), how/if GPS traces can be used and what existing tools/frameworks can be useful for your project.</p>
<p>This and all the other followup questions you and your students will have is probably more suitable for the mailing list, but if you are impatient, you can try the <a href="http://wiki.openstreetmap.org/wiki/Irc">Danish IRC channel</a>.</p>
