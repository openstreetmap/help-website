<p>You seem to use Potlatch which hasn't such a functionality as far as I can tell. But JOSM is able to copy relations. Just select the relation in the <a href="http://josm.openstreetmap.de/wiki/Help/Dialog/RelationList">relation list widget</a> and press the third button (labeled <em>create a copy of this relation [...]</em>).</p>
