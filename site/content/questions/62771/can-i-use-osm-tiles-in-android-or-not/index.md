<p>Can I use osm tiles in Android or not? I have found many opposite opinions.</p>
<p>Just simple question, can I use <a href="https://tile.openstreetmap.org/-------.png">https://tile.openstreetmap.org/-------.png</a> or not?</p>
<p><a href="https://operations.osmfoundation.org/policies/tiles/">https://operations.osmfoundation.org/policies/tiles/</a></p>
<p>Heavy use (e.g. distributing an app that uses tiles from openstreetmap.org) is forbidden without prior permission from the System Administrators. See below for alternatives. - what is heavy use</p>
<p>Highly Recommended: Do not hardcode any URL at tile.openstreetmap.org into an app. - this seems also not so clear, i dont have to hardcode it and still use it</p>
<p>Valid HTTP User-Agent identifying application. Faking another app’s User-Agent WILL get you blocked. - even osmdroid is asking for identifing of application - so seems that its possible and osmdroid is also using tile server by openstreetmap.org</p>
<p><a href="https://wiki.openstreetmap.org/wiki/Slippy_map_tilenames">https://wiki.openstreetmap.org/wiki/Slippy_map_tilenames</a> has also links to tile.openstreetmap.org</p>
<p>My app will use 19 zoom for very small views (i guess up to 5x5 tiles at most per one usage - tens of minutes)</p>
