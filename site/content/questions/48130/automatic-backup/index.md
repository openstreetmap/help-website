<div class="answer-body">
<p>Of course, but with OSM these are .osm files.<br />
You may download your desired region with wget as cronjob every night from OSM API (if it is not too big) or from Overpass API. This call</p>
<pre><code>wget &quot;http://overpass.osm.rambler.ru/cgi/xapi_meta?*[bbox=40.52215098562377,19.769210815429688,41.07624551474787,20.48194885253906]&quot; -O - &gt; $(date +%Y-%m-%d)_a_region_in_Albania.osm</code></pre>
<p>will result in a file named <strong>2016-02-15_a_region_in_Albania.osm</strong></p>
<p>Or you are already satisfied with the services download.geofabrik.de offers. For some recent days they have daily backups, then monthly and yearly ones. Just click on "raw directory index on their <a href="http://download.geofabrik.de/europe/slovenia.html#">download site</a>.</p>
