<div class="answer-body">
<p>All edits affect the master map data (aka "the planet"), so yes, it's not a good idea to remove things which other people will use. It's great that you've realised this and asked about it. Many people confuse the map on the main site as what OSM is about: this is just an illustration of one facet, the core of OSM is the shared data.</p>
<p>Now to answer your supplementary questions.</p>
<p>To create personal maps there are numerous techniques available. For Windows users the best place to start is with <a href="https://wiki.openstreetmap.org/wiki/Maperitive">Maperitive</a>, which allows you to choose what and how to render various features. You will need to download data to your own PC either from the main OSM site (suitable for small volumes of data), or from a number of other sites who provide OSM extracts as a service: notably <a href="http://downloads.cloudmade.com/">Cloudmade</a> and <a href="http://www.geofabrik.de/data/download.html">Geofabrik</a>. For other rendering approaches there is a <a href="https://wiki.openstreetmap.org/wiki/Category:Renderers">summary</a> page on the wiki.</p>
<p>To restore data to its previous state ('<a href="https://wiki.openstreetmap.org/wiki/Revert">reverts</a>') requires a good knowledge of OSM. Again there are a range of techniques available, but the easiest to use is the <a href="https://wiki.openstreetmap.org/wiki/JOSM">JOSM</a> <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/Reverter">revert plugin</a>. However, I would <strong>strongly suggest</strong> that you <strong>do not</strong> try to do this yourself: ask on the <a href="https://wiki.openstreetmap.org/wiki/Contact#IRC">IRC</a> channel or on a local mailing list and someone is likely to help.</p>
<p>Please don't let this discourage you from continuing to contribute. I note that your edits were in the Liverpool area: there's lots of detail which can be added, and there's probably stuff which is now out-of-date.</p>
