<p>Hi, I can see your latest changes and are well mapped.<br />
I expect you need to clear the cache from your browser … <a href="https://wiki.openstreetmap.org/wiki/Browser_cache">https://wiki.openstreetmap.org/wiki/Browser_cache</a><br />
If using a phone you could try using a Tor browser like <a href="https://onionbrowser.com/">https://onionbrowser.com/</a> that is less likely to be storing a cache for your local area if if it is you can choose a different circuit to get another new view.</p>
