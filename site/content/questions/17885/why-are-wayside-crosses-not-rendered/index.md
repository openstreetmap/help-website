<div class="answer-body">
<p>The POIs you mention are all wayside crosses; these aren't included in the standard map style and therefore don't appear on the map. (The standard map style simply cannot include all possible features - the map would become too crowded then.)</p>
<p>There are however specialized maps that show these objects or let you display an overlay with them, for example <a href="http://openstreetbrowser.org">http://openstreetbrowser.org</a> where you can activate the "culture and religion" overlay to see wayside crosses.</p>
