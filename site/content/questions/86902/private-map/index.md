<div class="answer-body">
<p>No. When you edit OpenStreetMap, your changes are pushed to the official database immediately. Creating a fictional city will get you banned quickly.</p>
<p>You might be interested in checking out this:</p>
<ul>
<li><a href="https://wiki.openstreetmap.org/wiki/OpenGeofiction">OpenGeofiction</a> is a sandbox where you can create fictional map data in a separate database.</li>
<li><a href="http://umap.openstreetmap.fr/">uMap</a> lets you create a personal layer with all kinds of points and lines, and this information does not interfere with the public database.</li>
</ul>
