<p>Your edit history suggests you're using Potlatch 2, so:</p>
<ul>
<li>select a road which is part of the relation</li>
<li>switch to Advanced tag view</li>
<li>on the left, click the little dropdown arrow next to the relation entry</li>
<li>select "Delete entire relation" from the menu</li>
</ul>
