<p>You cannot export the raw map data directly from the editing(!) API of such a big region (bbox). Try with a tiny area (e.g. only 100 × 100 m² - <a href="https://www.openstreetmap.org/api/0.6/map?bbox=-75.68393%2C45.47907%2C-75.68091%2C45.48131">example export</a>). If you need a big area: Try the alternatives mentioned by SomeoneElse or see the many <span>older questions on export</span>.</p>
