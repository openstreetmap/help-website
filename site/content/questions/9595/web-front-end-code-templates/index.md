<div class="answer-body">
<p>The web site and API are provided by a Ruby on Rails application. You can see <a href="http://git.openstreetmap.org/rails.git/">the code in the OSM Git repo</a> or <a href="https://github.com/openstreetmap/openstreetmap-website">at Github</a>. Note that the HTML pages you see are provided by Rails Views, which aren't quite the same as templates, but are similar enough to understand.</p>
