<p>If you open your GPX file with a text editor it should consist of a few lines of name and creator informtion followed by a large list on lats and longs each with date and time. if it does not have these time stamps it will not upload. you can see a good trace by down loading one from the traces list and then opening with a text editor.</p>

<li>This page</li>

<p>** specifically NaN or nan (i.e. "not a number") in fields that should have been numeric</p>
