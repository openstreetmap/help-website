<div class="answer-body">
<p>The OSRM routing on osm.org is provided by the Project OSRM demo server. This only provides car routing, not bike or foot routing.</p>
<p>It is possible to set up an instance of OSRM that will do bike or foot routing, but it has significant hardware requirements (probably over 128GB of RAM required), and thus far no-one has set up such a server and made it available to osm.org for free.</p>
