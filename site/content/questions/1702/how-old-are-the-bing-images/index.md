<div class="answer-body">
<p>The Bing images are sourced from different providers. Some are more current than others. If you have an image URL of, say</p>
<pre><code>http://ecn.t3.tiles.virtualearth.net/tiles/a1202022331312023.jpeg?g=587&amp;mkt=en-us</code></pre>
<p>You can retreive the HTTP headers using a browser plugin or an Unix commandline tool like</p>
<pre><code>$ HEAD &#39;http://ecn.t3.tiles.virtualearth.net/tiles/a1202022331312023.jpeg?g=587&amp;mkt=en-us&#39;</code></pre>
<p>This will return, among other things, a line that reads</p>
<pre><code>X-VE-TILEMETA-CaptureDatesRange: 6/17/2006-7/14/2006</code></pre>
<p>Which tells you that this specific tile was created from an image taken about 4.5 years ago.</p>
<p>OSM editors don't currently show that information but I assume that in the long run, ways will be found to make it readily availalbe.</p>
