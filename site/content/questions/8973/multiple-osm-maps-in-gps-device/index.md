<div class="answer-body">
<p>The maps are created with <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a>. Here the areas are selected and maps are created with the option 'Build my map' on the same page. When the maps arrive they come as executable and when running the file will install the requested areamap in MapSource. Here the map can be selcted as 'OSM World Routable' and from then you can select the map or parts of the map to upload to the GPS device.</p>
