<div class="question-body">
<p>Can you suggest me a power-user GPS application for Android?<br />
I've recently switched from my Sony Ericsson K800 to Android and really miss MapNav. Most of the android apps I've tried are really naive - there is a map, and there is a dot representing your position, and this is all. Some of them have online routing via Cloudmade or other service. Not something really impressive.<br />
</p>
<p>There are two cases I'll be using that app for:<br />
A) collect data for OSM<br />
B) when on a trip - checking and correcting OSM data<br />
For A) - I don't want anything with a embedded OSM editor! I like to make marks (in the app), take photos and then later enter all data in JOSM.<br />
</p>
<p>I'm often in regions with no access to Internet. So I need all maps to be prepared before and use them offline. Preferably use vector maps because of the scaling and routing.<br />
Also if using routing (and it is new to me and I don't find it really "a-must") it should be offline routing.<br />
About tracks - I keep lots of tracks, some of them with 5k+ nodes. I need to be able to easily tell which track is what.<br />
</p>
<p>MapNav was a perfect app with tons of options, but it won't run on Android. There are certain emulators but none of them get to run it smoothly. That's why I search for something that is android-native, but so far I haven't hit anything worthy. I've checked OsmAnd and it has troubles saving tracks. I've checked Maverick and it has troubles saving POIs.</p>
<p>I've created a list of features that I need and a list of features that it will be nice to have but not mandatory.</p>
<p>MUST HAVES</p>
<ul>
<li><p>Tracks<br />
-- record track points by distance traveled (for example "every 50 m") or by time (for example "every 10 s"). Combination of two can be possible with AND or OR.<br />
-- save and load tracks<br />
-- view saved tracks in list select active track<br />
-- export/import track formats - at least GPX<br />
-- display and update active track on the map<br />
-- ability to easily pause track recording (for example when you are standing still at one place)</p></li>
<li><p>Marks<br />
-- ability to quickly add new mark (or call it POI if you like) around my current location<br />
-- ability to add new mark with specific coordinates (useful for geocaching)<br />
-- easily export and import marks<br />
</p></li>
<li><p>Main display<br />
-- display current speed<br />
-- display distance between current position and a selected target position<br />
-- display distance between random two points<br />
</p></li>
<li><p>Map sources<br />
-- offline maps easily created<br />
-- OSM<br />
-- downloading Google Earth (or other sources) tiles from Internet</p></li>
</ul>
<p>OPTIONALS</p>
<ul>
<li><p>Tracks<br />
-- show altitude and speed profiling of a track<br />
-- edit track nodes (for example cut nodes out of the track, or split track in two)<br />
-- rename saved tracks<br />
-- list saved tracks with details (like length, timestamp of first node, timestamp of last node) and sorting<br />
-- display more than one track over the map at once</p></li>
<li><p>Marks<br />
-- ability to take photo and geotag/add it to a mark<br />
</p></li>
<li><p>Main display<br />
-- display current coordinates and altitude<br />
-- number of points in active track<br />
-- ability to show a "ruler" on the map<br />
-- display satelite status</p></li>
<li><p>Navigation<br />
-- ability to calculate route from point A to point B w/o Internet</p></li>
</ul>
</div>
