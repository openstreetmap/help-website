<p>OSM is just a big pile of data; it can't help you with your current location any more than a book that I'm currently reading can help me with mine.</p>
<p>What you probably want to ask is "how do I create a web site that asks the web browser that it is running in for the user's current location". A <a href="https://duckduckgo.com/html?q=html5%20geolocation">quick web search</a> suggests that there is lots of help out there for that. It's not really a question that has anything to do with OpenStreetMap.</p>
