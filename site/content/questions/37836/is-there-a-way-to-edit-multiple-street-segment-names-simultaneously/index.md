<div class="answer-body">
<p>Changing (the tags of) multiple street segments is a special case of changing multiple objects in general. See these previous questions:</p>
<ul>
<li><a href="/questions/23658/">changing-the-road-class-of-multiple-highways</a></li>
<li><a href="/questions/32495/">multiple-edits</a></li>
<li><a href="/questions/35819/">how-to-assign-a-tag-to-multiple-objects-at-once</a></li>
</ul>
<p>Depending on which editor you use there may be a feature to semiautomatically select multiple segments of a street with the same name (I know of such a feature in JOSM's plugins, IIRC). But that just eases the multiple selection.</p>
