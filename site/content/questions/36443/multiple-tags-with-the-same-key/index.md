<div class="answer-body">
<p>Personally, since "fixme" tags are intended to be used by mappers, so I'd stick all the text in one "fixme" tag.</p>
<p>However, <a href="http://taginfo.openstreetmap.org/search?q=fixme">taginfo</a> does find &gt; 1000 examples of "fixme2", but they seem to be as a result of one import in Poland. You could maybe also try something in the style of "FIXME:nsl" (to say that a particular tag needs refining (though again, that particular example was just one user performing a large mechanical edit in the UK).</p>
