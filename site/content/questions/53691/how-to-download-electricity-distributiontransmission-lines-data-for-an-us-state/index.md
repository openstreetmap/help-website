<p>At <a href="https://overpass-turbo.eu/">Overpass Turbo</a> enter "<strong>type:way &amp; (power=line | power=cable | power=minor_line) in Oregon</strong>" in the wizard, then click <em>Build and run query</em>, then in the <em>Export</em> menu select the output format most suitable to you.</p>
