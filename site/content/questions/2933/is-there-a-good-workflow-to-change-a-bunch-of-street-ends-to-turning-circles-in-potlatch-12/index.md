<p>Currently, in Potlatch 2 I have to do this:</p>
<ol>
<li>Click street.</li>
<li>Click street's end point (the whole street is still selected).</li>
<li>Hit Esc (now only the point is selected).</li>
<li>Click drop-down on the left.</li>
<li>Select Transport.</li>
<li>Scroll to the bottom.</li>
<li>Click Turning circle.</li>
<li>Repeat.</li>
</ol>
<p>Is there a way to select multiple nodes in different ways to at least have to do steps 4-7 once? If not, is there a way to select a node in one-click as opposed to my steps 1-3?</p>
<p>I don't think I can use R to my advantage because of the need to select the street in between working with nodes. If I could learn to select a single node with a single click, I could probably use R. That would be cool.</p>

<div class="answer-body">
<p>For the second and subsequent street ends, use the Potlatch2 "repeat" function. Select the end node, then press r. The repeat function will copy the tags from the previously selected node, to the currently selected node.<br />
</p>
<p>Subsequent turning_circle nodes can be marked this way.</p>
<p>Click street.<br />
Click street's end point (the whole street is still selected).<br />
</p>
<ol>
<li>Click street</li>
<li>Click street's end point</li>
<li>Press ESC (only the end node is selected)</li>
<li>Press r</li>
<li>Repeat</li>
</ol>
