<p>I suggest to use <strong>_macosx.zip = Installer for Garmin BaseCamp/RoadTrip (Mac OSX)</strong></p>
<p>Unzip it and place the .gmap folder to the default Garmin map folder. The location of this folder depends on the Windows version. In Vista or W7 this is: C:\ProgramData\GARMIN\Maps or C:\Users{user}\AppData\Roaming\GARMIN\Maps (please note this is a hidden folder!)</p>
<p>Windows XP: c:\documents and settings\all users\application data\garmin\maps</p>
<p>Or you can use <a href="http://www.javawa.nl/gmtk_en.html">http://www.javawa.nl/gmtk_en.html</a> to install the gmap maps.</p>
