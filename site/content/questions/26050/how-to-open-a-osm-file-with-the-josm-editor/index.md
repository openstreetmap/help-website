<div class="answer-body">
<p>First start JOSM, then go to the file menu and "open" or use the toolbar button.</p>
<p>However, you can download map data inside of JOSM - no need for a .osm file!</p>
<ul>
<li>You should read the information on the page <span><span>@Hendrikklaas</span></span> linked to. Most importantly <span>JOSM/Guide</span>.</li>
<li>Instead (in addition) you can go to <a href="http://learnosm.org">http://learnosm.org</a> and look inside step <a href="http://learnosm.org/en/beginner/start-josm/">Getting started editing</a>. Oh, are you reading there (there a sample.osm is mentioned)? "Open JOSM. Click the “Open” button in the upper left." is even described there.</li>
</ul>
<p>JOSM takes a while to get used to it, but it is definitely worth it! Go on, try, learn, ask (if you cannot find a previous question of the same topic).</p>
