<p>The maps in MAPS.ME update only rarely (when the application updates). If there are no application updates pending, try a different application - some of these (for example OsmAnd on Android) do have a mechanism for getting map updates to the user quicker.</p>
<p>You could try contacting the developers via <a href="https://github.com/mapsme/omim/issues">Github</a>, but don't expect a solution immediately.</p>
