<div class="answer-body">
<p>I presume you're referring to the OpenCycleMap rendering, rather than the standard ('Mapnik') OpenStreetMap data which does not show hills anyway.</p>
<p>OpenCycleMap uses elevation data from NASA's <a href="http://www2.jpl.nasa.gov/srtm/">SRTM</a> project. If hills are not showing up, <a href="http://osm.org/go/xxhaKr?layers=C">as appears to be the case for Naxos</a>, it is likely a failure of SRTM data rather than anything within OpenStreetMap itself.</p>
