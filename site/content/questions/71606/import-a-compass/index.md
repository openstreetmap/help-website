<div class="answer-body">
<p>If I'm understanding correctly that you need a picture of a map area with a depiction of a compass star on it, like often seen on paper maps, I'd just export the map you need to a png (that can be done from openstreetmap.org sidebar, under "Share") and add <a href="http://clipart-library.com/compass-graphic.html">the compass graphic</a> in an image editor. Done!</p>
<p>If you wan styled maps and other options, look into services here - <a href="https://wiki.openstreetmap.org/wiki/OSM_on_Paper">wiki.openstreetmap.org/wiki/OSM_on_Paper</a></p>
<p><a href="http://maperitive.net/">Maperitive</a> is a great free software for making whatever map you need in a printable format, where you can print exactly what you want.</p>
