<p>Southeast of Helsinki there is an island named Eestiluoto which has a tower (<a href="https://www.openstreetmap.org/#map=18/60.12434/25.22273).">https://www.openstreetmap.org/#map=18/60.12434/25.22273).</a> This tower is tagged as watch tower, but in fact it is a radar tower with a rotating antenna on the top. And there are several of this kind along the Finnish coastline. IMHO a tag "radar tower" should be created (at least I did not find something like this). I think, "watch tower" is wrong because there are no watching persons in the tower.</p>
