<ol>
<li><p><a href="https://wiki.openstreetmap.org/wiki/Key:baseball">https://wiki.openstreetmap.org/wiki/Key:baseball</a> shows <code>baseball:*=</code> in some unclear usage (with some over-namespacing and duplications). Under this syntax, it may be eg <code>baseball:home_plate=no</code>. For context, there are 7 wrongly spaced <code>baseball=home plate</code> instances for its location when it exists.</p></li>
<li><p><a href="https://wiki.openstreetmap.org/wiki/Proposed_features/pitch_markings">https://wiki.openstreetmap.org/wiki/Proposed_features/pitch_markings</a> will format it as <code>lines:baseball=no</code>.</p></li>
</ol>
