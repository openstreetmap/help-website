<div class="answer-body">
<p>There is an <a href="https://github.com/umap-project/umap/issues/125">open issue</a> on the uMap GitHub for "image only" markers that would allow customised outline shapes.</p>
<p>In the meantime the uMap interface allows for customisation of marker colour and allows reference to external <a href="https://wiki.openstreetmap.org/wiki/UMap/Guide/Customize_the_icon_of_a_marker">icons</a>. This is close to your second example link but without the border.</p>
