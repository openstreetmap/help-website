<p><span>@Harry Wood</span>: landuse=farm is older, but considered more ambiguous (because it predates the introduction of landuse=farmyard). Therefore, there are different opinions which tag should be preferred: The JOSM presets, for example, only include landuse=farmland. Taginfo counts <a href="http://taginfo.openstreetmap.de:8003/tags/landuse=farm">375 660 landuse=farm</a> and <a href="http://taginfo.openstreetmap.de:8003/tags/landuse=farmland">215 387 landuse=farmland</a>.</p>

<div class="answer-body">
<p>Using JOSM:</p>
<ul>
<li>download the area where you map</li>
<li>open the search (Ctrl+f)</li>
<li>search for the following string: landuse=farm user:c2r</li>
<li>edit the landuse value for all selected objects</li>
<li>upload the change with an informative changeset comment</li>
</ul>
<p>Be aware that this will modify all objects for which you were the last user to touch them. It's possible that other users have contributed to these objects, too. You should therefore be careful what exactly you edit, especially when the change is, as in the example of farm vs. farmland, mostly a matter of personal preference.</p>
