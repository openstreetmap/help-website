<div class="answer-body">
<p>If there is no other way attached to the end node of the way that you want to extend, just click the end node and press the "A" key ("Add/Extend"). The way will be selected and you can click the place where you want to place the new node, or the already existing node.</p>
<p>If the new node is on another way, it will be joined to it.</p>
<p>Press the "&lt;esc&gt;" key ("Escape") or select another mode (like "S" key for "Selection Tool") to finish adding nodes to the end of the way.</p>
<p>If the end node is part of other ways that finish at that same node, you have to select the desired way, using holding the &lt;control&gt; key "Add to Selection" when the Selection Tool is active, and selecting the end node and the way. Then proceed as usual.</p>
</div>
