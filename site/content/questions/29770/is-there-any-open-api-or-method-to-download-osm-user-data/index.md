<div class="answer-body">
<p>Hi, you are very welcome to do any research on the OSM community :)<br />
Please also check out this page, to know about previous works and how we understand a coop with the scientific community: <a href="https://wiki.openstreetmap.org/wiki/Research">https://wiki.openstreetmap.org/wiki/Research</a></p>
<p>Every user has a unique ID that is <a href="https://wiki.openstreetmap.org/wiki/Elements">tracked in edits</a> ("changesets"). So you can use the <a href="https://wiki.openstreetmap.org/wiki/API">API</a> to start with your analysis. But to do more/global processing, I highly recommend to make use of our <a href="https://wiki.openstreetmap.org/wiki/Planet.osm/full">history planet</a>, so you have a definite dump and can work locally. But currently there is kind of official user ranking out there. There are some unoffical stats, as <a href="http://hdyc.neis-one.org/?!i!">'how do you contribute'</a> for example.</p>
<p>But you need to pay attention on just using that simple metrices on the work of users/the community, as there might be local aspects that you need to keep in mind (e.g. imports or availability of aerial imagery has an influence on the workflow etc...) Please also note, that there are various other <a href="https://wiki.openstreetmap.org/wiki/Contact">communication channels</a> that are used differently on the local OSM communities.</p>
<p>P.S. Just one more hint: Please be aware that a lot of people are very concernd about their privacy etc. So AFAIK even the OSMF won't allow you to get more detailed informations e.g. "statistics about signup email account TLDs".</p>
