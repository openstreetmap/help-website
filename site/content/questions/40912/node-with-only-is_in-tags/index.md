<div class="answer-body">
<p>"is_in" is/was a work around problems when OSM didn't have boundary polygons for anything.</p>
<p>Given that AUS has such (boundaries) for both for the country and the state in question, the tags can be removed, particularly since they don't seem to be attached to anything useful.</p>
<p>Naturally simply leaving them untouched doesn't hurt either. Editing creates a new version so you don't really save any DB space, best to remove such stuff when you are editing the object in question in any case.</p>
