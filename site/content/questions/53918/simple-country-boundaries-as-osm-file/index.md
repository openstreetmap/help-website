<p>Use Natural Earth shapefiles <a href="http://www.naturalearthdata.com/downloads/">http://www.naturalearthdata.com/downloads/</a></p>
<p>These can be converted to OSM formats using ogr2ogr or possibly read into JOSM. (Low zoom levels on the main slippy map use this data already).</p>

<p>If you're after simplified country data, a source like Natural Earth is a better option. If you particularly want all the OSM border data and just the border data, the osmborder_filter program from <a href="https://github.com/pnorman/osmborder">https://github.com/pnorman/osmborder</a> is able to generate this.</p>
