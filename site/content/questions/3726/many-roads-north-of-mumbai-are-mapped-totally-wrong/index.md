<td><div class="item-right">
<div class="answer-body">
<p>It is OK to delete <strong>completely wrong</strong> roads. This does <strong>not</strong> cover situations where the road's name is wrong or misspelled (but there is a road near the drawn road). It is also not OK to delete roads where there is a road with the mapped name, but not on the exact position, or only partly on the correct position (e.g. the road in the map doesn't change the name like it does in reality). All of the mentioned errors should be <strong>corrected instead of deleted</strong> (this way we keep the history).</p>
</div>
