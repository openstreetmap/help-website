<div class="answer-body">
<p>You have successfully entered the address on the <a href="https://www.openstreetmap.org/way/573182587">building</a>. You have also given your business name to the building. Now it looks like the building is named <em>In Status - Mediação de Seguros</em>. If you business occupies the whole building you can just add an appropriate tag describing the business. Maybe <a href="https://wiki.openstreetmap.org/wiki/Tag:office%3Dinsurance"><code>office=insurance</code></a>?</p>
<p>If your business is only one tenant in the building better add a new node inside the building outline which you tag with the main tag <code>office=insurance</code> (or whatever is appropriate) and move the <code>name=</code> from the building outline to the new node. The name should be the name only and not contain a description of the business. I don't speak Portuguese but I guess the name is really only <em>In Status</em>, isn't it?</p>
<p>As to why the map is not showing the right direction is a different matter which has nothing to do with the mapping but rather with how you embed the map. Maybe someone else can give you a hint here.</p>
