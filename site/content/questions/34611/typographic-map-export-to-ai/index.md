<div class="answer-body">
<p>A possible course of action is:</p>
<ul>
<li>download London data from download.geofabrik.de</li>
<li>use "osmosis" software to filter out only streets (optional - do this only if you run out of memory in the next step)</li>
<li>load whole thing into "Maperitive" software (requires Windows, or if on Mac/Linux, requires Mono)</li>
<li>modify Maperitive drawing rules to only output labels for streets and nothing else</li>
<li>export to AI compatible SVG</li>
</ul>
