<p>Regarding what I found about Lacul Dracului this is a natural water in a collapsed cave. If you don't want to use<br />
<em>natural=water</em> because it is a pool of a stream/river, you can use<br />
<em>waterway=riverbank</em> to map that pool. Since it is inside a collapsed cave you could also add<br />
<em>embankment=yes</em> and - if matching<br />
<em>natural=cliff</em> (maybe only partly).</p>
<p>After all don't forget: mapping is about simplifying. :)</p>
