<p>Hi! A great tool for printing maps with a compass and a scale is <a href="https://maposmatic.osm-baustelle.de">MyOSMatic</a>, which allows you to design custom OpenStreetMap maps. However, adding a legend is not possible that way, since no complete "legend" exists for OSM Carto as far as I know.</p>
