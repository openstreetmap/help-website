<p>Hello and welcome to the OpenStreetMap community, we're glad to have you with us!</p>
<ol>
<li>Please start by reading the <a href="https://learnosm.org/en/beginner/">Beginner guide</a> and use the OSM wiki as a reference, most of your questions are already answered, as well as some much needed best practices.</li>
<li>Check if there is existing data and update it with information you think is beneficial to the map.</li>
<li>For a specific company, check the primary tags and the tags that can be used in combination. This page should get you started - <a href="https://wiki.openstreetmap.org/wiki/Tag:office%3Dcompany">https://wiki.openstreetmap.org/wiki/Tag:office%3Dcompany</a></li>
</ol>
