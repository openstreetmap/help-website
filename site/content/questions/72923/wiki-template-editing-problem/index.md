<p>I recently discovered <a href="https://wiki.openstreetmap.org/wiki/Data_items">wikibase</a>, a wikidata for the osm wiki. The point is to centralize documentation in machine-readable format, for translations or other tools (taginfo, etc). It's mainly used in the templates.</p>
<p>On item <a href="https://wiki.openstreetmap.org/wiki/Item:Q6491">Q6491</a>, which means building=train_station, you'll find a section "combination" with the tags you mentioned.</p>
<p>I don't know the details, but when you removed all combinations from template parameters, the template probably started to use the wikibase data.</p>
<p>On a side note, why don't you like these proposed combinations ?</p>
