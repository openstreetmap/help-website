<p>Have a look at viewranger <a href="http://www.viewranger.com">http://www.viewranger.com</a></p>
<p>As an example, I can export the keepright errors to GPX and import them to viewranger. Each waypoint shows up as a footballs against your chosen map (the standard map options include several versions of OSM).</p>
<p>It comes in both mobile and web versions so you can use it on a desktop or take it with you.</p>
<p>In the mobile versions, you can also see your current position or record your tracks.</p>
