<div class="answer-body">
<p>The general guidance on the <a href="https://wiki.openstreetmap.org/wiki/Key:building">building page</a> of the wiki is that the tag for the building should reflect what it was built as (or appears to have been built as). So if it was <em>built as</em> a fire station (and hasn't been heavily modified) I would tag the building as <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dfire_station">building=fire_station</a> and place the amenity tags as appropriate on nodes or <a href="https://wiki.openstreetmap.org/wiki/Key:building:part">building:part</a> areas.</p>
