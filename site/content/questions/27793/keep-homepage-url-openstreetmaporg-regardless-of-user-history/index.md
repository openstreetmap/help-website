<p>I'm unable to get back to the homepage URL (openstreetmap.org) after searching for a location or editing a map. Instead, the last used location shows up, along with its URL path.</p>
<p>Why do I want a plain old homepage URL? Because I don't want to have to close all my windows and tabs and delete all my user data from my browser just to do something like add the OpenStreetMap URL/icon to my browser toolbar or bookmarks, or to quickly grab the URL for a post or email.</p>
<p>To be clear, this request isn't just for me personally. This seems like a basic feature that many, many users might want.</p>

<p>The right location for your proposal is openstreetmap's trac. However, there is already my related bug report <a href="https://trac.openstreetmap.org/ticket/4942">Constantly updated permalink in URL bar (hash style) spams browsing history in Firefox</a> which was closed as "wontfix" (I guess yours would also be closed that way). Just workarounds are available. They would also work for your problem. Deleting (or not accepting) the "_osm_location" cookie from openstreetmap.org would work too, of course.</p>
