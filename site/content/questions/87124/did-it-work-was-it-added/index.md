<p>Map notes do not edit the map. They are just "notes" which may spur a keen mapper to add a feature. To act on a map note a mapper will really need to check the information in some way, which may not be easy without a survey. If you really want the map updated learn how to do it yourself. see <a href="https://wiki.openstreetmap.org/wiki/Beginners%27_guide">https://wiki.openstreetmap.org/wiki/Beginners%27_guide</a></p>
