<p>I would create three nodes very close to each other on the way.</p>
<p>The use of key_1, key_2 etc is deprecated or at least highly discouraged (see <a href="https://wiki.openstreetmap.org/wiki/Proposed_features/Remove_suffixed_name-tags_from_wiki">Proposed features/Remove suffixed name-tags from wiki</a>), so if you wanted to have multiple barrier types on one node, you'd want to use semicolons, eg barrier=cattle_grid;turnstyle;etc</p>
