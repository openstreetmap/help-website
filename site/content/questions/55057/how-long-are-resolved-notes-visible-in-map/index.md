<div class="answer-body">
<p>It can be specified in the underlying <a href="https://wiki.openstreetmap.org/wiki/API_v0.6#Map_Notes_API">notes API call</a> call. "7 is the default".</p>
<p>The <a href="https://www.openstreetmap.org">https://www.openstreetmap.org</a> homepage does (see your browser's network requests console), currently, not specify this parameter (it just specifies the <code>bbox</code>). Hence, notes are visible 7 days before they disappear from <em>this</em> map.</p>
</div>
