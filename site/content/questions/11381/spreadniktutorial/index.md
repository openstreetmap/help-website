<div class="answer-body">
<p>You can get that page via <a href="http://web.archive.org/web/20101018160608/http://weait.com/content/make-your-first-map">archive.org</a>. However, rather than following the prerequisite links from that page you might be better off following those on <a href="http://switch2osm.org/serving-tiles/manually-building-a-tile-server/">switch2osm</a> instead. Once you've got a working tile server, you'll then have generate_image.py and can edit that to define what area you want an image of.</p>
<p>Those pages will get you an OSM server and the OSM Mapnik 0.7 stylesheet. If you want to actually modify styles though, there are probably easier options. Search <a href="http://help.osm.org">help.osm.org</a> for "Tilemill" (which is an answer typically given to similar questions) and you'll see lots of questions and answers.</p>
</div>
