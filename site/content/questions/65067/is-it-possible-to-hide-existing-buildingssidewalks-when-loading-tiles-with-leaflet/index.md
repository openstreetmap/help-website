<p>Tiles are fixed images. If you do not like what is on them, you have to use another <a href="https://wiki.openstreetmap.org/wiki/Tile_servers">tile server</a> or create your own tiles.</p>
<p>One day in the future, the OSM servers might serve vector tiles that you can render as you wish (was discussed on SOTM the last weekend).</p>

<p>This answer is correct, but there <em>are</em> some ways to work around this. For example, you could try placing another layer on top of the OSM maps. If that layer is opaque on the area of interest, and transparent elsewhere, it will hide the undesired features. It's a crude solution, but perhaps "good enough" for some. :)</p>
