<div class="answer-body">
<p>If this window is primarily for vehicle use, then <strong><code>drive_through=yes</code></strong> is the correct tag -- but I think you're describing walk-up windows. And as far as I can tell there's no popular tag for them, so I'd say "be bold!"</p>
<p>That doesn't mean that you can't also discuss this issue on the tagging mailing list and submit a formal proposal there.</p>
<p>A few tagging possibilities that occur offhand:</p>
<ul>
<li><strong><code>takeaway:window=yes</code></strong> - not currently in use, but there is some use of both of the <strong><code>takeaway:*=yes</code></strong> and <strong><code>*:window=yes</code></strong> tagging forms</li>
<li><strong><code>takeaway:type=window</code></strong> - not currently in use but consistent with other <strong><code>*:type=*</code></strong> tagging forms, could possibly get complicated if someplace has a takeout window but also an indoor takeout counter</li>
<li><strong><code>walkup_window=yes</code></strong> - very rarely used (2 instances worldwide), does <em>not</em> use the word "takeaway" which might be good, because to my ear "takeaway" implies food and your question mentions both restaurants and shops</li>
</ul>
</div>
