<p><a href="https://www.mapillary.com/">Mapillary</a> (Provided by facebook)<br />
<a href="https://openstreetcam.org/">OSC-OpenStreetCam</a> (Provided by Grab)</p>
<hr />
These are used to add street level photos. You can create an account and add photos. These photos can be viewed in the ID editor and JOSM.
<hr />
In ID editor:<br />
Path:-<br />
OSM ID editor login -&gt; Map Data -&gt; Photo Overlays -&gt; Mapillary / OpenStreetCam (enable the feature)
