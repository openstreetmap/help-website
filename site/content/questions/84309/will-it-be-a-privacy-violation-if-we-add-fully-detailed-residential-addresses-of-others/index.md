<div class="answer-body">
<p>It definitely isn't a privacy violation to add addresses. Adding addresses is a great way to contribute because this is an area that's generally lacking in many regions. Addresses are necessary for people to find houses, send mail, etc., and are usually meant to be publicly-visible, so there's nothing wrong with adding them in OSM. They wouldn't work very well if they were private and kept from the public eye. :)</p>
<p>You shouldn't any personally-identifiable information, though. For example, you shouldn't tag a house with the name "Joe's house", or add a tag like <code>owner=Joe Smith</code>. Stick to the unit, house number, street, city, or any other relevant address info, and you should be fine.</p>
