<div class="answer-body">
<p>The short answer: because nobody has done the necessary work.</p>
<p>The long answer: the standard OSM map style is a complicated beast and the volunteers working on it only have a limited amount of time and they spend that on what they consider important (so part of the answer is that the current maintainers do not consider adding such rendering a high priority item).</p>
<p>You can naturally always volunteer your time for an implementation.</p>
<p>In the specific case there is an open issue <a href="https://github.com/gravitystorm/openstreetmap-carto/issues/791">https://github.com/gravitystorm/openstreetmap-carto/issues/791</a> which is however quite old.</p>

<div class="answer-body">
<p>As you're in the UK, it might be useful to mention that I've thought about adding some sort of support for embankments (in the first instance flood banks) to <a href="https://map.atownsend.org.uk/maps/map/map.html">this map</a>. What would need to happen is essentially this:</p>
<ol>
<li>Decide exactly which features should have the new tagging. This means looking at the tags that people use, and what combinations those tags appear in. Obviously, the fewer tag combinations are initially suggested, the easier it is to create a prototype.</li>
<li>Look at the location in which these tag combinations appear, and choose a rendering that would fit there. For example, <a href="https://www.openstreetmap.org/way/310911206#map=17/52.91287/-0.64852">here</a> there is a railway on an embankment, but it's the middle track of 3, so there's not much room for detail.</li>
<li>Design a rendering (i.e. a combination of CartoCSS properties) that represents the new thing. In this case something based on the bridge rendering or the verge rendering <a href="https://map.atownsend.org.uk/maps/map/map.html#zoom=15&amp;lat=-25.00275&amp;lon=135.01976">here</a> is probably a good start.</li>
<li>Write the necessary bit of lua to identify exactly the tag combinations that you want to capture, but no others, like e.g. <a href="https://github.com/SomeoneElseOSM/SomeoneElse-style/blob/master/style.lua#L469">here</a>.</li>
</ol>
<p>If you want to do this, feel free - you can clone both parts of the map style used and tinker to your heart's content - and if you would like any help with these steps, please ask!</p>
</div>
