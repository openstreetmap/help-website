<p>TL/DR: how to mark the old name of a recently (governmental) renamed street so that e.g. searches will still find it by the old name?</p>
<p>hello, last week the local government FINALLY renamed a street (it was named in honor of a Nazi prick). so i changed all name= and addr:street= to the new name (Lola-Fischel-Straße). Now the old name will disappear for good.</p>
<p>changeset: <a href="https://www.openstreetmap.org/changeset/42514920">https://www.openstreetmap.org/changeset/42514920</a> , my reference (in german): <a href="http://www.hannover.de/Service/Presse-Medien/Landeshauptstadt-Hannover/Aktuelle-Meldungen-und-Veranstaltungen/Stra%C3%9Fenumbenennung-in-Lola-Fischel-Stra%C3%9Fe">http://www.hannover.de/Service/Presse-Medien/Landeshauptstadt-Hannover/Aktuelle-Meldungen-und-Veranstaltungen/Stra%C3%9Fenumbenennung-in-Lola-Fischel-Stra%C3%9Fe</a></p>
<p>but</p>
<ul>
<li>the renaming was very recent, the street is very unimportant so not everybody will know the change right away...</li>
<li>there is still a sign with the old (crossed out) name.</li>
</ul>
<p>is there a way to tag it, so searches will find the street by the old name?</p>
