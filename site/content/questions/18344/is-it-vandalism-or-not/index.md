<div class="answer-body">
<p>The "name" tag should be chosen according to what is on the ground, i.e. what is on the street sign or, if there are no street signs, according to the name that you would hear used locally. - The "name" tag should not be chosen according to politics (what the official/national/... language is in theory).</p>
<p>Other languages can be placed in the "name:xx" tag, where "xx" is the language code - i.e. "name:ru" for Russian names and "name:ky" for Kyrgyz names.</p>
