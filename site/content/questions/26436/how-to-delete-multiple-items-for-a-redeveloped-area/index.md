<ul>
<li>Potlatch2: hold down Ctrl key, then drag a rectangle with the mouse. Shift drags allow to add or substract to/from the previous selection.</li>
<li>iD: hold down Shift key, then drag a rectangle with the mouse.</li>
<li>JOSM: in select mode (shortcut S) just drag a rectangle with the mouse. Ctrl or Shift drags allow to add or substract to/from the previous selection.</li>
</ul>
