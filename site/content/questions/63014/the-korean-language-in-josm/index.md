<p>Windows 10 changed default font configuration for Korean and Java has not been upated yet to take this into account, see <a href="https://bugs.openjdk.java.net/browse/JDK-8190907">JDK-8190907</a></p>
<p>The good news is we fixed the problem <a href="https://josm.openstreetmap.de/changeset/13791/josm">directly in JOSM 13791+</a>.</p>
<p>The May release will contain the fix.</p>
