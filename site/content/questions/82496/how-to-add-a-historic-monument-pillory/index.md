<p>According to Taginfo, currently there are 886 objects using the tag historic=pillory. I'm not sure which editor you're using but I presume you can add this tag manually if it's not present in the "map data pane".</p>
<p>If appropriate, you might also add</p>
<p>tourism=attraction</p>
<p>attraction=historic</p>
<p>Cheers</p>
