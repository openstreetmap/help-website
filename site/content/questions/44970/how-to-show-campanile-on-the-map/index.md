<div class="answer-body">
<p>To see what other people have mapped a certain feature as, you can use <a href="http://taginfo.openstreetmap.org/">taginfo</a>. In this case, if you search for "campanile", and then click <a href="http://taginfo.openstreetmap.org/search?q=campanile#values">values</a>, you'll see that the most popular value is the 559 <a href="http://taginfo.openstreetmap.org/tags/man_made=campanile">man_made=campanile</a>.</p>
<p>However as you've seen, not every feature is shown on any of the five styles on the osm.org site. However, if you just want to see the location of them, you can use the "<a href="http://overpass-turbo.eu/s/bcw">Overpass Turbo</a>" link from taginfo.</p>
