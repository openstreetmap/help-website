<div class="answer-body">
<p>Tag the general <a href="https://wiki.openstreetmap.org/wiki/Key:access">access</a> as usual, and then add a <a href="https://wiki.openstreetmap.org/wiki/Key:conditional">conditional restriction</a> such as <em>access:conditional=destination @ (weight &gt; 7.5)</em>.</p>

<div class="comment-text">
<p>I've always tended to write the conditional the other way around, so "maxweight:hgv=7.5" and "maxweight:hgv:conditional=none @ destination".</p>
<p>Taginfo stats for the UK are <a href="http://taginfo.openstreetmap.org.uk/tags/hgv=destination">hgv=destination</a> 634, <a href="http://taginfo.openstreetmap.org.uk/tags/maxweight%3Ahgv%3Aconditional=none%20%40%20destination">maxweight:hgv:conditional=none @ destination</a> 313, <a href="http://taginfo.openstreetmap.org.uk/search?q=maxweight%3D7.5">maxweight=7.5</a> variations around 6000. There are also 428 <a href="http://taginfo.openstreetmap.org.uk/tags/maxweight%3Aconditional=none%20%40%20destination">maxweight:conditional=none @ destination</a>. I suspect a large number of these "non-HGV" restrictions are actually mismapped (I know I'm responsible for a few).</p>
