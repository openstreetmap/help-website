<div class="answer-body">
<p>Certainly you and others can add this kind of information to the map.</p>
<p>For general information about contributing to the map see the <a href="https://wiki.openstreetmap.org/wiki/Beginners%27_guide" title="Beginners Guide">Beginners Guide</a></p>
<p>Specifically you would add areas (polygons) using the <a href="https://wiki.openstreetmap.org/wiki/Key:natural">Natural key</a> for example</p>
<ul>
<li>Natural forest: Key natural=wood</li>
<li>Natural bushes, small trees: natural=scrub</li>
<li>Natural mixed grassland/bushes: natural=heath</li>
<li>Forest plantation (man-made): landuse=forest</li>
</ul>
