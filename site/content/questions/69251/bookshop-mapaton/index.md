<div class="answer-body">
<p>All one needs to add data to OSM is an account, which is easy for people to set up. With iD editor, it is usually pretty easy for people with basic computer skills to add points like stores.</p>
<p>The bigger question is where will the data come from.</p>
<ul>
<li>If people add bookstores in their area which they visit or know about, that's pretty clearly welcomed. See <a href="https://wiki.openstreetmap.org/wiki/Mapping_parties">Mapping Parties</a> for some possible ideas.</li>
<li>If, however, there is some kind of list or other data source mappers would be drawing from, you should consult the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">Import/Guidelines</a> wikipage about issues of legality and processes to follow.</li>
</ul>
