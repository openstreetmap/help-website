<p>Done it, here is the new link: <a href="https://heatmap-external-a.strava.com/tiles/all/bluered/%7Bzoom%7D/%7Bx%7D/%7By%7D.png?px=256">https://heatmap-external-a.strava.com/tiles/all/bluered/{zoom}/{x}/{y}.png?px=256</a></p>
<p>I did open up Strava slide, had a look at the page information on my browser, tweaked a little the url that I found there and here we go, now it looks to be working fine! In case someone is interested, "all" and "bluered" values can be replaced with other values, in order to see the heatmap of a specific activity (cycling or running) or to see the heatmap with a different color.</p>
<p>Cycling heatmap: <a href="https://heatmap-external-a.strava.com/tiles/cycling/bluered/%7Bzoom%7D/%7Bx%7D/%7By%7D.png?px=256">https://heatmap-external-a.strava.com/tiles/cycling/bluered/{zoom}/{x}/{y}.png?px=256</a></p>
<p>Running heatmap: <a href="https://heatmap-external-a.strava.com/tiles/running/bluered/%7Bzoom%7D/%7Bx%7D/%7By%7D.png?px=256">https://heatmap-external-a.strava.com/tiles/running/bluered/{zoom}/{x}/{y}.png?px=256</a></p>
<p>Not so sure about that "?px=256" at the end of the url, only thing I know is that removing it messes pretty bad the heatmap layer.</p>
<p>I hope someone else will find this post useful. Could anyone update the links available on the Wiki Strava page? Thanks</p>

<p>There are at least three possible use cases:</p>
<ul>
<li><p>Aerial/Satellite imagery often has an offset. We usually use the GPS tracks <a href="https://www.openstreetmap.org/traces">OSM collects</a> to correct this offset by comparing roads which have multiple tracks on the imagery and a rendered image of the traces. If there are only few traces, it is very likely that you get wrong results by GPS tracks can have an error between 3 and 15 m. Strava has a better coverage in some area and you can use it as an alternative GPS track source.</p></li>
<li><p>Finding missing ways. You can find missing tracks and paths in forests or areas without good satellite/aerial imagery.</p></li>
<li><p>You can improve the accuracy of OSM ways which have been mapped using GPS tracks only. That was quite common in the pre-Bing era (We got the permission in November 2010, only Yahoo with its partially huge offsets was available before. Some regions got covered by Bing much later or never (e.g. rural parts of the alp in Switzerland got covered in 2013 or so). Nowadays forest tracks are still mapped using GPS tracks only because you cannot see them if the trees are not deciduous or imagery was taken in summer.</p></li>
</ul>
