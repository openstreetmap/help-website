<p>When I want to do this, I use the Level0 editor (<a href="http://level0.osmz.ru/).">http://level0.osmz.ru/).</a> Documentation at <a href="https://wiki.openstreetmap.org/wiki/Level0.">https://wiki.openstreetmap.org/wiki/Level0.</a> Basically, you just type (for example):</p>
<pre><code>node: 12.345, 6.789</code></pre>
<p>give it a changeset comment, click Upload to OSM, and it will create a node at that location, which you can then edit in iD. Or you can supply all the tags in Level0, e.g.</p>
<pre><code>node: 12.345, 6.789
name=My Node
amenity=bar</code></pre>

<div class="answer-body">
<p>Do you have a list of the points, or do you already have a KML or GPS file? GPX files can be added into the ID editor screen on your browser, so that the points will show up at the exact coordinates.</p>
<p>If you have a file in KML (Google Earth) format, you can convert it to GPX at <a href="https://gpx2kml.com/">gpx2kml.com</a>, and then downlaod the GPX file.</p>
<p>If you just have a list of coordinates, you can make a GPX file at <a href="http://www.gcgpx.cz/?lang=en">www.gcgpx.cz</a></p>
<p>Once you have the GPS or GPX file downloaded, just drag the file into the map in your ID editor browser window. It should show up as bright purple lines and points with labels. Now you can zoom in to each point and get things exactly right.</p>
