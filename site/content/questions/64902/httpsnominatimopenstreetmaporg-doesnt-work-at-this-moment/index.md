<p>Currently some OSM servers are moved from London to Amsterdam. The API and possible other servers run on backup hardware until the primary servers are up again. This will degrade performance and can also lead to sporadic problems. See <a href="https://lists.openstreetmap.org/pipermail/talk/2018-July/081009.html">https://lists.openstreetmap.org/pipermail/talk/2018-July/081009.html</a> . You can check the current state at <a href="https://twitter.com/osm_tech">https://twitter.com/osm_tech</a></p>
