<div class="answer-body">
<p>There is an article on the wiki for TMC <a href="https://wiki.openstreetmap.org/wiki/TMC">here</a>. It doesn't look like a simple scheme.</p>
<p>If you are bringing these in from another database/source then you will need to follow the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">import guidelines</a> and check license compatibility before you continue.</p>
<p>In the iD editor the relations applied to an element are shown below the presets and raw tags on that element. There is no TMC preset in iD so if you want to add these you will need to manually edit the relation tags according to the relevant scheme in the wiki.</p>
