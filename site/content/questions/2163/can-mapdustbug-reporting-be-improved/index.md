<div class="answer-body">
<p>Yes, it can. I've been working with Skobbler on improving MapDust (I wrote the MapDust interface for Potlatch2) and doing some analysis of the bugs reported so far. There are various suggestions I have made to Skobbler for improving their navigation application interface to generate better reports, and they are also adding more features to the MapDust API to e.g. allow us to filter out bugs with default text in Potlatch2.</p>
<p>Improvements should start showing up over the next few weeks.</p>
