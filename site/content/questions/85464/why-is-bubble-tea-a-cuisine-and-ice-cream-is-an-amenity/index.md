<p><code>ice_cream</code> exists as a cuisine value too, see <a href="https://taginfo.openstreetmap.org/tags/cuisine=ice_cream">https://taginfo.openstreetmap.org/tags/cuisine=ice_cream</a> and you can simply add it as a further cuisine value if necessary.</p>
<p>That there is a dedicated amenity value for a place that sells ice cream over the counter is mainly historic (the <code>bubble_tea</code> value was introduced in 2015, <code>amenity=ice_cream</code> pre-2010). Sure one could argue that a more consistent tagging would be <code>amenity=fast_food</code> + <code>cuisine=ice_cream</code> but there is no harm in using the dedicated tag.</p>
