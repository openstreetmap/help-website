<div class="answer-body">
<p>Whether watersheds should be mapped or not is in dispute. Someone created a wiki page for <a href="https://wiki.openstreetmap.org/wiki/Relation:watershed">type=watershed</a>, but that tag has not been officially proposed or voted upon. The documentation describes grouping together all of the waterway ways (like has been done with the river in China/Vietnam), but this doesn't actually describe a watershed (the entire area drained by the waterways) and is a bit of a misnomer.</p>
<p>In the end, there doesn't seem to be consensus either on how to map watersheds/drainage basins, nor whether such mapping should be done at all.</p>
