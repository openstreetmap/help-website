<p>OsMo is a quite new Service introduced in July 2014 where you can send your local position either to a central server in order to share it with a closed usergroup, or directly to another mobile device that displays the position of the first device.</p>
<p>see <a href="http://osmo.mobi">http://osmo.mobi</a> for more details.</p>
<p>So far we have two apps supporting that OsMo feature, one of them is the well known Osmand.</p>

<p>Only a workaround:</p>
<p>Try to install virtualbox or VMware or qemu on your PC and load an iso image from android-x86.org</p>
<p>Thus you can run an emulated android system on your desktop PC.</p>
<p>You can install then Osmand manually in that android system via the latest nightly apk from osmand.net</p>
