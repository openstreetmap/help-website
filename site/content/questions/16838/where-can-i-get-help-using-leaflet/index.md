<p>Leaflet has its own <a href="https://groups.google.com/forum/?fromgroups#!forum/leaflet-js">mailing list</a> where you can ask questions about using the library. If you think you've found a bug in Leaflet, you can report on the <a href="https://github.com/Leaflet/Leaflet/issues">Github issues page</a>, or to request enhancements use the <a href="http://leaflet.uservoice.com/">Leaflet UserVoice page</a>.</p>
