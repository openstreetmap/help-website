<td><div class="item-right">
<div class="answer-body">
<p>OpenStreetMap's current license, the Open Database License (ODbL), allows you to pick any license for "produced works" that you create from our data. Map images are "produced works", so yes, you can use any Creative Commons license for such images.</p>
<p>You still need to ensure that people are made aware that the raw data you used is originally from OpenStreetMap and licensed under the ODbL. In practice, that information would typically be listed alongside your own attribution.</p>
