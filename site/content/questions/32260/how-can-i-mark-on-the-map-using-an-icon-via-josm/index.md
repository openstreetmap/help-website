<p>JOSM is an OpenStreetMap editor and it primary purpose is to add and edit data that permamently resides in the OpenStreetMap database. On top of that, how an object entered in the database is displayed is not controlled by JOSM at all, except in JOSM itself.</p>
<p>If you are looking at adding a marker (not an object) to a OSM based map because you want to mark or point something out: have a look at umap.openstreetmap.fr</p>
