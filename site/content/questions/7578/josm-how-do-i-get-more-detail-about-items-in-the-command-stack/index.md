<div class="answer-body">
<p>You can right click on an item in the command stack, then choose "Select". This will select all of the objects added or modified by that command.</p>
<p>You can then go to <em>View</em> menu - &gt; <em>Info about element</em> (or Ctrl+I) or <em>Object History</em> (Ctrl+H). This will open the details for the selected objects in your browser window.</p>
<p>Though this doesn't work for objects that have been deleted, as it is not possible to select them. I don't think there's any other easy way of getting details for these. If not, it would be worth requesting that this feature is added to JOSM.</p>
