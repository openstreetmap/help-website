<p>For years now I have been using openmtbmap for collecting data.</p>
<p>It is very good but it is a dedicated map for mountain bikers as the name implies.</p>
<p>Unfortunately many POI's are not supported. Another drawback is that some attributes (e.g. opening-times and addresses) are not supplied at all, so I often collect the data twice. I do use the excellent walking_papers but it would be nice to have the data on my 60CSx.</p>
<p>NOTE: I am not a user of OSM-data. I know the areas where I am mapping.</p>
