<p>There's no automated way. In any case you need to determine if there is any kind of significant offset in the imagery you are using now, and if there is, correct for it first before moving anything (because you don't want to do it twice).</p>

<p>Discontinuities in imagery mosaics due to different sources / dates / equipment / orthonormalization etc, are completely normal and it is just bad luck when this happens in the middle of a city.</p>
<p>Solutions can be:</p>
<ul>
<li>using a different imager source (as the answer say)</li>
<li>realigning the imagery depending on which side is being used (a bit of a pita),</li>
</ul>
