<p>Hi,</p>
<p>sorry for the late answer.</p>
<p>Your question is not really specific, it is about legal requirements or the technical details ?</p>
<p>For the first part you should read the <a href="https://wiki.osmfoundation.org/wiki/Licence/Attribution_Guidelines#TV,_film,_or_video_productions">Attribution Guidelines</a>.</p>
<p>For the second, as hinted in the previous link, just leave the attribution as they are if you screenshot an OSM-based application that correctly displays attribution.</p>
<p>If you don't (want to) have end credits, you should at least display "Map data from OpenStreetMap." or something like that at some point.</p>
<p>Hope this helps.</p>
<p>Please ask if you have more specific questions.</p>
