<div class="answer-body">
<p>There seem to be at least two issues that are OSM related:</p>
<ul>
<li>the street name is wrong, see <a href="https://nominatim.openstreetmap.org/ui/details.html?osmtype=W&amp;osmid=326339842&amp;class=highway">https://nominatim.openstreetmap.org/ui/details.html?osmtype=W&amp;osmid=326339842&amp;class=highway</a> it is Via XXV Aprile 1945, not Via XXV Aprile</li>
<li>the unit value may be confusing things, without it we find <a href="https://nominatim.openstreetmap.org/ui/search.html?q=Via+25+Aprile+1945%2C+23%2C+24030+Brembate+di+Sopra%2C+Bergamo%2C+Italy">https://nominatim.openstreetmap.org/ui/search.html?q=Via+25+Aprile+1945%2C+23%2C+24030+Brembate+di+Sopra%2C+Bergamo%2C+Italy</a> (note that probably should be a building there or it isn't quite at the correct location).</li>
</ul>
<p>The third issue is that you are searching on Instagram: Facebook / Meta companies do not use OSM POI data, but prefer to use other providers. Obviously their data isn't completely correct as you can see.</p>
