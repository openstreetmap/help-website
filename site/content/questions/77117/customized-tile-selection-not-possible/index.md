<div class="answer-body">
<p>(to expand on the comment) <a href="http://garmin.openstreetmap.nl/">http://garmin.openstreetmap.nl/</a> has been broken or offering reduced functionality for some time.</p>
<p>It won't get "fixed" until someone volunteers to set up an equivalent alternative elsewhere and commits to maintaining it. See <a href="https://forum.openstreetmap.org/viewtopic.php?pid=802514#p802514">https://forum.openstreetmap.org/viewtopic.php?pid=802514#p802514</a> et al. Until that happens please use an alternative at <a href="https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download">https://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download</a> . If you wanted to set up an alternative yourself and commit to maintaining it, I'm sure the historic users of the old Dutch site would be grateful.</p>
