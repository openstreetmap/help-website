<p>Hello!</p>
<p>How can I edit Openfietsmap Lite map already downloaded and installed in BaseCamp/Mapsource via <a href="http://garmin.openstreetmap.nl">http://garmin.openstreetmap.nl</a> ?</p>
<p>I already know that I can change/edit map style like colours via editing *.typ files in tool like TYPViewer but I would like make some further modifications - for example remove some data like unwanted POIs, some roads etc. I admit that editing via Mkgmap syles is pretty difficult and not user friendly ( <a href="https://wiki.openstreetmap.org/wiki/Mkgmap/help/Custom_styles">https://wiki.openstreetmap.org/wiki/Mkgmap/help/Custom_styles</a> ) that's why I am looking for some other way to change some things.</p>
<p>Are there any editors where I can edit exported gmapsupp.img map from MapSource/Basecamp? And then install it in BaseCamp to see the changes I made?</p>
<p>I found GPSMapEdit 2.1 (<a href="http://www.geopainting.com/">http://www.geopainting.com/</a> ) but I do not know If I can edit final gmapsupp.img map file and then again export it after modifications to my Garmin device. Will it screw up something?</p>
<p>Thank You for help</p>

<p>I don't believe GPSMapEdit actually does what you want, it seems to be a more conventional waypoint/track editor. IMHO your best bet if you don't want to generate your own style and maps with mkgmap is to look at one of the many alternatives to openfietsmap for example <a href="https://extract.bbbike.org/">https://extract.bbbike.org/</a> (don't know how good their routing is) or <a href="https://openmtbmap.org/">https://openmtbmap.org/</a> (they have multiple variants available).</p>
