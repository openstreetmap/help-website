<div class="answer-body">
<p>"editing based on an uploaded GPS trace" works in <em>exactly</em> the same way that Potlatch 2 used to. With iD set as ypur default editor, go to the list of traces and select "Edit Map". Your trace (with any waypoints!) is visible in the editor.</p>
<p>Or are you asking a slightly different question? Potlatch 2 also allowed you to load other traces at the same time in the editor. I'm not aware that iD supports that (but could be wrong, and of course an overlay of "all GPS traces" is available in iD).</p>

<p>You do not have to upload the trace to OSM. Load the trace onto your PC zoom into OSM map and open iD, reduce the window size a little, then file search on your PC and just drag the gpx file into the open iD window. You should see the trace in the edit window. Maximise the mapping window and start editing.</p>
