<p>Well, using OSM data with a certain backround image is not a matter of OSM itself.</p>
<p>You need a framework like <a href="http://leaflet.cloudmade.com/">Leaflet</a> or <a href="http://openlayers.org/">OpenLayers</a>, thus you can publish rendered OSM data on a webbrowser maybe combined with an image.</p>
<p>Have a look at their websites, there are some examples how to use these frameworks for special purposes.</p>
