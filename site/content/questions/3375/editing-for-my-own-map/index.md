<div class="answer-body">
<p>I'd recommend checking out <a href="http://maperitive.net">Maperitive</a>, it's quite an easy way of selecting and exporting parts of a map.</p>
<p>For editing, Maperetive can export vector graphics (SVG) for Inkscape or Illustrator, or bitmap graphics for editing in any regular image editor. I haven't tried the SVG, but I assume that would be the best if you want to edit features on the map, not only add POIs on top.</p>
<p>If you want to export as editable SVG you'll have to download the OSM data (e.g. from <a href="http://download.geofabrik.de/osm/">Geofabrik</a>) for use in Maperitive, not use the default web-source.</p>

<p>As suggested, maperitive is an easy solution. But if you want to use mapnik, you can:</p>
<ul>
<li>store your edits in a .osm file</li>
<li>import this data in a postgis database</li>
<li>built a mapnik style using your own database</li>
</ul>
<p>I've done that and wroten <a href="http://wiki.openstreetmap.org/wiki/User:NicolasDumoulin/PlanStatique/DICRIM">some notes</a>. Let me know if you need more details.</p>
