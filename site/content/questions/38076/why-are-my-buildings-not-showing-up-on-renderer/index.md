<p>I've been trying to add some buildings (shops) in my local town to improve the coverage a bit. As this is my first edit I'm not really sure what I'm doing so I wanted to check that I'm not doing something wrong.</p>
<p>I created the buildings using iD and based the details I entered on an area of Bristol that looked similar (<a href="http://www.openstreetmap.org/#map=19/51.46078/-2.60175">here</a>). The buildings are showing up on the map data view (<a href="https://www.openstreetmap.org/#map=19/51.68168/-2.35629&amp;layers=D">here</a>), but aren't being rendered on the "standard" renderer. They have appeared on the "<a href="https://www.openstreetmap.org/#map=19/51.68168/-2.35663&amp;layers=H">Humanitarian</a>" renderer. I've tried refreshing the tile by visiting <a href="http://a.tile.openstreetmap.org/18/129356/86965.png/dirty">http://a.tile.openstreetmap.org/18/129356/86965.png/dirty</a> but that doesn't seem to make a difference.</p>
<p>As far as I can see, there aren't any significant differences between a simple building like <a href="http://www.openstreetmap.org/way/263438176">this one</a> (which is rendered) and the bank that I added <a href="https://www.openstreetmap.org/way/310241305">here</a>: everything I've read about this has suggested that the important thing is to have the building:yes tag, which both have.</p>
<p>Am I just being impatient or is there something that I've missed?</p>
<p>Thanks for the help.</p>
