<div class="answer-body">
<p>In short:</p>
<ul>
<li>Get an account on <a href="http://osm.org">osm.org</a></li>
<li>log in</li>
<li>on the map on osm.org go to the feature which is not correct</li>
<li>click on "edit"</li>
<li>select the feature by clicking on it</li>
<li>change the name (maybe also brand=, ref= or operator=) to the correct value</li>
<li>save the edit using a good comment</li>
</ul>
