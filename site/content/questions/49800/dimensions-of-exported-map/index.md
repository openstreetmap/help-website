<p>You could get very rough dimensions by using the scale rule on the main map page. My opinion, the best solution would be to install Viking <a href="https://sourceforge.net/projects/viking/">https://sourceforge.net/projects/viking/</a> It is free, it can display several OSM rederings, it as a print function ( well in windows, it does ) and there is a ruler tool that will measure point to point at any angle you want.</p>
