<div class="answer-body">
<p>Maybe you just have to wait a bit more for all the map tiles (images) to be refreshed.</p>
<p>If that does not fix your problem there is nothing you can do. The map cannot show all objects at the same time, so the creator of the map needs to make some decisions on what to show at which zoom level.</p>
<p><em>Automatic translation:</em><br />
Возможно, вам просто нужно немного подождать, пока обновятся все фрагменты карты (изображения).</p>
<p>Если это не решит вашу проблему, вы ничего не сможете сделать. Карта не может отображать все объекты одновременно, поэтому создатель карты должен принять решение о том, что показывать при каком уровне масштабирования.</p>
