This has been generated by the overpass-turbo wizard.
The original search was:
“highway=&#39;cycleway&#39; in qatar”

{{geocodeArea:qatar}}-&gt;.searchArea;
// gather results
(

  node[&quot;highway&quot;=&quot;residential&quot;](area.searchArea);

// print results
out body;
&gt;;
out skel qt;</code></pre>
<p>i tried to change the {geocodearea:qatar} to {25.2552,51.5635,25.2433,51.5434} but it didn't work how can i define a manual location ?</p>

<div class="answer-body">
<p>There's 2 ways to do a bounding box query, either for the <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_QL#Global_bounding_box_.28bbox.29">whole script</a>, or for <a href="https://wiki.openstreetmap.org/wiki/Overpass_API/Overpass_QL#Bounding_box">each query part</a>.</p>
<p>For the entire script, put something like this at the beginning (I've not checked the order):</p>
<pre><code>[bbox: 25.2552,51.5635,25.2433,51.5434];</code></pre>
<p>To do it on an individual query, put the corners inside parentheses (again, I didn't check if you have the correct order for Overpass API bounding boxes):</p>
<pre><code>node(25.2552,51.5635,25.2433,51.5434)[highway=residential];</code></pre>
