<div class="answer-body">
<p>There is no rendering of the database on the dev server.</p>
<p>Visiting <a href="http://master.apis.dev.openstreetmap.org/">http://master.apis.dev.openstreetmap.org/</a> and editing with Potlatch is the correct way to access the sandbox from Potlatch, no need to configure it.</p>
<p>It is necessary to create an account on the dev server, it does not reuse osm.org accounts. It sounds like you have already done this, just mentioning it for other readers.</p>
