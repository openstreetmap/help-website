<div class="answer-body">
<p>highway=<a href="https://wiki.openstreetmap.org/wiki/Tag:highway%3Dservice">service</a>, service=<a href="https://wiki.openstreetmap.org/wiki/Tag:service%3Ddriveway">driveway</a>, (and typically) access=private</p>
<p>Of course you can also add the tags surface, width, lit as well.</p>
<p>In general, those driveways are only added in case the house is far from the main road. "far" is different for each mapper.</p>
