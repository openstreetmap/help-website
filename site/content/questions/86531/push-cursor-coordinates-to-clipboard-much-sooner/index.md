<p>Hi, I am using OSM to get cursor coordinates in Clipboard. Doing it quite often takes a lot of time.</p>
<p>Here is OSM Process:</p>
<ul>
<li>Right Mouse Click On MAP to open Context Menu,</li>
<li>The option Show address is given,</li>
<li>Clicking it opens the Right Panel with Search Results,</li>
<li>Move the Cursor over the lat/long,</li>
<li>Click it using the Right Mouse button,</li>
<li>Context Menu opens,</li>
<li>Click Copy Link Address.</li>
</ul>
<p>The link is finally in the Clipboard.</p>
<p>It would be great (and probably not too hard to implement) if <strong>Copy Link Address</strong> is available in the <em>Context menu after the first Right Mouse Button Click</em>.</p>
