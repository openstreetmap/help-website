<div class="answer-body">
<p>It sounds like you want to create "your own map" incorporating OpenStreetMap data and not "update the map". There are a few options available (assuming you want some sort of web-based map similar to the maps displayed on the OpenStreetMap website. One is to use "<a href="http://umap.openstreetmap.fr/en/">uMap</a>" (see <a href="/questions/28287/what-is-the-best-solution-for-creating-personalised-maps-for-use-on-client-websites">this previous question</a>). <a href="http://switch2osm.org/">Switch2osm</a> also has a "<a href="http://switch2osm.org/using-tiles/">using tiles</a>" section - the <a href="http://leafletjs.com/">Leaflet</a> <a href="http://switch2osm.org/using-tiles/getting-started-with-leaflet/">examples</a> there might be useful. Another option might be to use <a href="https://www.mapbox.com/tilemill/">TileMill</a>.</p>
