<div class="answer-body">
<p>You could use the <span>search function of JOSM</span> (open e.g. by Ctrl+F): name="some name here" (note that you need quotes if the name includes spaces or some other special characters). Be careful: this simple search term (more complicated ones would be possible to filter more) does not only select streets and does not only select connected ways - so have a look what actually has been selected).</p>
