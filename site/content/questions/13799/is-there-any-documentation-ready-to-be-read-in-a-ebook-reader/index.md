<p>Hello again,</p>
<p>There is plenty of documentation about OSM and related projects. I could not find any documentation, tough, that is thought to be used as a book. Information is much, but is is scattered in hyperlinks and I have to follow loads of links to put everything together. I <strong>did</strong> actually found two books, but they are only available on paperback and I would happily avoid to use more sliced trees and to pay S&amp;H costs. I find quite odd that for an avant-garde technology project like this there is not a single e-book available. So probably I am missing something. Is there any OSM documentation available on .ePub format? What I really need is something I can browse and study while I am off-line and on the go. Thanks!</p>

<p>There is an ebook about OpenStreetMap on FLOSS Manuals. It is mostly a beginner's guide, and is available to download as Epub or PDF. <a href="http://en.flossmanuals.net/openstreetmap/">OpenStreetMap - FLOSS Manuals</a></p>
<p>There is also a "Beginning OpenStreetMap" guide on LearnOSM. It is available to download as PDF. <a href="http://www.learnosm.org/">LearnOSM</a></p>
