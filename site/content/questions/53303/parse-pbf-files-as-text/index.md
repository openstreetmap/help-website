<div class="comment-text">
<p>On Ubuntu use <code>base64</code>, it is located in the <code>coreutils</code> package and already preinstalled.</p>
<p>To transform any file into a base64 encoded ASCII file run <code>base64 origFile &gt; asciiFile</code>. To transform it from ASCII back into the original format run <code>base64 -d asciiFile &gt; origFile</code>. Note that a base64 encoded file will be around 33% larger.</p>
</div>
