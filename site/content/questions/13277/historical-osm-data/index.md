<div class="answer-body">
<p>The history data is dumped to special planet files located in <a href="http://planet.openstreetmap.org/full-experimental/">http://planet.openstreetmap.org/full-experimental/</a></p>
<p>Country level extracts can for example be found here <a href="http://odbl.poole.ch/extracts/">http://odbl.poole.ch/extracts/</a></p>
<p>There is an experimental osmosis plugin located here</p>
<p><a href="http://svn.openstreetmap.org/applications/utils/osmosis-history/">http://svn.openstreetmap.org/applications/utils/osmosis-history/</a></p>
<p>Note besides the files being unwieldly, a lot of the history file processing should be considered experimental at this point in time.</p>
<p>You can extract the data for specific points in time from the full history files, this should work for full history extracts too.</p>
<p>The alternative is to retrieve the individual historic planet dumps, the files with names in this format planet-YYMMDD.osm.bz2 (better in the compact .pbf binary format) and extract the region you are interested in.</p>
<p>The later solution will require substantially more space and computing power than getting it to work with a already existing full history extract, bur may be bit simpler.</p>
