<p>Hello, I'm thinking of using Openstreetmap on a Real Estate site in Russia, instead of Yandex Maps. But the current metro (subway) symbols would not fly here. Link shows how Russians are used to seeing metro stations on a map. <a href="https://yandex.ru/maps/213/moscow/?source=wizbiz_new_map_single&amp;ll=37.603441%2C55.765236&amp;z=17">https://yandex.ru/maps/213/moscow/?source=wizbiz_new_map_single&amp;ll=37.603441%2C55.765236&amp;z=17</a></p>
<p>Basically, bigger font. Symbol "M" in a circle and he circle has a set color depending on the line. The circle can be white, color is not necessary, just pretty cool. Look at what Google did for Moscow.</p>
<p>Has anybody done this, added custom symobols? Or is there a document, that would explain how to do this? I'd like to try and figure out how difficult it would be to add this.</p>
<p>Notice that Google Maps assigned a specific metro symbol for London, New York, Moscow, etc metros.</p>
<p>Regards,</p>

<div class="answer-body">
<p>You cannot change the symbols that are used on the standard map rendering. That is, unless you install your own version of the server and tweak the rendering (aka drawing) rules.</p>
<p>This is exactly what the French community did, see e.g. <a href="http://tile.openstreetmap.fr/?zoom=15&amp;lat=48.85496&amp;lon=2.36157&amp;layers=B0000000FFFFFF">http://tile.openstreetmap.fr/?zoom=15&amp;lat=48.85496&amp;lon=2.36157&amp;layers=B0000000FFFFFF</a> for how they render metro stations (already similar to what you want, an "M" in a circle).</p>
