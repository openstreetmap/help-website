<div class="answer-body">
<p>For such long period it makes sense to update the OSM data. There is no process to re-check the status in 2026 but hopefully someone will be vigilant then and update the data once again. One could also add a <code>note=</code> tag to certain elements to point mappers to the problem.</p>
<p>For the blocked part you should update the existing <code>highway</code>s. If they are being rebuilt the <a href="https://wiki.openstreetmap.org/wiki/DE:Tag:highway%3Dconstruction"><code>highway=construction</code></a> tag is probably appropriate. If they are merely closed add appropriate <a href="https://wiki.openstreetmap.org/wiki/DE:Key:access"><code>access</code></a> tags, e.g. <code>access=no</code>.</p>
<p>As far as I can see there are already mapped highways along the gpx track you shared. The section south of the parking lot <em>P+M Pforzheim Ost</em> is unroutable, though, as the track carries a <code>access=agricultural</code> tag wich closes is to all traffic but agricultural vehicles. This should probably be <code>motor_vehicle=agricultural</code> instead, leaving the track open for foot and bicycle traffic. I would not change the <a href="https://www.openstreetmap.org/relation/2083866">route relation of Enztalradweg</a> unless the new route is really signposted as such.</p>
