<div class="answer-body">
<p>You can use JOSM to select "Non-branching way sequences" for a series of lines. Easier than selecting multiple objects in either iD or there. That said, JOSM still has rectangular click-drag selection (aside from iD's lasso), more highly customizable selecting and hiding filters, selection history, undo selection, and many more powerful functionalities. Notably at a basic level, it differentiates [Ctrl] picking selection, and [Shift] add to selection; compared to iD using [Shift] for picking only. This prevents you from accidentally unselecting the same object.</p>
