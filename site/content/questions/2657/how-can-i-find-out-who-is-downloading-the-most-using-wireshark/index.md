If you are running Wireshark on Linux: Great. If you are a Windows user you want to get an AirPcap Stick from Cace Technologies (now Riverbed).</p><p>Next you configure your encryption key with Edit -&gt; Preferences -&gt; Protocols -&gt; IEEE 802.11.</p><p>If your bandwidth leecher is active you can probably identify him/her from the first tracefile. Use Statistics -&gt; Endpoints -&gt; IP and sort the columns to identify the top listener. Next get a 2 by 4 and bring the user in line.</p><p>If the user is not active (say, downloading after hours) configure a multi-file capture with Capture -&gt; Options.</p><p>Cace Pilot is a separate licensed tool to analyze multi-file captures. Start a new capture every 64 or 128 MB. If you don't have it or can not afford it or can't wait for the license: Look at the time stamps of the trace files.
