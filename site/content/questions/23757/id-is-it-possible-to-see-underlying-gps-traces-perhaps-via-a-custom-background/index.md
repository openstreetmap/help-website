<p>Within the background settings in iD, it is possible to define a custom background - the on-screen text says:</p>
<pre><code>&quot;Enter a tile template. Valid tokens are {z}, {x}, {y} for Z/X/Y scheme and {u} for quadtile scheme&quot;.</code></pre>
<p>Is there anywhere a layer of underlying GPS traces that is rendered that might be used as a custom background, so that users can see what traces have been recorded in an area, which in turn will help to see if there's any underlying offset in the canned background imagery sources such as Bing etc.?</p>
