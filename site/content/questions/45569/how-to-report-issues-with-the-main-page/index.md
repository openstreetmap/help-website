<p>Hi,</p>
<p>I noticed some issues with the main page:</p>
<ul>
<li>In the German translation, if I select an item which has a time on it (for example, a note), it says "Erstellt von ... vor vor 5 Monaten" where the original text would be "Createdd by ... about 5 months ago". Someone should replace it with "Erstellt von ... vor etwa 5 Monaten".</li>
<li>If I have a look at the change sets in the currently viewed region, I see a lot of changesets not really related to it. For example, some days ago, I was shown <a href="https://www.openstreetmap.org/changeset/34171114">this one</a>, despite of it not having any changes in the viewed region, but, due to its immense size, its bounding box touching my region. Maybe the displayed changesets shouldn't be determined with that bounding box, but by some other means sucha as presence of of objects located in the current view.</li>
</ul>
<p>How can I report these issues?</p>

<div class="answer-body">
<p>Generally you should file a new issue at <a href="https://github.com/openstreetmap/openstreetmap-website/issues">https://github.com/openstreetmap/openstreetmap-website/issues</a> .</p>
<p>However, in these two particular issues:</p>
<ul>
<li>Translations are handled at Translatewiki: see <a href="https://translatewiki.net/wiki/Translating:OpenStreetMap">https://translatewiki.net/wiki/Translating:OpenStreetMap</a></li>
<li>This is a long-standing issue and one which is unfortunately very complex to fix. It's already known and there's no need to make a duplicate issue. That said, if you'd like to contribute some code, <a href="https://github.com/ppawel/openstreetmap-watch-list">https://github.com/ppawel/openstreetmap-watch-list</a> is the place to start!</li>
</ul>
