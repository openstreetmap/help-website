<p>There are many ways of representing OSM data in PostgreSQL. The one used by the master database tends to be called "API DB", and the Osmosis utility comes with an SQL file to create it. However since the API is continuously improved, if you really want the latest schema, check <a href="https://github.com/openstreetmap/openstreetmap-website/tree/master/db.">https://github.com/openstreetmap/openstreetmap-website/tree/master/db.</a></p>
