<div class="answer-body">
<p>Good resources to start are <a href="https://wiki.openstreetmap.org/wiki/Beginners%27_guide">the beginner's wiki</a> and <a href="http://learnosm.org/">learnosm</a>.</p>
<p>If you want to just show your own data on top of OSM, <a href="http://umap.openstreetmap.fr/">umap</a> is often the answer. <a href="https://wiki.openstreetmap.org/wiki/UMap">More info</a>. If you have your own data you want to integrate into OSM itself, a good place to start is the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">Import guidelines</a>. In general, it is recomended to really get to know OSM before getting into that.</p>
