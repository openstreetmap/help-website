<div class="answer-body">
<p>The wiki page and editor presets for a <a href="https://wiki.openstreetmap.org/wiki/Tag:tourism%3Dpicnic_site">picnic site</a> suggest the already well-established tag <a href="http://taginfo.openstreetmap.org/keys/fireplace#values">fireplace=yes</a>. Therefore I suggest to use this tag, too (without the <em>tourism=picnic_site</em> if there is none).</p>
