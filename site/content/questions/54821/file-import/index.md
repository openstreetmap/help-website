<p>Hi Peter,</p>
<p>you should get familiar with OpenStreetMap mapping by mapping some features in the area around you using an editor for experienced mappers like JOSM before importing any data. Don't forget to read and understand the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">Import Guidelines</a> which apply to every import. If you violate these guidelines, your import will be reverted.</p>
