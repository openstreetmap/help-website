<div class="answer-body">
<p>Taginfo in the UK suggests a few ways of doing it - either a <a href="http://taginfo.openstreetmap.org.uk/keys/garden#values">value for "garden"</a> or a <a href="http://taginfo.openstreetmap.org.uk/keys/garden%3Atype#values">value for "garden:type"</a>, and some keys for which <a href="http://taginfo.openstreetmap.org.uk/search?q=beer#values">"beer_garden" is a value</a>. If enough pub beer gardens get mapped we can start worrying about tag standardisation but there are so few mapped currently that it matters less than getting more mapped.</p>
