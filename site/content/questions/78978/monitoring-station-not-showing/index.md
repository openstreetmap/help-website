<div class="answer-body">
<p>Currently monitoring stations are not rendered on the standard style map see <a href="https://github.com/gravitystorm/openstreetmap-carto/issues/2516">https://github.com/gravitystorm/openstreetmap-carto/issues/2516</a></p>
<p>You will either need to find a map that displays these, create one yourself, or display such data on top of a map, for example with umap <a href="https://wiki.openstreetmap.org/wiki/UMap">https://wiki.openstreetmap.org/wiki/UMap</a></p>
