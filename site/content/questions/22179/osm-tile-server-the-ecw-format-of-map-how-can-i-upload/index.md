<div class="answer-body">
<p>.ecw is a fairly common imagery format by ERDAS. It is proprietary, but there is a free SDK that works with gdal. <a href="http://www.gdal.org/frmt_ecw.html">http://www.gdal.org/frmt_ecw.html</a> has more information on making those work together.</p>
<p>You can use a variety of software to server OSM/Google numbered tiles. Mapproxy, mapserver, tilecache, and tilestache should all work, but how to set those up isn't really an OSM question and you'd be better off with their respective support channels.</p>
