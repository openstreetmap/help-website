<p>I'm not aware of a way to do it directly into the map.</p>
<p>I think your best bet would be to embed the uMap into another webpage and have the page counter there. You could pretty easily set up a Github Pages website or some other free service that would do the trick, and add the page counter code into that (e.g. <a href="https://github.com/brentvollebregt/hit-counter">this one</a>).</p>
