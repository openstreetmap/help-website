<p>If you're doing your own Openlayers based page then yes, see <a href="http://dev.openlayers.org/docs/files/OpenLayers/Control/Graticule-js.html">graticule</a> with <a href="http://openlayers.org/dev/examples/graticule.html">examples here</a>, though not using OSM data as the base layer. It isn't used on the currently available OpenStreetMap examples though, but there is an <a href="http://forum.openstreetmap.org/viewtopic.php?pid=47396#p47396">example on the forums</a> that I think you could save locally to obtain what you are after (thought you might want to take out the Osmarender section).</p>
