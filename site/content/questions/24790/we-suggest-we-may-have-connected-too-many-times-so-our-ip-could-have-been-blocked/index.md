<div class="answer-body">
<p>The <a href="https://wiki.openstreetmap.org/wiki/Nominatim_usage_policy">Nominatim usage policy</a> is defined in the wiki. Particularly there is a limit of <em>1 request/s</em>. This applies only to the Nominatim instance provided by OpenStreetMap. The instance provided by <a href="http://developer.mapquest.com/web/products/open/nominatim">MapQuest</a> has less strict conditions. And of course you can always deploy your <a href="https://wiki.openstreetmap.org/wiki/Nominatim/Installation">own Nominatim instance</a>.</p>
