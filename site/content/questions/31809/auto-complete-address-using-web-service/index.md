<p>No, OpenStreetMap does not provide any kind of type-ahead geocoder or address search. You would have to build that yourself based on a database import that you perform on your own server.</p>
<p>There is a third-party public service that does auto-complete from three characters here: <a href="http://photon.komoot.de/">http://photon.komoot.de/</a> - read the terms of use at the bottom of the page. The software used there is openly available so you can set up this service on your own machine if you want.</p>
