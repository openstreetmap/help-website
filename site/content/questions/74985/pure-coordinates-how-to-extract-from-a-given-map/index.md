<div class="question-body">
<p>From an OpenStreetMap site I can extract an URL which has for example this form: <a href="https://www.openstreetmap.org/?mlat=48.7726&amp;mlon=9.1688#map=12/48.7726/9.1688"><code>https://www.openstreetmap.org/?mlat=48.7726&amp;mlon=9.1688#map=12/48.7726/9.1688</code></a>.</p>
<p>I can also rightcklick on the map an then "show address", it shows the coordinates with an separating comma: <code>48.78642, 9.05533</code>. But when I copy and paste it, there is more: <a href="https://www.openstreetmap.org/#map=18/48.78642/9.05533"><code>https://www.openstreetmap.org/#map=18/48.78642/9.05533</code></a> !</p>
<p>But this all is not the syntax to type in in the search field, which should be (as far as I know) of the form: 50°8'7"N,7°8'7"E or 48.641956, 9.474610 .</p>
<p>How can I extract from a given site the coordinates which I or another person can <strong>directly</strong> type into the search field, i.e. the pure coordinates (without "www.openstreet..." and so) ?</p>
