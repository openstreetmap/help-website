<div class="answer-body">
<p>Geolocation is already available. In the toolbar on the right of the map, click on the 3rd icon down (the little arrow), for "Show My Location".</p>
<p>This will use your browser's geolocation, and show the map for your current location.</p>
<p>For feature requests, the correct place is to add an issue on <a href="https://github.com/openstreetmap/openstreetmap-website/issues">openstreetmap-website on Github</a>. It is worth searching to see if an issue has been reported previously.</p>
<p>Specifically, see this recent issue about options for geolocation: <a href="https://github.com/openstreetmap/openstreetmap-website/issues/373">Rationalize multiple "locate me" type functions</a></p>
