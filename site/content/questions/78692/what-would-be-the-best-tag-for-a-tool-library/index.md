<p>This is a rather new trend, so I'm wondering what is the best way to describe a tool sharing library as quite well described by its Wikipedia entry;</p>
<blockquote>
<p>"A tool library is an example of a Library of Things. Tool libraries allow patrons to check out or borrow tools, equipment and "how-to" instructional materials, functioning either as a rental shop, with a charge for borrowing the tools, or more commonly free of charge as a form of community sharing."</p>
</blockquote>
<p>Source: <a href="https://en.wikipedia.org/wiki/Tool_library">https://en.wikipedia.org/wiki/Tool_library</a><br />
See also: <a href="https://en.wikipedia.org/wiki/Library_of_Things">https://en.wikipedia.org/wiki/Library_of_Things</a></p>
<p>I've seen some examples described with the 'library' tag but I feel it doesn't perfectly fit what a tool library is. There is also the <a href="https://wiki.openstreetmap.org/wiki/Tag:shop%3Dtool_hire">'shop=tool_hire'</a> but the philosophy and the mood behind these shops is also sufficiently different to be separated.</p>
