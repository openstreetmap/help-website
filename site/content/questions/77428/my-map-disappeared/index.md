<p>The link you have posted is working for me.</p>
<p>Is you problem that you cannot edit the map any more? If you were not logged in when you created the map then you need a secret editing link to enter the map's edit mode. Maybe you can still retrieve that link from your browser history. If that is not possible you can download the complete map data (via the "share" button) and create a new map using that downloaded data. Make sure you are either logged in that time or save the editing link.</p>
