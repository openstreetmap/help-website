<div class="answer-body">
<p>Without wanting to set HOT policy, and given that there is likely no real guranteed to work solution, I would suggest to try avoid working on tiles which have active neighbours.</p>
<p>If you do that you can simply extend the road/area you are tracing in to the neighbouring tiles and then work on them. If you see a conflict in the making (aka somebody takes a neighbouring tile), I suspect the best thing to do is simply coordinate with the mapper in question (use the OSM message system, IRC etc).</p>
