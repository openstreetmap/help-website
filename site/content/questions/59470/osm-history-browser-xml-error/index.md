<p>Other standalone "large object viewer" options:</p>
<p><a href="http://osm.mapki.com/history/way.php?id=6117959">http://osm.mapki.com/history/way.php?id=6117959</a></p>
<p>works on nodes and ways, but not relations, and shows tags, values and (for ways) component nodes.</p>
<p>For relations:</p>
<p><a href="http://osmlab.github.io/osm-deep-history/#/relation/50288">http://osmlab.github.io/osm-deep-history/#/relation/50288</a></p>
<p>Works for nodes, ways and relations; shows tags and values only.</p>

<p>(for completeness) JOSM "download object" is only doing the same as <a href="https://www.openstreetmap.org/relation/3126449">https://www.openstreetmap.org/relation/3126449</a> , not <a href="https://www.openstreetmap.org/relation/3126449/history">https://www.openstreetmap.org/relation/3126449/history</a> . From there you can "view history" but it doesn't actually show you the history, only who has edited it. You then have to manually hunt out versions like you would with changes to a wiki page; you can't just search like you would be able to if <a href="https://www.openstreetmap.org/relation/3126449/history">https://www.openstreetmap.org/relation/3126449/history</a> didn't time out.</p>
