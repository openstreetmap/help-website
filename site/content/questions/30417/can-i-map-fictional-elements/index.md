<p>Definitely not! OpenStreetMap is intended for real, verifiable features. Non-factual data will be deleted.</p>
<p>You can, however, use the OSM code to set up your own fictional mapping website. This requires a fairly high degree of technical knowledge, but some people have already done it. One such site is <a href="http://opengeofiction.net/">opengeofiction.net</a>.</p>
