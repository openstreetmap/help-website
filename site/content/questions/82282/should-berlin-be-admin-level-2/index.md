<div class="answer-body">
<p>From the wiki for <a href="https://wiki.openstreetmap.org/wiki/Key:admin%20level?uselang=en"><code>admin_level</code></a>:</p>
<blockquote>
<p>In addition to the direct usage of this tag with the keys above, the numeric values used by admin_level=* are also used for tags:</p>
<p><code>capital=*</code></p>
<p><code>heritage=*</code></p>
</blockquote>
<p>So I assume the editor that added that meant to indicate that it is the capital of an <code>admin_level=2</code> area.</p>
