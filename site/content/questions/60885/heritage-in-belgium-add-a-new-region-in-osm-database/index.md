<div class="answer-body">
<p>Here are my thoughts (as someone who mapped quite some heritage in Flanders):</p>
<ul>
<li>as you present it, it seems like an import, as you will add the data that someone else provides you. Hence, you have to follow the <a href="https://wiki.openstreetmap.org/wiki/Import/Guidelines">import guidelines</a>.</li>
<li>The heritage items are certainly not complete in the other regions. I try to map as much as possible, but the data from Onroerend Erfgoed is not always up-to-date and I have contacted them already several times about demolished buildings. This means that I first visit the place in person before adding it.</li>
<li>I maintain the JOSM plugin <a href="https://josm.openstreetmap.de/wiki/Presets/BENELUX">Mapping in Belgium</a>, I can extend it for your regions, but I need some additional information, e.g the operator and the different categories they support. Please update the Heritage page <a href="https://wiki.openstreetmap.org/wiki/Key:heritage#Belgium">section for Belgium</a>.</li>
<li>You'll find my email address at the bottom of the <a href="http://www.osm.be/2017/04/01/en-project-heritage-crowd-sourced.html">article on heritage</a> I wrote for the Belgian website</li>
</ul>
<p>p.s. this is not the typical question for the this website, you can better contact the Belgian community on the <a href="https://lists.openstreetmap.org/listinfo/talk-be">mailing list</a>, or on the <a href="https://riot.im/app/#/room/#osmbe:matrix.org">Riot Channel</a></p>
