<div class="answer-body">
<p>Remote control doesn't work with Safari unfortunately. You can open a desktop editor (such as JOSM, Merkaartor or Potlatch 3) by downloading it and running it as a standalone app.</p>
<p>You can however create multipolygons just fine using iD, the default in-browser editor on openstreetmap.org - you don't need to run a different editor for that.</p>
