<p>You may be confusing things here: responses from the API, which as Andy points out are currently only available as XML (with the exception of the Notes API) and on the other hand authorisation of requests to the API with OAuth which has nothing to do with the format of the response.</p>
<p>See <a href="http://tools.ietf.org/html/rfc5849">http://tools.ietf.org/html/rfc5849</a> for OAuth, I believe MapBox has a JS implementation.</p>
