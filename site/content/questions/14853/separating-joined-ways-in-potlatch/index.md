They need to be separated, so that I can show the ditch (which is the real boundary of the coppice) and the footpath that runs in the gap between the coppice and the field. How do I separate the shared node into two, preferably using Potlatch 2, so that I can then move one of them?</p>

<div class="answer-body">
<p>The answer is Shift-J.</p>
<p>J joins two ways, when both ways are selected and they have compatible tags.</p>
<p>Shift-J converts a node shared by several ways into one node for each way, when the node is selected.</p>
