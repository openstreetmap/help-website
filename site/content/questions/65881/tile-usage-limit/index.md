<div class="answer-body">
<p>I guess there aren't any hard limits on purpose. If you put too much stress on the tile servers (or other OSM services) you will get blocked.</p>
<p>OSM's public services have limited resources. You shouldn't run into any problems for small, personal projects. However if you intent to run a business using OSM data then either <a href="https://switch2osm.org/serving-tiles/">setup your own tile server</a> or switch to a <a href="https://switch2osm.org/providers/">paid third-party provider</a>. The same guidelines apply for other services such as geocoding and routing.</p>
