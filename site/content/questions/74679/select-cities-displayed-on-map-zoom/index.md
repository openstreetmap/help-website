<p>Which specific map are you asking about?</p>
<p>Each map style has rules that decides what gets shown when, and those rules are different depending on what map style it is. For example, how OSM data gets converted into a Garmin map is completely different to the way it is handled in OsmAnd, which is also completely different to how it gets shown in the map styles at e.g. <a href="https://www.openstreetmap.de/karte.html">https://www.openstreetmap.de/karte.html</a> etc.</p>
