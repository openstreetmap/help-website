<p>Hallo Chuparuedes</p>
<ul>
<li>OSM.org has an export facility, but that does not help you to create maps for a Garmin device (since you ask for Mapsource, I assume you want to place the map on a Garmin device)</li>
<li>Did you read the instructions for creating a bounding box on <a href="http://extract.bbbike.org/extract.html">http://extract.bbbike.org/extract.html</a> ? Please provide more details in which step you get stuck.</li>
<li>Assuming you want a complete map, GPX is not possible. GPX is only for routes, not for complete maps. There are several maps to choose from for your Garmin device, all under the "Garmin" section of the format menu.</li>
</ul>
