<div class="answer-body">
<p>You can't. OSM does not have a review or an edit confirmation system. If you are using the in-browser editor iD you can set a flag that you would like a review, but your edits will be live immediately in any case, and there is no guarantee at all that the edits will ever be looked at (and most are not).</p>
<p>If you want to check if your edits work, you can</p>
<ul>
<li>have a look at the map and check if the objects are displayed as intended</li>
<li>check wiki.openstreetmap.org if the tags you used are actually "correct"</li>
<li>check one of the QA tools if it shows an error for the objects you created or edited, for example <a href="http://osmose.openstreetmap.fr/">http://osmose.openstreetmap.fr/</a></li>
<li>and a number of edit specific things, for example if you changed a road you might want to check if routing over it works and so on.</li>
</ul>
