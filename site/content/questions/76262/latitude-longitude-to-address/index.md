<p>The operation is called "reverse geocoding".</p>
<p>There are multiple geocoding engines out there that support this, however Nominatim is is what is used by the opensteetmap.org website. For more information see <a href="https://nominatim.org/">https://nominatim.org/</a></p>
<p>Note: use of the OSMF instance is subject to specific use policies: <a href="https://operations.osmfoundation.org/policies/nominatim/">https://operations.osmfoundation.org/policies/nominatim/</a></p>
