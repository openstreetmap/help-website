<p>In theory we could upload your data for you - but you'd be missing out!</p>
<p>The real strength of OpenStreetMap is mappers' local knowledge. You will make a much better map of your village than we ever could.</p>
<p>Mapping isn't difficult. Have a go (you'll probably find the 'Potlatch 2' editor, available if you hover over the Edit tab, much easier than the standard 'Potlatch 1') and come back here if you've got any problems - either that or ask in the online chatroom at <a href="http://irc.openstreetmap.org/"></a><a href="http://irc.openstreetmap.org"></a><a href="http://irc.openstreetmap.org">irc.openstreetmap.org</a>. It'd be great to have you on board.</p>

<p>Thanks to gnurk for offering to check a file.</p>
<p>I have since created a track and downloaded it in <em>.gpx format using Mapsource (purchased with Garmin) and also two free programs, GPSBabel and EasyGPS. Mapsource and EasyGPS both allow the track to be seen on screen. I note that the</em> .gpx file is in fact just a text file not unlike HTML.</p>
<p>Having got that far I decided that in the present weather it was going to be much more comfortable using the Potlatch 2 online software so I used that to draw my village on OpenStreetMap, which I have now done.</p>
