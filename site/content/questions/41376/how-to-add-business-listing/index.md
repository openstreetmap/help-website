<div class="answer-body">
<p>Hi Jacob, if you mean just to add one shop/company location, your own, to the OSM database (and, by this, to OSM-based maps)...</p>
<p>Please <a href="https://www.openstreetmap.org/welcome">just add it yourself</a> (please only objective data, no advert texts). You have already registered an account. See e.g. <a href="/questions/6/">how-do-i-get-started-with-openstreetmap</a>. Welcome to OSM! Of course, we hope that you could add a bit more useful data after you've added your business, but that is up to you of course.</p>
<p>Another option is to add a request for inclusion via a <span>Note</span> or indirectly via <a href="http://onosm.org/">http://onosm.org/</a> (more structured).</p>
