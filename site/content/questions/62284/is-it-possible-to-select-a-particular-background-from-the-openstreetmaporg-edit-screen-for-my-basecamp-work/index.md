<div class="answer-body">
<p>The backgrounds (e.g. the different aerial imagery) from the openstreetmap.org edit screen (likely you use the default <a href="https://wiki.openstreetmap.org/wiki/Editors">editor</a> iD) are <em>not</em> part of the OpenStreetMap data. We are just allowed to use them to add / change to our OpenStreetMap data (e.g. trace missing roads).</p>
<p>So, you can not get a OSM-based download/plugin for your basecamp which provides you those backgrounds. It could be that some aerial imagery providers offer ways <em>themselves</em> to be used in basecamp.</p>
