<div class="answer-body">
<p>In general querying for objects with specific properties is best done with the overpass API and the related UI, overpass turbo. For example at <a href="http://overpass-turbo.eu/">http://overpass-turbo.eu/</a></p>
<p>Typically you would either need to specify the location with a bounding box (aka all the objects in the box) or near to a location.</p>
<p>See</p>
<ul>
<li><a href="https://wiki.openstreetmap.org/wiki/Overpass_turbo">https://wiki.openstreetmap.org/wiki/Overpass_turbo</a></li>
<li><a href="https://wiki.openstreetmap.org/wiki/Overpass_API#Around">https://wiki.openstreetmap.org/wiki/Overpass_API#Around</a></li>
</ul>
