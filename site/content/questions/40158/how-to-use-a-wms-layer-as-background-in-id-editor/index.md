<div class="answer-body">
<p>iD now supports WMS since version 2.7.0 (March 2, 2018) (see <a href="https://github.com/openstreetmap/iD/issues/1141">https://github.com/openstreetmap/iD/issues/1141</a> and <a href="https://github.com/openstreetmap/openstreetmap-website/pull/1777">https://github.com/openstreetmap/openstreetmap-website/pull/1777</a> ).</p>
<p>However, there's isn't already support for custom WMS imagery (see <a href="https://github.com/openstreetmap/iD/issues/4977">https://github.com/openstreetmap/iD/issues/4977</a> ). So to use WMS, you would have to use your own local iD installation.</p>
</div>
