<p>OSM has no idea about "tweets". But it is surely possible to display any geolocated data <em>on top</em> of OSM using <a href="https://wiki.openstreetmap.org/wiki/Leaflet">Leaflet</a> or <a href="https://wiki.openstreetmap.org/wiki/OpenLayers">OpenLayers</a>, and to show them as a <a href="https://help.openstreetmap.org/search/?q=heatmap">heatmap</a>.</p>
