<div class="answer-body">
<p>It is either one thing or the other: you wrote you wanted to "customize in Illustrator". So drop or hide the stuff you don't want in Illustrator.</p>
<p>If you don't want to do that, and need full customization, then loading data in PostGIS and using e.g. QGIS to create custom maps is probably the only way, but I wouldn't classify that as a "simple way".</p>
<p>You can't have it all: either dive in deep and learn to make OpenStreetMap extractions yourself, or use existing data sources with their limitations.</p>
