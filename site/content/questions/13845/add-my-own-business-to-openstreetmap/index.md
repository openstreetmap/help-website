<div class="answer-body">
<p>Based on the statistic tool 'taginfo', OSM has currently 3 POI tagged as <a href="http://taginfo.openstreetmap.org/search?q=shop%3Dpackaging">'shop=packaging'</a>. Although it is not very common until now, you should adopt this tag. It might not be rendered on the official online map (with a special icon), you might not see your shop name. But, you will be present in the database and applications working with OSM data should be able to find your business if they search any value of the tag "shop".</p>
<p>If you don't know any thing about OSM, the simplest for you is to go to the main <a href="http://www.openstreetmap.org/">openstreetmap.org</a> page, create an account if it is not already the case, go to your shop location on the slippy map, click on the tab "Edit" (opening the online editor called Potlatch P2), enable aerial imagery in background (if it is not already displayed) and create a new POI (a simple node) tagged as "shop=packaging".</p>
<p>Since this tag is unusual, you cannot use one of the predefined tags in the editor : click one time on the map to create the node, then click on the tab "Advanced" on the left pane and add your own tag in the table where key is "shop" and value is "packaging". In addition, you can add more tags like "name=<em>" where '</em>' is your shop name. Even better, you can add the <a href="http://wiki.openstreetmap.org/wiki/Address">address</a> and <a href="http://wiki.openstreetmap.org/wiki/Phone">contact</a> informations with the tags "addr:housenumber=(the house number)", "add:street=(the street name)" and "phone=*".</p>
<p>When your POI is completed, type 's' to save your edits into the database. That's it ! It might take some minutes, hours or days until your data is imported into the multiple applications using OSM data.</p>

<p>OSMyBIZ is a dedicated web app for adding businesses to OpenStreetMap. New (as in not present on OSM yet) business are directly added to the OpenStreetMap data, changes are currently stored as OSM Notes with structured content.</p>
<p>See <a href="https://wiki.openstreetmap.org/wiki/OSMyBiz">https://wiki.openstreetmap.org/wiki/OSMyBiz</a> for more information.</p>
