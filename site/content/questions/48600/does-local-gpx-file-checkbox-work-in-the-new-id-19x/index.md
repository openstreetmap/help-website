<p>I seem to be having trouble toggling my local GPX file on and off in iD, and I think this started with the recent updates to iD 1.9.0 and 1.9.1.</p>
<p>What is happening is that I can check or uncheck the "Local GPX file" checkbox, but the GPX file still does (or does not display), until I also toggle one of the Map Features checkboxes below.</p>
<p>For example: GPX file is on. Uncheck "Local GPX file" - GPX is still displayed. Uncheck "Landuse Features" - landuse features and my GPX file are no longer displayed. Re-check "Landuse Features" - landuse features are displayed, GPX file is still off.</p>
<p>Does anyone else see this, or is it just me?</p>
