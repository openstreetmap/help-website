<p>The operations to create a building with the <a href="https://wiki.openstreetmap.org/wiki/JOSM/Plugins/BuildingsTools">JOSM building-plugin</a> shown in the picture in this post.<br />
How do they split the long side of the house so one can extrude with X?</p>
<p>The picture is from <a href="https://github.com/mapbox/mapping/issues/141">Github/Mapbox/Mapping</a> and their recipe to do this is</p>
<blockquote>
<p>B to start with buildings tool<br />
X to extrude building shapes as per the imagery.<br />
Shift+Ctrl to rotate buildings.<br />
Q to orthogonalize buildings<br />
Shift+J to merge polygons</p>
</blockquote>
