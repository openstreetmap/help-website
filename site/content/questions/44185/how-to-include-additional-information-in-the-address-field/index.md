<p>How to include additional information in the address field?</p>
<p>For example, a place has these address fields: street, number, district, city and ZIP code (based on iD editor).</p>
<p>If the place has other information such as: shop, room, apartment ... where you insert them?</p>
<p>That's because the street name is already captured automatically by the editor and the number field must contain only numbers (addr: housenumber).</p>
<p>I have put this information in the note field, but I understand that is not the best alternative. It should be created an additional information field (complementary) for this purpose.</p>
<p>One would inform whether there is any specific tag for this information?</p>

<div class="answer-body">
<p>OpenStreetMap does not limit the amount of fields you can have on an object; it is only the simplified user interface (aka "presets") of some editors that will offer you a limited set of fields. You can always add so-called tags (key-value pairs) yourself. See <a href="http://wiki.openstreetmap.org/wiki/Key:addr">http://wiki.openstreetmap.org/wiki/Key:addr</a> for suggestions about which tags to use.</p>
<p>If all else fails, use "addr:full" and write the full address like it would be written on a postal envelope.</p>
