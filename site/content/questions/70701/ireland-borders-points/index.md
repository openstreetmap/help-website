<div class="comment-text">
<p>You mentioned that the 208 crossings refer to private and public roads. I think that document claims to cover public roads (with a small number that are public on one side only, but none that are private on both sides).</p>
<p>Apparently the Irish Army has counted around 300 crossings, as reported here: <a href="https://www.belfasttelegraph.co.uk/news/northern-ireland/irish-army-identifies-300-border-crossing-points-37631474.html">https://www.belfasttelegraph.co.uk/news/northern-ireland/irish-army-identifies-300-border-crossing-points-37631474.html</a></p>
<p>I realise that as a leak of a confidential document this is very far from open data so you probably can't use it directly. But it does suggest that a large part of the difference is due to private crossings and other minor crossings - the report mentions "scores of private crossings".</p>
<p>The feature you mention isn't really surprising. The border was drawn using administrative boundaries defined in the 17th century, with no account taken of practicalities such as property divisions or the road network.</p>

<p>Hello, there!</p>
<p>I've just completed a personal (spatial) data analysis project in relation to Brexit called 'Identifying Irish Land Border Crossing Points Using OpenStreetMap Spatial Data Sets', which has yielded pretty good (and interesting) results, in my opinion.</p>
<p>If you're interested in seeing those results you can do so in the project report that you will find here: <a href="https://github.com/oriogain/Identifying-Irish-Land-Border-Crossing-Points-Using-OpenStreetMap-Spatial-Data-Sets/blob/master/Identifying%20Irish%20Land%20Border%20Crossing%20Points%20Using%20OpenStreetMap%20Spatial%20Data%20Sets.pdf">https://github.com/oriogain/Identifying-Irish-Land-Border-Crossing-Points-Using-OpenStreetMap-Spatial-Data-Sets/blob/master/Identifying%20Irish%20Land%20Border%20Crossing%20Points%20Using%20OpenStreetMap%20Spatial%20Data%20Sets.pdf</a></p>
<p>After reading the management summary, which is less than 2 pages long, you can, if you wish, skim through the other 20 pages which contain lots of pretty pictures.</p>
<p>All of the project materials (except for the OpenStreetMap roads network shape files that are too big for GitHub - so you'd have to go to Geofabrik for them) are also available for download in this project's GitHub repository here: <a href="https://github.com/oriogain/Identifying-Irish-Land-Border-Crossing-Points-Using-OpenStreetMap-Spatial-Data-Sets">https://github.com/oriogain/Identifying-Irish-Land-Border-Crossing-Points-Using-OpenStreetMap-Spatial-Data-Sets</a></p>
<p>I hope you find the results as interesting as I did!</p>
<p>Regards, Sean.</p>
