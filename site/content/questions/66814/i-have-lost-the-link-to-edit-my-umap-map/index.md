<p>You can write to the admins of umap and hope that they will help you. See "Feedback and Help" on <a href="https://wiki.openstreetmap.org/wiki/UMap">https://wiki.openstreetmap.org/wiki/UMap</a>.</p>
<p>If that does not help, you can at least partially rescue your data by accessing and saving the individual layers you have created</p>
<p><a href="http://umap.openstreetmap.fr/de/datalayer/681017/">http://umap.openstreetmap.fr/de/datalayer/681017/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/681016/">http://umap.openstreetmap.fr/de/datalayer/681016/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/681015/">http://umap.openstreetmap.fr/de/datalayer/681015/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/681013/">http://umap.openstreetmap.fr/de/datalayer/681013/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/680959/">http://umap.openstreetmap.fr/de/datalayer/680959/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/680946/">http://umap.openstreetmap.fr/de/datalayer/680946/</a> <a href="http://umap.openstreetmap.fr/de/datalayer/681040/">http://umap.openstreetmap.fr/de/datalayer/681040/</a></p>
<p>and uploading them into a new map (this time saving the edit link...)</p>
