<div class="answer-body">
<p>We can try - unfortunately (as you've noticed) the openstreetmap site isn't really designed with people just wanting to use the data in mind.</p>
<p>Maybe if we could try and take it one step at a time?</p>
<p>A previous answer to this question: <a href="/questions/1924/north-carolina-maps"></a><a href="/questions/1924/north-carolina-maps"></a><a href="/questions/1924/north-carolina-maps">https://help.openstreetmap.org/questions/1924/north-carolina-maps</a></p>
<p>suggested to look at the "OSM_Map_On_Garmin/Download" wiki page. On there there's a link to this site: <a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/">http://daveh.dev.openstreetmap.org/garmin/Lambertus/</a></p>
<p>and in there are lots of folders, each with a different date on it. There's one created every couple of weeks. The most recent folder is this one: <a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/">http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/</a></p>
<p>and in there there are 14 map downloadable maps (the ".img" files) and a "README" file: <a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/README"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/README"></a><a href="http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/README">http://daveh.dev.openstreetmap.org/garmin/Lambertus/30-01-2011/README</a></p>
<p>which asks "Which file do I use?" and goes on to explain that the size of the map that you can install depends on how much space there is on the SD card that you have in your Nuvi.</p>
<p>So the first questions therefore are:</p>
<ul>
<li>What sort of Garmin Nuvi do you have?</li>
<li>If you've already installed an SD or microSD card, how big was that?</li>
</ul>
