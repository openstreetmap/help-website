<div class="answer-body">
<p>OSM Inspector has a "redaction bot" view that shows objects deleted or changed by the redaction bot. Brighton seems to have been relatively unaffected:</p>
<p><a href="http://tools.geofabrik.de/osmi/?view=redactionbot&amp;lon=-0.14421&amp;lat=50.82845&amp;zoom=12">http://tools.geofabrik.de/osmi/?view=redactionbot&amp;lon=-0.14421&amp;lat=50.82845&amp;zoom=12</a></p>
<p>If you click on a red object on the map you can see its details in the upper right half of the screen. Once the red object has been remapped (or if it is considere a "false alarm"), you can "bin" it using the waste bin icon until there's no more red stuff left.</p>
<p>Detail information shown for deleted objects is from the old, CC-BY-SA based database and <strong>must not be used</strong> to add data to OpenStreetMap.</p>
