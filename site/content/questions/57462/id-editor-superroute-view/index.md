<div class="answer-body">
<p>A screenshot would likely be helpful to illustrate what you are exactly referring to.</p>
<p>But in any case I believe this is currently not possible to turn off and wouldn't make sense from an UI pov. The solution would be to have a dedicated route editing mode (or even just relation editing) in which it is known to which route you are adding.</p>
<p>Note that this is not really a question and that feature requests for iD should go to the projects issue tracker see <a href="https://github.com/openstreetmap/iD/issues/1575">https://github.com/openstreetmap/iD/issues/1575</a> for an existing issue on the topic.</p>
