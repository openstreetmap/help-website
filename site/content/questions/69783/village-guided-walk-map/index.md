<p>I like this version of openstreetmap. <a href="https://opentopomap.org/#map=15/52.33029/-0.20698">https://opentopomap.org/#map=15/52.33029/-0.20698</a> If a screen grab is not good enough this site also generates maps for garmin devices. The map will work with Garmin Basecamp which is available as a free download. Use the PC or Mac version for Basecamp. Basecamp software has map and route printing abilities. Basecamp is primarily for planning routes and tracks so you can send them to a Garmin. It also sends maps to Garmin devices, GPS , Sat Navs and Watches.</p>
