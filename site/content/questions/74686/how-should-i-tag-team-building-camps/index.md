<p>I'm afraid the best match would be some subkey of <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dcommunity_centre">amenity=community_centre</a>.</p>
<p>I use it for children summer camps venues. I guess what the facility you describe are analogous for adults.</p>
<p>The more "private" alternative <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Devents_venue">amenity=events_venue</a> could match as well.</p>
<p>You might want to start a discussion on the <a href="https://lists.openstreetmap.org/listinfo/tagging">tagging ML</a>, as I don't think it has ever been brought up.</p>
