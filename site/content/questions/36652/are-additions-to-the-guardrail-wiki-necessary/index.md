<div class="answer-body">
<p>Of course there is some potential for micromapping <code>barrier=guard_rail</code> (not <code>guardrail</code>), but I don't think help.osm.org is the place to invent new tagging styles. Currently there don't appear to be any commonly used subtags.</p>
<p>Note that there is an approved tag specifically for cable rails: <code>barrier=cable_barrier</code>.</p>
<p>As for <code>barrier=motorcycle_barrier</code>: Constructions designed to keep certain vehicles out of an area (bollards, cycle barriers, ...) are usually tagged as barriers in OSM, so the motorcycle barrier fits right in.</p>
