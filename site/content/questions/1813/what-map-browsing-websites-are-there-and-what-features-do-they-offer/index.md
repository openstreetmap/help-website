<h2 id="openstreetmap.org"><a href="http://www.openstreetmap.org/--"></a><a href="http://openstreetmap.org"></a><a href="http://openstreetmap.org">openstreetmap.org</a></h2>
<p><strong>Layers:</strong></p>
<ul>
<li><strong>Mapnik:</strong> The basic map that you all knwo</li>
<li><strong>Osmarender:</strong> also knwon as Tiles at Home</li>
<li><strong>Cycle map:</strong> highlightes cycle ways and cycling related amenities</li>
<li><strong>Straßen ohne Name / Streets without name(?):</strong> Well, the streets have names, but more POIs are shown than on Mapnik</li>
</ul>
<p><strong>Overlays:</strong></p>
<ul>
<li><strong>Data:</strong> see features</li>
</ul>
<p><strong>Features:</strong></p>
<ul>
<li>Permalink &amp; shotlink</li>
<li>Location search</li>
<li>Edit via Potlatch or desktop software</li>
<li>Export map regions in various formats</li>
<li>See recent changes in an area</li>
<li>Show data for map objects (in a rather technical way that does not appeal end users)</li>
</ul>
