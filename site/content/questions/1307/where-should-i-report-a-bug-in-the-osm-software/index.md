<td><div class="item-right">
<div class="answer-body">
<p>You should report bugs in trac using the <a href="http://trac.openstreetmap.org/newticket">new ticket form</a>. You will need you login with the username and password you use on the main site.</p>
<p>Before you report a bug, you should <a href="http://trac.openstreetmap.org/search">search trac</a> to see if the problem has already been reported - if it has you can add yourself to the CC list for the bug by checking the "Add to CC" checkbox and submitting. You will then get updates when the ticket is changed.</p>
<p>If you do report a bug, please try and set the component correctly so that the bug report can be sent to the best person to deal with it.</p>
<p>Some software, such as <a href="http://josm.openstreetmap.de/">JOSM</a> and <a href="http://www.merkaartor.be/">Merkaartor</a>, has its own project management system, so you should create tickets there if you find bugs in those applications.</p>
