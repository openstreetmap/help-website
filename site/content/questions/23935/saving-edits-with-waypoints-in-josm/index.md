<div class="answer-body">
<p>I guess by waypoints you mean GPX traces.</p>
<p>If the traces came from a local file, you just have to reopen that file again. If the traces came from the server (<em>[x] Raw GPS data</em>) you have to save the GPX Data layer to a separate GPX file and open that file again. The regular .osm file won't be used to store the traces.</p>

<div class="answer-body">
<p>I guess you are talking of saving into a .osm file.</p>
<p>Are you sure that you incorporated the waypoints into the <em>data</em> layer (which may not be the best choice since you may forget to delete the waypoints before you upload to osm.org)? Usually they are loaded via a <em>gpx</em> layer (if the waypoints are from a gps device). A gpx layer is not saved if you save the data layer. Have a look at the <a href="http://josm.openstreetmap.de/wiki/Help/Dialog/LayerList">layer list</a>.</p>
<p>If you want to load your waypoints again, just load the gpx file containing the waypoints again.</p>
