<p>Are you using the default editor iD? If you were not using any editor, <a href="https://www.openstreetmap.org/welcome">welcome</a>!</p>
<p>In case you mean <a href="https://wiki.openstreetmap.org/wiki/ID">iD</a>: Just click and hold the point with your mouse, drag it to a new location, release the mouse button. In case you are using a touch interface it may work slightly different, e.g. with a short tap on the point before and then the drag.</p>
<p>For other editors it usually is very similar.</p>
