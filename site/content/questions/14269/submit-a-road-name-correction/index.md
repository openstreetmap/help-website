<div class="answer-body">
<p>Try <a href="http://openstreetbugs.org">openstreetbugs</a> website</p>
<p>there you can submit a marker with a little text and error description.</p>
<p>With some luck, another mapper discovers that bug and does the correction.</p>
<p>OR:</p>
<p>Login at the main site osm.org with your user account</p>
<p>zoom to that position where the wrong street name is</p>
<p>choose "Edit" from the above screen edge (this is activating the Flash based editor Potlatch2)</p>
<p>click the street with the wrong name</p>
<p>somewhere at the left screen edge the street name is displayed among the other features of that way.</p>
<p>try to edit the name element</p>
<p>save your work with the so called button "Save" ... done.</p>

<p>Mapdust has some advantages and some disadvantages over OpenStreetBugs.<br />
</p>
<p>An advantage is that bugs appear directly in Potlatch for editors to see. A disadvantage is that many Mapdust bugs are created by Skobbler users with little or no idea what they're contributing to, so "bug reports" include things like "my GPS position is wrong".</p>
<p>If the area concerned in OSM is "well-gardened" to remove rubbish from Mapdust, then this is probably the best way to report it (but fixing it yourself would be preferable, of course!).</p>
