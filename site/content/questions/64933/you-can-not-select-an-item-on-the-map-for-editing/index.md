<p>Seems as if the landuse area in question was deleted and that the rendering in the standard style has not caught up yet. <a href="https://www.openstreetmap.org/#map=17/64.44088/76.50846&amp;layers=D">https://www.openstreetmap.org/#map=17/64.44088/76.50846&amp;layers=D</a> the data layer makes this clear.</p>
<p>On re-checking the landuse actually goes up to the road (somebody edited it 20 hours ago). If you still can't select it in iD, check that landuse features are not filtered out (click the "Map data" button on the right).</p>
