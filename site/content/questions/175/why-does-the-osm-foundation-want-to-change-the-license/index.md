<div class="answer-body">
<p>This is <a href="http://www.osmfoundation.org/wiki/License/We_Are_Changing_The_License">covered pretty extensively on the Wiki</a>, but basically the issue is that the Creative Commons Attribution Share-Alike 2.0 (CC-by-SA) licence is not really designed for databases, which Creative Commons themselves appear to acknowledge, so a more appropriate license is being sought.</p>
<p>Also read the <a href="http://www.osmfoundation.org/wiki/License/Why_CC_BY-SA_is_Unsuitable">explanation from the OSM Foundation</a></p>
