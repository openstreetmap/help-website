<div class="answer-body">
<p>My city is also maintaining roads in OpenStreetMap. As far as I know they do it very manually, though, using the editor <a href="https://wiki.openstreetmap.org/wiki/JOSM">JOSM</a>, just as many regular mappers do. Be aware that there are certain <a href="https://wiki.osmfoundation.org/wiki/Licence/Contributor_Terms">license requirements</a> on the data you add. If your data source ("xml file that has all the new streets on it that we got from our gis department") does not comply to the requirements laid out in the terms yoo must not use it to add roads to OSM or change them.</p>
<p>If the source is legit to use then by all means go ahead making appropriate changes in OSM. As long as you do it manually (you can overlay your data source on the map in the editor and draw the road by hand then) there shouldn't be a big problem. If you are talking about huge amounts of data that you would rather like to automatically import it gets tricky. There are many hurdles to overcome, technically and in alignment with the OSM community. You can start reading <a href="https://wiki.openstreetmap.org/wiki/Import">about that on the wiki</a>.</p>
