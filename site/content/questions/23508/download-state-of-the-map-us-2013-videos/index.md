<p>There's no supported way from Vimeo or OpenStreetMap but it's possible to grab the URL used by Vimeo and save the file by using an HTTP web debugger such as the tools built into Chrome (<a href="http://lists.openstreetmap.org/pipermail/talk/2013-June/067291.html">via Developer Tools→Network→ …</a>) or Fiddler2. In particular, after obtaining the URL of the video, one can save the file.</p>
