<p>Your question is not OpenStreetMap related. (If it were about OpenStreetMap then we would have to ask you to refrain from using the imagery as it is for personal use only, and data we store in OSM is for any use, personal, professional, commercial!)</p>
<p>JOSM is not the right tool for downloading and storing WMS data. You should look into free GIS software, like <a href="http://www.qgis.org">Quantum GIS</a>, for that.</p>
