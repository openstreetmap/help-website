<div class="answer-body">
<p><a href="http://code.google.com/p/mapsforge/">Mapsforge</a> currently provides a mapping library for use on Android. It's not "officially" part of OSM (but then very few useful OSM things are "official". the <a href="http://code.google.com/p/mapsforge/wiki/GettingStartedDevelopers">"getting started"</a> page does assume familiarity with both Eclipse and Maven, so if you're not, you might want to start a little further back. If it's Android development you're unfamiliar with, there are some excellent tutorials starting <a href="https://developer.android.com/sdk/index.html">here</a>.</p>
