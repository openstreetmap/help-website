<div class="answer-body">
<p>The "home" in your settings usually only gets used if you click the relevant link in your user menu.</p>
<p>For me the map always starts at the last location. This is remembered via a <a href="/questions/26960/privacy-how-do-i-stop-openstreetmap-from-remembering-my-searcheslocation">cookie</a> in your browser. Maybe something got messed up with the cookies. Just delete all <code>_osm_location</code> cookies (this may be possible via your browser interface, maybe only deleting all cookies is offered by your browser which will also work but it also affects other webpages). Then the map should not start anymore at Majorca. If there is no last location (by cookie) then the home location gets used.</p>
<p>You also could try to use another browser (which uses different cookies). It should not happen then.</p>
