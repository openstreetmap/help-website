<div class="answer-body">
<p>GPSMapEdit is a program external to OSM. Its website is <a href="http://www.geopainting.com/">here</a>. There's an FAQ <a href="http://www.geopainting.com/en/faq.html">here</a>, and details of supported formats <a href="http://www.geopainting.com/en/limitations.html#formats">here</a>.<br />
</p>
<p>There it says that you can export to Garmin GDB format. That's one of <a href="http://www.gpsbabel.org/">GPSBabel</a>'s <a href="http://www.gpsbabel.org/capabilities.html">supported formats</a>, so you could use that to complete the conversion to .osm.<br />
</p>
<p>However, what you get out depends very much on what data GPSMapEdit chooses to export in the first place. If you want to find out more about that, try the links from the "feedback" section of its <a href="http://www.geopainting.com/">website</a>.</p>
</div>
