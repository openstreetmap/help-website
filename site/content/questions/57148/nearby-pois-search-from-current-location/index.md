<p>You can use <a href="https://wiki.openstreetmap.org/wiki/Overpass_API">Overpass API</a> to search for nearby POIs. On Android you can use osmdroid + osmbonuspack to perform queries against Overpass API and display them as overlays on an OSM map.</p>
