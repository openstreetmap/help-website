<div class="answer-body">
<p>You can use <code>roof:direction=</code>. For a symmetric one, either down slope can be used.</p>
<p>There's <a href="https://taginfo.openstreetmap.org/tags/roof:shape=diagonal_pass">https://taginfo.openstreetmap.org/tags/roof:shape=diagonal_pass</a> apparently derived from <a href="https://wiki.openstreetmap.org/wiki/OSM-4D/Roof_table#Note:_How_to_tag_roof:orientation_for_roof:shape=side_hipped_and_similar_roof_shapes?">https://wiki.openstreetmap.org/wiki/OSM-4D/Roof_table#Note:_How_to_tag_roof:orientation_for_roof:shape=side_hipped_and_similar_roof_shapes?</a> --- it's only necessary in case of convex V-shape with a "ditch", or when it has uneven height.</p>
