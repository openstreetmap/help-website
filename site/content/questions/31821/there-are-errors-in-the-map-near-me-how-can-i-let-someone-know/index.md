<p>You can mark where OpenStreetMap is incorrect by placing Notes on the map giving an explanation of the problem. To do this:</p>
<ul>
<li>Find the place where the error occurs</li>
<li>Zoom in as much as possible and centre the map on the location</li>
<li>Click the Note icon on the right-hand side of the map. It looks like a speech bubble</li>
<li>Enter a description of the error, providing as much information as you can</li>
<li>Check the location of the marker, and drag it to a more accurate location if necessary</li>
<li>Click "Add Note"</li>
</ul>
<p>Now other mappers will be able to see the information you've entered and can correct the errors if you've provided enough information to do so.</p>
<p>It's also possible to fix any errors yourself. Start by reading the <a href="http://wiki.openstreetmap.org/wiki/Beginners%27_guide">Beginners' Guide</a> on the wiki</p>
