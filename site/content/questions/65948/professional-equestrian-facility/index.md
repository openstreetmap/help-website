<p><a href="https://www.openstreetmap.org/?mlat=59.5305&amp;mlon=17.6107#map=16/59.5305/17.6107">Here</a> is an area with various "horse maintenance" facilities which are, as far as I know, connected to the racecourse some hundred metres to the south. Currently, this is tagged with a node, place=hamlet, which is clearly wrong. But what should it be? <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dstables">amenity=stables</a> looks about right, but the wiki says its status is "not clear", and suggests using the tags under <a href="https://wiki.openstreetmap.org/wiki/Riding">Riding</a> instead. However, leisure=riding is explicitly about "spare time", while this area is used by professional jockeys, keepers etc.</p>
