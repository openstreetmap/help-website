<div class="answer-body">
<p>OpenStreetMap is a data project with many applications. It is unclear from your question which application you are using, as openstreetmap.org does not have a "markers" functionality. I assume you are talking about <a href="http://umap.openstreetmap.fr">http://umap.openstreetmap.fr</a> . It is possible to import other formats, according to <a href="https://wiki.openstreetmap.org/wiki/FR:UMap/Guide/Importer_un_fichier_de_donn%C3%A9es">this page</a> (in French). You can also use a csv file. Search this website for "<a href="https://help.openstreetmap.org/search/?csrfmiddlewaretoken=u0xTy4xWJQCh3WogysVGkVhYbA0Q2z5w&amp;q=umap+csv&amp;Submit=search&amp;t=question">umap csv</a>" to see questions related to that topic.</p>
<p>If you need help with another application, please update your question accordingly.</p>
