<div class="answer-body">
<p>OpenStreetMap itself only provides global "planet files" in the .osm.pbf format. Regional extracts in that format are provided as a service by third parties. Update frequency of these files is, therefore, dependent on which provider you plan to get these files from.</p>
<p>One of the most popular providers for OSM data is Geofabrik, who also offer an .osm.pbf download for <a href="https://download.geofabrik.de/north-america/us/colorado.html">Colorado</a>. If that's what you're using, the file will be updated daily.</p>
