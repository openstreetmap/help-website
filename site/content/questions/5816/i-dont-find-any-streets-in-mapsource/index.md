<div class="answer-body">
<p>It would help if you specified where you got the OSM Garmin maps from - did you download them from somewhere, or did you generate them yourself, if so how.</p>
<p>Anyway, how the Garmin map format works with address searching in the is not fully understood. So it is not properly supported by Mkgmap, the software used to generate most OSM based Garmin maps. So this means searching for a street or address might not work on some Garmin devices or in MapSource.</p>
<p>But there has recently been work on improving support for addressing in Mkgmap, so newer versions should work better with this. So try using some maps generated with a recent version of Mkgmap, possibly the 'locator' branch. You could try the maps from <a href="http://garmin.openstreetmap.nl/">Free worldwide routable Garmin maps</a>, I think they currently use a fairly new Mkgmap version, so have some addressing support.</p>
</div>
