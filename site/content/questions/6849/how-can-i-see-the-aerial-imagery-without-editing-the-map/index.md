<p>Not here. OpenStreetMap does not have any satellite imagery itself.</p>
<p>Our main provider of satellite imagery is Bing, Microsoft's mapping service. You can go to <a href="http://www.bing.com/maps/">www.bing.com/maps</a> to see these satellite images unencumbered by OSM data.</p>

<p>OpenStreetMap (OSM) itself does not offer/<span>is not about</span> satellite image layers. OSM is just about map data. However, since this map data can be freely used (while respecting the license), <span>many services</span> exist which offer a satellite/aerial imagery layer and OSM-based layers (at some you can overlay OSM-based transparent maps over sat images).</p>
