<div class="answer-body">
<p>This is easy in JOSM. Just start drawing a way using the first two nodes, then press <em>f</em> for <a href="http://josm.openstreetmap.de/wiki/Help/Action/FollowLine">follow line</a> and it will automatically extent the way to the next node. Consecutively pressing <em>f</em> will extent the way further.</p>
