<div class="answer-body">
<p>AFAIK not from the OpenStreetMap website. But, with you can using other tools like <a href="https://hiking.waymarkedtrails.org/#route?id=3110347">Waymarked trails</a> in which you can select a relation and download it as a GPX. That relation in particular seems to have some gaps though.</p>
<p>You can download a GPX with <a href="https://overpass-turbo.eu/s/12PT">overpass-turbo</a> too. Just select export and save it in GPX format.</p>
