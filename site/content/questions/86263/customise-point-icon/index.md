<div class="answer-body">
<p>OSM is a database of geographic information, to create a map in a particular style or with custom icons you do not need to and -should- not touch the underlying data (obviously if you want to correct data, yes then you should be editing it).</p>
<p>One reasonably fast way to get a map with custom icons for specific data would be to use a map created with Umap, see: <a href="https://wiki.openstreetmap.org/wiki/UMap">https://wiki.openstreetmap.org/wiki/UMap</a></p>
</div>
