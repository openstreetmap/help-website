<div class="answer-body">
<p>You can always see the raw code changes on our <a href="http://trac.openstreetmap.org/log/applications/editors/potlatch2">trac</a> website, but in brief, changes over the last fortnight include:</p>
<p><strong>Saving and loading</strong></p>
<ul>
<li>Proper conflict management if someone else has been editing the same area at the same time</li>
<li>Unobtrusive 'floating' alerts (Potlatch 1-style) on save and changeset close</li>
<li>Press Escape to revert an object to the last saved version</li>
</ul>
<p><strong>Relations</strong></p>
<ul>
<li>New 'actions' menu by each relation in Advanced mode - makes it easy to move objects from one relation to another</li>
<li>Edit relations when multiple items selected</li>
</ul>
<p><strong>User interface</strong></p>
<ul>
<li>New loading screen</li>
<li>Press 'J' to join all selected ways (same as clicking the Toolbox icon)</li>
<li>Press 'Q' to make selected ways right-angled (same as clicking the Toolbox icon)</li>
<li>Drag ways when multiple ways selected</li>
<li>Fix autocomplete issues in Advanced mode</li>
<li>Don't allow users to set empty keys</li>
<li>Support for embedded video in the Help dialogue (not in use yet)</li>
</ul>
<p><strong>Other changes</strong></p>
<ul>
<li>Be more selective as to the bugs displayed from MapDust</li>
<li>Tag preset changes</li>
<li>Minor bug-fixes, drawing and efficiency improvements</li>
</ul>
