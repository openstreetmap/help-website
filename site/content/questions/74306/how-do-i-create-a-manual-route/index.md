<div class="answer-body">
<p>As implied by TZorn comment, there are multiple ways of doing that.</p>
<p>The simplest that come to my mind, is to create a route (as a polyline) on <a href="https://wiki.openstreetmap.org/wiki/UMap">Umap</a>, then export it as GPX.</p>
<p>That GPX will be easy to import in a GPS navigation app, like <a href="https://wiki.openstreetmap.org/wiki/OsmAnd">OsmAnd</a>. This one can display and also follow a GPX track.</p>
<p>If you don't want straight lines, use a succession of markers in uMap instead of the polyline, then OsmAnd will compute the route between each point.</p>
<p>Hope that helps. Please give more details about your needs to get a more specific answer. :-)</p>
<p>Regards.</p>
