<p>Looking at taginfo, historic:civilization= <em>has many more uses (~1200) than civilization=</em> (30) (and historic:civilisation, note the s instead of z, is even less), so I'd recommend using that. <a href="http://taginfo.openstreetmap.org/search?q=civilization">http://taginfo.openstreetmap.org/search?q=civilization</a> Also, note that the value should be lower-case (i.e. -- mayan not Mayan)</p>
