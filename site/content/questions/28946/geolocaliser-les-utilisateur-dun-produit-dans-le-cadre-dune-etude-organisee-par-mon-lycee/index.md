<div class="answer-body">
<p>La librairie javascript <a href="http://leafletjs.com/">leaflet</a> est la plus fréquement utilisée avec OSM, et dispose de plugins qui rendent l'affichage de points (vos utilisateurs) sur une carte très facile.</p>
<p>Si vous avez besoin d'obtenir une latitude/longitude à partir d'une adresse, c'est vers les APIs de <a href="https://wiki.openstreetmap.org/wiki/FR:Nominatim">nominatim</a> qu'il faut vous tourner.</p>
<p>Dans les deux cas, il ne s'agit pas d'une "intégration dans osm" mais juste de l'utilisation d'outils et de données osm. Vous pouvez expérimenter avec la page web sur votre ordinateur en local, et éventuellement la transférer sur le site de votre école ou similaire.</p>
<p>D'une manière générale, utiliser les librairies et/ou les APIs de Google Maps n'est pas une bonne idée, surtout dans un cadre OSM. Les raisons tiennent essentièlement aux conditions légales d'utilisation, qui limitent ce qu'il est autorisé de faire voire qui transfèrement la propriété de vos données à Google.</p>
