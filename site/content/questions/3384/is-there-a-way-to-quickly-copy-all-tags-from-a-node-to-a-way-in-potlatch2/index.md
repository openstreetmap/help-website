<p>JOSM does have this feature: Select the node, then Edit menu -&gt; Copy (or Ctrl-C), then select the way, then use edit menu -&gt; "Paste Tags" (or Ctrl=Shift-V).</p>
<p>That will copy all of the tags across.</p>
