<p>Hi laksh, thanks for taking an interest into OpenStreetMap. OSM first of all provides a database with geoinformation that can - among other things - be assembled into a map. There is not "the" OSM map. That's why I am not sure what you are referring to with SDK.</p>
<p>But there are certainly many apps available for iOS which build on OSM. Have a look at our wiki for an <a href="https://wiki.openstreetmap.org/wiki/Apple_iOS">overview</a>. If that does not help, please be a bit more specific what you are looking for.</p>

<div class="answer-body">
<p>The simplest and easiest way of embedding an OSM-based map in an iOS app is probably via a webview using <a href="https://leafletjs.com/">Leaflet</a>. Using that, you can decide what tile source you want to use and what you want to display over the top of that. If you're familiar with displaying a web page in an iOS app, it should be straightforward.</p>
<p>There are other ways - proprietary and semi-proprietary services may use OSM data, which offer ice features but require you to use their SDK, but there are alternatives if you don't want to do that.</p>
