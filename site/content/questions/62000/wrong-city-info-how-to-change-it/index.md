<p>Hi there, it looks good to me. Milano, Roma and the main Italian cities are all mapped the same way</p>
<ul>
<li>Mi <a href="https://www.openstreetmap.org/relation/44915">https://www.openstreetmap.org/relation/44915</a></li>
<li>Roma <a href="https://www.openstreetmap.org/relation/41485">https://www.openstreetmap.org/relation/41485</a></li>
<li>Bg <a href="https://www.openstreetmap.org/relation/45681">https://www.openstreetmap.org/relation/45681</a></li>
<li>To <a href="https://www.openstreetmap.org/relation/43992">https://www.openstreetmap.org/relation/43992</a></li>
</ul>
<p>OSM is a standalone map, it's up to the various software developers using it to find out how to make our data work for them; we don't have to edit things so that they properly show up on third parties software. Feel free to directly get in touch with the Italian community at <a href="https://lists.openstreetmap.org/listinfo/talk-it">https://lists.openstreetmap.org/listinfo/talk-it</a> if you want to analyze it further.</p>
