<p>So when someone registers, they will type in their address, right ? Your server will then receive that address, store it in your database. Then you want to display a map with a marker for that address.</p>
<p>so you need</p>
<ul>
<li>a web form and an underlying database</li>
<li>a geocoder, e.g.Nominatim which translates the address to lat/lon</li>
<li>and some display technology for the map and the marker. This can be achieved with Leaftlet or OpenLayers Javascript libraries.</li>
</ul>
<p>I doubt that you can accomplish this all if you are not a programmer.</p>
