<div class="answer-body">
<p>The normal approach to this would be to place <a href="https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dgate">barrier=gate</a> on Penny Lane and set tags for <a href="https://wiki.openstreetmap.org/wiki/Key:access">access</a> appropriately (if known).</p>
<p>As mentioned <a href="https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dgate#On_a_way">further down</a> the gate page, the gate tags shouldn't go on the same node as the intersection as that would indicate that it blocks all incoming roads.</p>
</div>
