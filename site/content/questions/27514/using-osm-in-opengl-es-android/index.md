<p>As far as OSM is concerned, see <span>Tiles</span>. There you can see how you can calculate tilenames if you have coordinates (please note the usage policies of the tile services!). Tiles are just bitmaps.</p>
<p>How you can render bitmaps on 3d objects on Android is out of <span>OSM help's scope</span>, sorry. Maybe the game with OSM maps <span>Supertuxkart</span> is of some help to you.</p>
