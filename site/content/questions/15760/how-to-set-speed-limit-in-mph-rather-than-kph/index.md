I have no problem (in Potlatch2) in selecting the speed limit currently recorded ready to change it. However, all the options for new speed limit to select are in kph, even though the original speed limit is correctly recorded in mph.</p>
<p>How can I tell it to set the new speed limit to 50mph rather than 50kph (or do I have to select 80kph which would be about right)?</p>
