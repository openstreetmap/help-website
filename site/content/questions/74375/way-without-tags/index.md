<p>I don't know what the "nd ref" is about. But if you look at the way at <a href="https://www.openstreetmap.org/way/238291291">https://www.openstreetmap.org/way/238291291</a> you will see that it is part of a relation <a href="https://www.openstreetmap.org/relation/4743472">https://www.openstreetmap.org/relation/4743472</a> which defines an area of scrub vegetation.</p>
<p>In that case the way itself need not have any tags.</p>
