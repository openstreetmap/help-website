<div class="answer-body">
<p>You can search by address by appending <strong><code>search?query=</code></strong> to the URL, like this:</p>
<p><a href="https://www.openstreetmap.org/search?query=221b%20Baker%20Street%20London">https://www.openstreetmap.org/search?query=221b%20Baker%20Street%20London</a></p>
<p>...but that won't show a marker on the map, at least until you hover the mouse over one of the Nominatum results.</p>
<p>You can add a marker to the map by specifying latitude and longitude with <strong>mlat</strong> and <strong>mlon</strong> parameters:</p>
<p><a href="https://www.openstreetmap.org/?mlat=39.9155&amp;mlon=4.1745">https://www.openstreetmap.org/?mlat=39.9155&amp;mlon=4.1745</a></p>
<p>...but there's no way to do that with just an address.</p>
<p>However, if you're willing to do a two-step process, you can first query by address to get coordinates (you can use the Nominatum API directly, or just use the <strong><code>search?query=</code></strong> method and pull the top result's coordiates from the redirected URL after the <strong><code>#map=</code></strong> string) and then construct a marker URL with the <strong>mlat</strong> and <strong>mlon</strong> parameters.</p>
</div>
