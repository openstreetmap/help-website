<div class="answer-body">
<p>The <em>.prj</em> file is a file is a file that specifies what projection is used in the coresponding <em>.shp</em> file. It is optional as programs can often guess the projection in the shapefile.</p>
<p>However you can download <em>.prj</em> files for any known projection on <a href="http://spatialreference.org/">SpartialReferrence.org</a>. Download the <em>.prj</em> file, move it to the same directory as the <em>.shp</em> and rename it to <em>[shapefilename].prj</em> when the shapefile is called <em>[shapefilename].shp</em></p>
