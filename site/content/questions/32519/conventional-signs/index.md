<div class="answer-body">
<p>The different coloured arrows (I think you mean) are detailed on <a href="https://wiki.openstreetmap.org/wiki/UK_access_provisions">this wiki page</a>.</p>
<p>I don't recognise the first sign you described, but will see if I can work it out (and will edit this answer if I do).</p>
<p>I don't think there is a complete list of conventional signs as used in OSM. For the UK there is a probably incomplete <a href="https://wiki.openstreetmap.org/wiki/Road_signs_in_the_United_Kingdom">road sign wiki page</a>.</p>
<p>Edit: I might have misunderstood your original question. I think you mean what do the symbols you see on the website mean, and not what do the signs you see on footpaths mean so you can add them to OpenStreetMap.</p>
<p>So: The <a href="http://osm.org/go/0EHnD27QH?m=">two lines with a diagonal line</a> marks a gate, and for a key to the different coloured lines, click the "i" button in the right-hand toolbar.</p>

<p>For stiles just use the <a href="https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dstile">barrier=stile</a> tag.</p>
<p>For finding a specific tag there are several approaches. First, there is the <a href="https://wiki.openstreetmap.org/wiki/Map_features">map features wiki page</a> which lists many commonly used tags. Secondly, you can <a href="https://wiki.openstreetmap.org/w/index.php?search">search the wiki</a>. There is also a <a href="https://wiki.openstreetmap.org/wiki/Key:traffic_sign">wiki page about common traffic signs</a>. And then there is <a href="http://taginfo.openstreetmap.org/">taginfo</a> which is an automatically generated overview of all tags currently in use. Finally can also search the <a href="https://wiki.openstreetmap.org/wiki/Mailing_Lists">mailing lists</a> and <a href="http://forum.openstreetmap.org/">forum</a> or just use any Internet search engine you like. If none of them return a result, you can also <a href="https://wiki.openstreetmap.org/wiki/Any_tags_you_like">use your own tag</a>.</p>
