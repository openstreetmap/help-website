<p>Just wondering whether there is any smart shortcut to optimise the following process to require less keystrokes/mouseclicks?:</p>
<p>To draw L-shaped building in JOSM:</p>
<ul>
<li>Select the building plugin, by pressing B</li>
<li>Draw the first rectangle</li>
<li>Press S and select it (so that next rectangle is aligned in same direction)</li>
<li>Press B and draw other overlapping rectangles to describe the building shape</li>
<li>Press S and drag to select all the rectangles</li>
<li>Press Ctrl-J to join the overlapping areas</li>
</ul>
<p>Is there any plugin, mode, or setting which can be told that you always want to join overlapping buildings, or that the default alignment is to the last building you drew?</p>

<p>Not sure if I understand but what I do for a simple L shaped building is:</p>
<ul>
<li>press A for draw mode</li>
<li>draw building with 7 clicks on map (last = first)</li>
<li>press Q to make rectangular</li>
<li>click on building preset I have placed in my icon tray</li>
<li>confirm "building=yes" with enter key.</li>
</ul>
<p>I use neither the building plugin nor "join overlapping areas".</p>

<p>It's as quick for mot things (some take more clicks, some less), but not as precise. Try making a T-shaped building for example : the two segments of the way that should be exactly alligned are unlikely to be using your technique.
