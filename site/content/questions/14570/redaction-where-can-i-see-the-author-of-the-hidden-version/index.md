<p>You cannot access the contributions of non-agreeing people through the API any longer. (It is still possible for users with moderator privilege but this is intended for debugging only.) To find out details about a deleted object, you will have to download the pre-bot planet file (<a href="http://planet.openstreetmap.org/planet-120704.osm.bz2">http://planet.openstreetmap.org/planet-120704.osm.bz2</a>) and investigate things yourself.</p>
