<p>BCNorwich was so kind to update the map. But since your question was how to do it I want to add a bit of explanation.</p>
<p>You can do those changes yourself. All you have to do is create an account on openstreetmap.org, zoom to the area where you want to make the change and click the edit button there. This opens the editor iD. There is a tutorial you should follow to make your first steps. <a href="https://learnosm.org">Learn OSM</a> is another valuable resource to learn more about editing.</p>
