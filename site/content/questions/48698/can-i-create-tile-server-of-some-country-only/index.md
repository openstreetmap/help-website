<p>For users who are looking for such requirement. Here is simple solution to create all openstreetmap tiles for a single country <a href="https://github.com/gagan-bansal/osm-for-my-country">osm-for-my-country</a>. osm-for-my-country is a stack of all OSM related packages and has many scripts/commands for all tasks on a Linux machine. These are the tasks which can be achieved by this package:</p>
<ul>
<li><p>Install all necessary software like osm2pgsql, postgres, mapnik, kosmtik etc</p></li>
<li><p>Download your country data</p></li>
<li><p>Download and install map style openstreetmap-carto</p></li>
<li><p>Create all tiles for your country</p></li>
<li><p>Update tiles on daily basis</p></li>
<li><p>Serve tiles with node.js or you can serve these offline map tiles with any simple http serevr or apache</p></li>
</ul>
