<td><div class="item-right">
<div class="answer-body">
<p>Tags say what an object is in reality and what properties it has, so even though there are <a href="http://wiki.openstreetmap.org/wiki/Any_tags_you_like" title="Any tags you like">no static rules</a> on which tags <em>must</em> be used (you can even invent your own tags), it is sensible to use tags that are well-defined and used by a lot of people, because it makes it easier for programs to use the data.</p>
<p>To find a fitting tag for something you added, you can use the following resources:</p>
<ul>
<li>The <a href="http://wiki.openstreetmap.org/wiki/Map_Features" title="Map Features">Map Features</a>, which is a list of the most important features and tags sorted by category.</li>
<li>Check <a href="http://taginfo.openstreetmap.org/" title="Global Taginfo">Taginfo</a> for tags that are actually used, but be aware that the tag itself may not always be sufficient information to tell what it was meant for.</li>
<li>The <a href="http://wiki.openstreetmap.org/wiki/Proposed_features" title="Proposed features">Proposed features</a> for newly proposed tags.</li>
<li>Look for specific things on <a href="http://wiki.openstreetmap.org/wiki/Howto_Map_A" title="How to map A">"How to map a.."</a>.</li>
<li>Type in what you are looking for in the wiki search on the left (e.g. <a href="http://wiki.openstreetmap.org/w/index.php?title=Special%3ASearch&amp;search=waste+basket&amp;go=Go" title="Wiki search for &#39;waste basket&#39;">"waste basket"</a>).</li>
<li>Ask on the <a href="http://forum.openstreetmap.org">forum</a> or the <a href="http://lists.openstreetmap.org/listinfo/tagging" title="tagging mailinglist info">tagging mailinglist</a> (<a href="http://www.mail-archive.com/tagging@openstreetmap.org/" title="tagging mailinglist archive">archive</a>) or ask <a href="http://wiki.openstreetmap.org/wiki/Contact" title="Contact">anywhere else</a>.</li>
<li>Look at other objects in your area to get an idea on how to tag them.</li>
</ul>
<p>Many <a href="http://wiki.openstreetmap.org/wiki/Editing" title="Editing">editors</a> also have presets for tags, so you can just select the feature you want to add from a menu and it will automatically add the fitting tags.</p>
</div>

<div class="answer-body">
<p>Many editors already support templates (often called "presets") for common types of objects. This feature allows you to enter data without the need to look up tags. The attributes available there will often give you a good impression of the information you could add to OpenStreetMap.</p>
<p><br />
If you prefer adding tags manually, need to use less common attributes or want to discuss new tags, editor templates are probably not sufficient. The <a href="http://wiki.openstreetmap.org">OpenStreetMap Wiki</a> is the most comprehensive documentation of existing and proposed tags. You can often find the information you are looking for by using</p>
<ul>
<li><a href="http://wiki.openstreetmap.org/wiki/Map_Features">Map Features</a>, a table containing most of the commonly used tags</li>
<li><a href="http://wiki.openstreetmap.org/wiki/Category:Features">Category:Features</a>, an alphabetically sorted index of real-world objects, linked to the corresponding tags</li>
<li>the wiki's search box on the left that offers both full-text and page title search</li>
</ul>
<p>The pages documenting individual tags contain a wealth of information, including suggestions for "useful combinations": Tags that you can use to describe a feature in more detail.</p>
<p><br />
Want to know how other mappers tag? <a href="http://taginfo.openstreetmap.de/">Taginfo</a> lets you find out how often a tag is being used and what tags are often used together. Tools with a similar purpose are <a href="http://osmdoc.com/en/tags/">OSMdoc</a>, <a href="http://tagstat.hypercube.telascience.org">Tagstat</a> and <a href="http://tagwatch.stoecker.eu/">Tagwatch</a>.</p>
<p>Keep in mind, though, that there is no authoritative source for tags. Try to be open-minded and avoid unnecessary conflicts with other mappers - mapping should be fun!</p>

<p>There are some tags, such as name=* that can be added to a wide range of objects on the map, whether they be roads, or points of interest. Your best bet is to search the wiki for the type of object you are tagging and follow the advice on the relevant page.</p>
<p>For a list of the more common map features, see <a href="http://wiki.openstreetmap.org/wiki/Map_Features">http://wiki.openstreetmap.org/wiki/Map_Features</a></p>
