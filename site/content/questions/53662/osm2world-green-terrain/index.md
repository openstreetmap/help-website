<td><div class="item-right">
<div class="answer-body">
<p>Setting the <code>createTerrain</code> parameter to false in your OSM2World configuration file will remove the green default terrain entirely. This and other configuration options are documented on the <a href="https://wiki.openstreetmap.org/wiki/OSM2World/Configuration_file">OSM2World configuration file wiki page</a>.</p>
<p>Slightly OT: Please also remember to download the example textures and config file which are available as a separate download from <a href="http://osm2world.org/download/">osm2world.org/download/</a>. This will give you improved visuals compared to the flat colors you get from an unconfigured OSM2World.</p>
</div>

<p>The <a href="https://wiki.openstreetmap.org/wiki/OSM2World">OSM2World wiki</a> mentions a <a href="https://wiki.openstreetmap.org/wiki/OSM2World/Configuration_file">configuration file</a> which you can specify via option <code>--config &lt;file&gt;</code>. Inside the configuration file you can set a <code>backgroundColor</code> or alternatively a <code>backgroundImage</code>.</p>
