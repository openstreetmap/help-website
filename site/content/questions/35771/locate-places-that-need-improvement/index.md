<td><div class="item-right">
<div class="answer-body">
<p>There are a number of ways that other users can find what you have done. It is possible to use WhoDidIt at <a href="http://simon04.dev.openstreetmap.org/whodidit/">http://simon04.dev.openstreetmap.org/whodidit/</a> to create a RSS feed showing changes in an area. I do that for the area around my home which I know pretty well to alert me of changes I should look at to verify they seem to be an improvement.</p>
<p>Many data consumers (map rendering sites, travel planning and routing software) will pick up your change automatically though how long it will take for any particular data consumer will vary.</p>
<p>Just adding a new road, even if you don't know the name, speed limits, lane count, etc. is an improvement that will help others. For example, routers will note it and consider it in providing driving directions, etc.</p>
<p>If others are looking for areas where there are roads missing speed limits or lane counts, then there are tools like those from <a href="http://tools.geofabrik.de/osmi/">http://tools.geofabrik.de/osmi/</a> that will show that and more.</p>
<p>No map will ever be complete and perfect. I view the process as simply improving what is there. You have done that. Thanks!</p>

<div class="answer-body">
<p>While stf is right, no map is ever perfect and everybody has his or her own agenda, there are already some tools in place that show you where the map needs improvements:</p>
<ul>
<li>osm.org itself, e.g. missing buildings and house numbers can be seen there</li>
<li>The notes feature on osm.org allows non-mappers to indicate where the map has errors</li>
<li>Osmose, KeepRight, etc are QA tools that find incorrect or incompatible tags, intersecting buildings, etc.</li>
<li>Overpass, this allows you to write your own queries to see e.g. streets with missing maxspeed, lanes, etc.</li>
<li>Some of the ITO maps highlight streets with maxspeed, lanes, sidewalks FIXME tags, etc. see e.g. <a href="http://www.itoworld.com/map/12">http://www.itoworld.com/map/12</a> , <a href="http://www.itoworld.com/map/126">http://www.itoworld.com/map/126</a></li>
<li>The HOT task manager of the humanitarian OSM team</li>
<li>Maproulette.org</li>
</ul>
