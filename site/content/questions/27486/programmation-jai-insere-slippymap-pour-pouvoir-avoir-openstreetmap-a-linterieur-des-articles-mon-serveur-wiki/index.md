<p>Ne sachant pas quel logiciel/wiki/crm vous utilisez pour diffuser vos articles, il est difficile d'en dire plus. La syntaxe "&lt;slippymap ...=""/&gt;" n'est pas un standard OSM que je connait, peut-être un greffon Wordpress ?</p>
<p>Avec un peu de chance, le greffon que vous utilisez dispose des fonctions nécessaire, allez voir dans la doc correspondante. Au vu de la capture d'écran, je dirais que votre greffon se base sur la librairie <a href="http://www.openlayers.org/">OpenLayers</a>. Allez voir de ce coté si vous êtes prêt à améliorer le code du greffon.</p>
<p>Sinon, vous pouvez utiliser <a href="http://www.openlayers.org/">OpenLayers</a> ou <a href="http://leafletjs.com/">Leaflet</a> directement (la 2ème étant plus facile d'accès) pour faire ce que vous voulez. Ce sera plus facile à faire pour une page html normale, mais plus compliqué si ça doit être intégré à votre wiki/crm/etc.</p>
