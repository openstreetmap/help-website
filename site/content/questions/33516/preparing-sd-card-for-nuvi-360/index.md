<p>The wiki states that the Nüvi 3**-series can handle multiple gmapsupp-files with different names. So make a folder with the name 'Garmin' on your sd-card, copy the gmapsupp.img to that folder and voilà! If there is already a gmapsupp.img, first rename the new file to for example france_gmapsupp.img. After starting up the device, it will read the new map. Then it is possible to change the map-settings of the Nüvi so the new map is being displayed.</p>
