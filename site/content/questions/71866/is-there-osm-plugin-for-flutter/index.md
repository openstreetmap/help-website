<p>I guess that you're talking about <a href="https://en.wikipedia.org/wiki/Flutter_(software)">this</a>?</p>
<p>OSM is just a big pile of data that people can interact with in different ways. The can create maps, calculate routes, extract data (e.g. "how many social clubs in Stockholm") and do lots of other things. What is it that you want to do?</p>
<p>If you can load existing Javascript libraries in there via something like a webview, and you want to create maps, then look at using <a href="https://leafletjs.com/">Leaflet</a> in a webview.</p>
