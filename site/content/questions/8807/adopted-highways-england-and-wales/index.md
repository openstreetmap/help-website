<div class="answer-body">
<p>An adopted highway is a public right of way on private land, maintained at public expense. See for example <a href="http://www.eastsussex.gov.uk/roadsandtransport/roads/roadadoption/default.htm">how adoption works in East Sussex</a>.</p>
<p>As of today, there are :</p>
<ul>
<li>26 <a href="http://tagwatch.stoecker.eu/Great_britain/En/tagstats_designation_adopted_highway.html">highways with the "designation=adopted highway" tag</a>.</li>
<li>44 <a href="http://tagwatch.stoecker.eu/osmxapi/*%5Badopted=yes%5D">highways with the "adopted=yes" tag</a>.</li>
<li>33 <a href="http://tagwatch.stoecker.eu/osmxapi/*%5Badopted=no%5D">highways with the "adopted=no" tag</a>.</li>
</ul>
<p>From my personal experience, the "adopted=yes" tag seems more appropriate and it wins by numbers. But you might want to talk to users of those tags in order to work a consensus and let the world know about what solution is best... Folksonomic consensus through good communication is what Openstreetmap is about !</p>
</div>
