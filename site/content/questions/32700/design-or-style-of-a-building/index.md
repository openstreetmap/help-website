<div class="answer-body">
<p>There is <a href="http://taginfo.openstreetmap.org/keys/?key=building%3Aarchitecture#overview">building:architecture=*</a>, which was part of a now-withdrawn proposal for extending building tagging. Although that proposal was withdrawn, since this particular tag does make sense, and has nearly 5000 uses, that would be my recommendation. Wikipedia has a massive list of different <a href="https://en.wikipedia.org/wiki/Architectural_style">architectural style</a>s you might use for values.</p>
