<div class="answer-body">
<p>OpenStreetMap does not usually carry topographic data. If you see a map with elevation contours or hill shading, then whoever created that map from OpenStreetMap has added that information in from another source, possibly the free NASA SRTM dataset which might be used by other parties as well.</p>
<p>If you have a genuine bug report regarding OpenStreetMap, then <a href="http://www.openstreetbugs.org/">OpenStreetBugs</a> would be a good place to post it.</p>

<p>The Cycle Map layer on the openstreetmap map display is provided by <a href="http://opencyclemap.org">opencyclemap.org</a> and is based on openstreetmap vector data and SRTM elevation data. This digital elevation model has 90 meters resolution and is know to contain some errors. Apparently google uses the same data and therefore shows the same errors.</p>
