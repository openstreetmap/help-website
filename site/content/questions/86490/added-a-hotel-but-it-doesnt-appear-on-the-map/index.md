<p>the <a href="https://www.openstreetmap.org/node/10539012265">hotel</a> you recently added does appear on the default map.</p>
<p>The default map view on openstreetmap.org is updated is near real time. The "near" is important, it can be some minutes, hours for larger objects.</p>
