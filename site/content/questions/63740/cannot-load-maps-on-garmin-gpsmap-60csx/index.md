<p>Hello-</p>
<p>As mentioned above, I have a Garmin GPSMAP 60CSx. I have BaseCamp installed on my computer and have successfully used the OSM Switzerland map to get more detail and create a route. However, I cannot get this detail to transfer to my Garmin. When I plug it in, I can install the OSM Switzerland map onto the device (right-click on 'Internal Storage' in BaseCamp and use MapInstall). I can even see the OSM map under 'Settings' and 'Maps.' However, when I check the box to use the map I get no more detail.</p>
<p>I have also tried dropping the .zip file directly onto my 16gb sd card (moving the current image to another location since the 60CSx is funny about this). However, this yields no results either. In fact, it doesn't show up under 'Maps' in the settings at all.</p>
<p>Any help would be greatly appreciated.</p>
<p>-Nate</p>

<p>You may need the Garmin.img to be a of a different type for use a a Garmin GPS. You could download a Swiss version of this <a href="http://garmin.opentopomap.org/">http://garmin.opentopomap.org/</a> which are very user friendly. Once installed on the GPS plug-in the USB lead and Basecamp will probably display it. If you wanted it to be always available to Basecamp put a copy of that same file onto a memory stick and that works for me.</p>
