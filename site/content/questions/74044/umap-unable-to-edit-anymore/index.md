<div class="answer-body">
<p>To edit a map you need to either be logged in into your account and edit a map that you created while being logged in or you need a secret editing link. That link ends in 20 or so random characters. Once you used that link it redirects to the standard link of the map and stores a cookie in your browser enabling you to edit the map. If you deleted the cookie and lost that secret link you are out of luck to gain access.</p>
<p>As a workaround you might still be able to download the map data if you had enabled that option. If you did you can create a new map and import the data there.</p>
