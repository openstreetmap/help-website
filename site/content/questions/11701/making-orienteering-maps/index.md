<p>If by "orienteering map" you mean a map that is styled differently from the standard map you see at www.openstreetmap.org, then yes, of course this is possible. It does however require that you install software for making maps from data according to style rules that you define ("rendering software"). Type "own map style" in the search box above and read some of the answers to get an idea.</p>
