<div class="answer-body">
<p>It does exist, please take a look at the <a href="https://wiki.openstreetmap.org/wiki/Tag:amenity%3Dembassy">amenity=embassy</a> documentation. It mentions an <em>diplomatic</em> tag to specify <em>whether this is an embassy, an ambassador’s residence, a consulate etc. See current values in <a href="http://taginfo.openstreetmap.org/keys/diplomatic#values">taginfo</a>.</em></p>
<p>In your case please use <em>amenity=embassy</em> with the additional tag <em>diplomatic=consulate</em>.</p>
