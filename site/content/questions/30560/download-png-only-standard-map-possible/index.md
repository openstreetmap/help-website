<div class="answer-body">
<p>It is not possible to download map images in any other style than the standard style. When you download an image via the "share" button then it says: "Image will show standard layer..." - this is intended to make you aware of this situation.</p>
<p>This has been discussed on the German mailing list <a href="https://lists.openstreetmap.org/pipermail/talk-de/2014-February/107038.html">only a few days ago.</a></p>
<p>Workaround: Render your own map (plenty of infromation in this help system), or use the "bigmap" script to download and piece together a couple of tiles.</p>

<div class="comment-text">
<p>The export/share function runs on OpenStreetMap's own servers. This means it will only apply to the style generated on those same servers - the standard style. The cycle map, transport map, and MapQuest Open styles are all third-party styles run on independent servers, which the export/share function can't access.</p>
</div>
