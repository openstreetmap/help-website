<div class="answer-body">
<p>There are ready-made shape files for some regions e.g. at <a href="http://downloads.cloudmade.com">downloads.cloudmade.com</a> and <a href="http://download.geofabrik.de">download.geofabrik.de</a>.</p>
<p>Note that shape files contain raw data and not maps; do not expect to open a shape file in OpenJUMP and see a pretty map. You will have to apply a lot of styling work to get something that looks remotely pretty.</p>
<p>Also note that shape files can never contain <em>all</em> of OSM data because they are limited to a few columns and don't have a free-form tagging schema like OSM has.</p>
