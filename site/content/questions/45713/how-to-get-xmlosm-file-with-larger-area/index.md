<p>If the planet file is still too large for you (it will take much time to process it), you might have a look at the regional extracts provided by Geofabrik. <a href="http://download.geofabrik.de/">http://download.geofabrik.de/</a></p>
<p>If the programs which you use also accept PBF files, you might be better to use the PBF format because much programs are faster reading PBF than XML.</p>
