<p>Hi.</p>
<p>Unfortunately the iframe code from opentreetmap.org is only meant to display a simple map, optionally with a single marker. Here is the <a href="https://wiki.openstreetmap.org/wiki/Export">doc</a>.</p>
<p>I'm afraid you'll need to write your own code, using the <a href="https://leafletjs.com/">Leaflet</a> js framework for example.</p>
<p>Or I think you can use <a href="https://docs.facilmap.org/users/">facilmap</a>, based on OSM data. Here some <a href="https://npm.io/package/facilmap-frontend">code</a> that looks like what you want.</p>
<p>Regards.</p>
