<p>Perla_rt, is this a typical OSM rendering question, for a map based on OSM data ? Did you try to use the mouse wheel instead of the + - buttons ? And if you’re using a mobile try the usual enlargement or limiting tool. All methods should be available on any renderers engine. The map scrolls from 1000 m up to 500 - 200 - 50 m just by a move of the wheel, IMHO quick enough. That’s not a tool of the map but should come with the renderer’s product.</p>
