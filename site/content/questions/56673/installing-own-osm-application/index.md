<div class="answer-body">
<p>"Map" can mean many things of course, but let's imagine that it means "a slippy map a bit like the one that you see at <a href="https://www.openstreetmap.org/">OpenStreetMap.org</a>" There's a lot of information at <a href="https://switch2osm.org/">https://switch2osm.org/</a>. Specifically for a rendering server, see <a href="https://switch2osm.org/manually-building-a-tile-server-16-04-2-lts/">here</a>. For using tiles, as well as the section of <a href="https://switch2osm.org/">switch2osm.org</a>, see also the examples and documentation at <a href="http://leafletjs.com/">http://leafletjs.com/</a>.</p>
<p>You say "My own map with specific data". If that's geographic data for a specific area, then you can probably find what you want <a href="http://download.geofabrik.de/index.html">here</a>. The British Isles download is about 1Gb of OSM data and will "fit" on a modern desktop PC without problems. See <a href="https://wiki.openstreetmap.org/wiki/User:SomeoneElse/Ubuntu_1604_tileserver_load">here</a> for a bit of background. If you want "a selection of data (e.g. railways) but worldwide" then that's a slightly different question, and needs a bit more "data wrangling".</p>
<p>If you want to design or modify your own stylesheet, read Ircama's tutorials <a href="https://ircama.github.io/osm-carto-tutorials/">here</a>.</p>
