<div class="answer-body">
<p>If <a href="http://yourserveripaddress/hot/0/0/0.png">http://yourserveripaddress/hot/0/0/0.png</a> is working then the link between apache, mod_tile and renderd is (or at least at some stage was) working. One thing that might be happening is that requests are being coming in but are being processed very slowly.</p>
<p>Try:</p>
<pre><code>/etc/init.d/renderd restart
/etc/init.d/apache2 restart</code></pre>
<p>while all the while watching what's getting written to /var/log/syslog , and then try requesting <a href="http://yourserveripaddress/hot/1/0/0.png">http://yourserveripaddress/hot/1/0/0.png</a> which should show you a North America tile.</p>
