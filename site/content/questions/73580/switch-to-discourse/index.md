<p>Hi there,</p>
<p>I appreciate this OSQA instance since a long time. Both, as author for questions and answers, this works just fine. But the <a href="https://github.com/evgenyfadeev/askbot-devel">OSQA / askbot development</a> stopped for more then 10 years, while other (selfhosting) social platforms rise and seem to offer more interesting features:</p>
<ul>
<li>tag based topic categories</li>
<li>eMail / ML like interface</li>
<li>mobile friendly UI</li>
<li>content translation</li>
<li>integration of 3rd party services</li>
<li>migrations for OSQA / forums / mailinglists / ...</li>
</ul>
<p>All in all this looks good and IMHO this might lower the barrier to get help for new OSM users and contributors, while it might allow us to unite the discussions between OSM help / OSM forums / OSM mailinglists.</p>
<p>What does you think? Would it be worth to switch?</p>
</div>
