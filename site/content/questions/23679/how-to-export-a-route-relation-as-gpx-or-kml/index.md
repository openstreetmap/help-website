<p>One possible way to export GPX is:</p>
<ol>
<li>identify the relation ID of the route you want (if need be, use a search engine and search for "openstreetmap relation somthing other")</li>
<li>download JOSM editor</li>
<li>in JOSM, select "File -&gt; Download Object"</li>
<li>choose "Object type" = "relation", check "Download relation members"</li>
<li>click "Download object"</li>
<li>select "File -&gt; export to GPX"</li>
</ol>
</div>
