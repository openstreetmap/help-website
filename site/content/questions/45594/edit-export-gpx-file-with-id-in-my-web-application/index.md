<p>So it sounds "doable" - but as always...it is just a question of what you are trying to achieve.</p>
<p>If one just wanted to convert (many) GPX track nodes into a simple route then something like <a href="https://graphhopper.com/">GraphHopper</a> might be sufficient to get nodes onto the OSM graph.</p>
<p>As for converting the GPX into something iD can work with you are probably best heading over to the <a href="https://github.com/openstreetmap/iD/">iD github repo</a> to request an enhancement. The folks over there will no doubt be able to provide better feedback than on this forum.</p>
