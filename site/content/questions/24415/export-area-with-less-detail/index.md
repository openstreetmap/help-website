<p><strong>Is it possible to export the data contained in some region, but without too much detail?</strong></p>
<p>For instance, how can I get the data of the country boundaries in Europe? (in XML, or any computer-friendly format) The export function in OSM does not work, presumably because it would give me every possible piece of data everywhere in Europe, which is not what I want. I could also probably download the complete database and then extract only what I need, but that’s probably overkill.</p>
<p>There is probably a way to do it, the map you can see on OSM is like that, when you’re far away you can only see the countries and when you get closer you can see more.</p>
<p>Note that my question is not just about country boundaries in Europe, but more generally about exporting some part of the world, without the "small" details.</p>
