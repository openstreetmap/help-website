<div class="answer-body">
<p>I don't know where or if the feature is documented but <a href="https://tools.geofabrik.de/osmi/">OSM Inspector</a> appears to successfully link to specific elements within iD. Once an element is selected in the map pane the link appears as an icon (<img src="https://tools.geofabrik.de/img/to_id.png" alt="iD edit icon" />) on the right sidebar.</p>
<p>If your goal is to create a standalone, feature specific editor <a href="https://wiki.openstreetmap.org/wiki/MapComplete">MapComplete</a> might provide a suitable starting point.</p>

<p>To edit <strong>that</strong> object, url is<br />
<a href="https://www.openstreetmap.org/edit?editor=id&amp;node=">https://www.openstreetmap.org/edit?editor=id&amp;node=</a><strong>node id</strong><br />
<a href="https://www.openstreetmap.org/edit?editor=id&amp;way=">https://www.openstreetmap.org/edit?editor=id&amp;way=</a><strong>way id</strong><br />
<a href="https://www.openstreetmap.org/edit?editor=id&amp;relation=">https://www.openstreetmap.org/edit?editor=id&amp;relation=</a><strong>relation id</strong></p>
<p>Of course, this applies to whatever object. "amenity", "office", "shop" filter should be done upstream.</p>
<p>For edits that don't require visual feedback, I'd suggest Level0 editor.</p>
