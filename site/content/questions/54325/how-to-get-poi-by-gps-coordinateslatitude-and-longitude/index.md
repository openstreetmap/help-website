<div class="answer-body">
<p>It sounds as if you might be looking for Reverse Geocoding, something you can do with <a href="https://wiki.openstreetmap.org/wiki/Nominatim#Reverse_Geocoding">Nominatim</a>. Nominatim has a JSON API, so you can use standard web access methods to call it from Java or C#.</p>
<p>If you plan to publish software that makes frequent use of Nominatim, you should probably set up your own Nominatim server instead of relying on the donation-funded server operated by OSM itself - see the <a href="https://wiki.openstreetmap.org/wiki/Nominatim_usage_policy">usage policy</a>.</p>

<div class="answer-body">
<p>Do you want to get the POI(s) or "know which place is" the coordinates describe?</p>
<p>For retrieving all POIs of this area: Use <a href="https://wiki.openstreetmap.org/wiki/Overpass_API">Overpass API</a>. Let's say your coordinates are lat 51.56100 and lon 10.83576. The following Overpass API query will return <em>all</em> OSM <a href="https://wiki.openstreetmap.org/wiki/Elements">elements</a> within 25 of these coordinates:</p>
<pre><code>[out:json][timeout:25];
(
  node(around:25.0,51.56100,10.83576);
  way(around:25.0,51.56100,10.83576);
  relation(around:25.0,51.56100,10.83576);
);
out body;
&gt;;
out skel qt;</code></pre>
<p>You can view the result at <a href="http://overpass-turbo.eu/s/ly3">overpass turbo</a>. Note that you can retrieve these results programmatically (<a href="https://overpass-api.de/api/interpreter?data=%5Bout%3Ajson%5D%5Btimeout%3A25%5D%3B%0A%28%0A%20%20node%28around%3A25.0%2C51.56100%2C10.83576%29%3B%0A%20%20way%28around%3A25.0%2C51.56100%2C10.83576%29%3B%0A%20%20relation%28around%3A25.0%2C51.56100%2C10.83576%29%3B%0A%29%3B%0Aout%20body%3B%0A%3E%3B%0Aout%20skel%20qt%3B">example</a>) because overpass turbo is just a nice web frontend to Overpass API.</p>
<p>For knowing which place/address the coordinates describe: Use the reverse-geocoding of <a href="https://wiki.openstreetmap.org/wiki/Nominatim">Nominatim</a> or one of the other <a href="https://wiki.openstreetmap.org/wiki/Search_engines">OSM-based search engines</a>. Example call for Nominatim for the same coordinates as above: <a href="https://nominatim.openstreetmap.org/reverse?format=xml&amp;lat=51.56100&amp;lon=10.83576&amp;addressdetails=1">https://nominatim.openstreetmap.org/reverse?format=xml&amp;lat=51.56100&amp;lon=10.83576&amp;addressdetails=1</a></p>
