<p>I agree that you need to add some appropriate tags. The node is <a href="https://www.openstreetmap.org/node/4937586455">here</a> and the only "top level tag" it has is "landuse". At the very least I'd add a tag like "office", "shop" or similar (whatever is best for this establishment).</p>
<p>The <a href="https://lists.openstreetmap.org/listinfo/tagging">tagging mailing list</a> is where most of the people who revel in this sort of "how to tag" discussion are, so I'd perhaps ask there.</p>
