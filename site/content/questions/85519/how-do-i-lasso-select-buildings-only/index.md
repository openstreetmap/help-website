<p>Today I learnt that there's a lasso-selection capability within iD :)</p>
<p>The problem outlined here seems to be mentioned on <a href="https://github.com/openstreetmap/iD/issues/7375">this</a> iD issue (and also <a href="https://github.com/openstreetmap/iD/issues/8552">this one</a>). <p>Edit: Actually - it has been released! See answer above by Minh.</p>
