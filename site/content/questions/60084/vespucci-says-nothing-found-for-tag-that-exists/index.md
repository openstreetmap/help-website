<div class="answer-body">
<p>This is the <em>preset</em> search. You can only search for things that Vespucci has a preset for. However there seems to be no preset for historic=city_gate so the search fails.</p>
<p>If you already know the correct tag then skip the preset search. Just swipe to the right until you get to an activity where you can enter key and values directly.</p>
