<p>I know nothing more about We-Travel than I have read in the <a href="https://wiki.openstreetmap.org/wiki/We-Travel">We-Travel OpenStreetMap Wiki</a>.</p>
<p>But I see that Ecuador is covered by the Colombia and Peru maps on the <a href="http://we-travel.net/DownloadServer_FREE/">We-Travel Download Server</a>.</p>
<p>I tried to download the Colombia map, but "by coincidence" the free service was busy and they wanted my money to proceed.</p>
<p>Instead I successfully downloaded Colombia from the second wiki link: <a href="http://vserver2125.vserver-on.de/mapdownload/index.php">Free We-Travel Downloadserver</a></p>
<p>Zoom has to be enabled when building the map, so that is nothing you can do afterwards. The status of the maps mentioned in your question is:</p>
<ul>
<li>Peru No Zoom</li>
<li>Mexico No Zoom</li>
<li>Colombia Zoom 1-11</li>
</ul>
<p>In my taste those maps are outdated, but hopefully someone else can tell us if it's possible to build your own updated map specifically for Ecuador.</p>

<div class="answer-body">
<p>You cannot install OpenStreetMap osm-files directly on your We-Travel device, but have to use the <a href="http://we-travel.biz/wetravel/index.php?option=com_phocadownload&amp;view=category&amp;id=8:mapgenerator">We-Travel Mapgenerator</a> which is only available for Windows.</p>
<p>In order to use the map generator I have found that you have to install both <a href="http://www.java.com/">Java</a> and a <a href="http://www.mysql.com/">MySQL server</a>. Both are free.</p>
<p>Eventually I was able to run the map generator, but when trying to use an osm-file as input the conversion always exited with "ERROR OSM-CONVERT: java.lang.NullPointerException" after a while.</p>
<p>But, I have found that it is possible to download garmin maps in the menu and start the conversion. I guess that the garmin maps acually are OpenStreetMap maps stored in the garmin file format. And there is an Ecuador file available in the menu!</p>
<p>The file conversion is a very time consuming thing. According to entries on the <a href="http://we-travel.biz/wetravel/index.php?option=com_kunena&amp;view=listcat&amp;Itemid=46">We-Travel forum</a> it may take more than a week(!) to generate a file for a country like Germany. Since Ecuador is a smaller country than Germany with a much smaller population it will probably take less time. But expect it to take many hours.</p>
<p>The We-Travel map generator will ask you if you want to generate tiles for zoom or not, and generating many zoom levels will probably slow down the conversion significantly.</p>
