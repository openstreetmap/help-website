<p>Hi</p>
<p>I want to create a map of a local shopping High St. The requirements are:</p>
<ul>
<li>Slippy (but constrained to about 2km<sup>2</sup>)</li>
<li>Zoomable from level 16 to level 20-21 to allow all shop labels &amp; name to display clearly</li>
<li>Use OSMCarto 'standard' render</li>
<li>Each shop to be clickable to display a popup of its associated tags</li>
</ul>
<p>Would I need to follow <a href="https://switch2osm.org/manually-building-a-tile-server-16-04-2-lts/">Switch2OSM's</a> tile server instructions or is the an easier/simpler method? I'm going to assume the extra zoom levels will cause a bit of difficulty &amp; discount the likes of <a href="https://leafletjs.com/">leaflet</a>.</p>
<p>Ta</p>
