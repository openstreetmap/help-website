<p>Hi, I don't know about the verification code but you do have to confirm your email address, (maybe the code is in that email.</p>
<p>Please see webpage: <a href="https://wiki.openstreetmap.org/wiki/OpenStreetMap_account">https://wiki.openstreetmap.org/wiki/OpenStreetMap_account</a> ---- Extract from that page is below.</p>
<p>"You should now receive a confirmation email asking you to click a link to make sure the email address is really yours. If the confirmation email doesn't arrive, you should check your 'spam' folder, and perhaps refresh your inbox after a few minutes. --- If you still don't find the email or if you've forgotten the password for your OSM account, you can reset the password at <a href="https://www.openstreetmap.org/forgot-password.html">https://www.openstreetmap.org/forgot-password.html</a> "</p>
<p>Regards.</p>
