<div class="answer-body">
<p>The OSM database has always recorded the user who made an edit; it's just that it didn't publish who it was. So when a hitherto anonymous user flipped the switch, the database should immediately start to reveal his identity even on old edits.</p>
<p>Of course this does not affect old planet files on disk, that were generated before the user flipped the switch.</p>
