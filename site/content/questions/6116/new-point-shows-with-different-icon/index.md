<div class="answer-body">
<p>There are many ways of showing the same data!</p>
<p>Potlatch 2 uses one set of icons. The standard ('Mapnik') map style that you see on the main site uses another. If you choose the 'Osmarender' style (use the floating + icon on the right), that uses another. If you then choose the 'Cycle map' style, it uses another. Third-party sites such as <a href="http://open.mapquest.com/">MapQuest Open</a> - and many more - use further ones still. And so on.</p>
<p>That's the great thing about OpenStreetMap: the data is there for people to do what they like with. You shouldn't concentrate on "what it's going to look like", but rather, on faithfully recording "what's on the ground". Different maps will then choose to show it how they see fit.</p>
