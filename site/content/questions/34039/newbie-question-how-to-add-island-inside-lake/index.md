<p>I guess the question is more about multipolygons (i.e. how to create an object as a "hole" inside another object). The existing lake should became the outer ring, and the new island the inner ring, of a new multipolygon relation. How to create a multipolygon relation depends on the editor you use, the wiki contains an illustrated example with Potlatch: <a href="https://wiki.openstreetmap.org/wiki/Multipolygon">https://wiki.openstreetmap.org/wiki/Multipolygon</a></p>

<div class="answer-body">
<p>I didn't see the steps explicitly set out in the other answers, so in case it's helpful here goes:</p>
<p>In iD:</p>
<ol>
<li>Select the outer boundary way</li>
<li>Expand All relations, then click the plus to add a relation<br />
</li>
<li>Click New relation, then select Multipolygon</li>
<li>Add any tags to the multipolygon relation (name, natural=water, etc)</li>
<li>Select the outer boundary way again (which will shift you from editing the relation to editing the way), and under All relations, click where it says Role and enter "outer"</li>
<li>Create way for the island</li>
<li>Pick any tag (ie water, area, etc) in the Select feature type screen so you can get to the Edit feature screen--you can delete whatever tags it automatically added if you want</li>
<li>With the island way still selected, open All relations, click the plus, select the Multipolygon relation you just created, and where it says Role enter "inner"</li>
</ol>
<p>In Potlatch 2:</p>
<ol>
<li>Select the outer boundary way, at the bottom of the sidebar, click Advanced</li>
<li>In the Relation section at the bottom click Add to</li>
<li>In the pop up screen, click New relation</li>
<li>In the dropdown select Advanced &gt; Multipolygon (or just shift to the Advanced tab and enter type=multipolygon)</li>
<li>Add any other tags to the multipolygon</li>
<li>Close the pop up, and add "outer" as role</li>
<li>Create the island way</li>
<li>Click Add to again and select the Multipolygon you just created, and add "inner" as role</li>
</ol>
<p>One Potlatch 2 tip if you do a lot of relation editing there: you can repeat relations from the previously selected way with Shift-R</p>
