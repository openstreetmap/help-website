<p>Indeed, in OSM a way is a sequence of nodes. Linear, or circular if the end is the same as the beginning. But multiple ways will be used to represent a single road, with a shared "name" tag usually.</p>
<p>You can find more details about the OSM data model in the <a href="https://wiki.openstreetmap.org/wiki/Beginners_Guide_1.3">wiki</a>.</p>
