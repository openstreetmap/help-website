<p>I don't have JOSM installed on the computer I am writing from now. But I had the same problem already some time ago and opened tickets with OSC and JOSM: There are two ways removing the layer. You can right-click and delete or you can click the garbage can icon. One removes the layer permanently, the other only for the session (not sure anymore which did what). Unless there is something new in JOSM using the other method should solve your problem.</p>
