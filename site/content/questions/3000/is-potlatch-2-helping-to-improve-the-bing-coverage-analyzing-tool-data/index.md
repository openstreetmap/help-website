<td><div class="item-right">
<div class="answer-body">
<p>No.</p>
<p>This tool does not have access to information about what other users view with Potlatch2. It would require a change in the tool as well as a change in Potlatch2 to enable this. You can open a ticket on <a href="http://trac.openstreetmap.org">trac.openstreetmap.org</a> if you want to ask Potlatch developers to add this feature, and raise the issue with the developer of the tool - but there are some issues; one is a privacy issue (does the user want a third-party tool to know which areas he's looking at on Bing?) and the other a technical issue (Flash cannot easily access image headers and thus might have a problem in actually finding out whether there's coverage at all).</p>
