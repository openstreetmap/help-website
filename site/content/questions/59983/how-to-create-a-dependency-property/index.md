<p>Hi there,</p>
<p>I have created a number of user controls that I want to bind in <a href="https://goo.gl/kbk7aC">WPF</a>. Therefore I have created a Value property but when I am trying to bind it I get the following error message at runtime:</p>
<p>"Cannot bind the property 'Value' on the target control"</p>
<p>I have done some reading and <a href="http://msdn.microsoft.com/en-us/library/system.windows.data.binding.aspx">http://msdn.microsoft.com/en-us/library/system.windows.data.binding.aspx</a> states that the property used for binding must be a 'dependency property' (<a href="http://msdn.microsoft.com/en-us/library/ms752914.aspx).">http://msdn.microsoft.com/en-us/library/ms752914.aspx).</a></p>
<p>This is where I got lost. There is no mention of 'dependency properties' in the ABL docs. How do I create this? Do I really need this or is there another way to support data binding on user controls in WPF?</p>
<p>Thanks for any insight!</p>
