<p>The address is in Ohio, USA – 739 Portage Trail, Cuyahoga Falls, OH 44221.</p>

<p>I don’t know how your maps work but it often confuses our customers. We get many calls from them looking for us but are about 2 miles away. We have to explain that their map or GPS is incorrect. We have all the listings online with our correct address. We are NOT on the extension.</p>
<p>If you can forward this, we thank you in advance. If not, not a big deal.</p>

<div class="answer-body">
<p>Oldie but goldie; while Craig's List uses an OpenStreetMap based background map, it doesn't use OpenStreetMap data for its address search. So there is not really anything we can do short term except point back to Craig's List.</p>
<p>Note: the address in question "739 Portage Trail Cuyahoga Falls" can actually be found via the search box on openstreetmap.org but on inspection you will see that the result is actually from additional TIGER data and not from OSM data proper.</p>
<p>Naturally we would be happy if the company in question would make any necessary corrections to OSM and add their address (see <a href="https://www.openstreetmap.org/fixthemap">https://www.openstreetmap.org/fixthemap</a> ) but that is not going to help with Craig's List.</p>
