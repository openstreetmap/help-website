<div class="answer-body">
<p>There are lots of different ways of rendering OSM data, so "how things will be seen" will depend on which renderer is displaying the data. OSM data will look different depending on whether you're looking at a website (and depend on which one), a phone app (whichever one).</p>
<p>That said, although you can't add test data to OpenStreetMap, there are a few options. It's possible that what you want to do will be considered OK by <a href="https://opengeofiction.net/">https://opengeofiction.net/</a> (though please check with them first). Another option is to set up your own renderer and load some test data into that. The map legend on <a href="https://map.atownsend.org.uk/maps/map/map.html#zoom=17&amp;lat=-24.986394&amp;lon=135.17592">this map</a> is done that way and the code that looks after that is <a href="https://github.com/SomeoneElseOSM/SomeoneElse-style-legend">here</a>. Unfortunately there isn't a test rendering one of OSM's own dev sites that I am aware of.</p>
