<p>Simplest is to do a screen grab. The screen will have the copyright notice on it. It is free to use. <p>Hi</p>
<p>What is the purpose of the map? If it is for mapping, then you can use <a href="http://fieldpapers.org/">Fieldpapers</a>. Here you can select and area and tile size and print out maps for mapping purposes. If you just need a map, you can use <a href="http://openstreetmap.gryph.de/bigmap.html">Bigmap</a> for printing out a map. You draw a bounding box and it gives you output as a PNG file that you can then print out.</p>
