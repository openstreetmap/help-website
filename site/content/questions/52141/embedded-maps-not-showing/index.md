<p>This is the same problem as <a href="https://help.openstreetmap.org/questions/52023/embedded-maps-not-working">https://help.openstreetmap.org/questions/52023/embedded-maps-not-working</a> isn't it? The "people who can help" are likely to be site admins, and the best way to log website bugs is over at <a href="https://github.com/openstreetmap/openstreetmap-website/issues">https://github.com/openstreetmap/openstreetmap-website/issues</a>. A <a href="https://github.com/openstreetmap/openstreetmap-website/search?q=embedded&amp;type=Issues&amp;utf8=%E2%9C%93">quick search</a> doesn't show anything that obviously matches. Alternatively, you could try asking in #osm-dev on <a href="http://wiki.openstreetmap.org/wiki/IRC">IRC</a>.</p>
<p>When you say that "this does not appear to be an issue with my site and has nothing to do with the need for API key" this presumably means that you have obtained <em>your own</em> API key from Andy Allan, who hosts the cyclemap and travel tiles?</p>
