<div class="comment-text">
<p>Strictly speaking, you haven't download a "map", just the raw data that something can create a map from. A .osm file is a text file, so if you open it in a text editor you'll see the raw data. Most programs that "do things with OSM data" can read .osm files, but which of those you'd use depend on whether you want to view the data, edit it, or create some sort of map from it and then view that.</p>

<p>In addition to JOSM (see other answer) also many other programs can be used to view a <span>.osm file</span>. E.g. also</p>
<ul>
<li>our online editor <span>Potlatch2</span> (menu background → vector file) which needs no installation.</li>
<li><span>QGIS</span></li>
</ul>
<p>If you just want to "view" the file, just open it in any text editor - then you see the raw data (it is XML). But that is anything but a "map".</p>
