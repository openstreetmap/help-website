<div class="answer-body">
<p>The Mapnik tiles as displayed on <a href="http://www.openstreetmap.org">www.openstreetmap.org</a> are generated according to a style sheet that you can find in our <a href="http://trac.openstreetmap.org/browser/applications/rendering/mapnik">SVN repository</a> and you are welcome to submit change requests via <a href="http://trac.openstreetmap.org/">trac</a>. However, the omission of many POIs on that map happens for a reason - we don't want it to look too cluttered.</p>
<p>There is no individual customisation but of course you can <a href="http://wiki.openstreetmap.org">set up your own tileserver</a>.</p>
<p>And of course, if you want custom styles, there's nothing against just using the CloudMade system that you talk about - after all, it's the same data.</p>

<ul>
