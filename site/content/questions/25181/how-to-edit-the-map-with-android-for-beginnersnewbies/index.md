<div class="answer-body">
<p>There is currently no beginners editor for Android and it would seem unlikely that you would want to start off doing that. The best approach is to collect data/information with your device and then edit (on a desktop or laptop PC) with iD, P2 or JOSM.</p>
<p>For dedicated data collection I would recommend OSMTracker for Android (is in the market).</p>
<p>If you do want to edit you can use Vespucci, which does support Presets (the same ones as JOSM does), however you should be aware that the version in the market is rather dated. There is likely to be a new release soon, in the mean time you can use the beta version (mainly missing some translations compared to a relase version) from <a href="https://code.google.com/p/osmeditor4android/downloads/list">https://code.google.com/p/osmeditor4android/downloads/list</a></p>

<p>First of all: welcome! :-)</p>
<p>Disclaimer: I don't use an Android device myself - others may give a better answer. However: See <span>Android OSM editing software</span> in our docu wiki. I think Vespucci is the most popular one. However, I do not know how beginner-friendly it is. It <em>seems</em> (judging from screenshots) that it has no tag presets. Which means that you need to add the <span>tags</span> manually - you can look them up in <span>our wiki</span>.</p>
<p>You may also like <span>OsmAnd</span> for just displaying maps and recording tracks (and doing basic editing).</p>
<p>Also see our <span>beginner's guide</span> in general. Have fun and don't copy from other maps! :-)</p>
<p>Please add an own answer if you finally found a app which suits your needs. You may also add comments to my answer.</p>
