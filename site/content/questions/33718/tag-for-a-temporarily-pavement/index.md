<p>I think he is talking about what in The Netherlands is called a "bouwstraat". When a new neighbourhood is built, they pave the streets with the paving stones upside down. Then when all building work is done the paving stones get relaid right side up. (Concrete paving stones are usually beveled only on one side.)</p>

<div class="answer-body">
<p>I don't see any need to tag it as temporary. Just choose the best surface value for the current situation, and change it in a few years when they flip the stones over.</p>
<p>You could add a note tag mentioning that the surface is temporary so future contributors will know to check the surface once construction is complete, but I don't see any need to create a dedicated "temporary" tag.</p>
