<p>You are really asking two questions in one which is not really the done thing on this site.</p>
<ol>
<li>you need to give more details website, browser etc. before anybody can help you with the first problem</li>
<li>OSM is primarily a data collection project, as a consequence there are many services, projects and maps built with OSM data that are not centrally controlled. This makes answering what might seem like a simple question slightly more difficult than expected. I know of two global sites that display MTB routes: the cycle map displayed on openstreetmap.org and <a href="http://mtb.waymarkedtrails.org">http://mtb.waymarkedtrails.org</a> But as said, there are likely to be more.</li>
</ol>
