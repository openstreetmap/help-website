<ol>
<li>Download Osmand's latest Mapcreator from <a href="http://osmand.net">http://osmand.net</a></li>
<li>Download raw OSM data for the region you need via <a href="http://download.geoofabrik.de">http://download.geoofabrik.de</a></li>
<li>Create your own *.obf vector data map file and copy it to your device</li>
<li>Be aware that in certain countries thete have been massive data loss due to the Redaction Bot. Test this for your region first. geofabrik.de is also offering "old" map data before the bot run.</li>
</ol>
