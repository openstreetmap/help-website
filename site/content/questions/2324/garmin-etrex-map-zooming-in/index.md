<div class="answer-body">
<p>It sounds like you haven't selected all of the maps to transfer from MapSource to the eTrex. Depending on which maps you are using, it might have several different layers. ie the contours on one layer, and the OSM data (roads, paths etc) on another layer.</p>
<p>If you use the "Map Tool" in MapSource, then just click on the tiles you want, then it will probably only select the top layer of tiles, it sounds like this is what you have done. To make sure you select all of the tiles, the best way is to use the Map Tool, and drag to draw a box. Then this should select all of the tiles and different layers in that box.</p>
<p>Then send the maps to the eTrex, and all of the maps should be loaded onto it, with all of the details.</p>
