<div class="answer-body">
<p>It depends on exactly what happened to the previous locations.</p>
<ol>
<li><p>If they've ceased operations, I'd add a <strong>disused:</strong> <a href="https://wiki.openstreetmap.org/wiki/Lifecycle_prefix">lifecycle prefix</a> to the shop and/or amenity tag. If a different business reopens in the same location, the previous element can be re-used to help <a href="https://wiki.openstreetmap.org/wiki/Keep_the_history">keep the history</a>.</p></li>
<li><p>If they're still operating under another name, simply change the name and/or brand tags. (And if you don't know the new name, just omit the name tag -- someone will add it eventually!)</p></li>
</ol>
<p>If you're not sure, I'd go with #2, omit the name, and add a <strong>fixme</strong> tag or a note that will cue other mappers to survey the place.</p>
<p>If you question is "how to remove them all at once" there's no real shortcut here. You could propose an <a href="https://wiki.openstreetmap.org/wiki/Automated_edits">automated edit</a>, or set up a group mapping project using a <a href="https://wiki.openstreetmap.org/wiki/Tasking_Manager">tasking manager</a>, but at this scale it's probably easier to simply make the changes yourself, or with a few other mappers.</p>
