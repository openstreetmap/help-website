<p>It is hard to help you without knowing what problems you have. It would be helpful if you mentioned where in the installation you get errors and what errors you get. At current there are no known problems with installing gosmore on Ubuntu. Either compiling like on the wiki or installing from apt.</p>
<p>If you compile you can get a number of warnings, this is normal when compiling any program. What might be confusing to first time users is that you have to update the map. It will download a .pak file witch is the OSM data itself. Without this data the program is almost useless. All of this is covered in the wiki.</p>
