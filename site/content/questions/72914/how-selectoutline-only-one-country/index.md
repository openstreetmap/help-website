<div class="answer-body">
<p>The simplest way to highlight a country (or any other mapped entity), is to search for it on openstreetmap.org.</p>
<p>For example, I searched for Myanmar, clicked on the first result (translated to my language) and got this map : <a href="https://www.openstreetmap.org/relation/50371">https://www.openstreetmap.org/relation/50371</a>.</p>
<p>You'll see the border highlighted, and the capital as a point.</p>
<p>PS. I hope you're not using the "edit" part of openstreetmap to create your own map. Any change there is for everyone and will probably make a mess ! If you want to create a customized map, have a look at <a href="https://wiki.openstreetmap.org/wiki/UMap">uMap</a> or others.</p>
