<p>Looks like you were using Potlatch 1 (i.e., you clicked on the edit tab). In which case, you were probably using the Yahoo imagery.</p>
<p>Open the options box you were using to change the background. Yahoo is not in the list with the others, but has its own radio button below the list. Select the "Yahoo" radio button and then "ok".</p>
