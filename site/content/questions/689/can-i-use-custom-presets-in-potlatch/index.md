<div class="answer-body">
<p>One can set custom presets in Potlatch 1. Use CTRL+SHIFT+[number 1-9] to <strong>set</strong> a tag combination and SHIFT+[number 1-9] to <strong>use</strong> the combination.</p>
<p><strong>Example:</strong></p>
<ul>
<li>Create an untagged way and set <code>bridge=yes</code> and <code>layer=1</code>.</li>
<li>Press CTRL+SHIFT+1 to save the tag combination.<br />
<span class="small">(Under windows this will get saved in location C:\Documents and Settings\<em><strong>yourname</strong></em>\Application Data\Macromedia\Flash Player\#SharedObjects\SOMERANDOMDATA\openstreetmap.org)</span></li>
<li>Delete the previously created way if not needed</li>
<li>Pressing SHIFT+1 will add the bridge/layer tag combination to any selected way.</li>
</ul>
