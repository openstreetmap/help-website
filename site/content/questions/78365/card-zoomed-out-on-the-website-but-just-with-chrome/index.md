<p>This is a known problem with an unknown source. I still suspect that Chrome's iframe lazy-loading changes (at around end of 2019) introduced this problem.</p>
<p>See: <a href="https://github.com/openstreetmap/openstreetmap-website/issues/2531">https://github.com/openstreetmap/openstreetmap-website/issues/2531</a></p>
<p>and <a href="https://forum.openstreetmap.org/viewtopic.php?id=68432">https://forum.openstreetmap.org/viewtopic.php?id=68432</a></p>
<p>Unfortunately, there is no solution up until now, so you could help with e.g. posting stack traces from chromes developer tools (I'm not using anything by google, so I don't have Chrome here).</p>
