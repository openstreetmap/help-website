<p>I don't believe that is possible (closing the side bar) and I don't believe it would make a lot of sense as it is used essentially continuously. Naturally the downside is that it uses a lot of screen real estate. If this is a big issue for you, you might want to try JOSM (which however needs a bit more work to get up to speed).</p>
<p>In any case I can't really imagine the bar covering so much of teh screen that the circular popup is obscured if you pan the data and background to the right.</p>
