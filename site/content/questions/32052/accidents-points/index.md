<div class="answer-body">
<p>OpenStreetMap is not interested in data about road accidents; please do not add them to the database.</p>
<p>Of course, you can create a KML (or GeoJSON) file and then display the accident locations <em>on top of</em> an OpenStreetMap map, without uploading the locations to the OSM database. You will first have to project them to WGS84, and then you can use the OpenLayers or Leaflet libraries to make a web page with an OpenStreetMap map and your accident locations drawn on top.</p>
