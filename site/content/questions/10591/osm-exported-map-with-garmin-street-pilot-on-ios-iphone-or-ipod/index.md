<p>Hey Guys,</p>
<p>I have tried installing a map exported from <a href="http://garmin.openstreetmap.nl">garmin.openstreetmap.nl</a> for use with Garmin Street Pilot on my iPOd 4G (using bluetooth GPS), I installed the map as detailed in this <a href="http://www.noeman.org/gsm/garmin-tutorials/245054-how-install-any-garmin-map-iphone-garmin-streetpilot.html">thread</a>. The app is shortlising the correct addresses from the map, but when I click on one to start the routing the app crashes.</p>
<p>Does anyone have any ideas where to start troubleshooting this? Are there any Garmin experts out there who know what the difference in the img file structures are, between what Nuvi is using and what Street Pilot requires?</p>
<p>AFAIA the <a href="http://garmin.openstreetmap.nl">garmin.openstreetmap.nl</a> site exports the file in the NT format, and the Street Pilot app runs the NA NT format map that it came with just fine.</p>
