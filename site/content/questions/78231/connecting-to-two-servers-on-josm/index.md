<p>You can run JOSM with a second profile directory:</p>
<p><a href="https://josm.openstreetmap.de/ticket/2710#comment:8">https://josm.openstreetmap.de/ticket/2710#comment:8</a></p>
<p>I have 2 shortcuts setup, one to my main account and another to a secondary account. To use the secondary account I just click that shortcut.</p>
<p>A downside is that you'll have to configure plugins and such for each profile.</p>

<div class="comment-text">
<p>It's platform dependent. I've created .desktop files in .local/share/applications/ on a Linux system (the command goes on the Exec line, see <a href="https://specifications.freedesktop.org/desktop-entry-spec/desktop-entry-spec-latest.html">https://specifications.freedesktop.org/desktop-entry-spec/desktop-entry-spec-latest.html</a> ). On Windows, the command line in a shortcut could be edited to point to the additional profile(s). I don't know how OS X works.</p>
<p>Shell aliases are another way to do it. Creating the .desktop files adds them to compatible launchers.</p>
