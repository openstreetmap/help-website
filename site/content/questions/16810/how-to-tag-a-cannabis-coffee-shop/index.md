<p>I'd like to try to tag some "coffee shops" in the Netherlands. These shops sell cannabis. However, I couldn't find any good tag. In Amsterdam, a few are tagged, but as "shop=coffee", which is IMHO not very clear because it could easily suggest that they sell coffee (grains), which they don't. I guess the tag should also be able to represent places like "Medical lounges" in California.</p>
<p>Here are a few statistics on the tags that sound related:</p>
<ul>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=coffee">http://taginfo.openstreetmap.org/tags/shop=coffee</a> : 187</li>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=coffeeshop">http://taginfo.openstreetmap.org/tags/shop=coffeeshop</a> : 1</li>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=cannabis">http://taginfo.openstreetmap.org/tags/shop=cannabis</a> : 1</li>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=drug">http://taginfo.openstreetmap.org/tags/shop=drug</a> : 0</li>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=headshop">http://taginfo.openstreetmap.org/tags/shop=headshop</a> : 8</li>
<li><a href="http://taginfo.openstreetmap.org/tags/shop=smartshop">http://taginfo.openstreetmap.org/tags/shop=smartshop</a> : 3</li>
</ul>
<p>I'd rather go with shop=cannabis, does it sounds good? Any better suggestion?</p>
