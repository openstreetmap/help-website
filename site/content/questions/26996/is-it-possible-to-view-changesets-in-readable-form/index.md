<div class="answer-body">
<p>There are a bunch of options described <a href="https://wiki.openstreetmap.org/wiki/Quality_assurance">here</a>. It's likely that some of these will go some way to showing what you want - I find Whodidit <a href="http://zverik.osm.rambler.ru/whodidit/">here</a> or <a href="http://simon04.dev.openstreetmap.org/whodidit/">here</a> and <a href="https://wiki.openstreetmap.org/wiki/ITO_Map">ITO's tools</a> particularly useful.</p>
<p>It's difficult to "display a changset on the map" when a third dimension to represent time is what is really required. There would need to be some way to "view" changes to the tags on nodes or ways, or to their positions, or to indicate that something has been deleted - as a mapper wanting to see what a changeset has changed, how would you like that information to be represented?</p>
