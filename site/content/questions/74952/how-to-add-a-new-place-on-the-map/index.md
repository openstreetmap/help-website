<p>Hi</p>
<p>You simply need to create an account on openstreetmap.org and start adding any missing objects yourself.</p>
<p>Most things are very easy to add or modify. The default online editor has a good introduction to the concepts and there is lots of additional material to found here <a href="https://www.openstreetmap.org/help">https://www.openstreetmap.org/help</a></p>
<p>However please do not add information from other maps or data sources (google, Here and so on) for legal reasons except if you have obtained permission to do so.</p>
<p>There is a very active Russian community that will be glad to help you with any issues, see <a href="https://openstreetmap.community/">https://openstreetmap.community/</a> (click on Russia to get a list of possible channels).</p>
