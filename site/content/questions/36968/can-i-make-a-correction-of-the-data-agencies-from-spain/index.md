<p>You may just need to wait a bit longer for the changes that you have made to the data using the editor to be reflected by updated rendering of the data at openstreetmap.org The higher zoom levels are rendered more often.<br />
Changes to the coastline take longer.<br />
You could paste a link here to one of your edits if you want someone to check if all is well.</p>
<p>Refresh the browser too as it may be displaying a cached image instead of reloading the lastest image from osm.org</p>
