<p>I've used Potlatch recently with Mac OSX 10.7 with Click2Flash. It worked for me, but I'm running a later Mac with a later version of the OS.</p>
<p>Do you have the latest version of Flash? (I am not sure that Adobe even has the latest version available for OS 10.5)</p>
<p>I've found that JOSM works well enough to be vastly better than Potlatch on my Mac (a little funky using the touch pad on the MacBookPro, but usable).
