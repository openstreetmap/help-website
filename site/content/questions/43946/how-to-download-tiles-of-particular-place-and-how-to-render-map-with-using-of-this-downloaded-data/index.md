<div class="answer-body">
<p>Please see the <a href="https://wiki.openstreetmap.org/wiki/JTileDownloader">wiki page about JTileDownloader</a>. Scraping tiles from OSM's tile servers is against their <a href="https://wiki.openstreetmap.org/wiki/Tile_usage_policy">tile usage policy</a>. Therefore JTileDownloader has been blocked.</p>
<p>There are various <a href="https://wiki.openstreetmap.org/wiki/Tile_usage_policy#Alternative_OpenStreetMap_Tile_Providers">alternatives</a>:</p>
<ul>
<li>Download tiles from a different tile server, for example the ones provided by MapQuest. They usually have less strict usage policies.</li>
<li><a href="https://wiki.openstreetmap.org/wiki/Rendering">Render</a> your own tiles, for example by using <a href="https://wiki.openstreetmap.org/wiki/TileMill">TileMill</a> or <a href="https://wiki.openstreetmap.org/wiki/Rendering">Maperitive</a></li>
<li>Choose a <a href="https://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services">commercial tile provider</a>.</li>
</ul>
