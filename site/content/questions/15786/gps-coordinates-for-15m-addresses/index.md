<div class="answer-body">
<p>Two things:<br />
</p>
<ol>
<li>OSM does not contain all addresses in France today. According to the taginfo statistics tool for France, we have about <a href="http://taginfo.openstreetmap.fr/keys/addr%3Ahousenumber">800.000 addresses</a> (all being added manually, it is already a huge dataset). So, do not expect that OSM will provide you "1.5M addresses in France" today. All of them are imported manually into OSM by contributors either based on local survey or, most probably, from the French Cadastre. If you fill that some areas in France are lacking addresses, you can help by contributing and <a href="http://wiki.openstreetmap.org/wiki/Key:addr">adding them yourself</a>.<br />
</li>
<li>even if the database would contain all your required addresses, OSM is not providing "services" like the one you are asking. Either extract the data yourself : you can find <a href="http://wiki.openstreetmap.org/wiki/Planet#Country_and_area_extracts">database extracts per country by several providers</a> and use an extraction tool like <a href="http://wiki.openstreetmap.org/wiki/Osmosis">Osmosis</a> or <a href="http://wiki.openstreetmap.org/wiki/Osmium">Osmium</a> (see the list <a href="http://wiki.openstreetmap.org/wiki/Category:OSM_processing">here</a>). Or you can contact a company offering commercial services around OSM to do this for you (<a href="http://wiki.openstreetmap.org/wiki/Commercial_OSM_Software_and_Services">listed here</a>).</li>
</ol>
