<p>You could create your own server that generates the atom feed based upon an overpass query you regularly against the OSM database. see <a href="http://overpass-api.de/">http://overpass-api.de/</a></p>
<p>I'm not aware of an existing service that delivers RSS/Atom feeds on particular tags. There is a service that delivers an rss feed for any change in a certain area. <a href="http://zverik.osm.rambler.ru/whodidit/">http://zverik.osm.rambler.ru/whodidit/</a> You might get some ideas from that site as well.</p>
